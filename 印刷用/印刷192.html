問1．次の5社のECサイト月次データを読み、各社の実績に関する記述として正しいものを(1)〜(5)の中から1つ選べ。（6点）
あるグループ企業5社（A〜E）の自社ECサイトにおける当月実績である。A社はアクセス数200千件、購入件数4,000件、売上8,000千円、返品件数120件であった。B社はアクセス数150千件、購入件数3,600件、売上7,200千円、返品件数72件である。C社はアクセス数250千件、購入件数5,000件、売上9,000千円、返品件数200件であった。D社はアクセス数120千件、購入件数3,000件、売上6,600千円、返品件数60件にとどまる。E社はアクセス数180千件、購入件数3,600件、売上7,920千円、返品件数144件であった。なお、購入率はアクセス数に対する購入件数の割合、客単価は売上を購入件数で割った値、返品率は購入件数に対する返品件数の割合とする。
【表1】5社のECサイト月次データ
企業アクセス数（千件）購入件数（件）売上（千円）返品件数（件）
A2004,0008,000120
B1503,6007,20072
C2505,0009,000200
D1203,0006,60060
E1803,6007,920144
(1) 客単価が最も高いのはD社のみである。
(2) 返品率が最も高いのはC社である。
(3) 購入率が最も高いのはD社である。
(4) 客単価が最も低いのはB社である。
(5) 返品率が最も低いのはA社である。
問1(正解:(3))
各社の3指標を計算する。

【購入率】（購入件数÷アクセス数）
・A: 4,000÷200,000 = 2.0%　・B: 3,600÷150,000 = 2.4%　・C: 5,000÷250,000 = 2.0%　・D: 3,000÷120,000 = 2.5%　・E: 3,600÷180,000 = 2.0%

【客単価】（売上÷購入件数）
・A: 8,000千円÷4,000 = 2,000円　・B: 7,200千円÷3,600 = 2,000円　・C: 9,000千円÷5,000 = 1,800円　・D: 6,600千円÷3,000 = 2,200円　・E: 7,920千円÷3,600 = 2,200円

【返品率】（返品件数÷購入件数）
・A: 120÷4,000 = 3.0%　・B: 72÷3,600 = 2.0%　・C: 200÷5,000 = 4.0%　・D: 60÷3,000 = 2.0%　・E: 144÷3,600 = 4.0%

【選択肢の検討】
・(1) 客単価最高はD(2,200円)とE(2,200円)の同値であり「D社のみ」ではない → 誤
・(2) 返品率最高はC(4.0%)とE(4.0%)の同値であり、C社単独ではない → 誤
・(3) 購入率最高はD(2.5%)であり、B(2.4%)を上回る単独最高 → 正
・(4) 客単価最低はC(1,800円)であり、B(2,000円)ではない → 誤
・(5) 返品率最低はB・D(2.0%)であり、A(3.0%)ではない → 誤

したがって、正解は(3)である。
問2．次の6店舗の月次経営データ（表2・表3）を読み、ア〜オの記述の正誤を判定して、正しいものの組み合わせを(1)〜(5)の中から1つ選べ。（6点）

問2(正解:(5))
売上＝来客数（千人）×客単価（円）として各指標を計算する。

【売上（千円）】
A: 40×800 = 32,000　B: 30×1,000 = 30,000　C: 50×600 = 30,000　D: 25×1,200 = 30,000　E: 60×500 = 30,000　F: 36×900 = 32,400（最多）

【営業利益（千円）】（売上−原価−固定費）
・A: 32,000−19,200−10,000 = 2,800（最少）　・B: 30,000−16,500−9,000 = 4,500（最多）　・C: 30,000−15,000−12,000 = 3,000　・D: 30,000−19,500−7,500 = 3,000　・E: 30,000−17,400−9,600 = 3,000　・F: 32,400−20,088−8,000 = 4,312

【利益率】（営業利益÷売上）
・A: 2,800÷32,000 = 8.75%（最低）　・B: 4,500÷30,000 = 15.0%（最高）　・C: 10.0%　・D: 10.0%　・E: 10.0%　・F: 4,312÷32,400 ≒ 13.3%

【1日あたり来客数（人）】（来客数÷営業日数）
・A: 40,000÷25 = 1,600　・B: 30,000÷30 = 1,000（最少）　・C: 50,000÷25 = 2,000　・D: 25,000÷20 = 1,250　・E: 60,000÷24 = 2,500（最多）　・F: 36,000÷24 = 1,500

【記述の検討】
・ア 売上最多(F)≠営業利益最多(B) → 誤
・イ 利益率最高(B)＝1日あたり来客最少(B) → 正
・ウ 営業利益最少(A)＝利益率最低(A) → 正
・エ 1日あたり来客最多(E)≠売上最多(F) → 誤
・オ 営業利益最多(B)＝利益率最高(B) → 正

正しい記述はイ・ウ・オ。したがって、正解は(5)である。
