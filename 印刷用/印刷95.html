問1．下図のように、4本の棒の高さを低・中・高の3段階で表すことによって整数を表す規則がある。1から12までの整数とそれに対応する図形が示されている。この規則に従うとき、整数65を表す図形として正しいものを(1)〜(5)から1つ選べ。（6点）
【整数と図形の対応表】
問1(正解:(3))

【規則の発見】
4本の棒はそれぞれ「低」「中」「高」の3段階の高さをとる。これは3進法（基数3）の各桁に対応している。

棒の高さを数値に置き換える:
・低 = 0　・中 = 1　・高 = 2

左から順に、27の位、9の位、3の位、1の位を表す:
・整数 n = (左の棒) x 27 + (左中の棒) x 9 + (右中の棒) x 3 + (右の棒) x 1 + 1

【検証】
・1 = 0x27 + 0x9 + 0x3 + 0 + 1（全て低）
・2 = 0x27 + 0x9 + 0x3 + 1 + 1（右のみ中）
・4 = 0x27 + 0x9 + 1x3 + 0 + 1（右中のみ中）
・10 = 0x27 + 1x9 + 0x3 + 0 + 1（左中のみ中）

【整数65の計算】
65 - 1 = 64 を3進法で表す:
・64 = 2x27 + 1x9 + 0x3 + 1
・各桁: 2, 1, 0, 1 → 高, 中, 低, 中

【選択肢の検討】
・(1) 高中低低 → 2x27+1x9+0x3+0+1 = 64 → 不正解
・(2) 中高低中 → 1x27+2x9+0x3+1+1 = 47 → 不正解
・(3) 高中低中 → 2x27+1x9+0x3+1+1 = 65 → 正解
・(4) 高低中中 → 2x27+0x9+1x3+1+1 = 59 → 不正解
・(5) 高中中低 → 2x27+1x9+1x3+0+1 = 67 → 不正解

したがって、正解は(3)である。
問2．A、B、C、D、Eの5人が、5階建てマンションの1階から5階にそれぞれ1人ずつ住んでいる。次の条件がすべて成り立つとき、Dが住んでいる階として正しいものを(1)〜(5)から1つ選べ。（6点）
【条件】
I．BはAより上の階に住んでいる。
II．DはCのちょうど2階上に住んでいる。
III．Eは偶数階に住んでいる。
IV．Aは1階に住んでいない。
V．BとEは隣の階に住んでいない。
【マンション】
(1) 1階
(2) 2階
(3) 3階
(4) 4階
(5) 5階
問2(正解:(3))

【条件の整理】
条件II「DはCのちょうど2階上」より、(C, D)の組み合わせは:
・C=1階, D=3階
・C=2階, D=4階
・C=3階, D=5階

【場合分け】

場合1: C=1階, D=3階のとき
残りA, B, Eは2階, 4階, 5階に住む。
条件III「Eは偶数階」より、E=2階またはE=4階。

(a) E=2階: A, Bは4階と5階。条件I「B>A」よりA=4階, B=5階。
　　条件IV: A=4階 (1階でない) → 適合
　　条件V: |B-E|=|5-2|=3 (隣でない) → 適合 → 有効

(b) E=4階: A, Bは2階と5階。条件I「B>A」よりA=2階, B=5階。
　　条件V: |B-E|=|5-4|=1 (隣の階) → 不適合

場合2: C=2階, D=4階のとき
残りA, B, Eは1階, 3階, 5階。
条件III「Eは偶数階」→ 1, 3, 5はすべて奇数 → 不適合

場合3: C=3階, D=5階のとき
残りA, B, Eは1階, 2階, 4階。
条件III「Eは偶数階」より、E=2階またはE=4階。
条件IV「A=1階でない」→ Aは2階または4階。

(a) E=2階: A, Bは1階と4階。条件IV「A=1階でない」→ A=4階, B=1階。
　　条件I: B(1) > A(4) → 不適合

(b) E=4階: A, Bは1階と2階。条件IV「A=1階でない」→ A=2階, B=1階。
　　条件I: B(1) > A(2) → 不適合

【結論】
有効な配置は場合1(a)のみ: A=4階, B=5階, C=1階, D=3階, E=2階

したがって、Dが住んでいる階は(3) 3階である。
