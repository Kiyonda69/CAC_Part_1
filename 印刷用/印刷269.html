問1．下の図１のア〜オは、いずれも同じ大きさの正方形のマスを5個つなげて作った5種類のピースである。このうち4種類のピースを1枚ずつ、回転させてもよいが裏返さずに、互いに重ならないように隙間なく並べたところ、図２のような縦4マス・横5マスの長方形をちょうど敷き詰めることができた。このとき、使わなかったピースとして正しいものを、次の(1)〜(5)のうちから1つ選べ。（6点）
【図１】5種類のピース
【図２】敷き詰める長方形（縦4マス・横5マス）
(1) ア
(2) イ
(3) ウ
(4) エ
(5) オ
問1(正解:(1))
5個のマスからなるピース4枚で敷き詰めるので、マスの総数は 5×4＝20マスとなり、 縦4マス×横5マス＝20マスの長方形と一致する。したがって枚数の観点ではどの4枚を 選んでも矛盾はなく、実際に敷き詰められる組合せを探す必要がある。

【敷き詰め例の発見】
イ・ウ・エ・オの4枚を使うと、下の解説図１のように長方形をちょうど敷き詰めることができる （ピースは回転して使用している）。

【解説図１】イ・ウ・エ・オによる敷き詰め例

【アが使えないことの確認】
ア（コの字形）には中央に1マスのくぼみがある。このくぼみは他のピースの 「1マスだけ突き出た部分」で埋めるしかないが、アを長方形の縁に沿って置いても 内部に置いても、くぼみを埋めた残りの領域が5マスずつのまとまりに分割できなくなる。 アを含む4枚の組合せ（ア＋イウエ、ア＋イウオ、ア＋イエオ、ア＋ウエオ）を すべて調べても、4×5の長方形を敷き詰める並べ方は存在しない。

【選択肢の検討】
・(1) ア → 敷き詰めに使えない唯一のピース → 正解
・(2) イ → 解説図１のとおり使用されている → 不正解
・(3) ウ → 解説図１のとおり使用されている → 不正解
・(4) エ → 解説図１のとおり使用されている → 不正解
・(5) オ → 解説図１のとおり使用されている → 不正解

したがって、正解は(1)である。
問2．下の図３のア〜オは、問1と同じ5種類のピースである。この5種類のピースをすべて1枚ずつ、回転させてもよいが裏返さずに、互いに重ならないように隙間なく並べたところ、図４のような25マスの図形をちょうど敷き詰めることができた。このとき、★印のマスを覆っているピースとして正しいものを、次の(1)〜(5)のうちから1つ選べ。（6点）
【図３】5種類のピース（問1と同じ）
【図４】敷き詰める図形（25マス）
(1) ア
(2) イ
(3) ウ
(4) エ
(5) オ
問2(正解:(2))
5枚すべてを使うので 5×5＝25マスとなり、図４のマス数と一致する。 形が不規則な部分（上に突き出た2マスの帯）から順に決めていくのがポイントである。

【手順1: 上部の突出部を覆うピースを決める】
最上段の2マス（上から1行目）は左右が空白なので、ここからピースは 下方向にしかつながらない。この2マスを1枚のピースでまとめて覆う置き方を 試すと、いずれも2行目に埋められない欠けが生じて行き詰まる。 そこで左右を別々のピースで覆うと考えると、左のマスはエ（階段形）が 左下へ降りる形、右のマスはイ（T字形）の縦棒が3行目まで下へ伸びる形に限られる。

【手順2: 残りの領域を埋める】
イの横腕は2行目の右端2マスへ伸びる。すると右端の縦3マスと下辺2マスの L字にオがちょうど収まり、左下の2×2のかたまりを含む5マスにウが収まる。 最後に残る「くぼみを挟んだ5マス」にアがはまり、アのくぼみに イの縦棒の先端（★のマス）が刺さる形になる。

【解説図２】敷き詰め方（ただ1通り）
問2(正解:(2)) 解説のつづき

【手順3: ★のマスを確認する】
完成した敷き詰め方は解説図２のただ1通りであり、★のマス（上から3行目・左から4列目）を 覆っているのはイの縦棒部分である。★の左右のマスはアの両端であり、 アと取り違えないよう注意する。

【選択肢の検討】
・(1) ア → ★の両隣と下のマスを覆うが★自身は覆わない → 不正解
・(2) イ → 縦棒の先端が★のマスを覆う → 正解
・(3) ウ → 左下の領域を覆う → 不正解
・(4) エ → 左上の階段部分を覆う → 不正解
・(5) オ → 右端のL字部分を覆う → 不正解

したがって、正解は(2)である。
