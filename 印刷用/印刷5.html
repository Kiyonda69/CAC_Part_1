問1．下の展開図を折り畳んでできる立方体として、正しいものを(1)〜(5)の中から1つ選べ。（6点）
【展開図】
問1(正解:(1))

【展開図の折り畳みルール】
十字型展開図では、正面（□）を基準として折り畳みを考える。

・正面: □（空四角）
・左面: ■（塗り四角）← 正面の左に位置
・右面: △（空三角）← 正面の右に位置
・上面: ●（塗り円）← 正面の上に位置
・下面: +（プラス）← 正面の下に位置
・背面: ▲（塗り三角）← 下面のさらに下、正面の反対

【対面（反対面）のペア】
・上面(●) と 下面(+) は対面 → 同時に見えない
・正面(□) と 背面(▲) は対面 → 同時に見えない
・左面(■) と 右面(△) は対面 → 同時に見えない

【斜め右上から見たとき: 上面・正面・右面が見える】
正しい見え方: 上=●、正面=□、右=△

【選択肢の検討】
・(1) 上=●、正面=□、右=△ → 正しい配置 → 正解
・(2) 上=●、正面=■、右=△ → ■は左面。■(左)と△(右)は対面で同時に見えない → 不正解
・(3) 上=+、正面=□、右=△ → +は下面(●の対面)。上面に来ることはない → 不正解
・(4) 上=●、正面=□、右=▲ → ▲は背面(□の対面)。右面に来ることはない → 不正解
・(5) 上=●、正面=▲、右=△ → ▲は背面(□の対面)。正面に来ることはない → 不正解

したがって、正解は（1）である。
問2．下の立方体の各面には、●・○・■・□・▲・△の記号が1面ずつ描かれている。また、向かい合う面（対面）の関係は、上面(●)と下面(○)、正面(■)と背面(□)、左面(▲)と右面(△)である。この立方体の展開図として正しいものを、(1)〜(5)の中から1つ選べ。（6点）
【立方体（斜め右上から見た図）】
問2(正解:(4))

【立方体の完全な面配置】
問題より: 上=●、下=○、正面=■、背面=□、左=▲、右=△

【十字型展開図の折り畳みルール】
展開図の配置（P=上, Q=左, R=正面, S=右, T=下, U=背面）:
    [P]
[Q][R][S]
    [T]
    [U]

R（中央）= 正面(■)、Q（左）= 左面(▲)、S（右）= 右面(△)
P（上）= 上面(●)、T（下）= 下面(○)、U（最下）= 背面(□)

正解の展開図: P=●, Q=▲, R=■, S=△, T=○, U=□ → (4)

【各選択肢の検討】
・(1) P=○, T=● → 折ると上面=○、下面=●。目標は上面=●、下面=○で上下逆 → 不正解
・(2) Q=△, S=▲ → 折ると左面=△、右面=▲。目標は左面=▲、右面=△で左右逆 → 不正解
・(3) R=□, U=■ → 折ると正面=□、背面=■。目標は正面=■、背面=□で前後逆 → 不正解
・(4) P=●, Q=▲, R=■, S=△, T=○, U=□ → 折ると上=●, 左=▲, 正面=■, 右=△, 下=○, 背面=□ → 正解
・(5) T=□, U=○ → 折ると下面=□、背面=○。目標は下面=○、背面=□で逆 → 不正解

したがって、正解は（4）である。
