問1．一辺が1cmの立方体の積み木を、以下の条件に従って積み上げた。（6点）
底面は「前後2行×左右4列」のマス目に配置し、各位置に次の個数を積む。
  • 前列（手前）は、左から順に　2個・1個・3個・4個
  • 後列（奥）は、左から順に　1個・3個・2個・1個
この積み木の全体を、真正面（手前の行の方向）から眺めたとき、見えるシルエットはどれか。
（正面から見たシルエット）
問1（正解：(5)）

正面（手前の行の方向）から見たとき、各列で見えるのは前列と後列のうち高い方のみである。

【各列の高さを比較】
・左から1列目：前列2個 vs 後列1個 → 高い方は 2
・左から2列目：前列1個 vs 後列3個 → 高い方は 3
・左から3列目：前列3個 vs 後列2個 → 高い方は 3
・左から4列目：前列4個 vs 後列1個 → 高い方は 4

したがって、正面から見えるシルエットは左から順に「2段・3段・3段・4段」となる。

【選択肢の検討】
・(1)　[3, 3, 2, 4]　→ 1列目が3段（誤り。正しくは2段）
・(2)　[2, 4, 3, 3]　→ 2列目が4段（誤り。正しくは3段）
・(3)　[3, 2, 3, 4]　→ 1列目が3段・2列目が2段（いずれも誤り）
・(4)　[2, 3, 4, 4]　→ 3列目が4段（誤り。正しくは3段）
・(5)　[2, 3, 3, 4]　→ すべて一致　（正解）

したがって、正解は(5)である。
問2．縦4マス×横4マスの正方形の紙がある。この紙を以下の手順で折り、穴を開ける。（6点）
  • 【手順1】 紙の左半分を右半分の上に重ねるように折る（縦の中心線で折る）
  • 【手順2】 折った紙の下半分を上半分の上に重ねるように折る（横の中心線で折る）
  • 【手順3】 折り重なった紙の左上の位置（上端・左端の角）に、穴を1つ開ける
紙をすべて広げたとき、穴の開いた位置として正しいものはどれか。
（選択肢の黒丸が穴の位置を表す。上を1行目、左を1列目とする。）
（広げたときの穴の位置）
問2（正解：(1)）

折り畳みの各ステップで、穴が対応する位置を追跡する。

【手順1：左半分を右半分に重ねる】
・1列目は4列目の位置に重なる
・2列目は3列目の位置に重なる
折り後の紙は縦4行×横2列になる。
左端の列 = 元の3列目（表）と2列目（裏）
右端の列 = 元の4列目（表）と1列目（裏）

【手順2：下半分を上半分に重ねる】
・4行目は1行目の位置に重なる
・3行目は2行目の位置に重なる
折り後の紙は縦2行×横2列（計4層）になる。

【穴の位置の追跡】
折り重なった紙の左上（新1行目・新1列目）は、元の紙の次の4か所に対応する：
　・元の1行目・3列目（折り前の最表面）
　・元の1行目・2列目（手順1で3列目の裏側）
　・元の4行目・3列目（手順2で1行目の裏側）
　・元の4行目・2列目（手順1・2の両方で裏側）

穴は 1行目の2・3列、4行目の2・3列 の計4か所に開く。

【選択肢の検討】
・(1)　1・4行目の2・3列に穴 → 上記と一致　（正解）
・(2)　四隅（1・4行の1・4列）→ 新1行目・2列目の位置に穴を開けた場合の結果
・(3)　中央4個（2・3行の2・3列）→ 新2行目・1列目に穴を開けた場合の結果
・(4)　中行両端（2・3行の1・4列）→ 新2行目・2列目に穴を開けた場合の結果
・(5)　縦一列（全行の2列目）→ 折り畳みを1回しか行わなかった場合の誤り

したがって、正解は(1)である。
