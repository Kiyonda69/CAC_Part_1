問1．一辺が1cmの立方体の積み木を、以下の条件に従って積み上げた。（6点）
底面は「前後2行×左右4列」のマス目に配置し、各位置に次の個数を積む。
  • 前列（手前）は、左から順に　2個・1個・3個・4個
  • 後列（奥）は、左から順に　1個・3個・2個・1個
この積み木の全体を、真正面（手前の行の方向）から眺めたとき、見えるシルエットはどれか。
（正面から見たシルエット）
問1（正解：(5)）

正面（手前の行の方向）から見たとき、各列で見えるのは前列と後列のうち高い方のみである。

【各列の高さを比較】
・左から1列目：前列2個 vs 後列1個 → 高い方は 2
・左から2列目：前列1個 vs 後列3個 → 高い方は 3
・左から3列目：前列3個 vs 後列2個 → 高い方は 3
・左から4列目：前列4個 vs 後列1個 → 高い方は 4

したがって、正面から見えるシルエットは左から順に「2段・3段・3段・4段」となる。

【選択肢の検討】
・(1)　[3, 3, 2, 4]　→ 1列目が3段（誤り。正しくは2段）
・(2)　[2, 4, 3, 3]　→ 2列目が4段（誤り。正しくは3段）
・(3)　[3, 2, 3, 4]　→ 1列目が3段・2列目が2段（いずれも誤り）
・(4)　[2, 3, 4, 4]　→ 3列目が4段（誤り。正しくは3段）
・(5)　[2, 3, 3, 4]　→ すべて一致　（正解）

したがって、正解は(5)である。
問2．一辺が1cmの立方体の積み木を、以下の条件に従って積み上げた。（6点）
底面は「前後3行×左右3列」のマス目に配置し、各位置に次の個数を積む。
  • 前列（手前）は、左から順に　2個・1個・3個
  • 中列は、左から順に　4個・2個・1個
  • 後列（奥）は、左から順に　1個・3個・2個
この積み木の全体を、右側面（右端の列の方向）から眺めたとき、見えるシルエットはどれか。
ただし、シルエットは左側に前列、右側に後列が見える向きで示す。
（右側面から見たシルエット　左＝手前・右＝奥）
問2（正解：(1)）

右側面（右端の列の方向）から見たとき、各行（前・中・後）で見えるのはその行の最大高さだけである。

【各行の最大高さを求める】
・前列（左から2・1・3個）：最大は 3個
・中列（左から4・2・1個）：最大は 4個
・後列（左から1・3・2個）：最大は 3個

右側面から見ると左側に前列・右側に後列が映り、シルエットの高さは前から後へ「3段・4段・3段」となる（中列が最も高い山型）。

【選択肢の検討】
・(1)　[3, 4, 3]　→ すべて一致　（正解）
・(2)　[4, 3, 3]　→ 前列が4段（誤り。前列の最大は3段）
・(3)　[3, 3, 4]　→ 後列が4段（誤り。後列の最大は3段）
・(4)　[4, 4, 3]　→ 前列が4段（誤り。前列の最大は3段）
・(5)　[3, 4, 4]　→ 後列が4段（誤り。後列の最大は3段）

したがって、正解は(1)である。
