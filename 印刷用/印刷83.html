問1．次の図のような立方体の展開図を組み立てたとき、辺アイと重なる辺はどれか。（6点）
(1) 辺ウク
(2) 辺スセ
(3) 辺カキ
(4) 辺ケス
(5) 辺セコ
問1(正解:(3))

【解法】
展開図を組み立てたときの各面の配置を確認する。

③を正面として組み立てると：
・①は上面（③の上に折り上げる）
・②は左面（③の左に折る）
・④は右面（③の右に折る）
・⑤は背面（④のさらに右から後ろに折る）
・⑥は下面（③の下に折り下げる）

【頂点の対応】
組み立てると、以下の頂点が同一点に重なる：
・ア = ウ = キ（立方体の左上奥の頂点）
・イ = カ（立方体の右上奥の頂点）
・ク = シ = ス（立方体の左下奥の頂点）
・サ = セ（立方体の右下奥の頂点）

辺アイは①の上辺で、組み立てると立方体の上面奥の辺になる。
辺カキは⑤の上辺で、同じく立方体の上面奥の辺になる。
したがって、辺アイと辺カキは重なる。

【選択肢の検討】
・(1) 辺ウク → 立方体の左面奥の辺（辺キシと重なる）→ 不正解
・(2) 辺スセ → 立方体の下面奥の辺（辺サシと重なる）→ 不正解
・(3) 辺カキ → 立方体の上面奥の辺 → 正解
・(4) 辺ケス → 立方体の左面前下から奥の辺 → 不正解
・(5) 辺セコ → 立方体の右面前下から奥の辺 → 不正解

したがって、正解は(3)である。
問2．下の見本のようなL字型の図形がある。各マスにはアルファベットが書かれている。この見本を時計回りに90°回転させた図形として正しいものを、(1)〜(5)から1つ選べ。ただし、回転に伴い文字の向きも変わることに注意せよ。（6点）
【見本】
問2(正解:(4))

【解法】
L字型図形を時計回りに90°回転させるとき、以下の2点に注意する：
1. セルの位置が90°回転する
2. セル内の文字の向きも90°回転する（文字が右に傾く）

【位置の変化】
元の配置（座標）：
・F: (0, 2) → 左上
・G: (0, 1) → 左中
・H: (0, 0) → 左下
・J: (1, 0) → 右下

90°時計回り回転後の座標 (x,y)→(y,-x) を正規化：
・F: (2, 1) → 右上
・G: (1, 1) → 中上
・H: (0, 1) → 左上
・J: (0, 0) → 左下

回転後の図形：
・上段に H, G, F が左から右に並ぶ（各文字は90°時計回りに傾く）
・下段左に J が位置する（同様に90°時計回りに傾く）

【選択肢の検討】
・(1) 180°回転した配置 → 不正解
・(2) 鏡像を回転した配置（JがHの上に来ている）→ 不正解
・(3) 90°反時計回り回転（回転方向が逆）→ 不正解
・(4) 90°時計回り回転（位置・文字向きともに正しい）→ 正解
・(5) 形は似ているがFとGの位置が入れ替わっている → 不正解

したがって、正解は(4)である。
