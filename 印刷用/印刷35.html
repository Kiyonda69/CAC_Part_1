問1．図のように、黒と白の7つのマスからなる行が、ある規則にしたがって上から下のように変化している。図の規則と矛盾なく変化しているものは次のうちどれか。正しいものを1つ選び、解答用紙にマークせよ。ただし、選択肢では矢印を省略している。（6点）
問1(正解:(3))

【規則の発見】
例題の4行を観察すると、各行の次の行は以下の規則で生成されている:
「各マスの色は、そのマスと右隣のマス（最右端は最左端と比較）の色が異なれば黒、同じなら白」
これはXOR（排他的論理和）演算に相当する。

具体的に確認する:
・行1「B B W B W W B」の1番目(B)と2番目(B)は同色 → 次行1番目はW
・行1の2番目(B)と3番目(W)は異色 → 次行2番目はB
・行1の7番目(B)と1番目(B)は同色 → 次行7番目はW（端は循環）
→ 行2「W B B B W B W」と一致する。

【選択肢の検討】
各選択肢の行1→行2にこの規則を適用して確認する:

・(1) 行1「BWBWWBB」→ 規則適用:「BBBBBBW」→ 行2「WBBBWBW」と不一致 → 不正解
・(2) 行1「BBWBBWB」→ 規則適用:「WBBBBBB」→ 行2「BWWBWWB」と不一致 → 不正解
・(3) 行1「WBWBBWB」→ 規則適用:「BBBWBBB」→ 行2「BBBWBBB」と一致。行2→行3, 行3→行4も一致 → 正解
・(4) 行1「BWWBWBW」→ 規則適用:「BWBBBBW」→ 行2「WBWWWWW」と不一致 → 不正解
・(5) 行1→行2は一致するが、行2→行3で不一致（4番目のマスが異なる） → 不正解

したがって、正解は(3)である。
問2．1辺の長さが2の立方体ABCD-EFGHがある（ABCDが上面、EFGHが底面で、A-E, B-F, C-G, D-Hがそれぞれ対応する頂点である）。辺ABの中点をP、辺FGの中点をQ、辺DHの中点をRとする。三角形PQRの面積として正しいものを1つ選び、解答用紙にマークせよ。（6点）
(1)　3√3/2
(2)　3
(3)　2√3
(4)　√6
(5)　3√2
問2(正解:(1))

【座標の設定】
立方体の頂点に座標を設定する。
上面: A=(0,0,2), B=(2,0,2), C=(2,2,2), D=(0,2,2)
底面: E=(0,0,0), F=(2,0,0), G=(2,2,0), H=(0,2,0)

【中点の座標】
・P（ABの中点）= (1, 0, 2)
・Q（FGの中点）= (2, 1, 0)
・R（DHの中点）= (0, 2, 1)

【辺の長さの計算】
・PQ = √{(2-1)² + (1-0)² + (0-2)²} = √(1+1+4) = √6
・QR = √{(0-2)² + (2-1)² + (1-0)²} = √(4+1+1) = √6
・RP = √{(1-0)² + (0-2)² + (2-1)²} = √(1+4+1) = √6

3辺がすべて√6で等しいため、三角形PQRは正三角形である。

【面積の計算】
1辺√6の正三角形の面積 = (√3/4) × (√6)² = (√3/4) × 6 = 3√3/2

【別解：外積による計算】
ベクトル PQ = (1, 1, -2), PR = (-1, 2, -1)
PQ × PR = (1×(-1)-(-2)×2, (-2)×(-1)-1×(-1), 1×2-1×(-1)) = (3, 3, 3)
面積 = |PQ × PR| / 2 = √(9+9+9) / 2 = 3√3/2

したがって、正解は(1)である。
