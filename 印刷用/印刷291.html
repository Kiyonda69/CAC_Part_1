問1．ある商品に、原価の何割かの利益を見込んで定価をつけた。この商品を定価の1割5分引きで売ると原価の1割9分の利益があり、定価から550円引いて売ると原価の3割の利益がある。この商品の原価はいくらか。次の(1)〜(5)から1つ選べ。（6点）
(1) 6,545円   (2) 7,700円   (3) 5,000円   (4) 5,500円   (5) 7,150円
問1(正解:(4))
原価も定価も未知数であるうえ、一方の条件は「定価に対する割引率」、もう一方は「定価からの値引き額」で与えられ、利益はどちらも「原価に対する割合」で示されている。基準の異なる割合が混在しているため、原価を x 円、定価を y 円とおいて正確に立式する。

【立式】
・定価の1割5分引きで売ると原価の1割9分の利益 → 0.85y = 1.19x …①
・定価から550円引いて売ると原価の3割の利益 → y − 550 = 1.3x …②

【求解】
①より y = 1.19x ÷ 0.85 = 1.4x（1.19 = 0.85 × 1.4 に気づけば、定価は原価の4割増しと分かる）
これを②に代入して 1.4x − 550 = 1.3x → 0.1x = 550 → x = 5,500円（定価 y = 7,700円）

【別解（利益の差に着目）】
2つの売り方の利益の差は 1.3x − 1.19x = 0.11x。売価の差は (y − 550) − 0.85y = 0.15y − 550。y = 1.4x を使うと 0.21x − 550 = 0.11x となり、同じく x = 5,500円。

【検算】
定価7,700円の1割5分引き = 6,545円。利益 6,545 − 5,500 = 1,045円 = 5,500 × 0.19 → 原価の1割9分で一致。
定価から550円引き = 7,150円。利益 7,150 − 5,500 = 1,650円 = 5,500 × 0.3 → 原価の3割で一致。

【選択肢の検討】
・(1) 6,545円 → 1割5分引きの売価（0.85 × 7,700）との取り違え。不正解
・(2) 7,700円 → 定価との取り違え。不正解
・(3) 5,000円 → 2つの売価が異なることを見落とし、利益の差だけで 0.11x = 550 と誤って立式した値。不正解
・(4) 5,500円 → ①②をともに満たす。正解
・(5) 7,150円 → 550円引きの売価（7,700 − 550）との取り違え。不正解

したがって、正解は(4)である。
問2．ある店で、同じ商品を360個仕入れ、原価の5割の利益を見込んで定価をつけて売り出した。何個かは定価で売れたが、その後、定価の2割引にしたところ、定価で売れた個数より30個少ない個数が売れた。さらに、売れ残りのうち10個は汚損が見つかったため廃棄し、それ以外はすべて定価の半額にして売り切った。その結果、全体の利益は仕入総額のちょうど17.5％となった。定価の半額で売った個数は何個か。次の(1)〜(5)から1つ選べ。（6点）
(1) 115個   (2) 145個   (3) 80個   (4) 100個   (5) 90個
問2(正解:(5))
原価の実額は与えられていないが、利益の条件が仕入総額に対する割合で示されているため、原価を1個あたり x 円とおけば x は式の両辺から消える。定価で売れた個数を n 個とおくと、2割引で売れた個数は n − 30 個、廃棄が10個なので、半額で売った個数は 360 − n − (n − 30) − 10 = 380 − 2n 個と表せる。廃棄した10個は売上ゼロのまま原価だけが掛かる点に注意する。

【各売価を整理】
・定価 = 1.5x 円 → 1個あたりの利益 0.5x 円
・2割引の売価 = 1.5x × 0.8 = 1.2x 円 → 1個あたりの利益 0.2x 円
・半額の売価 = 1.5x × 0.5 = 0.75x 円 → 1個あたり 0.25x 円の損失
・廃棄 → 1個あたり x 円の損失（原価がまるごと損になる）

【立式と求解】
全体の利益 = 0.5x × n + 0.2x × (n − 30) − 0.25x × (380 − 2n) − x × 10
= (0.5n + 0.2n − 6 − 95 + 0.5n − 10)x = (1.2n − 111)x 円
これが仕入総額 360x 円の17.5％、すなわち 63x 円に等しいので、
1.2n − 111 = 63 → 1.2n = 174 → n = 145個
よって、半額で売った個数 = 380 − 2 × 145 = 90個（2割引は115個、廃棄10個で計360個）

【検算】
原価を800円とすると、仕入総額 = 288,000円。定価1,200円で145個 → 174,000円、960円で115個 → 110,400円、600円で90個 → 54,000円、廃棄10個 → 0円。売上合計 = 338,400円。
総利益 = 338,400 − 288,000 = 50,400円。50,400 ÷ 288,000 = 0.175 = 17.5％ となり、条件と一致する。

【選択肢の検討】
・(1) 115個 → 2割引で売れた個数（145 − 30）との取り違え。不正解
・(2) 145個 → 定価で売れた個数 n との取り違え。設問は半額で売った個数を問うている。不正解
・(3) 80個 → 半額で売った90個から廃棄の10個を誤ってさらに差し引いた値。廃棄分は半額販売とは別に数える。不正解
・(4) 100個 → 売れ残り全体（半額90個＋廃棄10個）との取り違え。不正解
・(5) 90個 → 1.2n = 174 → n = 145 から導かれる値。正解

したがって、正解は(5)である。
