問1．下図はある立方体の展開図であり、各面には異なる模様が1つずつ描かれている。この展開図を組み立てて立方体を作るとき、丸（●）が描かれた面と向かい合う面の模様として正しいものを、(1)～(5)から1つ選べ。（6点）
【展開図】
問1(正解:(1))

【展開図の組み立て】
この展開図は変則的な十字型で、以下のように配置されている。

丸（●）が描かれた面を基準（正面）として組み立てを考える。
・星（★）は丸の左にあるため → 左面
・三角（△）は丸の右にあるため → 右面
・ひし形（◇）は丸の下にあるため → 底面
・四角（□）は三角の上にあるため → 三角が右面になると、四角は上に折れて上面
・十字（＋）はひし形の下にあるため → ひし形が底面になると、十字は奥に折れて背面

【向かい合う面のペア】
・丸（●） ↔ 十字（＋）（正面 ↔ 背面）
・星（★） ↔ 三角（△）（左面 ↔ 右面）
・四角（□） ↔ ひし形（◇）（上面 ↔ 底面）

【選択肢の検討】
・(1) 十字（＋） → 丸の向かい側は背面であり、十字が背面 → 正解
・(2) 星（★） → 星は左面であり、丸（正面）とは隣接する面 → 不正解
・(3) 三角（△） → 三角は右面であり、丸とは隣接する面 → 不正解
・(4) ひし形（◇） → ひし形は底面であり、丸とは隣接する面 → 不正解
・(5) 四角（□） → 四角は上面であり、丸とは隣接しないが向かい側でもない → 不正解

したがって、正解は(1)である。
問2．下図は、3×3×3の格子空間に1辺1の立方体ブロックを配置したときの正面図と右側面図を示している。黒色の部分はブロックが見える位置、白色の部分はブロックが見えない位置を表す。2つの投影図がともに正しくなるようにブロックを配置するとき、必要なブロックの最小個数として正しいものを、(1)～(5)から1つ選べ。（6点）
(1) 5個
(2) 6個
(3) 7個
(4) 8個
(5) 9個
問2(正解:(3))

【解法の方針】
各高さ（y=1, 2, 3）ごとに、正面図と右側面図の条件を同時に満たす最小のブロック配置を求める。

正面図の位置(x, y)が黒 ⇔ 高さyにおいて、x列のどこかに少なくとも1つブロックがある。
右側面図の位置(z, y)が黒 ⇔ 高さyにおいて、z行のどこかに少なくとも1つブロックがある。

【各段の分析】

y=1（最下段）
・正面図: x=2のみ黒 → x=2の列に1個以上必要（必要列数: 1）
・右側面図: z=1, z=2が黒 → z=1行とz=2行それぞれに1個以上必要（必要行数: 2）
・最小ブロック数: max(1, 2) = 2個
・配置例: (2,1,1), (2,1,2)

y=2（中段）
・正面図: x=1, 2, 3すべて黒 → 3列すべてに必要（必要列数: 3）
・右側面図: z=2, z=3が黒 → 2行に必要（必要行数: 2）
・最小ブロック数: max(3, 2) = 3個
・配置例: (1,2,3), (2,2,2), (3,2,2)

y=3（最上段）
・正面図: x=1, x=3が黒 → 2列に必要（必要列数: 2）
・右側面図: z=1, z=2が黒 → 2行に必要（必要行数: 2）
・最小ブロック数: max(2, 2) = 2個
・配置例: (1,3,2), (3,3,1)

【合計】
最小ブロック数 = 2 + 3 + 2 = 7個

【最小性の証明】
各高さyにおいて、正面図でr列が黒、右側面図でc行が黒の場合、少なくともmax(r, c)個のブロックが必要である。これは、r列すべてにブロックを配置するにはr個以上、c行すべてに配置するにはc個以上必要なためである。

したがって、正解は(3)である。
