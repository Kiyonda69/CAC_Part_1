問1．下図は立方体の展開図であり、各面にア〜カの記号と模様が描かれている。この展開図を組み立てたとき、丸（ア）が描かれた面と向かい合う面の模様として正しいものを、下の(1)〜(5)のうちから一つ選べ。（6点）
【展開図】
問1(正解:(2))

【展開図の分析】
十字型展開図の各面の配置を確認する。

中央の面「ア（丸）」を基準に、展開図を組み立てる。
・ア（丸）の右隣 → イ（三角）
・ア（丸）の左隣 → エ（星）
・ア（丸）の上 → ウ（四角）
・ア（丸）の下 → オ（十字）
・オ（十字）のさらに下 → カ（菱形）

【対面関係の特定】
十字型展開図では、中央の面から2つ離れた面が向かい合う。
・ア（丸） ↔ カ（菱形）：中央とその2つ下
・イ（三角） ↔ エ（星）：左右の面
・ウ（四角） ↔ オ（十字）：上下の面

【選択肢の検討】
・(1) 三角 → イの模様。アとは隣接関係であり、向かい合わない
・(2) 菱形 → カの模様。アと向かい合う面であり、正解
・(3) 四角 → ウの模様。アとは隣接関係であり、向かい合わない
・(4) 十字 → オの模様。アとは隣接関係であり、向かい合わない
・(5) 星 → エの模様。アとは隣接関係であり、向かい合わない

したがって、正解は(2)である。
問2．下の(1)〜(5)は、いずれも6つの正方形を辺でつなげた図形である。このうち、折りたたんで立方体を組み立てることができないものを一つ選べ。（6点）
問2(正解:(3))

【展開図の判別方法】
立方体の展開図として成立するには、以下の条件を全て満たす必要がある。
・6つの正方形が辺でつながっている（連結）
・折りたたんだとき、面同士が重ならない
・折りたたんだとき、全ての面が立方体の6面を構成する

【各選択肢の検討】

・(1) 十字型：最も基本的な展開図の一つ。中央の面を底面にすると、上下左右と対面の計6面が正しく構成される。 → 組み立て可能

・(2) S字型：各段が1マスずつずれた階段状。折りたたむと各面が立方体の異なる面を構成する。 → 組み立て可能

・(3) 2行3列の長方形：6つの正方形が2行3列に並んでいる。この形は、折りたたむと面が重なってしまう。具体的には、上段の左端と下段の左端を折り上げると同じ方向を向き、立方体にならない。 → 組み立て不可能

・(4) 階段型：1マスずつ右にずれながら下に伸びる形。S字型と同様に、折りたたむと6面全てが異なる方向を向く。 → 組み立て可能

・(5) S字型の変形：上段2マス、中段3マス、下段1マスの構成。折りたたむと正しく立方体が構成される。 → 組み立て可能

【2行3列の長方形が不可能な理由】
2行3列の長方形を折りたたむ場合を考える。上段中央を固定して周囲を折ると、上段左と下段左がともに同じ方向に折れ、面が重なる。どの面を基準にしても、必ず2つの面が同じ位置に来てしまうため、立方体を構成できない。

したがって、正解は(3)である。
