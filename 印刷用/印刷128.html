問1．ある立体を真上から見ると図Aのようになり、アの方向から見ると図Bのようになった。この立体をイの方向から見た図として正しいものを1つ選び、解答用紙にマークせよ。（6点）
図A（真上から）
図B（アの方向から）
問1（正解: (4)）

与えられた立体は、次の2つのブロックからなる。

・ブロックA（右前・高い）：右半分（幅2）× 前半分（奥行2）× 高さ3

・ブロックB（左全域・低い）：左半分（幅2）× 全奥行（奥行4）× 高さ1


【上面図（図A）の確認】

真上から見ると、ブロックAとブロックBの足跡がL字形に見える。前半（手前）は全幅4単位、後半は左の幅2単位のみ。これが図Aに一致する。


【正面図（図B、ア方向）の確認】

ア方向（前面）から見ると：

・左半分（x:0-2）：ブロックBのみ → 高さ1

・右半分（x:2-4）：ブロックAのみ → 高さ3

左が低く右が高い階段形状となり、図Bに一致する。


【イ方向（右側面）の決定】

イ方向（右側）から見たとき、奥行き位置ごとの最大高さを調べる。

・前半（z:0-2）：ブロックA（高さ3）とブロックB（高さ1）が両方存在 → 最大高さ＝3

・後半（z:2-4）：ブロックBのみ（高さ1）→ 最大高さ＝1

よって、右側面図は手前（前部）が高さ3、奥（後部）が高さ1の階段形状となる。


・選択肢(1)(2)は高さが一定の長方形 → 誤

・選択肢(3)は前後が逆（後部が高い）→ 誤

・選択肢(5)は比率が違う（1/4の位置で段差）→ 誤


したがって、正解は(4)である。

問2．図のように、直方体ABCD-EFGHがある。AB=4、BC=3、AE=3とする。辺DHの中点をP、辺BFの中点をQとするとき、3点A、P、Qを含む平面でこの直方体を切断した。切り口の形として正しいものを1つ選び、解答用紙にマークせよ。（6点）
（切り口の形を選べ）
(1)　三角形
(2)　台形
(3)　平行四辺形
(4)　長方形
(5)　六角形
問2（正解: (3)）

直方体の各頂点を座標で表す（AB=4を x方向、BC=3を y方向、AE=3を z方向）。

A(0,0,0)、B(4,0,0)、C(4,3,0)、D(0,3,0)、E(0,0,3)、F(4,0,3)、G(4,3,3)、H(0,3,3)

P（辺DH中点）= (0, 3, 1.5)、Q（辺BF中点）= (4, 0, 1.5)


【切断面が通る点の特定】

3点A、P、Qを含む平面の法線ベクトルは

AP×AQ = (0,3,1.5)×(4,0,1.5) = (4.5, 6, -12)

平面の方程式：4.5x + 6y − 12z = 0（原点Aを通る）


各辺との交点を調べると：

・辺DC（底面後辺）：x=2, y=3, z=0 → この点をPと区別して R=(2,3,0) とする… いや、改めて確認すると辺DCでは 4.5(2)+6(3)=9+18=27 ≠ 0 なので交わらない。

・頂点G(4,3,3)：4.5×4+6×3−12×3 = 18+18−36 = 0 ✓ → Gも平面上にある


よって切り口の頂点は A、Q、G、P の4点であり、四角形 AQGP を形成する。


【断面形状の特定】

・AQ = (4, 0, 1.5)、PG = G − P = (4, 0, 1.5) → AQ = PG（平行かつ等長）

・AP = (0, 3, 1.5)、QG = G − Q = (0, 3, 1.5) → AP = QG（平行かつ等長）

2組の対辺が平行かつ等長なので、断面は平行四辺形である。


なお AQ · AP = 4×0 + 0×3 + 1.5×1.5 = 2.25 ≠ 0 なので長方形ではない。


したがって、正解は(3)である。
