問1．下表の各列A、B、C、Dには、それぞれ規則性がある。また、各行の4つの数値の間にも共通の関係がある。表中の?に入る数値として正しいものを、下の(1)～(5)のうちから1つ選べ。（6点）
【表1】
A B C D
行1 1 3 4 7
行2 2 5 7 17
行3 3 7 10 ?
※ 各列には行方向の規則性がある
※ 各行の4つの数値には共通の関係がある
(1) 25
(2) 27
(3) 29
(4) 31
(5) 35
問1(正解:(4))

【列方向の規則性を分析】
列A: 1, 2, 3 → 公差1の等差数列
列B: 3, 5, 7 → 公差2の等差数列
列C: 4, 7, 10 → 公差3の等差数列
列D: 7, 17, ? → 差分は10, ? （列の規則性だけでは一意に定まらない）

【行方向の関係を分析】
まず、列Cに注目すると:
行1: C = 1 + 3 = 4 = A + B
行2: C = 2 + 5 = 7 = A + B
行3: C = 3 + 7 = 10 = A + B
よって、C = A + B が成り立つ。

次に、列Dに注目すると:
行1: D = 7, A×B + C = 1×3 + 4 = 7
行2: D = 17, A×B + C = 2×5 + 7 = 17
よって、D = A × B + C が成り立つ。

【答えの計算】
行3: D = 3 × 7 + 10 = 21 + 10 = 31

【列方向での検証】
列D: 7, 17, 31 → 差分は10, 14
差分の差分: 14 - 10 = 4（一定の増加）
列A～Cの差分がそれぞれ1, 2, 3であることと整合する。

したがって、正解は(4) 31である。
問2．下表は、行方向と列方向の両方に規則性を持つ数表である。空欄A、B、Cに入る数値の合計（A + B + C）として正しいものを、下の(1)～(5)のうちから1つ選べ。（6点）
【表2】
列1 列2 列3 列4 列5
行1 1 3 5 7 9
行2 4 12 20 28 36
行3 9 A 45 63 B
行4 16 48 C 112 144
※ 行方向にも列方向にも一定の規則性がある
※ 複数の規則性を見つけて答えを導くこと
(1) 178
(2) 183
(3) 188
(4) 193
(5) 198
問2(正解:(3))

【列方向の規則性を分析】
列1: 1, 4, 9, 16 → 1, 4, 9, 16 = 1², 2², 3², 4²（平方数）
列2: 3, 12, A, 48 → 各項は列1の3倍か確認: 3=1×3, 12=4×3, 48=16×3
よって A = 9 × 3 = 27

列3: 5, 20, 45, C → 各項は列1の5倍か確認: 5=1×5, 20=4×5, 45=9×5
よって C = 16 × 5 = 80

列5: 9, 36, B, 144 → 各項は列1の9倍か確認: 9=1×9, 36=4×9, 144=16×9
よって B = 9 × 9 = 81

【行方向の規則性で検証】
行1: 1, 3, 5, 7, 9 → 公差2の等差数列（奇数列）
行2: 4, 12, 20, 28, 36 → 公差8の等差数列
行3: 9, 27, 45, 63, 81 → 公差18の等差数列
行4: 16, 48, 80, 112, 144 → 公差32の等差数列

【規則の一般化】
cell(i, j) = i² × (2j - 1)
行方向: i²を共通因子として、奇数列(1, 3, 5, 7, 9)を掛ける
列方向: (2j-1)を共通因子として、平方数(1, 4, 9, 16)を掛ける

【答えの計算】
A + B + C = 27 + 81 + 80 = 188

したがって、正解は(3) 188である。
