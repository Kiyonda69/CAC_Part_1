問1．下の図1のように、直線ℓと、1辺1cmの方眼のマス目に沿って描かれた図形が同じ平面上にある。図形は直線ℓをまたいでおり、ℓの左右両側に部分を持つ。この図形を、直線ℓを軸として1回転させてできる立体を、直線ℓを含む平面で切断したとき、切り口の形として正しいものを、次の(1)〜(5)のうちから1つ選べ。（6点）
【図1】直線ℓと図形（方眼の1マスは1辺1cm）
問1(正解:(5))
図形が回転軸ℓをまたいでいる点が本問の核心である。回転させると、ℓの右側の部分と左側の部分がそれぞれ別々にℓの周りに円を描く。したがって、ある高さで軸から距離rの位置に立体の材料があるのは、「その高さで、図形が左右どちらかの側に軸から距離rの部分を持つ」ときである。切り口（ℓを含む平面での断面）は必ず左右対称になり、各高さでの半幅は左右のうち軸から遠い方までの距離になる。

【高さごとに左右の張り出しを比べる】
図形の各段の、軸からの張り出しを読み取ると次のようになる。
・1段目（最下段）: 右3cm・左1cm → 遠い方は右の3cm → 切り口の幅6cm
・2段目: 右1cm・左2cm → 遠い方は左の2cm → 切り口の幅4cm
・3段目: 右2cm・左1cm → 遠い方は右の2cm → 切り口の幅4cm
・4段目（最上段）: 右1cm・左0cm → 半径1cm → 切り口の幅2cm
2段目は左側が、3段目は右側が半径を決めており、「遠い方」が段ごとに左右で入れ替わる。どちらか一方の側だけを見ていると正しい切り口は得られない。

【切り口の形】
以上より、切り口は下から幅6cm・4cm・4cm・2cm（各高さ1cm）の長方形を、軸をそろえて積んだ左右対称の形になる。これは(5)の図形である。

【選択肢の検討】
・(1) 元の図形のまま → 回転体の断面は左右対称になるはずで、非対称のままは誤り → 不正解
・(2) 2段目の幅が2cmしかない → 右側（1cm）だけを回転させ、左の2cmの張り出しを無視した形 → 不正解
・(3) 元の図形を左右に折り返しただけの鏡像 → 非対称のままで誤り → 不正解
・(4) 左右対称だが上下が逆（下から2・4・4・6cm） → 図形の上下を取り違えている → 不正解
・(5) 下から幅6・4・4・2cmの左右対称の階段形 → 正解

したがって、正解は(5)である。
問2．下の図2のように、直線ℓと、1辺1cmの方眼のマス目に沿って描かれた図形が同じ平面上にある。図形は直線ℓをまたいでおり、ℓの右側の部分には1辺1cmの正方形の穴A・穴Bがあいている。この図形を、直線ℓを軸として1回転させてできる立体を、直線ℓを含む平面で切断したとき、切り口の形として正しいものを、次の(1)〜(5)のうちから1つ選べ。（6点）
【図2】直線ℓと図形（方眼の1マスは1辺1cm）
問2(正解:(5))
問1と同じく、ある高さで軸から距離rの位置に立体の材料があるのは「その高さで、図形が左右どちらかの側に軸から距離rの部分を持つ」ときである。本問の核心は、この原理を穴に適用することにある。すなわち、右側の板の穴が切り口に残るのは、「同じ高さ・同じ軸からの距離に、左側にも材料がない」ときだけである。反対側に材料があれば、回転したときその材料が穴の位置の円環を掃くため、穴は立体の中で埋まって消える。

【切り口の外形】
右側の板は軸から5cm、左側は縦柱の外端まで4cmなので、どの高さでも最大半径は右の5cmである。よって切り口の外形は、幅10cm（左右5cmずつ）×高さ4cmの左右対称な長方形になる。左側の図形の凹凸（柱と橋の形）は、右の板がすべての高さ・半径を覆うため外形には現れない。

【穴Bは埋まる】
穴Bは軸から3〜4cm、下から3段目の高さにある。ところが左側の縦柱は軸から3〜4cmの位置で下から上まで全高にわたっており、穴Bと同じ高さ・同じ距離に材料がある。回転させると柱がこの半径の円環を一周掃くので、穴Bの位置にも材料が回り込み、切り口から穴Bは消える。

【穴Aは残る】
穴Aは軸から1〜2cm、下から2段目の高さにある。この高さで軸から1〜2cmの位置を見ると、右側は穴Aで材料がなく、左側にも材料がない（左の柱は軸から3〜4cm、橋は最下段のみ）。よって穴Aの位置はどの回転角でも空洞のままで、立体の内部にドーナツ状の空洞として残る。ℓを含む平面はこの空洞を左右2か所で横切るので、切り口には穴Aが左右対称の2か所に現れる。

【選択肢の検討】
・(1) 穴A・穴Bの両方が左右対称に残る → 左の縦柱が穴Bを埋めることを見落としている → 不正解
・(2) 穴が1つもない長方形 → 「反対側の材料で埋まる」理屈の過剰適用。穴Aと同じ高さ・距離には左側にも材料がなく、穴Aは埋まらない → 不正解
・(3) 穴Bだけが残る → 埋まる穴と残る穴の判定が逆 → 不正解
・(4) 元の図形の輪郭のまま → 回転体の断面は左右対称になるはずで、非対称のままは誤り → 不正解
・(5) 幅10cm×高さ4cmの長方形に、穴Aだけが左右対称の位置に残る → 正解

したがって、正解は(5)である。
