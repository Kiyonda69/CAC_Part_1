問1．下図のように、円の中の矢印と円の塗りつぶしが規則的に変化している。?に当てはまる図形として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
【変化の系列】
問1(正解:(3))
円の中の矢印の向きと、円の塗りつぶしの2つの属性が独立に変化している。

【矢印の向きの規則】
矢印の向きを8方位（東・南東・南・南西・西・北西・北・北東）で追うと、各ステップで時計回りに45°ずつ回転している。
・1: 東（→）
・2: 南東（↘）
・3: 南（↓）
・4: 南西（↙）
・5: 西（←）
・6: 西から45°時計回り → 北西（↖）

【塗りつぶしの規則】
円の塗りつぶしは黒と白の交互。
・1黒 → 2白 → 3黒 → 4白 → 5黒 → 6は白

【結論】
6番目は「白い円の中に北西向き（↖）の矢印」。

【選択肢の検討】
・(1) 北西・黒 → 向きは正しいが、色が誤り（6番目は白）
・(2) 西・白 → 色は正しいが、向きが Step5 から進んでいない
・(3) 北西・白 → 正解（両属性とも規則に合致）
・(4) 北・白 → 西から90°進んだ誤答（時計回り45°を超過）
・(5) 南西・白 → 西から反時計回りに戻した誤答

したがって、正解は(3)である。
問2．下図のように、「外形」「中の数字」「枠線の種類」の3つの属性がそれぞれ独立した規則で変化する図形が並んでいる。?に当てはまる図形として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
【変化の系列】
問2(正解:(1))
図形は「外形」「中の数字」「枠線の種類」の3つの属性がそれぞれ独立した規則で変化している。

【属性①：外形】
○ → □ → △ の周期3のサイクルを繰り返す。
・1=○, 2=□, 3=△, 4=○, 5=□, 6=△

【属性②：中の数字】
各ステップで1ずつ増加する。
・1, 2, 3, 4, 5, 6

【属性③：枠線の種類】
実線と点線が周期2で交互に現れる。
・1=実, 2=点, 3=実, 4=点, 5=実, 6=点

【結論】
6番目は「△・数字6・点線」。

【選択肢の検討】
・(1) △・6・点線 → 正解（3属性とも規則に合致）
・(2) △・6・実線 → 枠線が誤り（5=実だから6は点のはず）
・(3) ○・6・点線 → 外形が誤り。Step6は周期3で位置3(=△)なので○ではない
・(4) △・5・点線 → 数字が誤り。+1を忘れてStep5と同じ5にしている
・(5) □・6・点線 → 外形が誤り。Step5(=□)と同じにしてしまった誤答

したがって、正解は(1)である。
