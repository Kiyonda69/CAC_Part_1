問1．1辺1cmの立方体を、3×3のマス目の上に積み上げて立体を作る。【図1】は3×3のマス目を上から見た図であり、矢印アの方向から見た図を正面図、矢印イの方向から見た図を右側面図とする。正面図と右側面図がそれぞれ【図2】、【図3】のとおりであるとき、使用する立方体の個数の最大値として正しいものを、下の(1)～(5)のうちから一つ選べ。（6点）
【図1】マス目と視線方向（上から見た図）
(1)　12個　　　(2)　14個　　　(3)　15個　　　(4)　16個　　　(5)　18個
問1(正解:(2))

【基本的な考え方】
正面図は各x列について、y方向の最大の高さを表す。右側面図は各y列について、x方向の最大の高さを表す。

立方体の個数を最大にするには、各マス(x, y)の高さh(x, y)をできるだけ大きくする。
ただし、h(x, y)は正面図のx列の高さと右側面図のy列の高さの両方を超えてはならない。
よって、各マスの最大高さは h(x, y) = min(正面図[x], 右側面図[y]) となる。

【各マスの最大高さ】
正面図の高さ: x=1→2、x=2→3、x=3→1
右側面図の高さ: y=1→3、y=2→1、y=3→2

y=1 y=2 y=3
x=1 min(2,3)=2 min(2,1)=1 min(2,2)=2
x=2 min(3,3)=3 min(3,1)=1 min(3,2)=2
x=3 min(1,3)=1 min(1,1)=1 min(1,2)=1

【合計の計算】
2+1+2+3+1+2+1+1+1 = 14個

【検証】
・正面図: x=1列のmax(2,1,2)=2、x=2列のmax(3,1,2)=3、x=3列のmax(1,1,1)=1 → 一致
・右側面図: y=1列のmax(2,3,1)=3、y=2列のmax(1,1,1)=1、y=3列のmax(2,2,1)=2 → 一致

【選択肢の検討】
・(1) 12個 → 各マスの高さを最大にしていない場合の誤答
・(2) 14個 → 正解
・(3) 15個 → 正面図・右側面図の制約を一部無視した誤答
・(4) 16個 → 計算誤りによる誤答
・(5) 18個 → 正面図の高さの合計×3で計算した誤答

したがって、正解は(2)である。
問2．下図のように、1辺の長さが6の立方体ABCD-EFGHがある。辺AB上にAP = 2となる点P、辺BF上にBQ = 4となる点Q、辺GH上にGR = 4となる点Rをとる。3点P、Q、Rを通る平面でこの立方体を切断するとき、切断面の形状として正しいものを、下の(1)～(5)のうちから一つ選べ。（6点）
【立方体ABCD-EFGH】
(1)　三角形　　　(2)　六角形　　　(3)　四角形　　　(4)　五角形　　　(5)　七角形
問2(正解:(2))

【座標の設定】
A=(0,0,0)、B=(6,0,0)、C=(6,6,0)、D=(0,6,0)、E=(0,0,6)、F=(6,0,6)、G=(6,6,6)、H=(0,6,6)とする。
P=(2,0,0)、Q=(6,0,4)、R=(2,6,6)

【切断面の平面方程式】
ベクトルPQ=(4,0,4)、PR=(0,6,6)の外積から法線ベクトルを求めると(-24,-24,24)。
整理すると平面の方程式は x + y - z = 2 となる。

【各辺との交点の探索】
切断面を求めるには、立方体の各面ごとに平面との交線をたどる。

(1) 底面（z=0）では x+y=2。点Pから辺AB上を出発し、底面上で辺ADと交わる点を求めると (0,2,0)（辺AD上）。

(2) 左面（x=0）では y-z=2。点(0,2,0)から左面上で辺DHと交わる点を求めると (0,6,4)（辺DH上）。

(3) 奥面（y=6）では x-z=-4。点(0,6,4)から奥面上で辺GHと交わる点を求めると点R=(2,6,6)。

(4) 上面（z=6）では x+y=8。点Rから上面上で辺FGと交わる点を求めると (6,2,6)（辺FG上）。

(5) 右面（x=6）では y-z=-4。点(6,2,6)から右面上で辺BFと交わる点を求めると点Q=(6,0,4)。

(6) 前面（y=0）では x-z=2。点Qから前面上をたどると出発点P=(2,0,0)に戻る。

【切断面の頂点】
P(2,0,0) → (0,2,0) → (0,6,4) → R(2,6,6) → (6,2,6) → Q(6,0,4) → Pの6頂点を結ぶ六角形。

【選択肢の検討】
・(1) 三角形 → 3辺との交点のみ考えた誤答
・(2) 六角形 → 正解。平面が立方体の6面すべてと交わるため
・(3) 四角形 → 一部の面との交点を見落とした誤答
・(4) 五角形 → 交点の探索が不完全な誤答
・(5) 七角形 → 立方体は6面なので最大でも六角形。不可能

したがって、正解は(2)である。
