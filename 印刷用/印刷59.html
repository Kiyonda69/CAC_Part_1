問1．図は、3つの店舗P、Q、Rの売上高の前年比推移を示したものである。2020年の店舗Qの売上高が4000万円であったとき、2024年の店舗Qの売上高はおよそいくらになるか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
【売上高の前年比推移】
(1) 4960万円　　(2) 5380万円　　(3) 5820万円　　(4) 6140万円　　(5) 6500万円
問1(正解:(4))

【解法】
前年比推移のグラフから、店舗Qの各年の前年比を読み取る。

・2021年: +25%
・2022年: -10%
・2023年: +5%
・2024年: +30%

2020年の売上高4000万円に、各年の前年比を順に掛けていく。

【計算過程】
・2021年: 4000 x 1.25 = 5000（万円）
・2022年: 5000 x 0.90 = 4500（万円）
・2023年: 4500 x 1.05 = 4725（万円）
・2024年: 4725 x 1.30 = 6142.5（万円）

【注意点】
前年比は「前年の値に対する比率」であるため、単純に比率を加算するのではなく、各年の値に順に掛ける必要がある（複利的な計算）。

したがって、2024年の店舗Qの売上高はおよそ6140万円であり、正解は(4)である。
問2．問1の図を用いて、以下の問いに答えよ。2020年の3店舗P、Q、Rの売上高の合計が12000万円であり、P、Q、Rの売上高の比が3:2:1であったとする。このとき、2024年における店舗Pと店舗Rの売上高の差はおよそいくらか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
【売上高の前年比推移】（問1の図を再掲）
(1) 4280万円　　(2) 4950万円　　(3) 5320万円　　(4) 5540万円　　(5) 5780万円
問2(正解:(5))

【ステップ1: 2020年の各店舗の売上高を求める】
P:Q:R = 3:2:1 で合計12000万円なので、
・店舗P = 12000 x 3/6 = 6000（万円）
・店舗Q = 12000 x 2/6 = 4000（万円）
・店舗R = 12000 x 1/6 = 2000（万円）

【ステップ2: 店舗Pの2024年売上高を計算】
グラフから店舗Pの前年比: +10%, +20%, -5%, +15%
・2021年: 6000 x 1.10 = 6600（万円）
・2022年: 6600 x 1.20 = 7920（万円）
・2023年: 7920 x 0.95 = 7524（万円）
・2024年: 7524 x 1.15 = 8652.6（万円）

【ステップ3: 店舗Rの2024年売上高を計算】
グラフから店舗Rの前年比: +5%, +15%, +25%, -5%
・2021年: 2000 x 1.05 = 2100（万円）
・2022年: 2100 x 1.15 = 2415（万円）
・2023年: 2415 x 1.25 = 3018.75（万円）
・2024年: 3018.75 x 0.95 = 2867.8（万円）

【ステップ4: 差を求める】
8652.6 - 2867.8 = 5784.8 ≈ 5780（万円）

したがって、2024年における店舗Pと店舗Rの売上高の差はおよそ5780万円であり、正解は(5)である。
