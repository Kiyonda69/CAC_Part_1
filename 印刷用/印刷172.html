問1．下図は、1辺1cmの立方体ブロックを組み合わせて作った立体を、平面図・正面図・側面図（右側面）の3方向から見た図である。この立体として正しいものは、次のうちどれか。（6点）
【図1】3面図
問1(正解:(3))
3面図の読み取りから立体を特定する。

【3面図の読解】
・平面図: 外形は2×2の正方形（上から見た時、4つの位置すべてに何らかのブロックがある）。中央の水平線は、前半分と後半分で高さが異なることを示している。
・正面図: 2×2の正方形（前から見て、上下左右すべて埋まる）。
・側面図: 右上の1区画が欠けたL字形（右側から見て、上奥の位置が空いている）。

【立体の推定】
・側面図で「右上欠け」＝奥・上の位置が空いている。
・正面図が2×2の完全な正方形である以上、前列(手前)の上段ブロックは存在する。
・よって、欠けているのは「奥列の上段」の2個（左奥・右奥の上段）である。
・2×2×2の立方体から、奥の上端の1辺（2個）を取り除いた形となる。

【選択肢の検討】
・(1) 左列の上段2個が欠けている → 正面図が「左上欠けL」になる → 不正解
・(2) 前列の上段2個が欠けている → 側面図が「左上欠けL」になる → 不正解
・(3) 奥列の上段2個が欠けている → 全条件と一致 → 正解
・(4) 右列の上段2個が欠けている → 正面図が「右上欠けL」になる → 不正解
・(5) 右奥上の1個のみ欠け → 側面図・正面図とも完全な2×2正方形 → 不正解

したがって、正解は(3)である。
問2．下図は、1辺1cmの立方体ブロックを組み合わせて作った別の立体を、平面図・正面図・側面図（右側面）の3方向から見た図である。この立体として正しいものは、次のうちどれか。（6点）
【図2】3面図
問2(正解:(4))
3つのブロックが取り除かれた非対称な立体を3面図から特定する。

【3面図の読解】
・平面図: 外形は2×2、内部のL字線は奥左の1区画のみが他より高いことを示す。すなわち奥左の位置のみ高さ2、他3区画は高さ1。
・正面図: 右上が欠けたL字。前から見て上段の右側が空いている。
・側面図: 左上が欠けたL字（凹型）。右から見て上段の左側（＝前側の上段）が空いている。

【立体の推定】
・奥左のみ高さ2＝奥左の上段ブロック(0,1,1)は存在する。
・正面 右上欠け＝前から見て右上が空＝右列の上段(x=1, z=1)に手前・奥とも何もない＝(1,0,1)・(1,1,1)とも欠如。
・側面 左上欠け＝右から見て左上（前側上段）が空＝前列の上段(y=0, z=1)で(0,0,1)・(1,0,1)とも欠如。
・以上から、欠けているブロックは(0,0,1)、(1,0,1)、(1,1,1)の3個。
・残るブロックは5個で、奥左の柱(0,1,0)(0,1,1)＋底面3個(0,0,0)(1,0,0)(1,1,0)。

【選択肢の検討】
・(1) 奥左の上段を欠き、右上端を欠く → 平面図でL字内部線の位置が異なる → 不正解
・(2) 左列の上段2個と右奥上を欠く → 正面図が左上欠けL字 → 不正解
・(3) 右列の上段2個のみ欠け（3個でなく2個） → 平面図の内部線が縦の中央線になる → 不正解
・(4) 前列の上段2個と右奥上を欠く → 全条件と一致 → 正解
・(5) 前列の上段2個と左奥上を欠く → 平面図で右奥のみが高くなる → 不正解

したがって、正解は(4)である。
