問1．次の【資料】および【表】を読み、資料・表の内容と数値的に整合する記述として正しいものを、下の(1)〜(5)のうちから1つ選べ。（6点）
【資料】H市立図書館 月次運営報告書（2025年10月）
H市立図書館協議会は、市内5つの図書館（中央館、北支館、南支館、東支館、西支館）の2025年10月の運営状況について、開館日数、延べ来館者数、貸出冊数の3指標を集計した。
中央館は市内で最も規模の大きい図書館であり、10月は28日間開館し、延べ来館者数は14,000人、貸出冊数は21,000冊であった。支館では、北支館が25日間開館して4,500人が来館し、8,500冊を貸し出した。南支館は24日間開館し、来館者は6,000人、貸出冊数は7,200冊であった。東支館は26日間開館して5,200人を迎え、10,400冊を貸し出した。西支館は27日間開館し、8,100人が来館して13,500冊を貸し出した。
なお、運営指標として「1日あたり平均来館者数」「1日あたり平均貸出冊数」「来館者1人あたり貸出冊数」の3種類を定義している。指標の比較は各図書館ごとの値で行い、合計値ではなく平均値に基づく。
【表】市立5図書館の運営実績（2025年10月）
図書館 開館日数（日） 来館者数（人） 貸出冊数（冊）
中央館 28 14,000 21,000
北支館 25 4,500 8,500
南支館 24 6,000 7,200
東支館 26 5,200 10,400
西支館 27 8,100 13,500
合計 130 37,800 60,600

(1)5館のうち、1日あたり平均来館者数が最も多いのは西支館である。

(2)5館のうち、1日あたり平均貸出冊数が最も少ないのは北支館である。

(3)貸出冊数が最も多いのは中央館であり、その冊数は5館合計の半分以上を占めている。

(4)開館日数が最も多い図書館は西支館で、27日間開館した。

(5)来館者1人あたり貸出冊数が最大の図書館は東支館であり、その値はちょうど2.0冊である。

問1(正解:(5))

各選択肢の数値を表から計算し、資料・表の内容と整合するものを特定する。

【派生量の計算】
・1日あたり平均来館者数（＝来館者数÷開館日数）
　中央館：14,000÷28＝500.0人／北支館：4,500÷25＝180.0人／南支館：6,000÷24＝250.0人／東支館：5,200÷26＝200.0人／西支館：8,100÷27＝300.0人
・1日あたり平均貸出冊数（＝貸出冊数÷開館日数）
　中央館：21,000÷28＝750.0冊／北支館：8,500÷25＝340.0冊／南支館：7,200÷24＝300.0冊／東支館：10,400÷26＝400.0冊／西支館：13,500÷27＝500.0冊
・来館者1人あたり貸出冊数（＝貸出冊数÷来館者数）
　中央館：21,000÷14,000＝1.500冊／北支館：8,500÷4,500≒1.889冊／南支館：7,200÷6,000＝1.200冊／東支館：10,400÷5,200＝2.000冊／西支館：13,500÷8,100≒1.667冊

【選択肢の検討】
・(1) 1日あたり平均来館者数の最多は中央館の500.0人。西支館は300.0人で第2位 → 誤り
・(2) 1日あたり平均貸出冊数の最少は南支館の300.0冊。北支館は340.0冊で2番目に少ない → 誤り
・(3) 貸出冊数最多は確かに中央館（21,000冊）だが、5館合計60,600冊の半分は30,300冊。21,000冊＜30,300冊なので「半分以上」ではない → 誤り
・(4) 開館日数最多は中央館の28日。西支館は27日で第2位 → 誤り
・(5) 1人あたり貸出冊数の最大は東支館の2.000冊で、資料・表と完全に整合 → 正しい

したがって、正解は(5)である。
問2．次の【資料】および【表】を読み、T食品工場で「優良品」として評価される製品の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。（6点）
【資料】T食品工場 月次品質管理報告書（2025年10月）
T食品工場では5種類の主力製品（P1、P2、P3、P4、P5）を製造している。同工場では、月ごとに生産数・不良品数・顧客返品数・製造から出荷までの平均リードタイムを集計し、次の4つの「出荷基準」をすべて満たす製品のみを「優良品」として評価している（基準値ちょうどの場合は基準を満たすものとする）。

【出荷基準】
①　月間生産数が10,000個以上であること
②　不良品率（不良品数÷生産数）が1.0％以下であること
③　顧客返品率（返品数÷生産数）が0.5％以下であること
④　製造から出荷までの平均リードタイムが3.0日以内であること
なお、不良品率および返品率は小数点以下第4位を切り捨てて判定する。生産数・不良品数・返品数・リードタイムは下表のとおりである。
【表】製品別 月次生産実績（2025年10月）
製品 生産数（個） 不良品数（個） 返品数（個） 平均リードタイム（日）
P1 12,000 100 40 2.5
P2 15,000 120 50 3.2
P3 8,000 40 20 2.0
P4 11,000 110 30 2.8
P5 14,000 140 80 2.7

(1)P1 と P2

(2)P1 と P4

(3)P2 と P4

(4)P1、P3 と P5

(5)P2、P4 と P5

問2(正解:(2))

各製品について4つの出荷基準を順に検証する。

【基準ごとの検証】
・P1：生産12,000（≥10,000 ✓）／不良率 100÷12,000≒0.833%（≤1.0% ✓）／返品率 40÷12,000≒0.333%（≤0.5% ✓）／リード2.5日（≤3.0 ✓）→ 全基準充足（優良品）
・P2：生産15,000（✓）／不良率 120÷15,000＝0.800%（✓）／返品率 50÷15,000≒0.333%（✓）／リード3.2日＞3.0日（✗）→ 基準④で不適
・P3：生産8,000＜10,000（✗）→ 基準①で不適（この時点で他条件を検討する必要はない）
・P4：生産11,000（✓）／不良率 110÷11,000＝1.000%（≤1.0% ✓、境界値はOK）／返品率 30÷11,000≒0.272%（✓）／リード2.8日（✓）→ 全基準充足（優良品）
・P5：生産14,000（✓）／不良率 140÷14,000＝1.000%（✓）／返品率 80÷14,000≒0.571%＞0.5%（✗）→ 基準③で不適

【優良品】P1 と P4 の2製品のみが4基準すべてを満たす。

【選択肢の検討】
・(1) P2はリード3.2日で基準④違反 → 誤り
・(2) P1とP4は共に全基準を満たす → 正しい
・(3) P2は基準④違反のため優良品ではない → 誤り
・(4) P3は生産量8,000で基準①違反、P5は返品率0.571%で基準③違反 → 誤り
・(5) P2（基準④違反）およびP5（基準③違反）が含まれる → 誤り

したがって、正解は(2)である。
