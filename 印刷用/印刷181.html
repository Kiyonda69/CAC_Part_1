問1．下図は、ある航空機を真上から見た図（上面図）と前方から見た図（正面図）である。この航空機を機体の右側面から見た図として最も適切なものを、(1)〜(5)から1つ選べ。（6点）
【図1】航空機の上面図と正面図
右側面図として正しいものは？
問1(正解:(3))
上面図と正面図から、航空機の三次元的な特徴を読み取り、右側面図を構成する。

【上面図から読み取れる情報】
・機首は右側（細く尖った先端）
・機尾は左側（垂直尾翼の根元・水平尾翼が配置）
・主翼は胴体中央付近に左右対称に広がる（後退翼）
・水平尾翼は機尾近くに左右対称に広がる
・エンジンは主翼下の左右に2基

【正面図から読み取れる情報】
・垂直尾翼は胴体の上方に伸びる三角形
・エンジンは主翼より下方にぶら下がる円形
・主翼は水平に左右に伸びる

【右側面図に必要な特徴】
右側面（機体の右側から見た図）では、機首は右、機尾は左に来る（観察者は機体の右側に立ち、機首方向を右、機尾方向を左に見る）。
・機首：胴体の右端に尖った形
・垂直尾翼：胴体の後部（左側）の上方に三角形
・水平尾翼：胴体の後部（左側）に水平の薄い形
・主翼：胴体中央に水平の薄い形（横から見ると線状）
・エンジン：主翼下の下方に楕円

【選択肢の検討】
・(1) 機首が左を向いている → 上面図と方向が逆 → 不正解
・(2) 垂直尾翼が機首側（前方）にある → 上面図では尾翼は機尾側 → 不正解
・(3) 機首が右、垂直尾翼が後部上方、エンジンが下方 → すべての条件と一致 → 正解
・(4) 垂直尾翼が胴体の下方にある → 正面図では尾翼は上方 → 不正解
・(5) 機首が尖っておらず平らになっている → 上面図では機首は尖った形 → 不正解

したがって、正解は(3)である。
問2．下図は、立方体の展開図である。立方体の6つの面には、それぞれ航空に関係する文字（A：エアクラフト、B：ブルースカイ、C：コントロール、D：ダイヤモンド、E：エンジン、F：フラップ）が描かれている。この展開図を組み立て、Bの面が正面に、Cの面が下になるように立方体を置いた。このとき、右側面に見える文字として正しいものを、(1)〜(5)から1つ選べ。（6点）
【図2】立方体の展開図
右側面に見える文字は？
(1) F　　(2) D　　(3) A　　(4) E　　(5) C
問2(正解:(2))
展開図から立方体を組み立て、面の位置関係を空間的に把握する問題である。

【展開図の構造】
T字型展開図において、Cの面を中心とすると、上下左右に隣接する面はCを正面とした際の各方向の面になる。
・C：中央（正面）
・B：Cの上に隣接 → Cを正面に置いたときの上面
・F：Cの下に隣接 → Cを正面に置いたときの下面
・A：Cの左に隣接 → Cを正面に置いたときの左側面
・D：Cの右に隣接 → Cを正面に置いたときの右側面
・E：Dの右端、つまり折り返して裏面（後面）

【標準向きの面配置（Cを正面）】
・正面：C　・上：B　・下：F　・左：A　・右：D　・後：E

【操作: Bを正面、Cを下にする】
これは「立方体を観察者側へ前傾90度倒す」操作に相当する。すなわち、もともと上にあったBを正面に持ってきて、もともと正面だったCを下に倒し込む。
・正面：B（元の上面）
・上：E（元の後面）
・下：C（元の正面）
・後：F（元の下面）
・左：A（不変）
・右：D（不変）

前後方向の回転では左右は変化しないため、AとDの位置は標準向きと同じである。

【選択肢の検討】
・(1) F：操作後は後面 → 不正解
・(2) D：操作後も右側面のまま → 正解
・(3) A：操作後も左側面のまま → 不正解
・(4) E：操作後は上面 → 不正解
・(5) C：操作後は下面 → 不正解

したがって、正解は(2)である。
