問1．A、B、C、D、Eの5つの倉庫があり、それぞれの保管量は10トン、20トン、30トン、40トン、50トンのいずれかで、すべて異なる。次の条件がすべて成り立つとき、倉庫Cの保管量として正しいものを、次の(1)～(5)の中から1つ選べ。（6点）
【条件】
[ア]　倉庫Aと倉庫Bの保管量の合計は、倉庫Cと倉庫Dの保管量の合計に等しい。
[イ]　倉庫Bの保管量は、倉庫Cの保管量より20トン少ない。
[ウ]　倉庫Aの保管量は、倉庫Eの保管量より少ない。
[エ]　倉庫Dの保管量は、倉庫Bの保管量より少ない。
[オ]　倉庫Eの保管量は20トンか50トンである。
(1)　20トン
(2)　30トン
(3)　40トン
(4)　10トン
(5)　50トン
問1(正解:(3))

【条件の整理】
5つの倉庫に10, 20, 30, 40, 50トンを割り当てる。
・[イ] B = C - 20 より、(C, B) の候補は (30,10), (40,20), (50,30)
・[オ] E = 20 または 50

【場合分け】
(C=30, B=10 の場合)
・[ア] A + 10 = 30 + D → A = D + 20
・残り {20, 40, 50} から A, D, E を選ぶ
・[オ] E=20: A,D は {40,50}。A=D+20 → D=40,A=60(不可) または D=50,A=70(不可)
・[オ] E=50: A,D は {20,40}。A=D+20 → D=20,A=40 ✓
・[ウ] A < E → 40 < 50 ✓、[エ] D < B → 20 < 10 ✗ → 不成立

(C=40, B=20 の場合)
・[ア] A + 20 = 40 + D → A = D + 20
・残り {10, 30, 50} から A, D, E を選ぶ
・[オ] E=50: A,D は {10,30}。A=D+20 → D=10,A=30 ✓
・[ウ] A < E → 30 < 50 ✓、[エ] D < B → 10 < 20 ✓ → 成立

(C=50, B=30 の場合)
・[ア] A + 30 = 50 + D → A = D + 20
・残り {10, 20, 40} から A, D, E を選ぶ
・[オ] E=20: A,D は {10,40}。A=D+20 → D=10,A=30(不可) または D=40,A=60(不可) → 不成立

【結論】
唯一の解: A=30, B=20, C=40, D=10, E=50
したがって、倉庫Cの保管量は40トンであり、正解は(3)である。
問2．A～Eの5つのプロジェクトを第1週～第5週の各週に1つずつ割り当てる。次の条件がすべて成り立つとき、プロジェクトEが予定されている週として正しいものを、次の(1)～(5)の中から1つ選べ。（6点）
【条件】
[ア]　プロジェクトAはプロジェクトCより前の週に予定されており、AとCの間には少なくとも2週間の間隔がある。
[イ]　プロジェクトBはプロジェクトEの直前または直後の週に予定されている。
[ウ]　プロジェクトDは第1週と第5週には予定されていない。
[エ]　プロジェクトCはプロジェクトEより後の週に予定されている。
[オ]　プロジェクトAは第1週または第2週に予定されている。
[カ]　プロジェクトBは奇数番目の週に予定されている。
[キ]　プロジェクトDはプロジェクトEより後の週に予定されている。
(1)　第1週
(2)　第2週
(3)　第3週
(4)　第4週
(5)　第5週
問2(正解:(2))

【条件の整理】
・[オ] A = 第1週 or 第2週
・[ア] C ≥ A + 3（Aより少なくとも2週間後）
・[ウ] D ∈ {第2週, 第3週, 第4週}
・[カ] B ∈ {第1週, 第3週, 第5週}

【場合分け: Aの位置】
(A=第2週の場合)
・[ア] C ≥ 5 → C = 第5週
・[カ] B ∈ {1, 3, 5} から A=2, C=5 を除く → B ∈ {1, 3}
・B=1: [イ] |B-E|=1 → E=2(=A) → 不可
・B=3: [イ] E=2(=A) or E=4。E=4 → D=残り=1 → [ウ] D≠1 → 不可
→ A=第2週は不成立

(A=第1週の場合)
・[ア] C ≥ 4 → C = 第4週 or 第5週

C=第4週のとき:
・[カ] B ∈ {1,3,5} から A=1,C=4 を除く → B ∈ {3, 5}
・B=3: [イ] E=2 or 4(=C)。E=2 → D=5 → [ウ] D≠5 → 不可
・B=5: [イ] E=4(=C) or 6 → 不可
→ C=第4週は不成立

C=第5週のとき:
・[カ] B ∈ {1,3,5} から A=1,C=5 を除く → B = 3
・[イ] E=2 or E=4
・E=2: D=4 → [キ] D>E → 4>2 ✓、[エ] C>E → 5>2 ✓、[ウ] D=4 ✓
・E=4: D=2 → [キ] D>E → 2>4 ✗ → 不可
→ E=2, D=4 で成立

【結論】
唯一の解: A=第1週, E=第2週, B=第3週, D=第4週, C=第5週
したがって、プロジェクトEは第2週に予定されており、正解は(2)である。
