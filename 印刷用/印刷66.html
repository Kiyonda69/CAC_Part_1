問1．下図のように、一辺の長さが2cmの正方形ABCDが直線l上を滑ることなく転がる。正方形が1回転（元の向きに戻る）したとき、頂点Aが描く軌跡の長さとして正しいものを(1)〜(5)から1つ選べ。（6点）
【図1】正方形の回転と頂点Aの軌跡
(1)　2π√2 cm
(2)　3π cm
(3)　2π(1+√2) cm
(4)　π(2+√2) cm
(5)　π(3+√2) cm
問1(正解:(4))

【回転の分析】
正方形ABCDが直線l上を1回転（360°）するには、90°ずつ4回の回転が必要である。各回転では、地面に接している頂点を中心として正方形が倒れるように転がる。

辺DAが底辺の初期状態から、4段階の回転を追跡する。

【各段階での頂点Aの動き】
・段階1：頂点A（右下）を中心に90°回転 → Aは回転の中心なので動かない（軌跡の長さ = 0）
・段階2：頂点B（Aの隣）を中心に90°回転 → AはBから距離2cm → 半径2、角度90°の円弧 = 2 × π/2 = π cm
・段階3：頂点C（Aの対角）を中心に90°回転 → AはCから距離2√2 cm（対角線）→ 半径2√2、角度90°の円弧 = 2√2 × π/2 = π√2 cm
・段階4：頂点D（Aの隣）を中心に90°回転 → AはDから距離2cm → 半径2、角度90°の円弧 = 2 × π/2 = π cm

【合計の計算】
軌跡の長さ = 0 + π + π√2 + π = π(2 + √2) cm

【選択肢の検討】
・(1) 2π√2 ≒ 8.89 cm → 段階3の円弧の2倍であり不正解
・(2) 3π ≒ 9.42 cm → 全ての円弧が半径2と仮定した場合であり不正解
・(3) 2π(1+√2) ≒ 15.17 cm → 値が大きすぎて不正解
・(4) π(2+√2) ≒ 10.73 cm → 正しい計算結果と一致
・(5) π(3+√2) ≒ 13.87 cm → 段階1でも円弧を加算した場合であり不正解

したがって、正解は(4)である。
問2．下図のように、半径1cmの円板が一辺6cmの正三角形の外側に沿って滑ることなく転がり、正三角形の周りを1周する。このとき、円板の中心が描く軌跡で囲まれた領域の面積として正しいものを(1)〜(5)から1つ選べ。（6点）
【図2】円板が正三角形の外側を転がる様子
(1)　(9√3 + 18) cm²
(2)　(9√3 + 18 + 2π) cm²
(3)　(9√3 + 24) cm²
(4)　(9√3 + 20 + π) cm²
(5)　(9√3 + 18 + π) cm²
問2(正解:(5))

【軌跡の構成】
円板の中心は、正三角形の各辺から常に半径1cmだけ離れた位置を通る。この軌跡は以下の要素から構成される：
・直線部分：各辺に平行な線分（辺と同じ長さ6cm）が3本
・円弧部分：各頂点で120°の円弧（半径1cm）が3個

【面積の分解】
軌跡で囲まれた領域は、以下の3種類の部分に分解できる：

1. 元の正三角形の面積
S₁ = (√3/4) × 6² = 9√3 cm²

2. 3つの長方形（辺に沿った帯状領域）
各辺（長さ6cm）に対して、幅1cmの帯が外側にできる。
S₂ = 3 × 6 × 1 = 18 cm²

3. 3つの扇形（頂点での円弧部分）
正三角形の内角は60°なので、外側を回る際の転回角（外角）は180° - 60° = 120°である。
各頂点で半径1cm、中心角120°の扇形ができる。
3つの扇形の中心角の合計 = 120° × 3 = 360°（ちょうど完全な円）
S₃ = π × 1² = π cm²

【合計面積】
S = S₁ + S₂ + S₃ = 9√3 + 18 + π cm²

【選択肢の検討】
・(1) 扇形の面積が含まれていない → 不正解
・(2) 扇形の面積が2π（実際はπ）→ 不正解
・(3) 帯の面積24は長方形の幅を大きく見積もっている → 不正解
・(4) 帯の面積20は辺の長さの扱いに誤りがある → 不正解
・(5) 9√3 + 18 + π → 正しい計算結果と一致

したがって、正解は(5)である。
