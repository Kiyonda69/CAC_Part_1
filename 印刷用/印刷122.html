問1．下の図は、3つの属性（形・塗り・点の数）を持つ図形を、ある規則に従って3×3のマトリクスに配置したものである。各行・各列において、それぞれの属性は異なる値をとる。「?」のマスに入る図形として正しいものを、下の(1)～(5)のうちから1つ選べ。（6点）
【図1】図形マトリクス
※ 各図形には「形」「塗り」「点の数」の3つの属性がある
問1(正解:(3))

【3つの規則を発見する】
マトリクスの各マスの図形には、「形」「塗り」「点の数」の3つの属性があり、それぞれが独立した規則に従っている。各行・各列に3種類の値が1つずつ出現する（ラテン方陣構造）。

【規則1: 形の配置】
行1: △ → □ → ○
行2: ○ → △ → □
行3: □ → ○ → ?
各行・各列で形が1つずつ出現。行3には□と○があり残りは△。列3には○と□があり残りは△。→ ?の形は△（三角）

【規則2: 塗りの配置】
行1: 斜 → 黒 → 空
行2: 黒 → 空 → 斜
行3: 空 → 斜 → ?
各行・各列で塗りが1つずつ出現。行3には空と斜があり残りは黒。列3には空と斜があり残りは黒。→ ?の塗りは黒

【規則3: 点の数】
行1: 1, 2, 3
行2: 3, 1, 2
行3: 2, 3, ?
各行・各列に1, 2, 3が1つずつ出現。
行3には2, 3があり → ?=1。列3には3, 2があり → ?=1。一致を確認。

【答え】
形=△（三角）、塗り=黒、点=1個
したがって、正解は(3)である。
問2．下の図は、3×3の白黒パターンを3×3のマトリクスの各マスに配置したものである。各行および各列のパターンの間には共通の規則がある。「?」のマスに入るパターンとして正しいものを、下の(1)～(5)のうちから1つ選べ。（6点）
【図2】白黒パターンマトリクス
※ 各マスには3×3の白黒パターンが入る
問2(正解:(4))

【規則の発見】
各行の3つのパターンに着目する。1行目のパターンを仮にA, B, Cとすると、各マス（3×3=9マス）を1つずつ比較した場合、Cは「AとBの排他的論理和（XOR）」で得られることがわかる。

XOR規則: 2つのマスが同じ色→白、異なる色→黒

【1行目で検証】
A(1行1列)とB(1行2列)のXORがC(1行3列)と一致するか確認:
・(1,1)マス: A=黒, B=黒 → 同じ → XOR=白 → C=白（一致）
・(1,2)マス: A=白, B=黒 → 異なる → XOR=黒 → C=黒（一致）
・(2,2)マス: A=黒, B=黒 → 同じ → XOR=白 → C=白（一致）
（全9マスで一致を確認できる）

【列方向の規則も確認】
同じXOR規則が列方向にも成り立つ。
例: 1列目のA XOR D = G（3行1列のパターン）

【答えの導出】
?のパターンは2通りの方法で求められる:
・行方向: G(3行1列) XOR H(3行2列) = ?
・列方向: C(1行3列) XOR F(2行3列) = ?

いずれの計算からも同じ結果が得られる:
1段目: 黒白白
2段目: 白黒白
3段目: 黒黒白

したがって、正解は(4)である。
