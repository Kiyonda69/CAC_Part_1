問1．あるコンビニエンスストアチェーンでは、5つの支店（A〜E）の週間売上データを集計している。表1は各支店の第1週から第4週までの売上（単位：万円）を示したものである。4週間の平均売上が87万円以上の支店のうち、第4週の売上が最も高い支店として正しいものを1つ選び、解答用紙にマークせよ。(6点)
【表1】支店別週間売上データ（単位：万円）
支店 第1週 第2週 第3週 第4週
A 88 90 92 95
B 78 84 91 87
C 90 86 94 98
D 85 87 89 92
E 75 82 90 99
(1) 支店A
(2) 支店C
(3) 支店B
(4) 支店D
(5) 支店E
問1（正解：(2)）

【各支店の4週間平均を計算】
支店A：(88 + 90 + 92 + 95) / 4 = 365 / 4 = 91.2万円 → 87万円以上
支店B：(78 + 84 + 91 + 87) / 4 = 340 / 4 = 85.0万円 → 87万円未満
支店C：(90 + 86 + 94 + 98) / 4 = 368 / 4 = 92.0万円 → 87万円以上
支店D：(85 + 87 + 89 + 92) / 4 = 353 / 4 = 88.2万円 → 87万円以上
支店E：(75 + 82 + 90 + 99) / 4 = 346 / 4 = 86.5万円 → 87万円未満

【条件を満たす支店】
平均87万円以上の支店：A、C、D

【第4週の売上を比較】
支店A：95万円
支店C：98万円
支店D：92万円

第4週の売上が最も高いのは支店C（98万円）である。
正解：(2) 支店C
問2．表1のデータを用いて、各支店の週間成長率（前週比）を分析する。ある支店が「安定成長型」と判定されるのは、以下の2つの条件を同時に満たす場合である。
　条件1：第1週から第4週まで、毎週前週比でプラス成長している（3週連続でプラス）
　条件2：第1週から第4週までの総成長率（(第4週-第1週)/第1週）が20%以上である
「安定成長型」と判定される支店として正しいものを1つ選び、解答用紙にマークせよ。(6点)
【表1】支店別週間売上データ（単位：万円）（再掲）
支店 第1週 第2週 第3週 第4週
A 88 90 92 95
B 78 84 91 87
C 90 86 94 98
D 85 87 89 92
E 75 82 90 99
(1) 支店A
(2) 支店B
(3) 支店C
(4) 支店E
(5) 支店D
問2（正解：(4)）

【各支店の週間成長率を計算】

支店A：
第1→2週：(90-88)/88 = +2.3%
第2→3週：(92-90)/90 = +2.2%
第3→4週：(95-92)/92 = +3.3%
→ 3週連続プラス成長 → 条件1を満たす
総成長率：(95-88)/88 = 7/88 = 8.0% → 20%未満 → 条件2を満たさない

支店B：
第1→2週：(84-78)/78 = +7.7%
第2→3週：(91-84)/84 = +8.3%
第3→4週：(87-91)/91 = -4.4% → マイナス成長あり
→ 条件1を満たさない

支店C：
第1→2週：(86-90)/90 = -4.4% → マイナス成長あり
→ 条件1を満たさない

支店D：
第1→2週：(87-85)/85 = +2.4%
第2→3週：(89-87)/87 = +2.3%
第3→4週：(92-89)/89 = +3.4%
→ 3週連続プラス成長 → 条件1を満たす
総成長率：(92-85)/85 = 7/85 = 8.2% → 20%未満 → 条件2を満たさない

支店E：
第1→2週：(82-75)/75 = +9.3%
第2→3週：(90-82)/82 = +9.8%
第3→4週：(99-90)/90 = +10.0%
→ 3週連続プラス成長 → 条件1を満たす
総成長率：(99-75)/75 = 24/75 = 32.0% → 20%以上 → 条件2を満たす

正解：(4) 支店E
