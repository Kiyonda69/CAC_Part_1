問1．飛行機の進行方向を映し、ある方位面の断面を示す垂直線と、ある高度面の断面を示す水平線が表示されている計器がある。下図の左端は飛行機が設定された方位と高度を維持して飛行している適正な状態の表示である。続く4つの表示は時間経過に伴う計器の遷移を示しており、表示と表示の間 A・B・C で機首上げ・機首下げ・右旋回・左旋回のいずれか1つの操作が行われた。A・B・C で行われた操作の組み合わせとして正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
ABC
(1)機首下げ左旋回機首上げ
(2)機首上げ左旋回機首上げ
(3)機首上げ右旋回機首上げ
(4)機首上げ左旋回機首下げ
(5)機首下げ右旋回機首下げ
問1(正解:(2))

計器の表示は飛行機（中央の白い四角）から見た方位面・高度面の相対位置を示す。

【操作と線の動きの対応】
・機首上げ … 機体が高度面より上昇するため、水平線（高度面）は表示上「下」に動く。
・機首下げ … 機体が高度面より下降するため、水平線は表示上「上」に動く。
・右旋回 … 機体が方位面の右側へ移動するため、垂直線（方位面）は表示上「左」に動く。
・左旋回 … 機体が方位面の左側へ移動するため、垂直線は表示上「右」に動く。

【各遷移で動いた線】
・A：D1→D2 で水平線が「上」から「中央」へ＝下方向に移動 → 機首上げ
・B：D2→D3 で垂直線が「左」から「中央」へ＝右方向に移動 → 左旋回
・C：D3→D4 で水平線が「中央」から「下」へ＝下方向に移動 → 機首上げ

【選択肢の検討】
・(1) A=機首下げ → A の水平線は下方向に移動しているので不適。
・(2) A=機首上げ／B=左旋回／C=機首上げ → 全遷移と一致（正解）。
・(3) B=右旋回 → 垂直線は右方向に動いており不適。
・(4) C=機首下げ → C の水平線は下方向に動いており不適。
・(5) A=機首下げ／B=右旋回 → いずれも線の動きと反対方向で不適。

したがって、正解は(2)である。
問2．問1と同じ計器について、下図に示すように 5 つの表示が連続して観測された。表示と表示の間 A・B・C・D で機首上げ・機首下げ・右旋回・左旋回のいずれか1つの操作が行われた。線のずれ幅は均等に1段階ずつであり、どの遷移も垂直線か水平線のどちらか一方だけが動く。A・B・C・D で行われた操作の組み合わせとして正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
ABCD
(1)右旋回機首下げ左旋回機首上げ
(2)左旋回機首上げ右旋回機首下げ
(3)左旋回機首下げ右旋回機首上げ
(4)右旋回機首上げ左旋回機首下げ
(5)左旋回機首下げ左旋回機首上げ
問2(正解:(3))

操作と線の動きの対応は問1と同じ。本問では線のずれは「やや左／中央」「中央／やや上」の2段階のみで、各遷移は1段階だけ動く。

【各遷移で動いた線】
・A：D1→D2 で垂直線が「やや左」から「中央」へ＝右方向に移動 → 左旋回
・B：D2→D3 で水平線が「中央」から「やや上」へ＝上方向に移動 → 機首下げ
・C：D3→D4 で垂直線が「中央」から「やや左」へ＝左方向に移動 → 右旋回
・D：D4→D5 で水平線が「やや上」から「中央」へ＝下方向に移動 → 機首上げ

【選択肢の検討】
・(1) A=右旋回 → 垂直線は右方向に動いており不適。
・(2) B=機首上げ → 水平線は上方向（機首下げ方向）に動いており不適。
・(3) A=左旋回／B=機首下げ／C=右旋回／D=機首上げ → 全遷移と一致（正解）。
・(4) A=右旋回／B=機首上げ → ともに動きと反対で不適。
・(5) C=左旋回 → 垂直線は左方向に動いており不適。

【別解：観察ポイント】
D1とD5の表示は同一であり、4回の操作で元に戻ったことが分かる。これは「左右で打ち消し合う旋回操作（左旋回1回＋右旋回1回）」と「上下で打ち消し合うピッチ操作（機首下げ1回＋機首上げ1回）」が必ず含まれることを意味する。選択肢のうちこの条件を満たすのは(2)と(3)だが、Bの遷移で水平線が上に動く事実から(3)が正解と確定する。

したがって、正解は(3)である。
