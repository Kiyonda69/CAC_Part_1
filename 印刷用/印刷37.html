問1．正方形のタイルからなるいくつかのピースを組み合わせると、下の図のような形になった。この形にもう1つのピースを加えて、穴の無い長方形を完成させたい。加えるべきピースはどれか。正しいものを1つ選び、解答用紙にマークせよ。ただし、ピースは回転させてよいが、裏返してはならない。（6点）
問1(正解:(4))

【目標の特定】
図の形は5列×4行の範囲に収まっており、15マス分のタイルがある。5×4=20マスの長方形を完成させるには、5マス分のピースを加えればよい。

【空きの形の分析】
空いている5マスの位置は以下の通りである：
・行0の列2, 列3
・行1の列1, 列2
・行2の列2
これらを図示すると、F型ペントミノの形状になる。

【選択肢の検討】
・(1) T型ペントミノ：十字型の形状であり、空きの形と一致しない
・(2) F'型（Fの鏡像）：Fに似ているが左右が反転している。裏返しは禁止なので、回転だけではFの形にならない
・(3) P型ペントミノ：2×2ブロックに1マス追加した形状であり、空きの形と一致しない
・(4) F型ペントミノ：180度回転すると空きの形と完全に一致する。正解
・(5) W型ペントミノ：階段状の形状であり、空きの形と一致しない

したがって、正解は(4)である。
問2．一辺の長さがaの正四面体ABCDにおいて、辺ABの中点をM、辺CDの中点をNとする。線分MNについて、次の記述ア～エのうち正しいものをすべて含む組み合わせはどれか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
ア　MNの長さは a√2 / 2 である
イ　MNは辺ABと辺CDの両方に垂直である
ウ　線分MNの中点は正四面体ABCDの重心と一致する
エ　線分MNは面ABCと面ABDのなす二面角の二等分面上にある
(1)　アとイのみ
(2)　ア、イ、ウのみ
(3)　ア、イ、エのみ
(4)　ア、ウ、エのみ
(5)　ア、イ、ウ、エのすべて
問2(正解:(2))

正四面体の辺の長さをaとし、座標を設定する。
A=(0, 0, 0)、B=(a, 0, 0)、C=(a/2, a√3/2, 0)、D=(a/2, a√3/6, a√(2/3))

M = AB中点 = (a/2, 0, 0)
N = CD中点 = (a/2, a√3/3, a√(2/3)/2)
MNベクトル = (0, a√3/3, a√(2/3)/2)

【ア の検証】
MN² = (a√3/3)² + (a√(2/3)/2)² = a²/3 + a²/6 = a²/2
MN = a/√2 = a√2/2　→　正しい

【イ の検証】
AB = (a, 0, 0) → MN・AB = 0 → MN⊥AB
CD = (0, -a√3/3, a√(2/3)) → MN・CD = -a²/3 + a²/3 = 0 → MN⊥CD
→ 正しい

【ウ の検証】
重心G = (A+B+C+D)/4 = (a/2, a√3/6, a√(2/3)/4)
MN中点 = (a/2, a√3/6, a√(2/3)/4)
MN中点 = G → 正しい

【エ の検証】
面ABCと面ABDのなす二面角の二等分面は、辺ABを含む平面である。
面ABCの外向き法線: (0, 0, -1)
面ABDの外向き法線: (0, -2√2/3, 1/3)（正規化済み）
二等分面の法線方向: (0, -2√2/3, -2/3)
点Nからこの二等分面までの距離を計算すると √2/2 ≠ 0
→ 線分MNはこの二等分面上にない → 誤り

正しいのはア、イ、ウの3つ。したがって、正解は(2)である。
