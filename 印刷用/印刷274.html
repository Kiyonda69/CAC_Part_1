問1．下の図のように、長さの等しいマッチ棒を並べて、正六角形を横一列につないだ図形を作っていく。図1は正六角形が1個、図2は正六角形が2個、図3は正六角形が3個の図形であり、これ以降も同じ規則に従って正六角形を1個ずつ増やしていく。マッチ棒が100本あるとき、この作り方でできる図形のうち、正六角形の個数が最も多い図形を1つ作った。このとき、正六角形は何個できるか。（6点）
【図】マッチ棒で作る図形
(1)  16個
(2)  18個
(3)  19個
(4)  20個
(5)  21個
問1(正解:(3))
マッチ棒の本数の増え方の規則を見つけ、100本以内で作れる最大の図形を求める問題である。

【規則の発見】
各図に使われているマッチ棒の本数を数える。
・図1: 正六角形1個 → 6本
・図2: 正六角形2個 → 11本
・図3: 正六角形3個 → 16本
正六角形を1個増やすとき、隣の正六角形と縦の1辺を共有するため、新たに必要なマッチ棒は6本ではなく5本である。
したがって、正六角形n個の図形に使うマッチ棒の本数は
6 + 5×(n−1) = 5n + 1（本）

【100本で作れる最大の個数】
5n + 1 ≦ 100 を満たす最大の整数nを求める。
5n ≦ 99　→　n ≦ 19.8
よって n = 19 が最大である。
検算: 図19は 5×19 + 1 = 96本（100本以内）、図20は 5×20 + 1 = 101本（100本を超える）。
したがって、正六角形は最大で19個できる。

【選択肢の検討】
・(1) 16個 → 共有辺を考えずに1個あたり6本として 100÷6 = 16.6… と計算した誤り
・(2) 18個 → (100−6)÷5 = 18.8 の切り捨て値をそのまま答え、最初の1個を足し忘れた誤り
・(3) 19個 → 5n+1 ≦ 100 より正しい
・(4) 20個 → 最初の図形の「+1本」を忘れて 5n ≦ 100 とした誤り
・(5) 21個 → 100÷5 = 20 に、さらに1個を加えてしまった誤り

したがって、正解は(3)である。
問2．下の図のように、長さの等しいマッチ棒を並べて、1辺がマッチ棒1本分の小さな正三角形をすき間なく組み合わせた大きな正三角形を作っていく。図1は1辺がマッチ棒1本分、図2は1辺がマッチ棒2本分、図3は1辺がマッチ棒3本分の大きな正三角形であり、これ以降も同じ規則に従って1辺を1本分ずつ長くしていく。マッチ棒をちょうど135本すべて使ってこの図形を1つ作ったとき、図形の中にある小さな正三角形のうち、図1の正三角形と同じ向きのものは全部で何個あるか。（6点）
【図】マッチ棒で作る図形
(1)  27個
(2)  36個
(3)  45個
(4)  55個
(5)  81個
問2(正解:(3))
マッチ棒の本数の規則から図形の大きさを逆算し、さらに小さな正三角形の個数を数える2段階の問題である。

【手順1: マッチ棒の本数の規則】
各図に使われているマッチ棒の本数を数える。
・図1: 3本
・図2: 9本（増分6本）
・図3: 18本（増分9本）
増分は 6, 9, 12, … と3本ずつ増えていく（図nで新たに加わる最下段の分が3n本）。
したがって、図nに使うマッチ棒の本数は
3×(1 + 2 + … + n) = 3n(n+1)/2（本）

【手順2: 135本になる図の特定】
3n(n+1)/2 = 135　→　n(n+1) = 90　→　n = 9
（9×10 = 90。nは連続する2整数の積なのでn = 9のみが解）
つまり、1辺がマッチ棒9本分の大きな正三角形である。

【手順3: 図1と同じ向きの小さな正三角形を数える】
図1と同じ向き（頂点が上）の小さな正三角形は、上の段から順に 1個, 2個, 3個, … と並び、最下段（9段目）には9個ある。
1 + 2 + 3 + … + 9 = 45（個）
なお、逆向き（頂点が下）の小さな正三角形は 1 + 2 + … + 8 = 36個であり、両者の合計は81個である。

【選択肢の検討】
・(1) 27個 → 大きな正三角形の周上のマッチ棒の本数（9×3 = 27本）と混同した誤り
・(2) 36個 → 逆向き（頂点が下）の小さな正三角形の個数を答えた誤り
・(3) 45個 → 1 + 2 + … + 9 = 45 で正しい
・(4) 55個 → n(n+1) = 90 から n = 10 と取り違えて 1 + 2 + … + 10 を計算した誤り
・(5) 81個 → 向きを区別せず、小さな正三角形をすべて数えた誤り（45 + 36 = 81個）

したがって、正解は(3)である。
