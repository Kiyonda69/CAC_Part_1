問1．4種類のおもりP、Q、R、Sがある。図1は天秤の釣り合い条件を示している。以下の(ア)～(エ)の条件がすべて成り立つとき、おもりSの重さとして正しいものを1つ選び、解答用紙にマークせよ。（6点）
【図1】天秤の釣り合い条件
(1) 8g
(2) 12g
(3) 14g
(4) 16g
(5) 20g
問1（正解:(2)）
各条件を順に使って、おもりの重さを決定していく。

【解法】
条件(エ)より：P = 12g

条件(イ)より：Q + Q + Q = P
3Q = 12
Q = 4g

条件(ア)より：P + Q = R
R = 12 + 4 = 16g

条件(ウ)より：R + S = P + P + Q
16 + S = 12 + 12 + 4
16 + S = 28
S = 12g

したがって、Sの重さは12gである。
正解：(2) 12g
問2．5種類のおもりA、B、C、D、Eがある。図2は天秤の釣り合い条件を示している。以下の(ア)～(オ)の条件がすべて成り立つとき、おもりEの重さとして正しいものを1つ選び、解答用紙にマークせよ。（6点）
【図2】天秤の釣り合い条件
(1) 6g
(2) 8g
(3) 9g
(4) 10g
(5) 12g
問2（正解:(4)）
連立方程式を立てて解く。

【条件の式化】
(ア) A + B = C + D
(イ) B + C = D + E
(ウ) A + D = 2B + E
(エ) 2C = A + E
(オ) A + B + C + D + E = 50

【解法】
(ア)より：A + B - C - D = 0 ... [1]
(イ)より：B + C - D - E = 0 ... [2]
(ウ)より：A + D - 2B - E = 0 ... [3]
(エ)より：2C - A - E = 0 ... [4]

[1] + [2]より：A + 2B - 2D - E = 0 ... [5]
[3]より：A - 2B + D - E = 0 ... [3']
[5] + [3']より：2A - D - 2E = 0
よって D = 2A - 2E ... [6]

[4]より：A = 2C - E ... [7]
[1]に代入：(2C - E) + B - C - D = 0
B + C - E - D = 0 ... [8]

[2]と[8]を比較：B + C = D + E より [8]は矛盾なし

問2（正解:(4)）続き

[6]を[1]に代入：A + B = C + (2A - 2E)
B = C + A - 2E ... [9]

[7]を[9]に代入：B = C + (2C - E) - 2E = 3C - 3E ... [10]

(オ)に[6][7][10]を代入：
(2C - E) + (3C - 3E) + C + (2(2C - E) - 2E) + E = 50
2C - E + 3C - 3E + C + 4C - 2E - 2E + E = 50
10C - 7E = 50 ... [11]

[2]に[6][10]を代入：
(3C - 3E) + C = (2(2C - E) - 2E) + E
4C - 3E = 4C - 2E - 2E + E = 4C - 3E（恒等式、条件は整合）

連立方程式を整理すると、整数解として：
A = 14g, B = 6g, C = 12g, D = 8g, E = 10g

【検算】
(ア) A + B = 14 + 6 = 20, C + D = 12 + 8 = 20 （成立）
(イ) B + C = 6 + 12 = 18, D + E = 8 + 10 = 18 （成立）
(ウ) A + D = 14 + 8 = 22, 2B + E = 12 + 10 = 22 （成立）
(エ) 2C = 24, A + E = 14 + 10 = 24 （成立）
(オ) 合計 = 14 + 6 + 12 + 8 + 10 = 50g （成立）

したがって、Eの重さは10gである。
正解：(4) 10g
