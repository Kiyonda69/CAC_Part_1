問1．次の条件で定まる立方体について、指定された3点を通る平面で切断したときの断面の形として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
【立方体の構成】
  • 立方体 ABCD-EFGH において、ABCD は上面の正方形、EFGH は下面の正方形である。
  • 頂点を上から見たとき、A・B・C・D は反時計回りに並んでおり、A の真下が E、B の真下が F、C の真下が G、D の真下が H である。
【切断する3点】
  • 頂点 A
  • 頂点 C
  • 辺 FG の中点 M
上記の3点 A・C・M を通る1つの平面で立方体を切断したときに現れる切り口（断面）の形として最も適切なものを選べ。
問1(正解:(1))

立方体の一辺の長さを 1 とし、座標を A(0,0,1)、B(1,0,1)、C(1,1,1)、D(0,1,1)、E(0,0,0)、F(1,0,0)、G(1,1,0)、H(0,1,0) と置く。すると、辺 FG の中点は M(1, 0.5, 0) である。

【切断面の特定】
3点 A(0,0,1)、C(1,1,1)、M(1, 0.5, 0) を通る平面で立方体を切ると、平面と立方体の各面との交わりにより断面の輪郭が決まる。各面ごとに見ていくと:
・上面 ABCD 上では 対角線 AC が断面の一辺となる。
・側面 BCGF 上では 頂点 C と 辺 FG 上の点 M を結ぶ線分 CM が断面の一辺となる。
・底面 EFGH 上では 点 M から 辺 EF 上の点 N(0.5, 0, 0) まで、平面が底面と交わる線分 MN が断面の一辺となる。
・側面 ABFE 上では 点 N と 頂点 A を結ぶ線分 NA が断面の一辺となる。
よって断面は 4頂点 A・C・M・N からなる四角形となる。

【辺の長さの計算（一辺=1）】
・AC = √((1-0)²+(1-0)²+0²) = √2 ≈ 1.414（上面の対角線）
・CM = √(0²+(0.5)²+1²) = √1.25 ≈ 1.118（側面 BCGF を通る斜辺）
・MN = √((0.5)²+(0.5)²+0²) = √0.5 ≈ 0.707（底面 EFGH 内の線分）
・NA = √((0.5)²+0²+1²) = √1.25 ≈ 1.118（側面 ABFE を通る斜辺）

【形状の判定】
・MN の方向ベクトルは (-0.5, -0.5, 0)、AC の方向ベクトルは (1, 1, 0) であり、互いに平行 ⇒ 1組の対辺 AC ∥ MN を持つ 台形。
・残る2辺 CM = NA = √1.25 で長さが等しい ⇒ 平行でない辺の長さが等しい 等脚台形。
・AC ≠ MN なので長方形・平行四辺形ではない。

【選択肢の検討】
・(1) 等脚台形 → 1組の平行辺と等しい2脚を持つ。条件に一致（正解）
・(2) 正三角形 → 断面の頂点数が4なので不適
・(3) 平行四辺形 → 対辺の片方（AC と MN）の長さが異なるので不適
・(4) 長方形 → 角度が直角ではない（内角は約71.6°と108.4°）ので不適
・(5) 一般の四角形 → 対辺 AC ∥ MN という平行関係があるため「一般」ではない

したがって、正解は(1)である。
問2．次の条件で定まる立方体について、指定された3点を通る平面で切断したときの断面の形として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
【立方体の構成】
  • 立方体 ABCD-EFGH において、ABCD は上面の正方形、EFGH は下面の正方形である。
  • 頂点を上から見たとき、A・B・C・D は反時計回りに並んでおり、A の真下が E、B の真下が F、C の真下が G、D の真下が H である。
【切断する3点】
  • 辺 AB の中点 P
  • 辺 CG の中点 Q
  • 辺 HE の中点 R
上記の3点 P・Q・R を通る1つの平面で立方体を切断したときに現れる切り口（断面）の形として最も適切なものを選べ。
問2(正解:(2))

立方体の一辺を 1 とし、A(0,0,1)、B(1,0,1)、C(1,1,1)、D(0,1,1)、E(0,0,0)、F(1,0,0)、G(1,1,0)、H(0,1,0) とおく。
・P = 辺 AB の中点 = (0.5, 0, 1)
・Q = 辺 CG の中点 = (1, 1, 0.5)
・R = 辺 HE の中点 = (0, 0.5, 0)

【切断面の方程式】
3点 P・Q・R を通る平面の方程式は、法線ベクトルを計算すると n=(-1, 1, 1) となり、整理すると −x + y + z = 0.5。
この平面は立方体の中心 (0.5, 0.5, 0.5) を通る（−0.5+0.5+0.5=0.5 が成立）ことに注目する。

【各辺との交点】
立方体の12本の辺について、平面と交わる点をすべて求めると:
・辺 AB の中点 P(0.5, 0, 1)　　・辺 BC の中点 (1, 0.5, 1)
・辺 CG の中点 Q(1, 1, 0.5)　　・辺 GH の中点 (0.5, 1, 0)
・辺 HE の中点 R(0, 0.5, 0)　　・辺 AE の中点 (0, 0, 0.5)
以上の 6つの中点 を順に結ぶと六角形ができる。

【正六角形であることの証明】
・隣り合う2つの中点間の距離は、すべて √((0.5)²+(0.5)²) = √0.5 ≈ 0.707 で等しい。
・各内角を計算すると、いずれも 120° となる（例えば点 P での角度は、両側に伸びるベクトル (−0.5, 0.5, 0) と (0.5, 0.5, −0.5) のなす角を計算すると 120°）。
・6辺の長さがすべて等しく、6つの内角がすべて 120° なので、断面は 正六角形。

【選択肢の検討】
・(1) 五角形 → 立方体の6つの面すべてと交わるので頂点数は6（不適）
・(2) 正六角形 → 6辺すべて √0.5、6つの内角すべて 120°（正解）
・(3) 正方形 → 頂点数が異なる（不適）
・(4) 長方形 → 頂点数が異なる（不適）
・(5) 一般の六角形 → 6辺・6内角がすべて等しいので「一般」ではない（不適）

【補足】
立方体の中心を通り、互いに平行でない3つの辺（AB、CG、HE）の中点を通る平面は、立方体の対称性から必ず正六角形を切り出す。これは空間幾何の有名な事実である。

したがって、正解は(2)である。
