問1．下の図のように、立方体ABCD-EFGHを、辺AD・DC・CG・GF・FE・EAの中点を通る平面で切断したところ、切り口は正六角形となった。この立方体を、前面ABFEを中央にした十字型の展開図に開いたとき、切り口の線（切断面の縁）はどのように現れるか。最も適切なものを下の選択肢から1つ選びなさい。（6点）
【図1】立方体と展開図の対応
問1(正解:(1))
立方体ABCD-EFGHを、辺AD・DC・CG・GF・FE・EAの中点を通る平面で切断したときの正六角形の切り口が、十字型展開図上のどの位置に現れるかを問う問題である。

【切り口が通過する6つの面と切り落とされる頂点】
平面が通過する6本の辺をたどると、各面で「2つの中点を結ぶ線分」が現れる。各面の切り取り頂点（切り落とされる頂点）は次のとおり。
・上面ABCD: 辺AD・DCの中点を結ぶ → 頂点Dが切り落とされる
・後面DCGH: 辺DC・CGの中点を結ぶ → 頂点Cが切り落とされる
・右面BCGF: 辺CG・GFの中点を結ぶ → 頂点Gが切り落とされる
・下面EFGH: 辺GF・FEの中点を結ぶ → 頂点Fが切り落とされる
・前面ABFE: 辺FE・EAの中点を結ぶ → 頂点Eが切り落とされる
・左面ADHE: 辺EA・ADの中点を結ぶ → 頂点Aが切り落とされる

切り落とされる6頂点はA・C・D・E・F・G。残るB・Hは体対角線BHの両端点であり、切断面はBHに垂直な中点平面である。

【展開図上の各パネルの頂点配置】（前面ABFEを中央）
・上面パネル: A=左下、B=右下、C=右上、D=左上
・前面パネル: A=左上、B=右上、F=右下、E=左下
・下面パネル: E=左上、F=右上、G=右下、H=左下
・左面パネル: D=左上、A=右上、E=右下、H=左下
・右面パネル: B=左上、C=右上、G=右下、F=左下
・後面パネル: C=左上、D=右上、H=右下、G=左下

【切り取り位置のまとめ】
・上面パネル: D=左上 → 左上コーナーを切り取る
・前面パネル: E=左下 → 左下コーナー
・下面パネル: F=右上 → 右上コーナー
・左面パネル: A=右上 → 右上コーナー
・右面パネル: G=右下 → 右下コーナー
・後面パネル: C=左上 → 左上コーナー

【選択肢の検討】
・(1) 上=左上、前=左下、下=右上、左=右上、右=右下、後=左上 → 上記と完全一致 正解
・(2) 体対角線AGに垂直な切断（残る頂点A・G）。切り取られる頂点はB・C・D・E・F・H で、上面では C=右上、前面では F=右下が切り取られる → 不正解
・(3) 体対角線CEに垂直な切断（残る頂点C・E）。上面では A=左下、前面では B=右上が切り取られる → 不正解
・(4) 体対角線DFに垂直な切断（残る頂点D・F）。上面では B=右下、前面では A=左上が切り取られる → 不正解
・(5) 中央4面に水平帯。これは正方形断面（垂直軸の中点を通る平面）であり、正六角形ではない → 不正解

したがって、正解は(1)である。
