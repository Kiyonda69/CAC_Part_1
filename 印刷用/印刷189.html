問1．次の5図書館の年間利用統計を読み、各館の実績に関する記述として正しいものを(1)〜(5)の中から1つ選べ。（6点）
ある市の5つの図書館（A〜E）の年間統計である。A図書館は蔵書150千冊、年間貸出240千冊、登録者40千人、開館日数300日であった。B図書館は蔵書120千冊、貸出198千冊、登録者36千人、開館日数300日である。C図書館は蔵書200千冊、貸出280千冊、登録者56千人、開館日数280日であった。D図書館は蔵書90千冊、貸出168千冊、登録者24千人、開館日数280日にとどまる。E図書館は蔵書125千冊、貸出225千冊、登録者50千人、開館日数250日であった。なお、1人あたり年間貸出冊数は貸出冊数を登録者数で割った値、1日あたり貸出冊数は貸出冊数を開館日数で割った値、蔵書回転率は貸出冊数を蔵書数で割った値とする。
【表1】5図書館の年間利用統計
図書館蔵書数（千冊）年間貸出（千冊）登録者数（千人）開館日数（日）
A15024040300

(1) 1人あたり年間貸出冊数が最も多いのはA図書館である。
(2) 蔵書回転率が最も高いのはE図書館である。
(3) 1日あたり貸出冊数が最も多いのはC図書館である。
(4) 登録者1人あたり年間貸出冊数はC図書館がB図書館を上回る。
(5) 1日あたり貸出冊数が最も少ないのはE図書館である。
問1(正解:(3))
各館の導出指標を計算する。

【1人あたり年間貸出冊数】（貸出÷登録者）
・A: 240÷40 = 6.0冊　・B: 198÷36 = 5.5冊　・C: 280÷56 = 5.0冊　・D: 168÷24 = 7.0冊　・E: 225÷50 = 4.5冊

【1日あたり貸出冊数】（貸出÷開館日数）
・A: 240,000÷300 = 800冊　・B: 198,000÷300 = 660冊　・C: 280,000÷280 = 1,000冊　・D: 168,000÷280 = 600冊　・E: 225,000÷250 = 900冊

【蔵書回転率】（貸出÷蔵書）
・A: 240÷150 = 1.60　・B: 198÷120 = 1.65　・C: 280÷200 = 1.40　・D: 168÷90 ≒ 1.87　・E: 225÷125 = 1.80

【選択肢の検討】
・(1) 1人あたり貸出が最も多いのはD(7.0冊)であり、A(6.0冊)ではない → 誤
・(2) 蔵書回転率が最も高いのはD(約1.87)であり、E(1.80)ではない → 誤
・(3) 1日あたり貸出が最も多いのはC(1,000冊)である → 正
・(4) C(5.0冊)はB(5.5冊)を下回るため、上回ってはいない → 誤
・(5) 1日あたり貸出が最も少ないのはD(600冊)であり、E(900冊)ではない → 誤

したがって、正解は(3)である。
問2．次の6工場の月間生産データを読み、ア〜オの記述の正誤を判定して、正しいものの組み合わせを(1)〜(5)の中から1つ選べ。（6点）
ある企業の6工場（P〜U）の当月実績である。P工場は生産数8,000個、不良品160個、稼働時間200時間、作業員40人、電力使用量2,400kWh。Q工場は生産数9,000個、不良品270個、稼働時間250時間、作業員50人、電力使用量3,150kWh。R工場は生産数7,200個、不良品108個、稼働時間160時間、作業員45人、電力使用量2,880kWh。S工場は生産数10,000個、不良品250個、稼働時間200時間、作業員40人、電力使用量2,500kWh。T工場は生産数6,000個、不良品210個、稼働時間200時間、作業員40人、電力使用量2,700kWh。U工場は生産数8,400個、不良品84個、稼働時間240時間、作業員35人、電力使用量1,680kWh。なお、不良品率は不良品数を生産数で割った値、時間あたり生産数は生産数を稼働時間で割った値、1人あたり生産数は生産数を作業員数で割った値、製品1個あたり電力使用量は電力使用量を生産数で割った値とする。

ア　不良品率が最も高い工場は、時間あたり生産数が最も少ない工場でもある。

問2(正解:(2))
4種の導出指標を全工場について計算する。

【不良品率】（不良品÷生産数）
P: 2.0%　Q: 3.0%　R: 1.5%　S: 2.5%　T: 3.5%（最高）　U: 1.0%（最低）

【時間あたり生産数】（生産数÷稼働時間）
P: 40　Q: 36　R: 45　S: 50（最多）　T: 30（最少）　U: 35（個/h）

【1人あたり生産数】（生産数÷作業員数）
P: 200　Q: 180　R: 160　S: 250（最多）　T: 150（最少）　U: 240（個/人）

【製品1個あたり電力使用量】（電力÷生産数）
P: 0.30　Q: 0.35　R: 0.40　S: 0.25　T: 0.45（最多）　U: 0.20（最少）（kWh）

【記述の検討】
・ア 不良品率最高(T)＝時間あたり生産数最少(T) → 一致 → 正
・イ 1人あたり最多(S)≠不良品率最高(T) → 誤
・ウ 電力最少(U)＝不良品率最低(U) → 一致 → 正
・エ 時間あたり最多(S)＝1人あたり最多(S) → 一致 → 正
・オ 電力最多(T)≠1人あたり最多(S) → 誤

正しい記述はア・ウ・エ。したがって、正解は(2)である。
