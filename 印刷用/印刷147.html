問1．次の表の「？」に当てはまる数字として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
603572
402550
201530
60？72
(1) 25　　(2) 35　　(3) 30　　(4) 40　　(5) 45
問1(正解:(2))

各列の数値を上から下へ観察すると、最下行の値が最上行の値と一致する規則性がある。

【規則の発見】各列で row1 = row4
・第1列: 60, 40, 20, ? → 上から60, 40, 20と20ずつ減少し、最下行は再び 60（=row1）
・第3列: 72, 50, 30, ? → 上から72, 50, 30と22, 20ずつ減少し、最下行は再び 72（=row1）
・どの列でも row4 が row1 と同じ値になっている。

【別の見方】各列で「下方向の減少量の合計」=「最下行への戻り量」
・第1列: 減少量 (60→40→20) = 20+20 = 40 ＝ 戻り量 (20→60) = 40 ✓
・第3列: 減少量 (72→50→30) = 22+20 = 42 ＝ 戻り量 (30→72) = 42 ✓
・第2列: 減少量 (35→25→15) = 10+10 = 20 → 戻り量 = 20 → ? = 15+20 = 35

【検算】
第2列が row4 = row1 = 35 を満たすか確認: 35 = 35 ✓
したがって、空欄に入る数字は 35 である。

【選択肢の検討】
・(1) 25 → row2と同じ値、規則に合わない
・(2) 35 → 上記の通り正解
・(3) 30 → 規則に合わない
・(4) 40 → 第1列のrow2と混同した誤答
・(5) 45 → 規則に合わない

したがって、正解は(2)である。
問2．以下に1から12の数を表す図形が並んでいる。53を表す図形として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
問2(正解:(1))

各円は8つのセクター（45°ずつ）に分割されており、塗りつぶされたセクターの位置が2進数のビット位置を表している。

【規則の発見】8セクター = 8ビットの2進数
円の12時方向のセクターから時計回りに、それぞれ 1, 2, 4, 8, 16, 32, 64, 128 の値を持つ。
塗りつぶされたセクターの値の合計が、その円の表す数となる。

【参照円から規則を確認】
・1 = 12時セクター（=1） のみ塗りつぶし
・2 = 1時方向セクター（=2） のみ
・3 = 12時+1時 = 1+2 = 3 ✓
・4 = 3時方向セクター（=4） のみ
・5 = 12時+3時 = 1+4 = 5 ✓
・8 = 4-5時方向セクター（=8） のみ
・12 = 3時+4-5時 = 4+8 = 12 ✓

【53の2進数表現】
53 = 32 + 16 + 4 + 1
= 12時セクター（1）+ 3時セクター（4）+ 6時セクター（16）+ 7-8時セクター（32）
よって、これら4つのセクターが塗りつぶされた円が正解。

【選択肢の検討】
・(1) 12時+3時+6時+7-8時 = 1+4+16+32 = 53 ✓
・(2) 12時+1時+6時+7-8時 = 1+2+16+32 = 51
・(3) 12時+3時+4-5時+7-8時 = 1+4+8+32 = 45
・(4) 12時+3時+6時+9時 = 1+4+16+64 = 85
・(5) 1時+3時+6時+7-8時 = 2+4+16+32 = 54

したがって、正解は(1)である。
