問1．下の表は、ブラックボックスPとブラックボックスQのそれぞれの入力と出力の関係を示している。下図のように、ブラックボックスPの出力をブラックボックスQの入力とする直列接続を考える。この直列接続に3を入力したとき、最終的な出力の値として正しいものを(1)～(5)から1つ選べ。（6点）

問1(正解:(2))

【ブラックボックスPの規則】
Pの出力を見ると、5, 7, 9, 11, 13 と等差数列（公差2）になっている。
入力 x に対して P(x) = 2x + 3 であることがわかる。
・P(1) = 2(1) + 3 = 5
・P(2) = 2(2) + 3 = 7
・P(3) = 2(3) + 3 = 9

【ブラックボックスQの規則】
Qの出力を見ると、0, 3, 8, 15, 24 で、階差は 3, 5, 7, 9（等差数列、公差2）。
これは二次関数のパターンであり、Q(x) = x² - 1 であることがわかる。
・Q(1) = 1 - 1 = 0
・Q(2) = 4 - 1 = 3
・Q(3) = 9 - 1 = 8

【直列接続の計算】
入力3 → P(3) = 2(3) + 3 = 9
P の出力 9 を Q に入力 → Q(9) = 9² - 1 = 81 - 1 = 80

【選択肢の検討】
・(1) 63 → Q(8) = 63 であり、P(3) = 8 と誤った場合の値
・(2) 80 → Q(9) = 80 で正解
・(3) 99 → Q(10) = 99 であり、P(3) = 10 と誤った場合の値
・(4) 120 → Q(11) = 120 であり、P(4) の値を使った場合の値
・(5) 168 → Q(13) = 168 であり、P(5) の値を使った場合の値

したがって、正解は(2)である。
問2．下の表は、ブラックボックスFとブラックボックスGの入出力の関係を示している。ただし、表中の(ア)～(ウ)は未知の値である。下図のように、ブラックボックスGの出力をブラックボックスFの入力とする直列接続 H(x) = F(G(x)) を考えるとき、H(4) の値として正しいものを(1)～(5)から1つ選べ。（6点）

問2(正解:(1))

【ブラックボックスGの規則】
Gの既知の出力: G(1)=2, G(2)=5, G(3)=8, G(5)=14
階差を見ると、G(2)-G(1)=3, G(3)-G(2)=3 で等差（公差3）。
G(5)-G(3)=6（2ステップ分で3×2=6）も一致する。
よって G(x) = 3x - 1 であり、G(4) = 3(4) - 1 = 11（ウの値）。

【ブラックボックスFの規則】
Fの既知の出力: F(1)=3, F(3)=15, F(5)=35
階差: F(3)-F(1)=12, F(5)-F(3)=20（2ステップごと）
二階差分: 20-12=8（2ステップごと）→ 二次関数のパターン。
F(x) = ax² + bx + c とすると、
・F(1) = a + b + c = 3
・F(3) = 9a + 3b + c = 15
・F(5) = 25a + 5b + c = 35
連立方程式を解くと a=1, b=2, c=0 → F(x) = x² + 2x
（ア）= F(2) = 4 + 4 = 8、（イ）= F(4) = 16 + 8 = 24

【H(4) の計算】
H(4) = F(G(4)) = F(11)
F(11) = 11² + 2(11) = 121 + 22 = 143

【選択肢の検討】
・(1) 143 → F(11) = 143 で正解
・(2) 132 → 11² + 11 = 132 であり、F(x) = x² + x と誤った場合の値
・(3) 121 → 11² = 121 であり、F(x) = x² と誤った場合の値
・(4) 99 → F(9) = 99 であり、G(4) = 9 と誤った場合の値
・(5) 80 → F(8) = 80 であり、G(3) = 8 を G(4) と混同した場合の値

したがって、正解は(1)である。
