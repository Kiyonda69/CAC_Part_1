問1．5人の航空大学校受験者A、B、C、D、Eが、月曜日から金曜日の5日間に、それぞれ異なる1日ずつ体力測定を受ける。下図の条件が与えられているとき、Dが体力測定を受ける曜日として正しいものを選べ。（6点）
【条件関係図】矢印は「より早い曜日に受ける」を表す
(1) 月曜日
(2) 火曜日
(3) 水曜日
(4) 木曜日
(5) 金曜日
問1（正解：(5)）

【解法：順序制約の整理】
条件①②から B < A < C（日数の早い順）という不等式が成り立つ。
条件③から E < C、条件⑤から C < D となる。
したがって、次の順序関係が確立される：
   B < A < C < D（4つの人物の順序が確定）
   E < C（EはCより前）

【Dの曜日を確定する】
B < A < C < D の4段階の順序があるため、Dは少なくとも4番目以降。
・もし D = 木曜日（4日目）なら、C ≤ 3日目、A ≤ 2日目、B ≤ 1日目
  → B=月(1)、A=火(2)、C=水(3)、D=木(4) と決まる。
  → E < C=水(3) なので E = 月 or 火。しかし月と火はB・Aが占有。矛盾。
・よって D = 金曜日（5日目） が確定。

【残りの曜日を確定する】
D=金曜日のとき、B・A・C・Eは月〜木の4日で決まる。
C < D=金、B < A < C なので C = 木曜日（4日目）が確定。
（もし C = 水(3)なら B・A・E が1〜2日目に入れず矛盾）

B・A・Eは月〜水（1〜3日）に入る。B < A の条件から：
  ・(B=月,A=火,E=水)：|B-E|=|1-3|=2 → 条件④「連続しない」を満たす ✓
  ・(B=月,A=水,E=火)：|B-E|=|1-2|=1 → 条件④に違反 ✗
  ・(B=火,A=水,E=月)：|B-E|=|2-1|=1 → 条件④に違反 ✗

唯一の有効な配置：B=月、A=火、E=水、C=木、D=金

したがって、Dが体力測定を受ける曜日は (5) 金曜日 である。
問2．3桁の暗証番号（各桁は1〜9の異なる数字）を当てるゲームを行った。各回の試行に対し、数字・位置ともに一致する桁数を「ヒット（H）」、数字は合っているが位置が異なる桁数を「ブロー（B）」として下表の結果が得られた。暗証番号として正しいものを選べ。（6点）
ヒット（H）：正しい桁に正しい数字がある個数
ブロー（B）：数字は正しいが桁の位置が異なる個数
【試行結果表】
試行 入力した数字 ヒット（H） ブロー（B）
第1回 1 2 3 1 0
第2回 4 5 6 1 0
第3回 7 8 9 0 1
第4回 3 2 7 2 0
(1) 2 4 7
(2) 2 7 4
(3) 4 2 7
(4) 4 7 2
(5) 7 4 2
問2（正解：(3)）

【第4回（3 2 7、2H 0B）から開始】
2ヒット0ブローなので、{3,2,7}のうち2つが正しい位置にある。残り1つは暗証番号に含まれない。
場合分け：
  (a) 3が1桁目 かつ 2が2桁目（→ 7は番号に含まれない）
  (b) 3が1桁目 かつ 7が3桁目（→ 2は番号に含まれない）
  (c) 2が2桁目 かつ 7が3桁目（→ 3は番号に含まれない）

【第1回（1 2 3、1H 0B）で場合を絞る】
0ブローなので{1,2,3}のうち1つが正位置にあり、残りは番号に含まれない。
・場合(a)：1桁目=3で試行の1桁目=1(不一致)、2桁目=2でヒット。だが3が1桁目なので、試行の3桁目=3もブロー扱い → 1H 1Bとなり矛盾。除外。
・場合(b)：1桁目=3。試行の3桁目=3がブロー(3は1桁目) → 0Bに矛盾。除外。
・場合(c)：2が2桁目、7が3桁目。試行の2桁目=2 → 1H。3は番号に含まれないので試行の3桁目=3はブロー0。1H 0B ✓

場合(c)のみ残る：2桁目=2、3桁目=7、3は番号に含まれない。

【第2回（4 5 6、1H 0B）で1桁目を決定】
5・6は番号に含まれない（0ブロー確定）。1ヒットは4が1桁目に一致する場合のみ。
→ 1桁目=4 が確定。

【第3回（7 8 9、0H 1B）で確認】
7は3桁目（試行では1桁目）→ ブロー ✓。8・9は番号に含まれない。0H 1B ✓

したがって、暗証番号は (3) 4 2 7 である。
