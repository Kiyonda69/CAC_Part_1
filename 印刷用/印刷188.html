問1．下図のように、一辺の長さが等しい立方体ABCD-EFGHがある。頂点Bを端点とする3本の辺の、B以外の端点A・C・Fの3点を通る平面でこの立方体を切断する。このときの切り口（断面）の形として正しいものを、次の(1)〜(5)から1つ選べ。（6点）
【図1】立方体ABCD-EFGHと切断面（△ACF）
問1(正解:(1))
立方体の一辺の長さを a とする。切断に使う3点 A・C・F は、いずれも頂点Bを共有する3つの面の「面の対角線」の端点である。

【3辺の長さを求める】
AとC、AとF、CとFは、それぞれ立方体の異なる面上の対角線で結ばれる。
・AC：上面ABCD上の対角線 → 長さ √2 a
・AF：側面ABFE上の対角線 → 長さ √2 a
・CF：側面BCGF上の対角線 → 長さ √2 a
よって AC＝AF＝CF＝√2 a となり、3辺がすべて等しい。

【切り口の形】
3辺が等しい三角形は正三角形である（各内角は60°）。頂点Bの3つの面の角はそれぞれ直角だが、切り口の三角形はそれらの面そのものではなく、3本の面対角線を結んだ三角形なので直角は現れない。

【選択肢の検討】
・(1) 正三角形 → 3辺が等しく正しい
・(2) 直角二等辺三角形 → 直角の角に引きずられた誤り。切り口に直角はない
・(3) 正方形 → 切り口は三角形であり四角形ではない
・(4) 鈍角二等辺三角形 → 各内角は60°なので鈍角は生じない
・(5) 長方形 → 切り口は三角形であり四角形ではない

したがって、正解は(1)である。
問2．問1と同じ立方体ABCD-EFGHにおいて、辺ABの中点をP、辺ADの中点をQとする。3点P・Q・Gを通る平面でこの立方体を切断する。このときの切り口（断面）は何角形になるか。正しいものを次の(1)〜(5)から1つ選べ。（6点）
【図2】立方体ABCD-EFGHと3点P・Q・G
問2(正解:(4))
切り口を求めるには、平面が立方体の各面とどこで交わるかを順にたどればよい。一辺の長さを2とし、A(0,0,2)、B(2,0,2)、C(2,2,2)、D(0,2,2)、E(0,0,0)、F(2,0,0)、G(2,2,0)、H(0,2,0)と座標をとる。

【3点と切断面の式】
P＝辺ABの中点(1,0,2)、Q＝辺ADの中点(0,1,2)、G(2,2,0)。この3点を通る平面の式は 2x＋2y＋3z＝8 となる。

【各面との交線を追う】
・上面(z=2)：PとQを結ぶ → 1辺目
・前面(y=0)：Pから辺BF上の点(2,0,4/3)へ → 2辺目
・右面(x=2)：辺BF上の点からGへ → 3辺目
・後面(y=2)：Gから辺DH上の点(0,2,4/3)へ → 4辺目
・左面(x=0)：辺DH上の点からQへ → 5辺目
底面(z=0)では平面が頂点Gに接するだけで交線を作らない。
よって平面は5つの面と交わり、切り口は五角形になる。

【選択肢の検討】
・(1) 三角形 → 平面は5面と交わるので3辺ではない
・(2) 平行四辺形 → 4辺ではない
・(3) 正六角形 → 6つの中点を通る場合の形であり、本問は底面で交線を作らないため6辺にならない
・(4) 五角形 → 5面と交わり5辺。正しい
・(5) 台形 → 4辺ではない

したがって、正解は(4)である。
