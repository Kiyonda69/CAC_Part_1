問1．ある管制空域において、5機の航空機A機〜E機がレーダーに映っている。各機体は4方角（北・東・南・西）と3距離圏（第1圏・第2圏・第3圏）の交点に1機ずつ存在する。各パイロットから管制官への報告を下表に示す。ただしC機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。（6点）
【レーダー図】
機体報告内容
A機「私は北方向の第2圏にいる。北方向には私を含めて2機がいる。」
B機「私は東方向の第1圏にいる。第1圏には私を含めて2機がいる。」
C機？（通信不良）
D機「私は南方向の第2圏にいる。第2圏には私を含めて2機がいる。」
E機「私は北方向の第1圏にいる。」
ア．C機の報告がない場合、C機の位置として2通りが考えられる。
イ．C機の報告がない場合、C機の位置として4通りが考えられる。
ウ．C機が「私は東方向にいる」と報告した場合、C機の位置が確定する。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問1（正解：(3)）

【各機体の位置を確定する】
各パイロットの報告（C機除く）から位置を求める。
・A機の報告：「北方向の第2圏、北方向は計2機」→ A機＝（北、第2圏）と確定。
・B機の報告：「東方向の第1圏、第1圏は計2機」→ B機＝（東、第1圏）と確定。
・D機の報告：「南方向の第2圏、第2圏は計2機」→ D機＝（南、第2圏）と確定。
・E機の報告：「北方向の第1圏」→ E機＝（北、第1圏）と確定。

【各圏・方向の充足状況を整理する】
・北方向：A機（第2圏）＋E機（第1圏）＝2機 → 満員（A機の報告より）
・第1圏：B機（東）＋E機（北）＝2機 → 満員（B機の報告より）
・第2圏：A機（北）＋D機（南）＝2機 → 満員（D機の報告より）

【C機の候補を絞り込む】
C機は北方向・第1圏・第2圏に入れないため、第3圏に存在する。
方角は 東・南・西 のいずれかで3通りが候補。

【各命題の検討】
ア．C機の位置として2通りが考えられるか？
　→ 候補は3通りのため誤り。ア：誤り。
イ．C機の位置として4通りが考えられるか？
　→ 候補は3通りのため誤り。イ：誤り。
ウ．「東方向にいる」の報告でC機が確定するか？
　→ 東・第3圏のみ（1通り）→ 確定する。ウ：正しい。

したがって、正解は(3)　ウのみ正しい　である。
問2．問1と同じ条件（4方角×3距離圏、各交点に1機ずつ）で、5機A機〜E機が存在する。各パイロットの報告を下表に示す。B機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。（6点）
【レーダー図】
機体報告内容
A機「私は北方向の第2圏にいる。北方向には私を含めて2機がいる。」
B機？（通信不良）
C機「私は南方向の第1圏にいる。第1圏には私を含めて2機がいる。」
D機「私は東方向の第2圏にいる。第2圏には私を含めて2機がいる。東方向には私のみである。」
E機「私は北方向の第1圏にいる。」
ア．B機の報告がない場合、B機の位置として2通りが考えられる。
イ．B機の報告がない場合、B機以外の4機の位置は全て確定する。
ウ．B機が「私は第3圏にいる」とのみ報告した場合、B機の位置が確定する。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問2（正解：(4)）

【各機体の位置を確定する】
各パイロットの報告（B機除く）から位置を求める。
・A機の報告：「北方向の第2圏、北方向は計2機」→ A機＝（北、第2圏）と確定。
・C機の報告：「南方向の第1圏、第1圏は計2機」→ C機＝（南、第1圏）と確定。
・D機の報告：「東方向の第2圏、第2圏は計2機、東方向は1機のみ」
　→ D機＝（東、第2圏）と確定。
・E機の報告：「北方向の第1圏」→ E機＝（北、第1圏）と確定。
　→ A・C・D・E機の4機が全て一意に確定する。

【各圏・方向の充足状況を整理する】
・北方向：A機（第2圏）＋E機（第1圏）＝2機 → 満員（A機の報告より）
・東方向：D機（第2圏）のみ → D機の報告より、東方向に他の機はない
・第1圏：C機（南）＋E機（北）＝2機 → 満員（C機の報告より）
・第2圏：A機（北）＋D機（東）＝2機 → 満員（D機の報告より）

【B機の候補を絞り込む】
・北方向は満員 → B機は北方向に入れない。
・東方向はD機のみ → B機は東方向に入れない。
・第1圏は満員 → B機は第1圏に入れない。
・第2圏は満員 → B機は第2圏に入れない。
　よってB機は第3圏かつ 南・西 方向のいずれかで2通りが候補。

【各命題の検討】
ア．B機の位置として2通りが考えられるか？
　→ 南・第3圏 と 西・第3圏 の2通り。ア：正しい。
イ．B機以外の4機の位置は全て確定するか？
　→ A・C・D・E機は全て一意に確定する。イ：正しい。
ウ．「第3圏にいる」の報告だけでB機が確定するか？
　→ 南・第3圏 と 西・第3圏 の2通りが残り確定しない。ウ：誤り。

したがって、正解は(4)　ア、イが正しい　である。
