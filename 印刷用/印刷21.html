問1．下図のように、正六角形の内部を6等分した図形が規則的に変化している。?に当てはまる図形として正しいものを(1)〜(5)から1つ選べ。（6点）
【変化の系列】
問1（正解:(2)）
正六角形の内部が6つの扇形に等分されており、隣り合う2つの扇形が塗りつぶされている。

【扇形の番号付け】
扇形を時計回りに1〜6と番号付けする（1=右上、2=右、3=右下、4=左下、5=左、6=左上）。

【規則の発見】
各ステップで、塗りつぶされた2つの隣接扇形が時計回りに1つずつ移動している。
・A: 扇形1+2（右上・右）が塗りつぶし
・B: 扇形2+3（右・右下）が塗りつぶし → 1ステップ時計回り
・C: 扇形3+4（右下・左下）が塗りつぶし → 1ステップ時計回り
・D: 扇形4+5（左下・左）が塗りつぶし → 1ステップ時計回り
・?: 扇形5+6（左・左上）が塗りつぶし → 1ステップ時計回り

【選択肢の検討】
・(1) 扇形4+5 → Dと同じ（直前の状態）
・(2) 扇形5+6 → 正解
・(3) 扇形6+1 → 1ステップ余計に進んだ状態
・(4) 扇形3+4 → Cと同じ（2ステップ前）
・(5) 扇形1+2 → Aと同じ（最初の状態）

したがって、正解は(2)である。
問2．下図は、正方形を対角線で4分割した図形を3行3列に配置したものである。横方向・縦方向にそれぞれ規則が存在するとき、?に入る図形として正しいものを(1)〜(5)から1つ選べ。（6点）
【3×3 図形行列】
問2（正解:(3)）
各セルは正方形を対角線で4分割した図形であり、「上・右・下・左」の4つの三角形で構成されている。

【横方向の規則（列による塗りつぶし数）】
・列1: 1つの三角形が塗りつぶされている
・列2: 2つの三角形が塗りつぶされている
・列3: 3つの三角形が塗りつぶされている

【縦方向の規則（行による塗りつぶし開始位置）】
・行1: 「上」の三角形から時計回りに塗りつぶし開始
・行2: 「右」の三角形から時計回りに塗りつぶし開始
・行3: 「下」の三角形から時計回りに塗りつぶし開始

【?の特定】
?は「行3・列3」のセルであるから、「下」から時計回りに3つの三角形が塗りつぶされる。
下→左→上 の順に3つ → 「下・左・上」が塗りつぶされ、「右」のみ未塗り。

【選択肢の検討】
・(1) 上+右+下（左なし） → 「右」が含まれるので不正解
・(2) 右+下+左（上なし） → 「下」から始まっていないので不正解
・(3) 上+下+左（右なし） → 正解
・(4) 上+右+左（下なし） → 「下」が含まれないので不正解
・(5) 全塗り → 塗りつぶし数が4つで不正解

したがって、正解は(3)である。
