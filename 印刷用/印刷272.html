問1．下の図のように、方眼のマス目のいくつかを黒く塗りつぶして作った図形がある。次の(1)〜(5)のうち、この図形を紙面上で回転させたとき、元の図形とぴったり重ね合わせることができるものはどれか。ただし、図形を裏返してはならない。（6点）
【図1】元の図形
問1(正解:(3))
元の図形も5つの選択肢も、黒く塗られたマスはすべて6個であり、マスの数では区別できない。紙面上の回転で一致するかどうかは、図形の「裏表（左右の向き）」が保たれているかどうかで決まる。

【判定の着眼点】
・元の図形は、まっすぐ3マス並んだ「軸」を持ち、その一端には直角に突き出た「2マスの頭」、もう一端には「1マス曲がってさらに1マス折れるL字のしっぽ」が付いた階段状の図形である。
・軸が縦、頭が上になるように図形を回転させて置くと、元の図形では頭は左へ突き出し、しっぽは右へ折れる。
・紙面上でどのように回転させてもこの左右の関係は変わらない。一方、図形を裏返したもの（鏡像）では関係が逆になり、頭は右、しっぽは左になる。

【選択肢の検討】
・(1) 軸を縦・頭を上にすると、頭が右・しっぽが左 → 元の図形を裏返したもの（鏡像） → 一致しない
・(2) 同様に頭が右・しっぽが左 → 鏡像を90度回転したもの → 一致しない
・(3) 頭が左・しっぽが右 → 元の図形を時計回りに90度回転したもの → 一致する
・(4) 頭が右・しっぽが左 → 鏡像を180度回転したもの → 一致しない
・(5) 頭が右・しっぽが左 → 鏡像を270度回転したもの → 一致しない

したがって、正解は(3)である。
問2．下の図のように、方眼のマス目のいくつかを黒く塗りつぶして作った図形がある。次の(1)〜(5)のうち、この図形を紙面上で回転させたとき、元の図形とぴったり重ね合わせることができるものはどれか。ただし、図形を裏返してはならない。（6点）
【図2】元の図形
問2(正解:(1))
黒く塗られたマスはどの図形も9個で、全体の輪郭もよく似ているため、マスの数や大まかな形では判別できない。特徴的な部分を決めて、その位置関係が保たれているかを調べる。

【判定の着眼点】
・元の図形は、幅1マスの帯が階段状に折れ曲がった一筆書き状の図形で、一方の端に「まっすぐ3マス並んだ頭」がある。
・頭が上（水平）になるように図形を回転させて置くと、元の図形では帯は頭の右端から下へ伸びる。回転させてもこの関係は変わらないが、裏返した図形（鏡像）では帯は頭の左端から伸びる。
・また、元の図形は端から端まで一本につながった帯であり、途中に枝分かれはない。

【選択肢の検討】
・(1) 頭を上にすると帯は右端から伸びる → 元の図形を反時計回りに90度回転したもの → 一致する
・(2) 頭を上にすると帯は左端から伸びる → 鏡像を90度回転したもの → 一致しない
・(3) 頭を上にすると帯は左端から伸びる → 鏡像を180度回転したもの（元の図形の上下反転） → 一致しない
・(4) 頭を上にすると帯は左端から伸びる → 鏡像を270度回転したもの → 一致しない
・(5) 一見(1)と同じに見えるが、右下の端の1マスが1つ内側にずれており、帯の途中に枝分かれができている → 一致しない

(5)のような「1マスだけ違う図形」は、端の部分どうしを1マスずつ突き合わせて確かめるとよい。

したがって、正解は(1)である。
