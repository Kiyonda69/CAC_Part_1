問1．下の3つの図は、各面に1〜6の数字が一つずつ書かれた同じサイコロを、異なる方向から見たものである。このサイコロを、前面（数字「2」の面）を中央に置いた十字型の展開図に開いたとき、空欄(ア)・(イ)・(ウ)に入る数の組合せとして正しいものを、下の選択肢(1)〜(5)のうちから一つ選びなさい。（6点）
【図1】サイコロの3つの視点
【図2】十字型展開図（前面=2 を中央に配置）
(1) (ア, イ, ウ) = (5, 4, 6)
(2) (ア, イ, ウ) = (4, 6, 5)
(3) (ア, イ, ウ) = (4, 5, 6)
(4) (ア, イ, ウ) = (6, 5, 4)
(5) (ア, イ, ウ) = (5, 6, 4)
問1(正解:(3))
3つの視点から、サイコロの各面の対応関係を順に決定する問題である。

【視点1から判明する3面】
視点1: 上=1, 前=2, 右=3。これにより前面と右面・上面の3面が確定する。残り3面（左、後、下）は{4, 5, 6}のいずれかとなる。

【視点2から下面を決定】
視点2: 上=3, 前=2, 右=6。前面が「2」のままなので、視点1から前後軸まわりに左へ90°回転（左に転がす）したものと判る。
左回転では「上→左、左→下、下→右、右→上」となる。視点1の右=3が新しい上=3となるのは整合し、新しい右=6は視点1の下面に相当するから、下=6が確定する（よって対面: 上↔下 = 1↔6）。

【視点3から左面を決定】
視点3: 上=4, 前=2, 右=1。前面が「2」のままなので、視点1から前後軸まわりに右へ90°回転（右に転がす）したもの。
右回転では「上→右、右→下、下→左、左→上」。新しい上=4は視点1の左面に相当するから、左=4が確定する（よって対面: 左↔右 = 4↔3）。

【後面の決定】
残る面は1つ（後面）であり、未使用の数字は5。よって後=5。対面: 前↔後 = 2↔5。

すべての対面の組: 1↔6、2↔5、3↔4（標準的な西洋サイコロと同じ配置）。

【十字型展開図】
上=1、左(ア)=4、前=2、右=3、後(イ)=5、下(ウ)=6。

【選択肢の検討】
・(1) (5,4,6): 左=5、後=4 だが、視点3より左=4が確定 → 不正解
・(2) (4,6,5): 後=6、下=5 だが、視点2より下=6が確定 → 不正解
・(3) (4,5,6): すべて整合 → 正解
・(4) (6,5,4): 左=6、下=4 だが、視点3より左=4 → 不正解
・(5) (5,6,4): 左=5、下=4 だが、視点3より左=4 → 不正解

したがって、正解は(3)である。
問2．下の4つの図は、各面に1〜6の数字が一つずつ書かれた同じサイコロを、異なる方向から見たものである（このサイコロは標準的な西洋サイコロとは限らない）。このサイコロを、前面（数字「2」の面）を中央に置いた十字型の展開図に開いたとき、空欄(ア)・(イ)・(ウ)に入る数の組合せとして正しいものを、下の選択肢(1)〜(5)のうちから一つ選びなさい。（6点）
【図1】サイコロの4つの視点
【図2】十字型展開図（前面=2 を中央に配置）
(1) (ア, イ, ウ) = (3, 6, 4)
(2) (ア, イ, ウ) = (6, 3, 4)
(3) (ア, イ, ウ) = (6, 4, 3)
(4) (ア, イ, ウ) = (4, 3, 6)
(5) (ア, イ, ウ) = (3, 4, 6)
問2(正解:(2))
4つの視点から、対面の組（top↔bottom, front↔back, left↔right）を順に決定する非標準サイコロの問題である。

【視点1から3面の確定】
視点1: 上=1, 前=2, 右=5。これにより上面・前面・右面の3面が確定する。
残り3面（下、後、左）は{3, 4, 6}のいずれかに対応する。

【視点2: 前後軸まわりの左回転（roll left）】
視点2: 上=5, 前=2, 右=4。前=2が共通なので、視点1から前後軸まわりに90°左回転した姿勢である。
左回転では「上→左、左→下、下→右、右→上」と対応するため:
・新しい上=4の位置にあるのは視点1の右=5 → 整合
・新しい右=4は視点1の下面に相当する → よって下=4が確定（対面: 上↔下 = 1↔4）

【視点3: 上下軸まわりの右回転（yaw right）】
視点3: 上=1, 前=5, 右=3。上=1が共通なので、視点1から上下軸まわりに90°右回転した姿勢である。
右回転では「前→左、左→後、後→右、右→前」と対応するため:
・新しい前=5の位置にあるのは視点1の右=5 → 整合
・新しい右=3は視点1の後面に相当する → よって後=3が確定（対面: 前↔後 = 2↔3）

【視点4: 前後軸まわりの右回転で確認】
視点4: 上=6, 前=2, 右=1。前=2が共通なので前後軸まわりに90°右回転した姿勢。
右回転では「上→右、右→下、下→左、左→上」と対応するため:
・新しい上=6は視点1の左面に相当する → よって左=6が確定（対面: 左↔右 = 6↔5）

すべての対面の組: 1↔4、2↔3、5↔6（標準的な西洋サイコロとは異なる配置）。

【十字型展開図】
上=1、左(ア)=6、前=2、右=5、後(イ)=3、下(ウ)=4。

【選択肢の検討】
・(1) (3,6,4): 左=3、後=6 だが、視点3より後=3、視点4より左=6 → 不正解
・(2) (6,3,4): 左=6、後=3、下=4。すべて整合 → 正解
・(3) (6,4,3): 後=4、下=3 だが、視点2より下=4、視点3より後=3 → 不正解
・(4) (4,3,6): 左=4、下=6 だが、視点4より左=6、視点2より下=4 → 不正解
・(5) (3,4,6): 左=3 だが、視点4より左=6 → 不正解

したがって、正解は(2)である。
