問1．図のような正六角形ABCDEFの頂点に次のア～オの操作のいずれか1つを行ったとき、操作後の頂点の並びが元の正六角形と一致する（回転や裏返しにより元の配置に戻せる）操作はどれか。正しいものの組み合わせを1つ選び、解答用紙にマークせよ。（6点）
ア　AをCに、CをEに、EをAに移し、BをDに、DをFに、FをBに移す。
イ　AとD、BとE、CとFをそれぞれ入れ替える。
ウ　AとCを入れ替え、BとFを入れ替える。DとEはそのまま。
エ　BとFを入れ替え、CとEを入れ替える。AとDはそのまま。
オ　AとBを入れ替え、DとEを入れ替える。CとFはそのまま。

(1)ア、ウ　　(2)イ、エ　　(3)ア、イ、エ　　(4)ア、ウ、オ　　(5)イ、エ、オ
問1(正解:(3))

【解法のポイント】
正六角形の対称操作（回転・裏返し）は全部で12通りある。操作後の頂点配置がこの12通りのいずれかに該当すれば、元の正六角形と一致する。

【正六角形の対称操作】
・回転: 0度、60度、120度、180度、240度、300度の6通り
・裏返し: 頂点を通る軸3本、辺の中点を通る軸3本の計6通り
・合計12通りの対称操作

【各操作の検証】
・ア: (ACE)(BDF) = 120度回転に対応 → 対称操作である
・イ: (AD)(BE)(CF) = 180度回転に対応 → 対称操作である
・ウ: (AC)(BF) → AとC、BとFを入れ替えるとA-Cは2つ離れた頂点、B-Fは隣接頂点。この置換は12通りの対称操作のいずれにも該当しない → 対称操作でない
・エ: (BF)(CE) → AとDを固定し、B-F、C-Eを入れ替える。これはA-D軸に関する鏡映（裏返し）に対応 → 対称操作である
・オ: (AB)(DE) → AとB、DとEを入れ替えるが、CとFが固定。正六角形の対称軸でC-Fを通る軸は(BD)(AE)の入れ替えを生じるため、(AB)(DE)は対称操作に該当しない → 対称操作でない

したがって、正六角形の対称操作であるのはア、イ、エの3つであり、正解は(3)である。
問2．6つの停留所P, Q, R, S, T, Uがあり、3つのバス路線X, Y, Zが運行している。各路線は2つ以上の停留所を通る。次のことが分かっているとき、確実にいえるものはどれか。正しいものの組み合わせを1つ選び、解答用紙にマークせよ。（6点）
(I)　各停留所には少なくとも1つの路線が通る。
(II)　路線Xは3つの停留所を通り、路線Yは3つの停留所を通り、路線Zは2つの停留所を通る。
(III)　PとQには路線Xのみが通る。
(IV)　Rには路線Xと路線Yの2つが通る。
(V)　Sには路線Yのみが通る。
(VI)　TとUにはそれぞれ少なくとも1つの路線が通るが、路線Xは通らない。
ア　路線Zが通る停留所のうち少なくとも1つには、路線Yも通る。
イ　Rには路線X、Y、Zの3つすべてが通る。
ウ　TとUのうち路線Yが通るのはちょうど1つである。
エ　2つ以上の路線が通る停留所はちょうど2つである。

(1)ア、イ　　(2)ア、エ　　(3)イ、ウ　　(4)ア、ウ　　(5)ウ、エ
問2(正解:(4))

【条件の整理】
・路線X（3停留所）: 条件(III)よりP, Qを含み、条件(IV)よりRを含む → X = {P, Q, R}
・条件(VI)よりTとUはXに含まれないので、X = {P, Q, R} で確定。
・路線Y（3停留所）: 条件(IV)よりRを含み、条件(V)よりSを含む。3つ目は条件(III)よりP,Qは不可（Xのみ）、R,Sは既に含む → TまたはUのいずれか1つ。
・路線Z（2停留所）: 条件(III)よりP,Qは不可、条件(V)よりSは不可 → {R, T, U}から2つ選択。

【有効な配置の列挙】
条件(I)（全停留所にルートあり）を満たす配置は以下の4通り:
・配置1: Y={R,S,T}, Z={R,U} → Rに3路線
・配置2: Y={R,S,T}, Z={T,U} → Rに2路線、Tに2路線
・配置3: Y={R,S,U}, Z={R,T} → Rに3路線
・配置4: Y={R,S,U}, Z={T,U} → Rに2路線、Uに2路線

【各命題の検証】
・ア: Zの停留所の少なくとも1つにYも通る
　→ 全4配置で成立（配置1: Rに両方、配置2: Tに両方、配置3: Rに両方、配置4: Uに両方）→ 確実にいえる
・イ: Rに3路線すべて
　→ 配置1,3では成立するが、配置2,4では不成立 → 確実にはいえない
・ウ: T,Uのうち路線Yが通るのはちょうど1つ
　→ Yの3つ目はTかUのいずれか一方 → 全4配置で成立 → 確実にいえる
・エ: 2路線以上の停留所がちょうど2つ
　→ 配置1,3ではRのみ（1つ）、配置2,4では2つ → 確実にはいえない

したがって、確実にいえるのはア、ウであり、正解は(4)である。
