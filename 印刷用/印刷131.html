問1．下の図のように半径3の円の内部に半径1の円が点Pで接している。半径1の円を半径3の円周の内部を滑らないように転がしていくときの点Pの軌跡はどれか。正しいものを1つ選び解答用紙にマークせよ。（6点）
【図1】半径3の円内で転がる半径1の円
実線：大円（半径3）
破線：小円の初期位置（半径1）
点P：接点
問1(正解:(4))
このタイプの問題は、ハイポサイクロイド（内転円）と呼ばれる古典的な曲線を扱っています。

【軌跡の性質】
大きな円の内側で小さな円が滑らずに転がるとき、接点の軌跡はハイポサイクロイドという曲線になります。この曲線の形は大円と小円の半径比によって決まります。

【半径比による軌跡の分類】
・半径比が2:1（大:小）→ 直線
・半径比が3:1 → 3つの尖点を持つ曲線（デルトイド）
・半径比が4:1 → 4つの尖点を持つ曲線（アストロイド）

【本問での計算】
大円の半径 R = 3、小円の半径 r = 1
半径比 R/r = 3/1 = 3
したがって、軌跡は3つの尖点を持つデルトイドになります。

【選択肢の検討】
・(1) 直線 → 半径比2:1の場合のみ。本問では不正解。
・(2) 円 → 転がる際にずれが生じるため不可能。
・(3) 2つの山を持つ曲線 → パターン不一致。
・(4) 3つの尖点を持つデルトイド → 半径比3:1に対応。正解。
・(5) 別パターン → 尖点の配置が異なる。

したがって、正解は(4)である。
問2．下の図のように半径4の円の内部に半径1の円が点Pで接している。半径1の円を半径4の円周の内部を滑らないように転がしていくときの点Pの軌跡はどれか。正しいものを1つ選び解答用紙にマークせよ。（6点）
【図2】半径4の円内で転がる半径1の円
実線：大円（半径4）
破線：小円の初期位置（半径1）
点P：接点
問2(正解:(1))
問1と同様にハイポサイクロイドの軌跡を問う問題ですが、今回は円の半径比が異なります。

【轨跡の分析】
大きな円の内側で小さな円が滑らずに転がるとき、接点の軌跡はハイポサイクロイドになり、その形状は半径比によって一意に決まります。

【半径比の計算】
大円の半径 R = 4、小円の半径 r = 1
半径比 R/r = 4/1 = 4
したがって、軌跡は4つの尖点を持つアストロイド（astroid）になります。

【アストロイドの特徴】
・4つの鋭い尖点を持つ
・正方形に内接する曲線
・対称性が高い（4回対称）
・数学的には（x^(2/3) + y^(2/3) = a^(2/3)）で表される

【選択肢の検討】
・(1) 4つの尖点を持つアストロイド → 半径比4:1に対応。正解。
・(2) 円 → 転がる際に軌跡がずれるため不可能。
・(3) 5つの尖点 → 半径比5:1に対応する異なるパターン。
・(4) 3つの尖点（デルトイド） → 半径比3:1に対応する問1のパターン。
・(5) 直線 → 半径比2:1の場合のみ。本問では不正解。

【結論】
問1（半径比3:1）→ デルトイド（3尖点）
問2（半径比4:1）→ アストロイド（4尖点）

したがって、正解は(1)である。
