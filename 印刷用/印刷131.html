問1．下の図のように半径3の円の内部に半径1の円が点Pで接している。半径1の円を半径3の円周の内部を滑らないように転がしていくときの点Pの軌跡はどれか。正しいものを1つ選び解答用紙にマークせよ。（6点）

問1（正解:(4)）

半径Rの円の内部を半径rの円が滑らずに転がるとき、初め接点Pにあった点が描く軌跡をハイポサイクロイド（内転擺線）という。

【性質】
R/r = n（自然数）のとき、ハイポサイクロイドはn個の尖点をもち、すべての尖点は半径Rの外円に接する。

・R/r = 2 → 直径（直線）に退化
・R/r = 3 → 3尖点（デルトイド）
・R/r = 4 → 4尖点（アステロイド）

【本問】
R = 3, r = 1 より R/r = 3。よって軌跡は3個の尖点をもつ星形（デルトイド）である。

各尖点は外円上にあり、隣り合う尖点の中心角は120°となる。

したがって、正解は(4)である。

【参考】媒介変数表示
R = 3, r = 1のハイポサイクロイドは次式で与えられる。
x(t) = 2 cos t + cos 2t
y(t) = 2 sin t − sin 2t
t = 0, 2π/3, 4π/3 で尖点となり、各尖点の位置は外円上の3点(3, 0), (−3/2, 3√3/2), (−3/2, −3√3/2)である。

問2（正解:(1)）

半径Rの円の外部を半径rの円が滑らずに転がるとき、初め接点Pにあった点が描く軌跡をエピサイクロイド（外転擺線）という。

【性質】
R/r = n（自然数）のとき、エピサイクロイドはn個の尖点をもち、すべての尖点は半径Rの内円上にある。

・R/r = 1 → 1尖点（カーディオイド／心臓形）
・R/r = 2 → 2尖点（ネフロイド／腎臓形）
・R/r = 3 → 3尖点（3尖エピサイクロイド）

【本問】
R = 2, r = 1 より R/r = 2。よって軌跡は2個の尖点をもつ腎臓形（ネフロイド）である。

2つの尖点は内円上で互いに正反対の位置（中心角180°）にあり、軌跡はそれらを結ぶ直径を対称軸とする。

したがって、正解は(1)である。

【参考】媒介変数表示
R = 2, r = 1のエピサイクロイドは次式で与えられる。
x(t) = 3 cos t − cos 3t
y(t) = 3 sin t − sin 3t
t = 0, π で尖点となり、各尖点の位置は内円上の2点(2, 0), (−2, 0)である。最遠点は(0, ±4)で、原点から距離4(= R + 2r)の位置にある。
