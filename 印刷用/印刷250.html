問1．次の資料1は、ある飛行場の運航規程のうち「使用滑走路の選定」に関する抜粋であり、資料2（次のページ）はこの飛行場の滑走路の配置と現在の気象である。資料1の規程に従って計算するとき、訓練機が使用すべき滑走路と、そのときの正面風成分の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。（6点）
【資料1】運航規程「使用滑走路の選定」（抜粋）
第1条　滑走路の各使用方向は、その方向の磁方位を10で割った2桁の数字で呼ぶ（例：磁方位090°へ向かって離着陸する方向は「滑走路09」、磁方位270°へ向かう方向は「滑走路27」）。1本の滑走路は、両端のどちらの方向からも使用できる。
第2条　風向は「風が吹いてくる方位」で表す。
第3条　風向と使用方向の磁方位との角度差（0°〜180°で表す）を求め、下の係数表を用いて次のとおり計算する。
　　　　・正面風成分 ＝ 風速 × 正面風係数
　　　　・横風成分 ＝ 風速 × 横風係数
　　　　正面風係数が負のときは追い風であり、計算結果の絶対値を「追い風成分」とする。
第4条　横風成分と追い風成分がいずれも機体の制限値以下となる方向のみ使用できる。
第5条　使用できる方向が複数あるときは、正面風成分が最も大きい方向を使用する。
角度差 0° 30° 45° 60° 90° 120° 135° 150° 180°
正面風係数 1.00 0.87 0.71 0.50 0.00 −0.50 −0.71 −0.87 −1.00
横風係数 0.00 0.50 0.71 0.87 1.00 0.87 0.71 0.50 0.00
（資料2と選択肢は次のページ）
【資料2】滑走路の配置と現在の気象
・滑走路は2本: 09/27（磁方位090°/270°）、18/36（磁方位180°/360°）
・現在の風: 風向120°・風速12ノット（ガストなし）
・訓練機の制限値: 横風成分12ノット、追い風成分5ノット
(1) 滑走路09 ・ 正面風成分 約10.4ノット
(2) 滑走路09 ・ 正面風成分 約6.0ノット
(3) 滑走路18 ・ 正面風成分 約10.4ノット
(4) 滑走路18 ・ 正面風成分 約6.0ノット
(5) 使用できる方向はない
問1(正解:(1))
規程の係数表を4つの使用方向すべてに適用し、制限値で絞り込む資料読取問題である。

【手順1: 各方向の角度差】
風向は120°。各使用方向の磁方位との角度差（0°〜180°）は次のとおり。
・滑走路09（090°）: |120−90| ＝ 30°
・滑走路18（180°）: |120−180| ＝ 60°
・滑走路27（270°）: |120−270| ＝ 150°
・滑走路36（360°）: |120−360| ＝ 240° → 360−240 ＝ 120°

【手順2: 成分の計算と判定】（風速12ノット）
・滑走路09: 正面風 12×0.87＝＋10.44　横風 12×0.50＝6.00 → 制限内で使用可
・滑走路18: 正面風 12×0.50＝＋6.00　横風 12×0.87＝10.44（≦12） → 制限内で使用可
・滑走路27: 正面風 12×(−0.87)＝−10.44 → 追い風10.44ノット（＞5） → 使用不可
・滑走路36: 正面風 12×(−0.50)＝−6.00 → 追い風6.00ノット（＞5） → 使用不可

【手順3: 使用方向の決定】
使用できる方向は09と18の2つ。第5条により正面風成分が最も大きい方向を使用するので、
滑走路09（正面風成分 10.44 ≒ 約10.4ノット）を使用する。

【選択肢の検討】
・(1) 滑走路09・約10.4ノット → 上記のとおり正解
・(2) 滑走路09・約6.0ノット → 係数表の正面風係数と横風係数を取り違えた誤り（6.0は09の横風成分）
・(3) 滑走路18・約10.4ノット → 滑走路の選定を誤り、さらに18の横風成分を正面風成分と取り違えた誤り
・(4) 滑走路18・約6.0ノット → 18の正面風成分としては正しい値だが、第5条（正面風成分が最大の方向を使用）に反する
・(5) 使用できる方向はない → 横風成分を風速そのもの（12ノット）と比較するなどの誤判定。実際は09と18の2方向が使用できる

したがって、正解は(1)である。
問2．問1と同じ飛行場（滑走路09/27・18/36）に、ある到着機が着陸を計画している。資料3は問1の運航規程に追加される規程であり、資料4は当日の時刻別の風の予報である。この到着機の制限値は、横風成分15ノット、追い風成分5ノットである。問1の資料1・係数表と資料3の規程に従って計算するとき、どの方向の滑走路も使用できず、この到着機が着陸できない時刻として正しいものを、下の(1)〜(5)のうちから1つ選べ。（6点）
【資料3】追加規程（ガストの取り扱い）
・風速が「ガスト（瞬間最大風速）」を伴って報じられている場合、横風成分はガスト値を用いて計算し、正面風成分および追い風成分は平均風速を用いて計算する。
・ガストの報がない場合は、すべての成分を平均風速で計算する。
【資料4】時刻別風予報
時刻 風向 平均風速 ガスト
09時 090° 10ノット ―
10時 135° 12ノット 22ノット
11時 180° 14ノット 20ノット
12時 225° 10ノット 16ノット
13時 330° 16ノット 24ノット
(1) 09時
(2) 10時
(3) 11時
(4) 12時
(5) 13時
問2(正解:(2))
各時刻について4方向すべての成分を計算する。資料3により、横風成分はガスト値、正面風・追い風成分は平均風速で計算する（制限: 横風15ノット・追い風5ノット）。

【09時】風向090°・平均10ノット・ガストなし（すべて平均風速で計算）
・09（角度差0°）: 正面風＋10.0、横風0.0 → 使用可
・18・36（角度差90°）: 正面風0.0（追い風も0で制限内）、横風10.0（≦15） → 使用可
→ 着陸できる

【10時】風向135°・平均12ノット・ガスト22ノット
風向135°は4つの使用方向すべてと角度差45°または135°になる点が急所。
・09（45°）: 正面風 12×0.71＝＋8.5、横風 22×0.71＝15.6（＞15） → 使用不可
・18（45°）: 同じく横風15.6（＞15） → 使用不可
・27（135°）: 追い風 12×0.71＝8.5（＞5） → 使用不可
・36（135°）: 追い風8.5（＞5） → 使用不可
→ 4方向すべて使用できず、着陸できない

【11時】風向180°・平均14ノット・ガスト20ノット
・18（角度差0°）: 正面風＋14.0、横風0.0 → 使用可（09・27は横風20.0で不可、36は追い風14.0で不可）

【12時】風向225°・平均10ノット・ガスト16ノット
・18・27（角度差45°）: 正面風＋7.1、横風 16×0.71＝11.4（≦15） → 使用可（09・36は追い風7.1で不可）

【13時】風向330°・平均16ノット・ガスト24ノット
・36（角度差30°）: 正面風 16×0.87＝＋13.9、横風 24×0.50＝12.0（≦15） → 使用可
　（27は横風 24×0.87＝20.9で不可、09・18は追い風で不可）

【選択肢の検討】
・(1) 09時 → 滑走路09が使用できる
・(2) 10時 → 上記のとおり全方向が使用できず正解
・(3) 11時 → 滑走路18が使用できる。ガスト20ノットに惑わされないこと（18の横風は0）
・(4) 12時 → 滑走路18・27が使用できる
・(5) 13時 → 滑走路36が使用できる。平均16・ガスト24と風は最も強いが、横風成分は制限内

風の強さ（ガスト値）だけでは判定できず、風向と滑走路方位の角度差が決定的であることを見抜く必要がある。10時は風が斜め45°から吹くため、ガストによる横風超過と追い風超過が全方向で同時に生じる。

したがって、正解は(2)である。
