問1．ある商品に、原価の3割の利益を見込んで定価をつけた。しかし売れなかったので、定価の2割引にして売ったところ、120円の利益が得られた。この商品の原価はいくらか。次の(1)〜(5)から1つ選べ。（6点）
(1) 2,400円   (2) 1,200円   (3) 3,000円   (4) 6,000円   (5) 4,000円
問1(正解:(3))
原価を x 円とおき、定価と実際の売価を順に式で表す。

【立式】
・定価：原価の3割の利益を見込む → 定価 = x × (1 + 0.3) = 1.3x 円
・売価：定価の2割引 → 売価 = 1.3x × (1 − 0.2) = 1.3x × 0.8 = 1.04x 円
・利益：売価 − 原価 = 1.04x − x = 0.04x 円

【求解】
0.04x = 120 より x = 120 ÷ 0.04 = 3,000円

【検算】
原価3,000円 → 定価 3,000 × 1.3 = 3,900円 → 売価 3,900 × 0.8 = 3,120円
利益 = 3,120 − 3,000 = 120円 となり、条件と一致する。

【選択肢の検討】
・(1) 2,400円 → 利益率を0.05と計算した誤り（120 ÷ 0.05）。不正解
・(2) 1,200円 → 「3割 − 2割 = 1割の利益」と誤り、0.1x = 120 とした値。割引は定価（1.3x）に掛かるため誤り。不正解
・(3) 3,000円 → 0.04x = 120 を満たす。正解
・(4) 6,000円 → 1.3 × 0.8 を 1.02 と誤計算した値（120 ÷ 0.02)。不正解
・(5) 4,000円 → 1.3 × 0.8 を 1.03 と誤計算した値（120 ÷ 0.03)。不正解

したがって、正解は(3)である。
問2．ある店で、同じ商品を240個仕入れ、原価の2割5分の利益を見込んで定価をつけて売り出した。何個かは定価で売れたが、売れ残りが出たため、売れ残った分はすべて定価の2割引にして売り切った。その結果、全体の利益は仕入総額のちょうど15％となった。定価で売れた個数は何個か。次の(1)〜(5)から1つ選べ。（6点）
(1) 120個   (2) 96個   (3) 180個   (4) 192個   (5) 144個
問2(正解:(5))
原価の実額は与えられていないが、利益の条件が仕入総額に対する割合で示されているため、原価を1個あたり x 円とおけば x は式の両辺から消える。定価で売れた個数を n 個とおく。

【1個あたりの利益を整理】
・定価 = x × 1.25 = 1.25x 円 → 定価で売れた分の利益は1個あたり 0.25x 円
・割引後の売価 = 1.25x × 0.8 = x 円 → 割引後の売価は原価とちょうど等しい
　つまり、売れ残って割引で売った分の利益は1個あたり0円である。

【立式と求解】
全体の利益 = 定価で売れた分の利益のみ = 0.25x × n 円
仕入総額 = 240x 円 なので、条件より
0.25x × n = 240x × 0.15 = 36x
両辺を x で割って 0.25n = 36 → n = 36 ÷ 0.25 = 144個

【検算】
原価を800円とすると、仕入総額 = 192,000円。定価1,000円で144個 → 利益 200 × 144 = 28,800円。割引価格800円で96個 → 利益0円。
総利益28,800円 ÷ 192,000円 = 0.15 = 15％ となり、条件と一致する。

【選択肢の検討】
・(1) 120個 → 割引販売分にも「2割5分 − 2割 = 5分」の利益が残ると誤り、0.25n + 0.05(240 − n) = 36 とした値。2割引は定価1.25xに掛かるため利益は0。不正解
・(2) 96個 → 売れ残り（割引で売った）個数 240 − 144 = 96 との取り違え。不正解
・(3) 180個 → 利益率15％の基準を仕入総額でなく定価売上（1.25x × 240）と誤った値。不正解
・(4) 192個 → 「2割引だから2割が売れ残った」と早合点した 240 × 0.8 の値。不正解
・(5) 144個 → 0.25n = 36 を満たす。正解

したがって、正解は(5)である。
