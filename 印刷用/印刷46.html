問1．表は、ある地域の5つの農場（A～E）の年間収穫データを示したものである。1ヘクタールあたりの出荷額が高い年度順に農場を並べたものはどれか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
【農場別年間収穫データ】
農場 作付面積
(ha)		 収穫量
(t)		 農薬使用量
(kg)		 従業員数
(人)		 出荷額
(万円)
A1284368420
B181087212540
C872166504
D151056010480
E1060255360
(1) C→A→E→D→B
(2) E→C→A→D→B
(3) C→E→D→A→B
(4) A→C→E→D→B
(5) C→E→A→D→B
問1(正解:(5))

【解法のポイント】
各農場の出荷額を作付面積で割り、1ヘクタールあたりの出荷額を求めて比較する。

【計算過程】
・A: 420 / 12 = 35.0万円/ha
・B: 540 / 18 = 30.0万円/ha
・C: 504 / 8 = 63.0万円/ha
・D: 480 / 15 = 32.0万円/ha
・E: 360 / 10 = 36.0万円/ha

【順位の決定】
高い順に並べると: C(63.0) → E(36.0) → A(35.0) → D(32.0) → B(30.0)

【選択肢の検討】
・(1) C→A→E→D→B → AとEが逆（A=35.0 < E=36.0）
・(2) E→C→A→D→B → CとEが逆（C=63.0 > E=36.0）
・(3) C→E→D→A→B → DとAが逆（D=32.0 < A=35.0）
・(4) A→C→E→D→B → Aが1位は誤り
・(5) C→E→A→D→B → 正解

したがって、正解は(5)である。
問2．表は、ある企業の5つの支店（甲～戊）の上半期・下半期の売上高と経費、および従業員数を示したものである。従業員1人あたりの年間利益額（年間利益＝上半期利益＋下半期利益、各半期の利益＝売上高－経費）が大きい順に支店を並べたものはどれか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
【支店別半期業績データ】　　　　（単位：百万円）
支店 上半期 下半期 従業員数
(人)
売上高 経費 売上高 経費
甲18012024015030
乙25017020013040
丙1509019011025
丁30021028019050
戊20014022014035
(1) 甲→丙→戊→乙→丁
(2) 丙→甲→戊→乙→丁
(3) 丙→甲→乙→戊→丁
(4) 丙→戊→甲→乙→丁
(5) 丁→甲→戊→乙→丙
問2(正解:(2))

【解法のポイント】
各支店の上半期・下半期それぞれの利益を求め、年間利益を算出した後、従業員数で割る必要がある。年間売上高の大きさと1人あたり利益額は必ずしも一致しない点に注意。

【計算過程】
甲: 上半期利益 = 180 - 120 = 60、下半期利益 = 240 - 150 = 90
　　年間利益 = 150、1人あたり = 150 / 30 = 5.00百万円

乙: 上半期利益 = 250 - 170 = 80、下半期利益 = 200 - 130 = 70
　　年間利益 = 150、1人あたり = 150 / 40 = 3.75百万円

丙: 上半期利益 = 150 - 90 = 60、下半期利益 = 190 - 110 = 80
　　年間利益 = 140、1人あたり = 140 / 25 = 5.60百万円

丁: 上半期利益 = 300 - 210 = 90、下半期利益 = 280 - 190 = 90
　　年間利益 = 180、1人あたり = 180 / 50 = 3.60百万円

戊: 上半期利益 = 200 - 140 = 60、下半期利益 = 220 - 140 = 80
　　年間利益 = 140、1人あたり = 140 / 35 = 4.00百万円

【順位の決定】
大きい順: 丙(5.60) → 甲(5.00) → 戊(4.00) → 乙(3.75) → 丁(3.60)

【注意】
年間利益の総額だけを見ると丁(180)が最大だが、従業員数(50人)が多いため1人あたりでは最下位となる。甲と乙は年間利益が同額(150)だが、従業員数の差(30人 vs 40人)により1人あたりでは甲が上位となる。

したがって、正解は(2)である。
