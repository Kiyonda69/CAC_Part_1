問1．下の【図1】は、ある飛行場を真上から見た地図である。地図は北を上、東を右とする。パイロットの自機（◆）は地図の位置にあり、機首を真東（東向き、図の矢印の方向）に向けて止まっている。パイロットが操縦席の正面の窓から前方（機首の向く側）を見たとき、4つの施設 A〜D は左から右へどの順に並んで見えるか。正しい並びを下の(1)〜(5)から1つ選べ。（6点）
【図1】飛行場の地図（北が上・自機は東向き）
（左）……（右）の順
(1)　A → B → C → D
(2)　C → A → B → D
(3)　D → B → C → A
(4)　A → C → B → D
(5)　A → C → D → B
問1(正解:(4))
自機は東を向いている。操縦席から前方（東）を見ると、地図の「北側」は自分の左手、「南側」は右手になる。つまり北を上にした地図を、東が上に来るように頭の中で時計回りに90°回し、上＝前方として左右の並びを読み取ればよい。各施設について、機首方向（東）を基準にした左右のずれ（相対方位）を考える。

【左右の判定】
・前方（東）に対し、より北寄りの施設ほど左、より南寄りの施設ほど右に見える。
・A 管制塔：北へ大きく寄る → もっとも左。
・C ターミナル：やや北寄り → 左から2番目。
・B 格納庫：やや南寄り → 右から2番目。
・D 給油車：南へ大きく寄る → もっとも右。

よって左から右は A → C → B → D。

【選択肢の検討】
・(1) A→B→C→D：中央2つ（B・C）の左右を取り違えている。
・(2) C→A→B→D：A と C の前後（北寄りの度合い）を逆にしている。
・(3) D→B→C→A：左右をまるごと反転（後方から見た並び）にしてしまっている。
・(4) A→C→B→D：北寄り＝左で正しく並べたもの。正解。
・(5) A→C→D→B：右側 B・D の左右を取り違えている。

したがって、正解は(4)である。
問2．下の【図2】は、別の飛行場を真上から見た地図である。地図は北を上、東を右とする。自機（◆）は機首を南東（南東向き、図の矢印の方向）に向けて止まっている。パイロットが操縦席の正面の窓から前方（機首の向く側、正面に対して左右90°までの範囲）を見たとき、窓に見える施設を左から右へ並べたものとして正しいものを下の(1)〜(5)から1つ選べ。なお、機首の真横より後ろにある施設は窓には見えないものとする。（6点）
【図2】飛行場の地図（北が上・自機は南東向き）
（左）……（右）の順
(1)　P → R → S → Q → T
(2)　T → P → R → S → Q
(3)　Q → S → R → P
(4)　P → S → R → Q
(5)　P → R → S → Q
問2(正解:(5))
自機の機首は南東（北を0°として時計回りに135°）を向いている。各施設の方位を機首方向からの左右のずれ（相対方位）に直し、（1）真横より後ろにあるものは窓に見えないので除外し、（2）残りを左から右へ並べる。

【前後の判定】
・T レーダー塔は自機の北西、すなわち機首（南東）のちょうど真後ろ側にある。前方の窓（正面±90°）の外なので見えない。これがこの問題の最大のポイント。
・残る P・Q・R・S はいずれも前方側にあり、窓に見える。

【左右の判定】
機首（南東）を基準に、反時計回り側（おおまかに北東寄り）が左、時計回り側（南西寄り）が右になる。
・P 管制塔：もっとも左。
・R 格納庫：左から2番目。
・S ターミナル：ほぼ正面（中央）。
・Q 給油車：もっとも右。

よって窓に見える並びは P → R → S → Q（T は含まない）。

【選択肢の検討】
・(1) P→R→S→Q→T：並び自体は前方側で正しいが、後方の T を加えてしまっている。
・(2) T→P→R→S→Q：見えない T を含め、しかも左端に置いている。
・(3) Q→S→R→P：左右をまるごと反転させている。
・(4) P→S→R→Q：中央の R と S の左右を取り違えている。
・(5) P→R→S→Q：T を正しく除外し、左右の順も正しい。正解。

したがって、正解は(5)である。
