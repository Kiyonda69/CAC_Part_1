問1．下図は、立方体（正六面体）の展開図であり、六つの面にそれぞれアルファベット A・B・C・D・E・F が記されている。この展開図を組み立てて立方体を作るとき、面 B と向かい合う（平行に対面する）面はどれか。正しいものを1つ選べ。（6点）
【図1】立方体の展開図
(1) A　　(2) D　　(3) C　　(4) E　　(5) F
問1(正解:(2))

【展開図の組み立て】
中央の面 C を前面（基面）として立方体を組み立てると、各面は次のように配置される。
・C の上にある A を上方向へ折り込む → 上面 = A
・C の下にある E を下方向へ折り込む → 下面 = E
・C の左にある B を左方向へ折り込む → 左面 = B
・C の右にある D を右方向へ折り込む → 右面 = D
・E のさらに下にある F は、E が下面となった後に背面方向へ折り返される → 背面 = F

【対面ペアの整理】
立方体の平行な対面の組は以下の3組である。
・上面 A ⇔ 下面 E
・前面 C ⇔ 背面 F
・左面 B ⇔ 右面 D

【選択肢の検討】
・(1) A → 上面。B（左面）とは隣接する。誤り
・(2) D → 右面。B（左面）と平行な対面。正解
・(3) C → 前面。B（左面）とは隣接する。誤り
・(4) E → 下面。B（左面）とは隣接する。誤り
・(5) F → 背面。B（左面）とは隣接する。誤り

したがって、正解は(2)である。
問2．問1と同じ展開図を組み立てた立方体を、面 E が底面となるように机の上に置く。このとき、前面（手前側の面）には C が、右側面には D が見えている。この立方体を次の手順で転がしたとき、上面（最上面）に現れる文字はどれか。正しいものを1つ選べ。（6点）

【手順】
① 手前から奥方向へ 90° 転がす（前面が下面となるように倒す）。
② 続いて、①の状態から右方向へ 90° 転がす（右側面が下面となるように倒す）。
【図2】展開図（再掲）と初期配置
(1) A　　(2) D　　(3) B　　(4) F　　(5) C
問2(正解:(3))

【初期配置】
問1で求めたとおり、展開図を組み立てた立方体の対面ペアは次のとおり。
・A ⇔ E　・C ⇔ F　・B ⇔ D
初期状態で「底面 E・前面 C・右面 D」なので、自動的に「上面 A・背面 F・左面 B」となる。

【①手前から奥方向へ 90° 転がす】
前面と下面（底面）を結ぶ軸で回転。各面は次のように遷移する。
・上面 → 前面　：A が前面に
・前面 → 下面　：C が下面に
・下面 → 背面　：E が背面に
・背面 → 上面　：F が上面に
・左右は不変　：左 B・右 D
→ この時点で、上 F・前 A・右 D・下 C・後 E・左 B。

【②右方向へ 90° 転がす】
前面・背面は不変で、上下左右が回る。
・上面 → 右面　：F が右面に
・右面 → 下面　：D が下面に
・下面 → 左面　：C が左面に
・左面 → 上面　：B が上面に
・前後は不変　：前 A・後 E
→ 最終状態: 上 B・前 A・右 F・下 D・後 E・左 C。

【検算】
対面ペアは必ず保たれる。最終状態のペアは（上 B・下 D）（前 A・後 E）（右 F・左 C）で、初期の A⇔E、B⇔D、C⇔F と一致する。

【選択肢の検討】
・(1) A → 最終前面。誤り
・(2) D → 最終下面。誤り
・(3) B → 最終上面。正解
・(4) F → ①直後の上面で止まっている誤答
・(5) C → ①直後の下面。誤り

したがって、正解は(3)である。
