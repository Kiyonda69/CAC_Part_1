問1．下の【図1】のように、1辺の長さが2の正方形ABCDが、直線ℓの上を滑ることなく矢印の向きに回転しながら進み、正方形が再びもとの向き（頂点Aが左下にくる向き）になったところで止まる。このとき、頂点Aが描く軌跡として最も妥当なものを、下の(1)〜(5)のうちから1つ選べ。（6点）
【図1】
問1(正解:(3))
多角形が直線の上を滑ることなく転がるとき、多角形の頂点は「そのとき接地して回転の中心となっている頂点」を中心とする円弧を順に描く。正方形の場合、1回の回転角はつねに90°なので、描かれる弧はすべて四分円（中心角90°）である。

【頂点Aの動きの追跡】
・1回目の回転: 中心は頂点B。半径はAB＝2（辺の長さ）
・2回目の回転: 中心は頂点C。半径はAC＝対角線＝2√2 ≈ 2.83
・3回目の回転: 中心は頂点D。半径はAD＝2（辺の長さ）
・4回目の回転: 回転の中心がA自身になるため、Aは動かず弧を描かない

したがって軌跡は「四分円が3つ」で、中央の弧だけが半径2√2と大きく、最も高くふくらむ。

【解説図】頂点Aの軌跡（数値は各弧の半径）
【選択肢の検討】
・(1) 半円の山が2つ → 円が転がるとき（サイクロイド）との混同。正方形では弧は3つで中央だけ大きい → 不正解
・(2) 中央が水平な直線 → 2回目の回転でもAは中心Cから2√2の距離を保って弧を描くので、水平移動にはならない → 不正解
・(3) 四分円3つ・中央だけ半径が大きい → 上の追跡と一致 → 正解
・(4) 全体が1つの大きな半円 → 回転の中心が3回入れ替わることを無視している → 不正解
・(5) 中央がへこんだ形 → 中央の弧は半径2√2で両端の弧より高くなるはずで、へこむことはない → 不正解

したがって、正解は(3)である。
問2．下の【図2】のように、縦の辺の長さが1、横の辺の長さが2の長方形PQRSが、直線ℓの上を滑ることなく矢印の向きに回転しながら進み、長方形が再びもとの向き（頂点Pが左下にくる向き）になったところで止まる。このとき、頂点Pが描く軌跡の長さとして正しいものを、下の(1)〜(5)のうちから1つ選べ。ただし、円周率をπとする。（6点）
【図2】
(1)  5π/2
(2)  (2＋√5)π/2
(3)  (4＋√5)π/2
(4)  (3＋√5)π/2
(5)  (3＋√5)π
問2(正解:(4))
長方形が直線の上を滑ることなく転がるときも、頂点Pは「接地して回転の中心となっている頂点」を中心とする円弧を順に描く。長方形の内角は90°なので、1回の回転角はつねに90°（四分円）である。弧の長さは「半径 × π/2」で求められる。

【頂点Pの動きの追跡】
・1回目の回転: 中心は頂点Q。半径はPQ＝2（横の辺） → 弧長 2・(π/2)＝π
・2回目の回転: 中心は頂点R。半径はPR＝対角線＝√(2²＋1²)＝√5 → 弧長 √5・(π/2)＝√5π/2
・3回目の回転: 中心は頂点S。半径はPS＝1（縦の辺） → 弧長 1・(π/2)＝π/2
・4回目の回転: 回転の中心がP自身になるため、Pは動かず長さは0

よって軌跡の長さは
　π ＋ √5π/2 ＋ π/2 ＝ (2＋√5＋1)π/2 ＝ (3＋√5)π/2（≈ 8.22）

【解説図】頂点Pの軌跡（数値は各弧の半径）
【選択肢の検討】
・(1) 5π/2 ＝ (2＋2＋1)π/2 → 2回目の半径を対角線√5ではなく辺の2としてしまった誤り → 不正解
・(2) (2＋√5)π/2 → 3回目の半径1の弧を見落とした誤り → 不正解
・(3) (4＋√5)π/2 → 4回目にも半径1の弧を描くと誤って加えたもの（4回目はP自身が中心で動かない） → 不正解
・(4) (3＋√5)π/2 → 上の計算と一致 → 正解
・(5) (3＋√5)π → 各回転を180°（半円）として計算した誤り（実際は90°） → 不正解

したがって、正解は(4)である。
