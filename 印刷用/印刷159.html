問1．下の立方体を赤い点線で切って広げたとき、どの形になりますか。(1)〜(5)から1つ選べ。（6点）
問1(正解:(1))
立方体の6面を T(上)、B(下)、F(前)、K(奥)、L(左)、R(右)と表記する。

【切る辺の整理】
赤点線で示された切る辺は次の7本である。
・上面と奥・左・右の3辺(T-K, T-L, T-R)
・下面と奥・左・右の3辺(B-K, B-L, B-R)
・奥と左の縦辺(K-L)

【残す辺(つながったまま)】
12本のうち上記7本を切るので、残る5本が「ヒンジ(蝶番)」となる。
・上前(T-F)、下前(B-F)、前左縦(F-L)、前右縦(F-R)、奥右縦(K-R)

【展開の手順】
F面を平面に置く。F面を中心として、
・T面はT-F辺を軸にFの上方向へ広げる
・B面はB-F辺を軸にFの下方向へ広げる
・L面はF-L辺を軸にFの左方向へ広げる
・R面はF-R辺を軸にFの右方向へ広げる
さらに、K面はK-R辺で繋がっているので、Rの右方向へ展開される。

【できあがる形】
Fを中心にT・B・L・Rが十字に配置され、Rの右にKが付く。
すなわち「4マスの横一列の中央上下に1マスずつ付き、横一列が4マス」となる形である。これは選択肢(1)の形である。

【選択肢の検討】
・(1) 中央上下対称の十字+1形 → 正解
・(2) 4マス列の左端上・右端下に1マスずつ付く形 → ヒンジ位置が異なる
・(3) 階段形(2-2-2型) → ヒンジが連続しない、対応せず
・(4) 3-3オフセット形 → 上面の上下に枝分かれする展開ではない
・(5) L字型(2-3-1型) → 左上の2マスが横並びになる展開ではない

したがって、正解は(1)である。
問2．下の立方体を赤い点線で切って広げたとき、どの形になりますか。(1)〜(5)から1つ選べ。（6点）
問2(正解:(4))
立方体の6面を T(上)、B(下)、F(前)、K(奥)、L(左)、R(右)と表記する。

【切る辺の整理(7本)】
・上面の3辺: T-K, T-L, T-R
・下前(B-F)、下奥(B-K)
・前左縦(F-L)、奥右縦(K-R)

【残す辺(5本のヒンジ)】
T-F、F-R、R-B、B-L、L-K の5本が連続的につながっている。
これは「T-F-R-B-L-K」と一筆書きの順序で隣接面が連なる構造である。

【展開の手順】
Fを中心に置く。残った5本が一本鎖に繋がっているため、ジグザグ状の展開図となる。
・F → 上にT(T-F辺)
・F → 右にR(F-R辺)
・R → 下にB(R-B辺)
・B → 下にL(B-L辺、ただしBの向きが回転しているのでLはBの「下方向」へ)
・L → 右にK(L-K辺、Lの向きに従ってKはLの「右方向」へ)

【できあがる形】
以下の階段状(ジグザグ)になる。
・1段目: T
・2段目: F、R
・3段目: B(Rの真下)
・4段目: L、K(Bの下、その右)
これが選択肢(4)の縦長ジグザグ階段形である。

【選択肢の検討】
・(1) 1-4-1 で枝が同じ側 → 鎖状の連結ではない
・(2) プラス形 → 中心面に4面が直接つながる構造、本問とは異なる
・(3) 3-3オフセット → 横方向の鎖で構造が異なる
・(4) 縦長ジグザグ階段 → 正解
・(5) 横階段(2-2-2) → 横向きの階段、本問の鎖の向きと異なる

したがって、正解は(4)である。
