問1．ある管制空域において、5機の航空機A機〜E機がレーダーに映っている。各機体は4方角（北・東・南・西）と3距離圏（第1圏・第2圏・第3圏）の交点に1機ずつ存在する。各パイロットから管制官への報告を下表に示す。ただしB機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。（6点）
【レーダー図】
機体報告内容
A機「私は北方向の第2圏にいる。」
B機？（通信不良）
C機「私は南方向の第3圏にいる。」
D機「私は西方向の第1圏にいる。第1圏には私を含めて3機がいる。」
E機「私は南方向の第1圏にいる。」
ア．B機の報告がなくても、5機全ての位置が確定する。
イ．B機の報告がない場合、B機の位置として2通りが考えられる。
ウ．B機の報告がない場合、B機以外の4機の位置は全て確定する。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問1（正解：(5)）

【各機体の位置を確定する】
各パイロットの報告（B機除く）から位置を求める。
・A機の報告：「北方向の第2圏」→ A機＝（北、第2圏）と確定。
・C機の報告：「南方向の第3圏」→ C機＝（南、第3圏）と確定。
・E機の報告：「南方向の第1圏」→ E機＝（南、第1圏）と確定。
・D機の報告：「西方向の第1圏。第1圏は計3機。」
　→ D機＝（西、第1圏）と確定。
　→ 第1圏の3機はD機（西）・E機（南）・B機のいずれか。
　→ B機は第1圏に位置することが分かる。

【B機の候補を絞り込む】
B機は第1圏にいるが、方角は不明。
・西・第1圏 → D機が占有 → 不可
・南・第1圏 → E機が占有 → 不可
・北・第1圏 → 空き → 候補①
・東・第1圏 → 空き → 候補②
よってB機の候補は（北・第1圏）または（東・第1圏）の2通り。

【各命題の検討】
ア．B機の報告なしで全位置確定するか？
　→ B機の位置が2通りあるため確定しない。ア：誤り。
イ．B機の位置として2通りが考えられるか？
　→ 北・第1圏 と 東・第1圏 の2通り。イ：正しい。
ウ．B機以外の4機の位置は全て確定するか？
　→ A機・C機・D機・E機は報告から全て一意に確定する。ウ：正しい。

したがって、正解は(5)　イ、ウが正しい　である。
問2．問1と同じ条件（4方角×3距離圏、各交点に1機ずつ）で、5機A機〜E機が存在する。各パイロットの報告を下表に示す。B機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。（6点）
【レーダー図】
機体報告内容
A機「私は北方向の第3圏にいる。第3圏には私を含めて2機がいる。」
B機？（通信不良）
C機「私は南方向の第1圏にいる。第1圏には私を含めて2機がいる。」
D機「私は西方向の第3圏にいる。」
E機「私は北方向の第1圏にいる。北方向には私を含めて2機がいる。」
ア．B機の報告がない場合、B機の位置として2通りが考えられる。
イ．B機の報告がない場合、B機の位置として3通りが考えられる。
ウ．B機が「私は第2圏にいる」とのみ報告した場合、B機の位置が確定する。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問2（正解：(2)）

【各機体の位置を確定する】
各パイロットの報告（B機除く）から位置を求める。
・D機の報告：「西方向の第3圏」→ D機＝（西、第3圏）と確定。
・E機の報告：「北方向の第1圏、北方向は計2機」
　→ E機＝（北、第1圏）と確定。北方向にもう1機存在する。
・A機の報告：「北方向の第3圏、第3圏は計2機」
　→ A機＝（北、第3圏）と確定。E機の「もう1機」はA機。
　→ 第3圏の2機はA機（北）・D機（西）と確定。
・C機の報告：「南方向の第1圏、第1圏は計2機」
　→ C機＝（南、第1圏）と確定。
　→ 第1圏の2機はC機（南）・E機（北）と確定。

【B機の候補を絞り込む】
・第3圏は2機（A機・D機）→ B機は第3圏に入れない。
・第1圏は2機（C機・E機）→ B機は第1圏に入れない。
　よってB機は第2圏に存在する。
・北方向は2機（A機・E機）→ B機は北方向に入れない。
　よってB機の方角は 東・南・西 のいずれか。
・東・第2圏、南・第2圏、西・第2圏 の3通りが候補。

【各命題の検討】
ア．B機の位置として2通りが考えられるか？
　→ 候補は3通りのため誤り。ア：誤り。
イ．B機の位置として3通りが考えられるか？
　→ 東・第2圏、南・第2圏、西・第2圏 の3通り。イ：正しい。
ウ．「第2圏にいる」の報告だけでB機が確定するか？
　→ 第2圏と分かっても方角（東・南・西）が3通り残るため確定しない。ウ：誤り。

したがって、正解は(2)　イのみ正しい　である。
