問1．ある管制空域において、5機の航空機A機〜E機がレーダーに映っている。各機体は4方角（北・東・南・西）と3距離圏（第1圏・第2圏・第3圏）の交点に1機ずつ存在する。各パイロットから管制官への報告を下表に示す。ただしB機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。（6点）
【レーダー図】
機体報告内容
A機「私は北方向の第2圏にいる。」
B機？（通信不良）
C機「私は南方向の第3圏にいる。」
D機「私は西方向の第1圏にいる。第1圏には私を含めて3機がいる。」
E機「私は南方向の第1圏にいる。」
ア．B機の報告がなくても、5機全ての位置が確定する。
イ．B機の報告がない場合、B機の位置として2通りが考えられる。
ウ．B機の報告がない場合、B機以外の4機の位置は全て確定する。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問1（正解：(5)）

【各機体の位置を確定する】
各パイロットの報告（B機除く）から位置を求める。
・A機の報告：「北方向の第2圏」→ A機＝（北、第2圏）と確定。
・C機の報告：「南方向の第3圏」→ C機＝（南、第3圏）と確定。
・E機の報告：「南方向の第1圏」→ E機＝（南、第1圏）と確定。
・D機の報告：「西方向の第1圏。第1圏は計3機。」
　→ D機＝（西、第1圏）と確定。
　→ 第1圏の3機はD機（西）・E機（南）・B機のいずれか。
　→ B機は第1圏に位置することが分かる。

【B機の候補を絞り込む】
B機は第1圏にいるが、方角は不明。
・西・第1圏 → D機が占有 → 不可
・南・第1圏 → E機が占有 → 不可
・北・第1圏 → 空き → 候補①
・東・第1圏 → 空き → 候補②
よってB機の候補は（北・第1圏）または（東・第1圏）の2通り。

【各命題の検討】
ア．B機の報告なしで全位置確定するか？
　→ B機の位置が2通りあるため確定しない。ア：誤り。
イ．B機の位置として2通りが考えられるか？
　→ 北・第1圏 と 東・第1圏 の2通り。イ：正しい。
ウ．B機以外の4機の位置は全て確定するか？
　→ A機・C機・D機・E機は報告から全て一意に確定する。ウ：正しい。

したがって、正解は(5)　イ、ウが正しい　である。
問2．5機の航空機A機〜E機が、ある空港から1機ずつ、異なる時刻に出発した。下図は出発した順番を示している。各パイロットから管制官への報告を下表に示す。ただしC機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。（6点）
【出発順序】
機体報告内容
A機「私はC機より早く、E機より遅く出発した。」
B機「D機は私の直後に出発した。」
C機？（通信不良）
D機「私は最初でも最後でもない順番で出発した。」
E機「私はA機の直前に出発した。」
ア．C機の報告がなくても、5機全ての出発順が確定する。
イ．C機の報告がない場合、出発順の組み合わせとして2通りが考えられる。
ウ．C機の報告がない場合、A機が何番目に出発するかは確定する。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問2（正解：(2)）

【各報告が与える制約】
・A機：「E機より遅く、C機より早く出発」 → E < A < C　…①
・B機：「D機は私の直後」 → B出発直後にD出発（D = B + 1）　…②
・D機：「最初でも最後でもない」 → D ≠ 1番、D ≠ 5番　…③
・E機：「A機の直前に出発」 → E出発直後にA出発（A = E + 1）　…④

【出発順の候補を列挙する】
④より（E, A）は連続する2番目：(1,2)、(2,3)、(3,4)、(4,5) のいずれか。

（E,A）＝（1番, 2番）の場合：
残りの3、4、5番目はB、D、C。②③よりD=B+1かつD≠5。
→ (B,D)=(3,4)のとき C=5。①よりE(1)<A(2)<C(5)　✓　→ 有効
　　出発順: E→A→B→D→C

（E,A）＝（3番, 4番）の場合：
残りの1、2、5番目はB、D、C。②③よりD=B+1かつD≠5。
→ (B,D)=(1,2)のとき C=5。①よりE(3)<A(4)<C(5)　✓　→ 有効
　　出発順: B→D→E→A→C

その他の（E,A）の組み合わせは制約を満たさない（省略）。

【各命題の検討】
ア．全機の出発順が確定するか？
　→ 2通りの有効な順序が存在するため確定しない。ア：誤り。
イ．出発順の組み合わせが2通りか？
　→ 「E→A→B→D→C」と「B→D→E→A→C」の2通り。イ：正しい。
ウ．A機が何番目か確定するか？
　→ 順序①ではA機は2番目、順序②ではA機は4番目。確定しない。ウ：誤り。

したがって、正解は(2)　イのみ正しい　である。
