問1．次の【資料】および【表】を読み、文中の空欄（ア）〜（エ）に入る語句・数値の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。（6点）
【資料】N地方の2024年度 観光振興レポート
N地方では域内のA県〜E県（5県）について、毎年「観光入込客数」「うち宿泊客数」「観光消費額」「観光関連雇用者数」の4指標を集計し、地域経済への貢献を評価している。2024年度の集計結果によれば、5県合計の観光入込客数は（ア）万人にのぼり、地方全体の観光消費額は9,000億円となった。県別にみると、観光消費額が最も多いのは（イ）県で、地方全体の約26.7％を占めている。一方、宿泊率（宿泊客数÷観光入込客数）に着目すると、5県の中で最も宿泊率が高いのは（ウ）県（40.0％）であり、観光客の多くが日帰りではなく宿泊を伴っていることがわかる。さらに、観光関連雇用者1人あたりの観光消費額（観光消費額÷観光関連雇用者数）は、地方全体で約（エ）万円に達した。

問1(正解:(1))

【表】の数値を用いて、資料文中の各空欄に入る値を順に確定する。

【各空欄の確定】
・（ア）5県の観光入込客数の合計：1,200 + 800 + 1,500 + 600 + 900 = 5,000万人（表の合計欄と一致）。
・（イ）観光消費額が最も多い県：A県 2,400億円（B 1,800、C 1,800、D 1,200、E 1,800）。2,400 ÷ 9,000 ≒ 0.267（約26.7％）で資料文の記述と整合 → A県。
・（ウ）宿泊率（宿泊客数÷観光入込客数）：
　　A 300/1,200 = 25.0％、B 320/800 = 40.0％、C 300/1,500 = 20.0％、D 210/600 = 35.0％、E 180/900 = 20.0％
　　最大はB県（40.0％）。
・（エ）観光関連雇用者1人あたり観光消費額：
　　9,000億円 ÷ 200千人 = 9,000×10⁸ ÷ 2×10⁵ = 4,500,000円 = 450万円。

【選択肢の検討】
・(1) ア=5,000、イ=A、ウ=B、エ=450 → すべて整合 → 正しい
・(2) イ=Cは誤り（C県の観光消費額は1,800億円でA県2,400億円より少ない）。
・(3) ウ=Dは誤り（D県の宿泊率は35.0％。最高はB県の40.0％）。
・(4) ア=1,310は宿泊客数の合計であり、観光入込客数の合計ではない。
・(5) エ=45は単位の取り違え（9,000億÷200千人＝450万円が正しい。45は20万人で割った場合の値ではなく、計算ミス）。

したがって、正解は(1)である。
問2．次の【資料】および【表】を読み、文中の空欄（ア）〜（オ）に入る数値・語句の組合せとして正しいものを、次ページの(1)〜(5)のうちから1つ選べ。（6点）
【資料】Y社 統合業績レポート（2023〜2025年度）
食品メーカーY社は「製造」「販売」「物流」の3部門で事業を展開しており、2023〜2025年度の3年間にわたって部門別の業績を集計している。2025年度の全社売上高は（ア）億円であり、2023年度の合計2,500億円から3年間で約（イ）％増加した（成長率＝（2025年度－2023年度）÷2023年度×100、小数第2位を四捨五入）。
部門別にみると、2025年度の営業利益率（営業利益÷売上高×100）は製造16.4％、販売14.4％、物流12.5％であり、最も高いのは（ウ）部門である。一方、2025年度に従業員1人あたり営業利益（営業利益÷従業員数）が最も高いのは（エ）部門であり、その値は約3,939万円に相当する（1億円＝10,000万円）。なお、3部門合計の従業員数は2023年度の1,100人から2025年度の（オ）人へと増加した。

問2(正解:(5))

【表】の値を用いて、資料文中の各空欄を順に確定する。

【各空欄の確定】
・（ア）2025年度の全社売上高：1,400 + 900 + 600 = 2,900億円。
・（イ）2023→2025年度の成長率：(2,900 − 2,500) ÷ 2,500 × 100 = 16.0％ → 16.0％。
・（ウ）2025年度の営業利益率：
　　製造 230÷1,400 = 16.43％ ≒ 16.4％、販売 130÷900 = 14.44％ ≒ 14.4％、物流 75÷600 = 12.5％。
　　最大は製造部門。
・（エ）2025年度の1人あたり営業利益（万円換算）：
　　製造 230億÷640人 ≒ 0.3594億/人 ≒ 3,594万円
　　販売 130億÷330人 ≒ 0.3939億/人 ≒ 3,939万円
　　物流 75億÷220人 ≒ 0.3409億/人 ≒ 3,409万円
　　最大は販売部門。
・（オ）2025年度の3部門合計従業員数：640 + 330 + 220 = 1,190人。

【選択肢の検討】
・(1) ア=2,500は2023年度の合計値で、2025年度の値ではない → 誤り
・(2) イ=13.8は (2,900−2,500)÷2,900×100 ≒ 13.8％。分母を2025年度にした計算ミス → 誤り
・(3) ウ=販売は誤り（2025年度の営業利益率最高は製造16.4％）。
・(4) オ=90は2023→2025年度の従業員「増加数」（1,190−1,100）であり、2025年度の従業員数（1,190人）ではない → 誤り
・(5) ア=2,900、イ=16.0、ウ=製造、エ=販売、オ=1,190 → すべて整合 → 正しい

したがって、正解は(5)である。
