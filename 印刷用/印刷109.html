問1．次の展開図を組み立てて立方体を作るとき、面Eと向かい合う面の記号はどれか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
(1) A
(2) C
(3) B
(4) D
(5) F
問1(正解:(2))

【展開図の組み立て】
面Cを前面（基面）として考える。
・面A は Cの上に接続 → 上面
・面F は Cの下に接続 → 下面
・面B は Cの左に接続 → 左面
・面D は Cの右に接続 → 右面
・面E は Dの右にさらに接続 → Dを右側面として折り畳むと、EはさらにDの右側で折り曲がって立方体の背面に回り込む

したがって、組み立てた立方体の各面対応は次のようになる。
・前面＝C　背面＝E
・上面＝A　下面＝F
・左面＝B　右面＝D

【対面の特定】
立方体において「前面」と「背面」は対面関係にある。
よって、面Eと向かい合う面は面C である。

【選択肢の検討】
・(1) A → 上面。Eと隣接しており対面ではない。誤り
・(2) C → 前面。Eの対面（背面と前面の関係）。正解
・(3) B → 左面。Eと隣接しており対面ではない。誤り
・(4) D → 右面。Eは展開図上でDの右に連結するがDと隣接するため対面ではない。誤り
・(5) F → 下面。Eと隣接しており対面ではない。誤り

したがって、正解は(2)である。
問2．立方体の展開図にアルファベットを図のように記入する。この立方体を組み立て、図のように鏡をおく。目の方向からみると鏡に映る立方体はどのように見えるか。正しいものを1つ選び、解答用紙にマークせよ。（6点）
問2(正解:(2))

【展開図から立方体の組み立て】
展開図を組み立てると、各面は次のように対応する。
・N = 前面、V = 背面（NとVは対面）
・K = 左面、M = 右面（KとMは対面）
・T = 上面、U = 下面（TとUは対面）

【鏡像の原理】
鏡は立方体の左側に置かれ、目は右側にある。目は鏡に映った立方体の像を見る。

(a) 見える面の変化: 鏡は立方体の左面に面しているため、鏡像には前面(N)・上面(T)に加えて、実物の左面(K)が映る。右面(M)ではなく左面(K)が映ることに注意。

(b) 文字の反転: 鏡に映ると、各面の文字は左右反転（鏡文字）になる。T・N・K のすべてが反転する。

【選択肢の検討】
・(1) 側面にMの鏡文字 → 左面Kではなく右面Mが表示されており誤り
・(2) 上面に反転T、前面に反転N、側面に反転K → 正解
・(3) 前面のNが反転されていない → 鏡像では全ての文字が反転するため誤り
・(4) 上面のTが反転されていない → 鏡像では全ての文字が反転するため誤り
・(5) 全ての文字が反転されていない → 鏡像の原理に反するため誤り

したがって、正解は(2)である。
