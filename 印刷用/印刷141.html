問1．次の「倉庫在庫管理月次レポート」を読み、空欄（ア）に当てはまる数値として正しいものを(1)〜(5)から1つ選べ。（6点）
倉庫在庫管理月次レポート（2026年3月分）

当倉庫では商品A〜Dの4品目を管理している。当月末時点の在庫数は以下の通りである。
・4品目の合計在庫数は1,200個であった。
・商品Aの在庫数は商品Bの在庫数の2倍であった。
・商品Cの在庫数は商品Aの在庫数より80個少ない（ア）個であった。
・商品Dの在庫数は商品Bと商品Cの在庫数の合計と等しかった。
(1) 260個  (2) 280個  (3) 240個  (4) 220個  (5) 300個
問1（正解: (1)）

商品BをxとするとA=2x、C=2x−80、D=x+(2x−80)=3x−80。
合計: 2x＋x＋(2x−80)＋(3x−80) = 8x−160 = 1,200 → x=170

【各商品の在庫数】
・商品A = 2×170 = 340個
・商品B = 170個
・商品C = 340−80 = 260個（= ア）
・商品D = 170＋260 = 430個
合計確認: 340＋170＋260＋430 = 1,200個 ✓

【選択肢の検討】
・(2)280個: A=360, D=450 → 合計1,260 ≠ 1,200 → 不正解
・(3)240個: A=320, D=410 → 合計1,140 ≠ 1,200 → 不正解
・(4)220個: A=300, D=390 → 合計1,080 ≠ 1,200 → 不正解
・(5)300個: A=380, D=470 → 合計1,350 ≠ 1,200 → 不正解

したがって正解は(1)の260個である。
問2．次の「月間商品販売実績レポート」を読み、空欄（ウ）に当てはまる数値として正しいものを(1)〜(5)から1つ選べ。（6点）
月間商品販売実績レポート（2026年3月分）

当月は商品P〜Tの5品目の販売実績をまとめた。
・5品目の月間販売合計は500個であった。
・商品Pの販売数は商品Qの販売数より30個多かった。
・商品Qの販売数は商品Rの販売数の1.5倍であった。
・商品Rと商品Sの販売数の合計は160個であった。
・商品Sの販売数は商品Tの販売数の2倍であった。
・商品Tは（ウ）個販売された。
(1) 26個  (2) 28個  (3) 30個  (4) 34個  (5) 40個
問2（正解: (4)）

商品TをtとするとS=2t、R=160−2t、Q=1.5×(160−2t)=240−3t、P=Q+30=270−3t。
合計: (270−3t)＋(240−3t)＋(160−2t)＋2t＋t = 670−5t = 500 → t=34

【各商品の販売数】
・商品T = 34個（= ウ）
・商品S = 2×34 = 68個
・商品R = 160−68 = 92個
・商品Q = 1.5×92 = 138個
・商品P = 138＋30 = 168個
合計確認: 168＋138＋92＋68＋34 = 500個 ✓

【解法のポイント】
各商品の関係式をTの変数tで統一表現し、合計式でtを求める。
直接Tから遡れないため、全体合計を方程式の軸とすることが鍵となる。

したがって正解は(4)の34個である。
