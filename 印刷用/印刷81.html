問1．9個のマス目がついた透明なシートがある。マス目の中には記号がかかれている。下の例のように、2枚のシートを重ねると、両方の記号が見える。（6点）
【例】2枚のシートの重ね合わせ
次のアからオまでのシートのうち3枚を重ねたところ、下の目標パターンのように見えた。重ねたシートの組み合わせは次のうちどれか。正しいものを1つ選び、解答用紙にマークせよ。ただし、シートは回転させないものとする。
【シート一覧】
【目標パターン】
(1) アとイとウ　　(2) アとウとエ　　(3) イとウとオ
(4) アとエとオ　　(5) イとエとオ
問1(正解:(1))

【解法】
各シートの記号配置を確認し、3枚を重ねて目標パターンが再現できる組み合わせを探す。

【目標パターンの分析】
目標パターンには9つの記号がある：
・1行目：●（黒丸）、□（白四角）、▲（黒三角）
・2行目：◇（白ひし形）、○（白丸）、■（黒四角）
・3行目：△（白三角）、▼（逆黒三角）、●（黒丸）

【各シートの確認】
・シートア：●(左上)、○(中央)、●(右下) → 3個
・シートイ：□(上中)、◇(左中)、△(左下) → 3個
・シートウ：▲(右上)、■(右中)、▼(下中) → 3個
・シートエ：▲(右上)、○(中央)、△(左下) → 3個
・シートオ：●(左上)、□(上中)、■(右中)、▼(下中) → 4個

【検証】
ア＋イ＋ウを重ねると：
・アの●(左上) + イの□(上中) + ウの▲(右上) → 1行目が完成
・イの◇(左中) + アの○(中央) + ウの■(右中) → 2行目が完成
・イの△(左下) + ウの▼(下中) + アの●(右下) → 3行目が完成
すべてのセルが一致し、記号の衝突もない。

他の選択肢は、目標パターンの全記号を再現できないか、同じセルに異なる記号が重なる。

したがって、正解は(1)である。
問2．9個のマス目がついた透明なシートがある。マス目の中には記号がかかれている。下の例のように、シートは回転させることができ、回転後に重ねると異なるパターンが得られる。ただし、裏返すことはできないものとする。（6点）
【例】回転と重ね合わせ
次のアからオまでのシートのうち3枚を選び、必要に応じて回転させて重ねたところ、下の目標パターンのように見えた。重ねたシートの組み合わせは次のうちどれか。正しいものを1つ選び、解答用紙にマークせよ。
【シート一覧】
【目標パターン】
(1) アとウとエ　　(2) イとウとオ　　(3) アとエとオ
(4) ウとエとオ　　(5) アとイとウ
問2(正解:(5))

【解法】
各シートを0°、90°、180°、270°に回転させた場合の記号配置を検討し、3枚の組み合わせで目標パターンを再現する。

【目標パターンの分析】
・1行目：■（黒四角）、▲（黒三角）、◇（白ひし形）
・2行目：△（白三角）、空白、○（白丸）
・3行目：●（黒丸）、□（白四角）、空白

【各シートの検討】
・シートア（回転なし）：■(左上)、○(右中) → 2個
・シートイ（回転なし）：▲(左中)、●(右下) → 2個
　→ 90°回転すると：▲(上中)、●(左下) → 目標の位置に一致
・シートウ（回転なし）：◇(右上)、△(左中)、□(下中) → 3個

【検証：ア(0°) ＋ イ(90°回転) ＋ ウ(0°)】
シートイを90°時計回りに回転させると：
・元の▲(左中) → 回転後▲(上中)
・元の●(右下) → 回転後●(左下)

3枚を重ねると：
・アの■(左上) + イ(90°)の▲(上中) + ウの◇(右上) → 1行目完成
・ウの△(左中) + 空白(中央) + アの○(右中) → 2行目完成
・イ(90°)の●(左下) + ウの□(下中) + 空白(右下) → 3行目完成
すべてのセルが一致し、衝突もない。

他の選択肢はいかなる回転の組み合わせでも目標パターンを再現できない。

したがって、正解は(5)である。
