問1．下図は立方体の展開図であり、各面にア〜カの記号が書かれている。展開図上の頂点Pに黒丸（●）の印がつけられている。この展開図を折り曲げて立方体を組み立てたとき、頂点Pに集まる3つの面の記号の組み合わせとして正しいものを、次の(1)〜(5)の中から1つ選べ。（6点）
【展開図】
(1)　ア, オ, イ
(2)　ウ, エ, イ
(3)　ア, エ, イ
(4)　ア, ウ, イ
(5)　ア, ウ, カ
問1(正解:(4))

【展開図の折り畳み分析】
展開図の中心にある「エ」を正面として立方体を組み立てると、各面の配置は次のようになる。

・エ → 正面
・イ（エの上） → 上面
・カ（エの下） → 底面
・ウ（エの左） → 左側面
・オ（エの右） → 右側面
・ア（ウの左） → 背面

【対面関係】
・エ ↔ ア（正面 ↔ 背面）
・イ ↔ カ（上面 ↔ 底面）
・ウ ↔ オ（左側面 ↔ 右側面）

【頂点Pの追跡】
展開図上でPはイの左上角にある。イが上面に折り畳まれるとき、イの下辺（エとの境界）が上面の手前側になり、イの上辺が上面の奥側になる。またイの左辺がウ側（左側面側）になる。

したがって、Pはイの左上角 → 上面の奥・左の頂点 → 背面・左側面・上面が集まる頂点に対応する。

よって、頂点Pに集まる3面は「ア（背面）、ウ（左側面）、イ（上面）」である。

【選択肢の検討】
・(1) ア, オ, イ → 背面・右側面・上面が集まる頂点（奥・右・上の角）。Pとは異なる頂点。
・(2) ウ, エ, イ → 左側面・正面・上面が集まる頂点（手前・左・上の角）。Pとは異なる頂点。
・(3) ア, エ, イ → アとエは対面関係にあり、同じ頂点に集まることは不可能。
・(4) ア, ウ, イ → 背面・左側面・上面が集まる頂点。Pと一致する。正解。
・(5) ア, ウ, カ → 背面・左側面・底面が集まる頂点（奥・左・下の角）。Pとは異なる頂点。

したがって、正解は(4)である。
問2．下図は立方体の展開図であり、各面にア〜カの記号が書かれている。展開図上の4つの頂点にP, Q, R, Sの印がつけられている。この展開図を折り曲げて立方体を組み立てたとき、Pと同じ頂点に重なる印として正しいものを、次の(1)〜(5)の中から1つ選べ。（6点）
【展開図】
(1)　Qのみ
(2)　Sのみ
(3)　Rのみ
(4)　QとR
(5)　RとS
問2(正解:(3))

【展開図の折り畳み分析】
このT字型展開図を「エ」を正面として組み立てると、各面の配置は次のようになる。

・エ → 正面
・イ（エの上） → 上面
・ア（イの左） → 左側面（上面の左辺から折り下げ）
・ウ（イの右） → 右側面（上面の右辺から折り下げ）
・オ（エの下） → 底面
・カ（オの下） → 背面（底面の奥辺から折り上げ）

【対面関係】
・エ ↔ カ（正面 ↔ 背面）
・イ ↔ オ（上面 ↔ 底面）
・ア ↔ ウ（左側面 ↔ 右側面）

【各マークの追跡】

P（アの左上角）:
アは上面（イ）の左辺から折り下げて左側面になる。アの左上角は、折り畳み後に背面・左側面・底面が集まる頂点（奥・左・下の角）に移動する。
→ 3D座標: (0, 1, 0)

Q（イの右上角 = ウの左上角）:
イは上面。イの右上角は上面の奥・右の頂点。これは背面・右側面・上面が集まる頂点に対応する。
→ 3D座標: (1, 1, 1)

R（オの左下角 = カの左上角）:
オは底面。オの下辺（カとの境界）が背面側になるため、オの左下角は背面・左側面・底面が集まる頂点に移動する。
→ 3D座標: (0, 1, 0) → Pと同じ頂点!

S（カの右下角）:
カは背面。カの下辺は底面から折り上げた奥側。カの右下角は背面・右側面・上面が集まる頂点に対応する。
→ 3D座標: (1, 1, 1) → Qと同じ頂点（Pとは異なる）

【結論】
PとRは同じ頂点(0,1,0)に集まり、QとSは別の頂点(1,1,1)に集まる。
Pと同じ頂点に重なるのはRのみ。

したがって、正解は(3)である。
