問1．図は、ある自治体のスポーツ施設における年間利用者の年代の内訳を性別に示したものである。30代の利用者数は、利用者数全体のおよそ何%か。正しいものを1つ選び、解答用紙にマークせよ。（6点）
【性別・年代別スポーツ施設利用者数】
（架空の自治体統計データより作成）
(1) 27.3%　　(2) 28.9%　　(3) 29.5%　　(4) 30.6%　　(5) 33.8%
問1(正解:(4))

【解法】
30代の利用者数を男女それぞれ計算し、合計する。

【計算過程】
・男性の30代: 8,640 x 32% = 8,640 x 0.32 = 2,764.8人
・女性の30代: 3,360 x 27% = 3,360 x 0.27 = 907.2人
・30代合計: 2,764.8 + 907.2 = 3,672人

・全体に占める割合: 3,672 / 12,000 x 100 = 30.6%

【選択肢の検討】
・(1) 27.3% → 不正解
・(2) 28.9% → 不正解
・(3) 29.5% → 不正解
・(4) 30.6% → 正解
・(5) 33.8% → 不正解

したがって、正解は(4)である。
問2．下の表は、5つの地域（P～T）における3種類の農作物（米、野菜、果物）の出荷額を示したものである。次の記述ア～オのうち、正しいものの組み合わせを1つ選び、解答用紙にマークせよ。（6点）
【地域別農作物出荷額】
地域 米（億円） 野菜（億円） 果物（億円） 合計（億円）
P 85 120 45 250
Q 150 90 60 300
R 70 200 80 350
S 110 75 165 350
T 95 140 65 300
（架空の統計データより作成）
ア．米の出荷額が最も大きい地域は、合計出荷額でも最大である。
イ．果物の出荷額が合計出荷額の40%を超える地域がある。
ウ．野菜の出荷額の5地域合計は、米の出荷額の5地域合計より大きい。
エ．合計出荷額に占める米の割合は、どの地域でも30%以上である。
オ．果物の出荷額が最も小さい地域は、野菜の出荷額も最も小さい。
(1) ア、エ　　(2) ア、オ　　(3) イ、エ　　(4) ウ、オ　　(5) イ、ウ
問2(正解:(5))

【各記述の検討】

ア．米の出荷額が最も大きい地域は、合計出荷額でも最大である。
・米の最大: Q（150億円）
・合計の最大: R、S（ともに350億円）
・一致しないため、正しくない

イ．果物の出荷額が合計出荷額の40%を超える地域がある。
・P: 45/250 = 18.0%
・Q: 60/300 = 20.0%
・R: 80/350 = 22.9%
・S: 165/350 = 47.1% → 40%を超える
・T: 65/300 = 21.7%
・Sが40%を超えるため、正しい

ウ．野菜の出荷額の5地域合計は、米の出荷額の5地域合計より大きい。
・野菜合計: 120 + 90 + 200 + 75 + 140 = 625億円
・米合計: 85 + 150 + 70 + 110 + 95 = 510億円
・625 > 510 のため、正しい

エ．合計出荷額に占める米の割合は、どの地域でも30%以上である。
・R: 70/350 = 20.0% → 30%未満
・30%未満の地域があるため、正しくない

オ．果物の出荷額が最も小さい地域は、野菜の出荷額も最も小さい。
・果物最小: P（45億円）
・Pの野菜: 120億円（最小はSの75億円）
・一致しないため、正しくない

正しい記述はイとウであるから、正解は(5)である。
