問1．標準的なサイコロ（向かい合う面の数字の和が7のもの）を水平な机の上に置き、上面が1、南向きの面が2、東向きの面が3となるようにする。このサイコロを滑らないように次の順序で転がしたとき、最終的に上面に現れる数字はどれか。なお、各ステップでサイコロは1マス分（一辺分）だけその方向に倒れるものとする。（6点）
① 東へ1回 → ② 北へ1回 → ③ 東へ1回 → ④ 南へ1回
（問題文には図はない。各自で初期状態のサイコロを想定し、上面・底面・南北・東西の6面の数字を書き出して、転がす操作を順に追って解答すること。）
問1(正解:(3))
サイコロは図示されていないため、まず初期状態の6面を全て書き出すことが要点である。

【初期状態の整理】
標準サイコロは対面の和が7。上面=1、南=2、東=3と与えられているので、
・底=6（1の対面）
・北=5（2の対面）
・西=4（3の対面）
となる。

【転がし操作の規則】
サイコロを「東」に倒すと、上→東、東→底、底→西、西→上 と4面が時計回りにずれ、南北は不変。「北」に倒すと、上→北、北→底、底→南、南→上 と4面がずれ、東西は不変。逆方向は逆回転で同様。

【各ステップの追跡】
初期：上1・底6・南2・北5・東3・西4
①東へ1回：上=旧西=4、東=旧上=1、底=旧東=3、西=旧底=6（南北不変）→ 上4・底3・南2・北5・東1・西6
②北へ1回：上=旧南=2、北=旧上=4、底=旧北=5、南=旧底=3（東西不変）→ 上2・底5・南3・北4・東1・西6
③東へ1回：上=旧西=6、東=旧上=2、底=旧東=1、西=旧底=5（南北不変）→ 上6・底1・南3・北4・東2・西5
④南へ1回：上=旧北=4、南=旧上=6、底=旧南=3、北=旧底=1（東西不変）→ 上4・底3・南6・北1・東2・西5

【検算】
最終状態でも対面の和はすべて7：上4+底3=7、南6+北1=7、東2+西5=7 ✓

【選択肢の検討】
・(1) 2 → ③の操作を省略した場合の上面（誤）
・(2) 3 → ②③の操作を省略した場合の上面（誤）
・(3) 4 → 4手順を正しく追跡した結果（正解）
・(4) 5 → 北と南を取り違えた場合の値（誤）
・(5) 6 → ④を行わない場合の上面（誤）

したがって、正解は(3)である。
問2．標準的なサイコロ（向かい合う面の数字の和が7のもの）を水平な机の上に置き、上面が1、南向きの面が2、東向きの面が3となるようにする。このサイコロを滑らないように次の順序で7回転がしたとき、最終的な「上面の数字」と「東向きの面の数字」の組み合わせとして正しいものはどれか。なお、各ステップでサイコロは1マス分（一辺分）だけその方向に倒れるものとする。（6点）
① 東へ1回 → ② 東へ1回 → ③ 北へ1回 → ④ 西へ1回
→ ⑤ 南へ1回 → ⑥ 南へ1回 → ⑦ 東へ1回
（問題文には図はない。各自で初期状態の6面を書き出し、各転がし操作で上・底・東西・南北のうちどの4面が入れ替わるかを整理してから順に追うこと。）
問2(正解:(3))
問1と同じ初期状態（上1・底6・南2・北5・東3・西4）から、7回の操作を順に追跡する。

【各ステップの追跡】
初期：上1・底6・南2・北5・東3・西4
①東：上=西=4、東=旧上=1、底=旧東=3、西=旧底=6 → 上4・底3・南2・北5・東1・西6
②東：上=西=6、東=旧上=4、底=旧東=1、西=旧底=3 → 上6・底1・南2・北5・東4・西3
③北：上=南=2、北=旧上=6、底=旧北=5、南=旧底=1 → 上2・底5・南1・北6・東4・西3
④西：上=東=4、西=旧上=2、底=旧西=3、東=旧底=5 → 上4・底3・南1・北6・東5・西2
⑤南：上=北=6、南=旧上=4、底=旧南=1、北=旧底=3 → 上6・底1・南4・北3・東5・西2
⑥南：上=北=3、南=旧上=6、底=旧南=4、北=旧底=1 → 上3・底4・南6・北1・東5・西2
⑦東：上=西=2、東=旧上=3、底=旧東=5、西=旧底=4 → 上2・底5・南6・北1・東3・西4

【最終状態】
上=2、東=3、底=5、西=4、南=6、北=1
対面の和：上2+底5=7、南6+北1=7、東3+西4=7 ✓

【選択肢の検討】
・(1) 上=1, 東=4 → ②段階で操作を止めた場合の対称形（誤）
・(2) 上=3, 東=5 → ⑦の最後の東を行わない（6回で止めた）場合（誤）
・(3) 上=2, 東=3 → 7手順すべて正しく追跡した結果（正解）
・(4) 上=6, 東=5 → ⑥⑦を省略した場合（誤）
・(5) 上=4, 東=5 → ⑤⑥⑦を省略した場合（誤）

したがって、正解は(3)である。
