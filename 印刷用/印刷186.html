問1．標準的なサイコロ（向かい合う面の数字の和が7のもの）を水平な机の上に置き、上面が1、南向きの面が2、東向きの面が3となるようにする。このサイコロを滑らないように次の順序で転がしたとき、最終的に上面に現れる数字はどれか。なお、各ステップでサイコロは1マス分（一辺分）だけその方向に倒れるものとする。（6点）
① 東へ1回 → ② 北へ1回 → ③ 東へ1回 → ④ 南へ1回
（問題文には図はない。各自で初期状態のサイコロを想定し、上面・底面・南北・東西の6面の数字を書き出して、転がす操作を順に追って解答すること。）
問1(正解:(3))
サイコロは図示されていないため、まず初期状態の6面を書き出すことが要点である。
【初期状態】 対面の和は7なので、上=1・南=2・東=3 から 底=6・北=5・西=4。
【転がしの規則】 「東」に倒すと 上→東→底→西→上 の4面が回り、南北は不変。「北」に倒すと 上→北→底→南→上 の4面が回り、東西は不変。逆方向は逆回り。
【各ステップの追跡】（数字は 上・底・南・北・東・西 の順）
初期：1・6・2・5・3・4
①東：上=旧西=4 → 4・3・2・5・1・6
②北：上=旧南=2 → 2・5・3・4・1・6
③東：上=旧西=6 → 6・1・3・4・2・5
④南：上=旧北=4 → 4・3・6・1・2・5
【検算】 対面の和：上4+底3=7・南6+北1=7・東2+西5=7 ✓
【選択肢の検討】
・(1)2＝③省略時／(2)3＝②③省略時／(4)5＝南北取り違え／(5)6＝④省略時（いずれも誤）
・(3)4＝4手順を正しく追跡した結果（正解）
したがって、正解は(3)である。
問2．標準的なサイコロ（向かい合う面の数字の和が7のもの）を水平な机の上に置き、上面が1、南向きの面が2、東向きの面が3となるようにする。このサイコロを滑らないように次の順序で7回転がしたとき、最終的な「上面の数字」と「東向きの面の数字」の組み合わせとして正しいものはどれか。なお、各ステップでサイコロは1マス分（一辺分）だけその方向に倒れるものとする。（6点）
① 東へ1回 → ② 東へ1回 → ③ 北へ1回 → ④ 西へ1回
→ ⑤ 南へ1回 → ⑥ 南へ1回 → ⑦ 東へ1回
（問題文には図はない。各自で初期状態の6面を書き出し、各転がし操作で上・底・東西・南北のうちどの4面が入れ替わるかを整理してから順に追うこと。）
問2(正解:(3))
問1と同じ初期状態から、7回の操作を順に追跡する（数字は 上・底・南・北・東・西 の順）。
初期：1・6・2・5・3・4
①東：上=旧西=4 → 4・3・2・5・1・6
②東：上=旧西=6 → 6・1・2・5・4・3
③北：上=旧南=2 → 2・5・1・6・4・3
④西：上=旧東=4 → 4・3・1・6・5・2
⑤南：上=旧北=6 → 6・1・4・3・5・2
⑥南：上=旧北=3 → 3・4・6・1・5・2
⑦東：上=旧西=2 → 2・5・6・1・3・4
【最終状態】 上=2・東=3（底5・西4・南6・北1）。対面の和：2+5=7・6+1=7・3+4=7 ✓
【選択肢の検討】
・(1)上1東4＝②段階の対称形／(2)上3東5＝⑦省略（6回で停止）／(4)上6東5＝⑥⑦省略／(5)上4東5＝⑤⑥⑦省略（いずれも誤）
・(3)上2東3＝7手順を正しく追跡した結果（正解）
したがって、正解は(3)である。
