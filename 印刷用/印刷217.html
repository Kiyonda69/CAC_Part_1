問1．積み木を組んでできた立体を真上から見て、各マスに積み上げた立方体の段数を記したものが【図1】である。0は積み木が無いことを表す。この立体の右側に鏡を垂直に立てて映すとき、鏡に映った立体を同じように真上から見た図として正しいものを、(1)〜(5)から一つ選べ。（6点）
【図1】立体を真上から見た図と鏡の位置
【選択肢】鏡に映った立体を真上から見た図
問1(正解:(4))
鏡を立体の右側に立てて映すと、左右が反転する。真上から見た図では、各行の数字をそのまま左右逆順に並べ替えればよい。

元の図（各行）：
3 1 0 ／ 2 0 1 ／ 1 2 2
左右反転（鏡像）：
0 1 3 ／ 1 0 2 ／ 2 2 1
これは選択肢(4)と一致する。

【選択肢の検討】
・(1) 元の図のまま。反転していないので誤り。
・(2) 上下も左右も入れ替えた180度回転。鏡像ではない。
・(3) 行と列を入れ替えた転置（対角線対称）。左右の鏡像とは異なる。
・(4) 各行を左右逆にした図。これが鏡像であり正解。
・(5) 行の並びだけを逆にした上下反転（前後方向の鏡像）。右側の鏡では起こらない。

したがって、正解は(4)である。
問2．【図2】は、積み木を組んでできた立体を真上から見て各マスの段数を記したものである。黒く塗ったマスには、目印のついた立方体が1段だけ置かれている。この立体の右側に鏡を垂直に立てて映すとき、鏡に映った立体を真上から見た図として正しいものを、(1)〜(5)から一つ選べ。（6点）
【図2】立体を真上から見た図（黒マス＝目印）と鏡の位置
【選択肢】鏡に映った立体を真上から見た図
問2(正解:(1))
問1と同じく、右側の鏡では左右が反転する。数字の配置を各行で左右逆順にすると同時に、黒い目印のマスも左右反転で列が移動する。

元の図：2 0 3 1 ／ 1 3 0 2 ／ 0 2 1 3 ／ 3 1 2 0
目印は上から2行目・左端（左から1列目）にある。
左右反転後：1 3 0 2 ／ 2 0 3 1 ／ 3 1 2 0 ／ 0 2 1 3
目印は上から2行目・右端（右から1列目）へ移る。
これは選択肢(1)と一致する。

【選択肢の検討】
・(1) 数字配置・目印の位置がともに左右反転と一致。正解。
・(2) 元の図のまま。反転していない。
・(3) 180度回転。目印が反対側の行・列へ移っており誤り。
・(4) 行と列を入れ替えた転置。配置が一致しない。
・(5) 上下反転。目印が下方の行へ移っており、右側の鏡では起こらない。

数字配置と目印の位置の両方が一致するのは(1)のみ。よって正解は(1)である。
