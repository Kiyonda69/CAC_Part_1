問1．以下の説明を読み、与えられた状況の表示の変化の直後に実施する適切な操作として正しいものを1つ選び、解答用紙にマークせよ。（6点）
大型貨物船の船橋（ブリッジ）には、船体の前後方向の傾き（トリム）を監視するための機械式傾斜指示装置が設置されている。この装置は、横一列に並んだ4つの円形パネルで構成されている。各パネルの内部には錘のついた振り子が収められており、船首が沈む方向に傾くと振り子が動いてパネルの表面に暗色の印が現れ（表示：●）、傾きが解消されると明色に戻る（表示：○）仕組みである。この装置は純粋に機械的な構造で動作し、電力を必要としない。4つのパネルはそれぞれ反応する傾斜角度の閾値が異なっており、左端のパネルが最も敏感でわずかな船首沈下でも暗色を示し、右に行くほど閾値が高くなるため、大きく傾かないと暗色にならない。したがって、船首が沈む方向に傾くほど、左から順に暗色のパネルが増えていく。貨物船はプロペラ効率と操舵性能の観点から、船首がわずかに沈んだ状態で航行するのが最適とされており、左から2つが暗色の「●●○○」が適正トリムの表示である。「○○○○」は船尾が沈みすぎた状態、「●●●●」は船首が沈みすぎた状態を意味する。パネルの各表示は1段階の変化に対応しており、バラストタンク（船体に海水を注排水する区画）の1回の操作で1段階のトリム変化が生じる。
【状況】
【バラスト操作】
ア：船首バラストタンクから排水する
イ：船尾バラストタンクから排水する
ウ：船首・船尾の両方のバラストタンクに注水する
エ：船首バラストタンクに注水し、船尾バラストタンクから排水する
オ：船尾バラストタンクに注水し、船首バラストタンクから排水する
(1) ア　　　　(2) エ　　　　(3) オ　　　　(4) ウ　　　　(5) イ
問1(正解:(3))

【表示の読み取り】
・4つのパネルは左端が最も敏感で、船首沈下が大きくなるほど左から順に暗色（●）が増える
・適正トリム「●●○○」はレベル2（左2つが暗色）
・変化後「●●●●」はレベル4（全て暗色）で、船首が大きく沈んだ危険な状態
・適正に戻すにはトリムを2段階改善する（船首を浮かせる方向へ）必要がある

【各操作の効果】
・ア：船首バラスト排水 → 船首が軽くなり1段階改善 → ●●●○（不十分）
・イ：船尾バラスト排水 → 船尾が軽くなりさらに船首側へ傾く → 悪化
・ウ：両方注水 → 全体が重くなるが前後バランスは変わらない → ●●●●のまま
・エ：船首注水＋船尾排水 → 船首がさらに重く、船尾がさらに軽く → 大幅悪化
・オ：船尾注水＋船首排水 → 船首が軽く（−1）かつ船尾が重く（−1）で合計2段階改善 → ●●○○

【ポイント】
・2段階の補正が必要なため、1方向のみの操作（ア：−1段階）では不十分
・オは「排水」と「注水」を逆方向に同時に行うため2段階分の効果がある

したがって、正解は(3)である。
問2．以下の説明を読み、与えられた状況の表示の変化の直後に実施する適切な操作の組み合わせとして正しいものを1つ選び、解答用紙にマークせよ。（6点）
ある化学プラントの反応炉には、炉内の温度と圧力をそれぞれ監視する指示灯が設置されている。温度指示灯と圧力指示灯はそれぞれ4つの円形パネルが横一列に並んだ構造で、いずれも測定値が各パネルの閾値を超えると暗色（●）を示し、下回ると明色（○）を示す。左端のパネルが最も閾値が低く、右に行くほど閾値が高い。温度・圧力ともに「●●○○」が適正範囲である。オペレーターは以下の5つの操作から同時に2つまで実行でき、効果は加算される。温度と圧力がともに適正表示に戻る操作の組み合わせを選べ。
操作 内容 温度への影響 圧力への影響
ア 加熱ヒーターを起動する 1段階上昇 1段階上昇
イ 冷却水を注入する 1段階下降 1段階下降
ウ 排気弁を開く 変化なし 1段階下降
エ 触媒を追加投入する 1段階上昇 変化なし
オ 撹拌速度を上げる 1段階下降 1段階上昇
【状況】
(1) ア、ウ　　　(2) イ、オ　　　(3) ア、エ　　　(4) エ、イ　　　(5) エ、ウ
問2(正解:(5))

【状況の分析】
・温度: ●●○○ → ●○○○（1段階低下、低い側に逸脱）→ 温度を1段階上げる必要がある
・圧力: ●●○○ → ●●●○（1段階上昇、高い側に逸脱）→ 圧力を1段階下げる必要がある
・必要な変化: (温度 +1, 圧力 -1)

【単独操作の検討】
・ア(温度+1, 圧力+1)、イ(温度-1, 圧力-1)、オ(温度-1, 圧力+1) → 温度と圧力が同方向または逆方向に連動し、片方だけ補正できない
・ウ(温度±0, 圧力-1) → 圧力は下がるが温度が戻らない
・エ(温度+1, 圧力±0) → 温度は上がるが圧力が戻らない
・単独操作では両方を同時に適正に戻せない

【各選択肢の効果】
・(1) ア、ウ → ア(+1,+1) + ウ(0,-1) = (+1, 0) → 圧力が戻らない → 不正解
・(2) イ、オ → イ(-1,-1) + オ(-1,+1) = (-2, 0) → 温度がさらに低下 → 不正解
・(3) ア、エ → ア(+1,+1) + エ(+1,0) = (+2, +1) → 両方過剰 → 不正解
・(4) エ、イ → エ(+1,0) + イ(-1,-1) = (0, -1) → 温度が戻らない → 不正解
・(5) エ、ウ → エ(+1,0) + ウ(0,-1) = (+1, -1) → 温度・圧力ともに適正に復帰 → 正解

【ポイント】
・温度と圧力で逆方向の補正が必要な場合、ア（両方+1）やイ（両方-1）は使えない
・エ（温度のみ+1）とウ（圧力のみ-1）はそれぞれ独立に1つのパラメータだけを変化させるため、組み合わせることで逆方向の補正が実現できる

したがって、正解は(5)である。
