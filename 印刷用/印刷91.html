問1．以下の説明を読み、与えられた状況の表示の変化の直後に実施する適切な操縦操作として正しいものを1つ選び、解答用紙にマークせよ。（6点）
APAIとはApproach Path Angle Indicator（進入経路角度指示装置）の略で、着陸しようとする航空機に対して降下角度が適切かどうかを光の色で知らせる装置である。その外見は、滑走路の脇の平らな地面に横一列に並んだ4つの箱型ユニットである。それぞれのユニットは金属製の頑丈な筐体に収められており、大きさは一つあたりおおよそ小型のスーツケース程度である。各ユニットの前面には横長のレンズ窓があり、そこから滑走路に進入してくる航空機に向かって光を放っている。4つのユニットは数メートルずつ間隔をあけて一直線に設置されており、遠くから見ると滑走路の脇にぽつぽつと低い構造物が並んでいるように見える。
このライトの最大の特徴は、見る角度によって色が変わることである。各ユニットには特殊なレンズが組み込まれており、一定の角度より上から見ると明色（白色）に、下から見ると暗色（黒色）に見える仕組みになっている。電気的に色を切り替えているのではなく、光の屈折を利用した光学的な構造である。4つのユニットはそれぞれ明暗が切り替わる境界の角度が少しずつずらして設定されており、パイロットの目線の角度に応じて明色と暗色の組み合わせが自然に変化する。パイロットから見て左2つが明色、右2つが暗色という「○○●●」の状態が適正降下角度であり、多くの空港では3度に設定されている。パイロットはこの「○○●●」を維持しながら降下していくのである。
【操縦操作】
ア：操縦桿を引き、推力を上げる
イ：操縦桿と推力を維持する
ウ：推力を上げる
エ：操縦桿を押す
オ：推力を下げる
【状況】
(1) ア、ウ　　　　(2) イ　　　　(3) ア　　　　(4) ウ　　　　(5) オ
問1(正解:(3))

【表示の読み取り】
・4つのユニットは境界角度が少しずつ異なる。進入角度が低いほど暗色（●）に見えるユニットが増える
・正常状態「○○●●」は適正降下角度（約3度）にいることを示す
・変化後「○●●●」は進入角度が低下し、やや低い位置にいることを示す

【必要な操作の分析】
・進入角度を上げて正常な降下経路に戻る必要がある
・操縦桿を引いて機首を上げることで降下角度を浅くし、高度を回復できる
・しかし、機首上げだけでは速度が低下し失速の危険がある
・そのため、推力も同時に上げて速度を維持する必要がある

【選択肢の検討】
・(1) ア、ウ → アは機首上げ+推力増で適切だが、ウ（さらに推力増）を重ねると速度超過となる → 不正解
・(2) イ → 現状維持では低角度のまま地面に接近し危険 → 不正解
・(3) ア → 操縦桿を引いて機首上げ、同時に推力を上げて速度を維持。安全に角度を回復できる → 正解
・(4) ウ → 推力を上げるだけでは速度は増すが降下角度はほぼ変わらない → 不正解
・(5) オ → 推力を下げるとさらに降下し、危険が増す → 不正解

したがって、正解は(3)である。
問2．以下の説明を読み、与えられた状況の表示の変化の直後に実施する適切な操縦操作の組み合わせとして正しいものを1つ選び、解答用紙にマークせよ。（6点）
問1のAPAI（進入角度指示灯）に加え、滑走路手前には進入速度指示灯が設置されている。これも4つの箱型ユニットが横一列に並ぶ構造であるが、こちらは航空機の接近速度に応じて表示が変わる。速度が各ユニットの閾値を超えると明色（○）から暗色（●）に変わり、速度が高いほど左から順に暗色のユニットが増える。適正速度範囲では「●●○○」が正常な表示状態である。
パイロットは以下の5つの操作を使用できる。同時に2つまでの操作を行うことができ、その効果は加算される。進入角度と速度がともに正常な表示に戻る操作の組み合わせを選べ。
操作 内容 進入角度への影響 速度への影響
ア 推力を上げる 1段階上昇 1段階上昇
イ 推力を下げる 1段階下降 1段階下降
ウ 操縦桿を引く 1段階上昇 変化なし
エ スピードブレーキを展開する 変化なし 1段階下降
オ スピードブレーキを格納する 変化なし 1段階上昇
【状況】
(1) ア、エ　　　(2) イ、オ　　　(3) ア、ウ　　　(4) ウ、エ　　　(5) イ、ア
問2(正解:(4))

【状況の分析】
・進入角度: ○○●● → ○●●●（1段階低下、低い側に逸脱）→ 角度を1段階上げる必要がある
・速度: ●●○○ → ●●●○（1段階上昇、速い側に逸脱）→ 速度を1段階下げる必要がある
・必要な変化: (角度 +1, 速度 -1)

【各選択肢の効果】
・(1) ア、エ → ア(角度+1, 速度+1) + エ(角度±0, 速度-1) = (角度+1, 速度±0) → 速度が戻らない → 不正解
・(2) イ、オ → イ(角度-1, 速度-1) + オ(角度±0, 速度+1) = (角度-1, 速度±0) → 角度がさらに低下 → 不正解
・(3) ア、ウ → ア(角度+1, 速度+1) + ウ(角度+1, 速度±0) = (角度+2, 速度+1) → 両方過剰 → 不正解
・(4) ウ、エ → ウ(角度+1, 速度±0) + エ(角度±0, 速度-1) = (角度+1, 速度-1) → 両方正常に戻る → 正解
・(5) イ、ア → イ(角度-1, 速度-1) + ア(角度+1, 速度+1) = (角度±0, 速度±0) → 何も変わらない → 不正解

【ポイント】
・角度と速度で逆方向の補正が必要な場合、推力の操作（ア・イ）は両方に同時に影響するため使えない
・操縦桿（ウ）は角度のみ、スピードブレーキ（エ）は速度のみに影響するため、これらを組み合わせることで独立に制御できる

したがって、正解は(4)である。
