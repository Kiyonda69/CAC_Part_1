問1．次は5つのカフェ（A〜E）の店舗紹介と営業情報である。下記の【希望条件】をすべて満たすカフェを、選択肢(1)〜(5)の中から1つ選べ。（6点）
【資料1】各カフェの店舗紹介

カフェA：駅前ビル2階の落ち着いた雰囲気のカフェ。全席で終日Wi-Fiが無料で利用できる。ブース席を6席完備し、深夜0時まで営業。サンドイッチやパスタなどの食事メニューは開店から閉店まで終日提供している。

カフェB：木のぬくもりを活かした内装で、店内全域でWi-Fiを無料開放している。半個室のブース席が4席あり、食事メニューは終日提供。閉店時刻は20時で、夕方以降は静かに過ごしたい来店客に好まれている。

カフェC：深夜0時まで営業する24時間に近い長時間営業の店舗。ブース席を備え、食事メニューは終日提供。Wi-Fiは混雑緩和のため、平日の11時から14時のランチタイムに限り利用可能となっている。

カフェD：オープンスペース型のフロア構造で、開放感のある内装が特徴。全席で終日Wi-Fiを利用でき、食事メニューも終日提供。閉店時刻は23時。個室席およびブース席は設けていない。

カフェE：4部屋の個室を備え、深夜0時まで営業するナイトカフェ。終日Wi-Fiの利用が可能。食事メニューは11時から14時のランチタイムのみで、それ以外の時間帯はドリンクとデザートのみの提供となる。

問1(正解:(1))
各カフェについて、希望条件①〜④の充足状況を確認する。

【条件チェック表】
・A：Wi-Fi終日○ / 24時閉店○ / ブース席○ / 食事終日○ → 4条件すべて○
・B：Wi-Fi終日○ / 20時閉店× / ブース席○ / 食事終日○ → 条件②不適合
・C：Wi-Fi一部時間のみ× / 24時閉店○ / ブース席○ / 食事終日○ → 条件①不適合
・D：Wi-Fi終日○ / 23時閉店○ / 個室・ブースなし× / 食事終日○ → 条件③不適合
・E：Wi-Fi終日○ / 24時閉店○ / 個室○ / 食事ランチタイムのみ× → 条件④不適合

【判定の要点】
・カフェCは閉店時刻が深夜0時で長時間営業だが、Wi-Fiは「平日11時〜14時のランチタイムに限り利用可能」と記載されており「終日」ではない。
・カフェEは個室を備え深夜まで営業するが、食事は「11時〜14時のランチタイムのみ」で、それ以外はドリンク・デザートのみ。
・カフェDは終日Wi-Fi・23時閉店・終日食事を満たすが、店舗紹介に「個室席およびブース席は設けていない」と明記されており条件③に該当しない。

したがって、4条件すべてを満たすカフェはAのみであり、正解は(1)である。
問2．次は5つの医療保険プラン（P〜T）の説明と保障内容である。45歳の被保険者が下記の【加入条件】をすべて満たすプランを選ぶとき、選択肢(1)〜(5)の中から1つ選べ。（6点）
【資料1】各プランの説明

プランP：手厚い保障を売りとする総合医療保険。入院日額5,000円、通院給付金は1日2,500円。がん診断一時金は100万円で、先進医療特約も標準付帯。月額保険料は3,800円。加入年齢上限は70歳。

プランQ：バランス重視型の医療保険。入院日額5,000円、通院給付金は1日3,000円と充実。がん診断一時金100万円、先進医療特約付き。月額保険料は3,200円。加入年齢上限は60歳。

プランR：保険料を抑えた割安プラン。月額2,800円で入院日額4,000円、通院給付金1日3,000円、がん診断一時金100万円、先進医療特約付き。加入年齢上限は65歳。

プランS：入院保障に重点を置いた厚手プラン。入院日額6,000円、通院給付金1日2,000円、がん診断一時金100万円、先進医療特約付き。月額保険料は3,500円。加入年齢上限は55歳。

プランT：通院重視のバランス型プラン。入院日額5,000円、通院給付金1日2,500円、がん診断一時金80万円、先進医療特約付き。月額保険料は3,500円。加入年齢上限は65歳。

被保険者は45歳。加入年齢上限が45歳未満のプランは加入不可となる。
各プランについて加入条件①〜⑤の充足状況を確認する。

【条件チェック表】
・P：①月額3,800円× ②入院5,000円○ ③通院2,500円○ ④がん100万円○ ⑤特約○ ／年齢70歳○ → 条件①不適合
・Q：①月額3,200円○ ②入院5,000円○ ③通院3,000円○ ④がん100万円○ ⑤特約○ ／年齢60歳○ → 5条件すべて○
・R：①月額2,800円○ ②入院4,000円× ③通院3,000円○ ④がん100万円○ ⑤特約○ ／年齢65歳○ → 条件②不適合
・S：①月額3,500円○ ②入院6,000円○ ③通院2,000円× ④がん100万円○ ⑤特約○ ／年齢55歳○ → 条件③不適合
・T：①月額3,500円○ ②入院5,000円○ ③通院2,500円○ ④がん80万円× ⑤特約○ ／年齢65歳○ → 条件④不適合

【判定の要点】
・プランPは月額3,800円で①の上限3,500円を超過。保障内容は条件を満たすが保険料で除外される典型例。
・プランRは保険料が最も安い2,800円で魅力的だが、入院日額が4,000円と②の5,000円基準に満たない。
・プランSは入院日額6,000円と最も手厚いが、通院給付金が1日2,000円で③の2,500円基準を下回る。
・プランTはQと月額・入院日額・通院給付金が同水準で紛らわしいが、がん診断一時金が80万円で④の100万円基準を満たさない。
・プランQはすべての条件を満たし、加入年齢上限60歳のため45歳の被保険者も加入可能。

したがって、5条件すべてを満たすプランはQのみであり、正解は(2)である。
