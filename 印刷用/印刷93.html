問1．ある管制空域において、5機の航空機A機～E機がレーダーに映っている。各機体は4方角（北・東・南・西）と3距離圏（第1圏・第2圏・第3圏）のすべて異なる交点に存在する。各パイロットから管制官への報告を下表に示す。ただしB機のみ通信不良のため報告が不明（?）である。次のア～ウについて、正しいものの組み合わせを選べ。（6点）
報告内容
A機 「私は東方向の第2圏にいる。第2圏には私を含めて2機がいる。」
B機 ？（通信不良）
C機 「私は東方向の第1圏にいる。第1圏には私を含めて2機がいる。」
D機 「私は南方向の第3圏にいる。」
E機 「私は西方向の第2圏にいる。西方向には私を含めて1機がいる。」
ア．B機の報告がない場合、B機の位置として2通りが考えられる。
イ．B機の報告がない場合、B機の位置として3通りが考えられる。
ウ．B機が「私は北方向にいる」とのみ報告した場合、B機の位置が確定する。
(1)　アのみ正しい
(2)　ア、ウが正しい
(3)　イのみ正しい
(4)　イ、ウが正しい
(5)　ウのみ正しい
問1(正解:(2))

【確定情報の整理】
各報告から確定する位置:
・A機: 東方向・第2圏
・C機: 東方向・第1圏
・D機: 南方向・第3圏
・E機: 西方向・第2圏

【制約条件の適用】
・A機の報告: 第2圏には2機 → A機(東・第2圏)とE機(西・第2圏)で2機。B機は第2圏にいない。
・C機の報告: 第1圏には2機 → C機(東・第1圏)ともう1機。B機は第1圏の可能性あり。
・E機の報告: 西方向には1機 → E機のみ。B機は西方向にいない。

B機が第2圏でなく、西方向でもない位置で、第1圏にもう1機必要:
→ B機は第1圏にいなければならない（第1圏を2機にするため）
→ 第1圏で空いている方向: 北・南（東はC機、西は除外）
→ B機の候補: 北方向・第1圏、南方向・第1圏 の2通り

【各選択肢の検討】
・ア: 2通り → 正しい
・イ: 3通り → 誤り（2通り）
・ウ: 「北方向にいる」→ 北方向・第1圏に確定 → 正しい

したがって、ア、ウが正しく、正解は(2)である。
問2．5人の生徒A～Eのうち、ちょうど2人が嘘つきで、残りの3人は正直者である。正直者は常に真実を述べ、嘘つきは常に嘘を述べる。5人はそれぞれ以下のように発言した。次のア～ウについて、正しいものの組み合わせを選べ。（6点）
発言内容
A 「CとEは両方とも正直者だ。」
B 「Aは嘘つきだ。」
C 「BとDのうち、少なくとも1人は嘘つきだ。」
D 「Eは嘘つきだ。」
E 「Aは正直者だ。」
ア．Aは正直者である。
イ．Bは嘘つきである。
ウ．Cは嘘つきである。
(1)　アのみ正しい
(2)　イのみ正しい
(3)　ウのみ正しい
(4)　ア、イが正しい
(5)　イ、ウが正しい
問2(正解:(4))

【場合分けによる推論】
嘘つきはちょうど2人。AとEの発言に注目する。
・E:「Aは正直者だ」、B:「Aは嘘つきだ」→ BとEは矛盾するため、一方が正直者で他方が嘘つき。

【場合1: Aが嘘つきの場合】
・A(嘘つき):「CとEは両方正直者」→ 嘘 → CまたはEの少なくとも1人が嘘つき
・B(?)：「Aは嘘つき」→ 真 → Bは正直者
・E(?)：「Aは正直者」→ 偽 → Eは嘘つき
・嘘つき: A, E（2人）→ B, C, Dは正直者
・C(正直者):「BとDのうち少なくとも1人は嘘つき」→ B,D共に正直者なので偽。矛盾。
→ 場合1は不成立

【場合2: Aが正直者の場合】
・A(正直者):「CとEは両方正直者」→ 真 → C, E共に正直者
・E(正直者):「Aは正直者」→ 真 ✓
・嘘つき2人はB, Dの中から選ぶ。A, C, Eが正直者なので、B, Dが嘘つき。
・B(嘘つき):「Aは嘘つき」→ 嘘 → Aは正直者 ✓
・C(正直者):「BとDのうち少なくとも1人は嘘つき」→ B,D共に嘘つきなので真 ✓
・D(嘘つき):「Eは嘘つき」→ 嘘 → Eは正直者 ✓
→ 場合2は成立。嘘つきはBとD。

【結論】
・ア: Aは正直者 → 正しい
・イ: Bは嘘つき → 正しい
・ウ: Cは嘘つき → 誤り（Cは正直者）

したがって、ア、イが正しく、正解は(4)である。
