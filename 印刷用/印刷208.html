【条件】下記をすべて満たす商品を抽出する
  • 条件1：単価が1,500円以上である
  • 条件2：4〜6月の合計販売数が400個以上である
  • 条件3：6月の販売数が4月の販売数の1.2倍以上である

（1）245　　（2）295　　（3）330　　（4）365　　（5）410
問1(正解:(4))
各商品について4つの条件を順に検証する。

【条件別の絞り込み】
・条件1（単価1,500円以上）を満たさない：P-101(850), P-102(1,200), P-105(950) を除外
・残る候補：P-103, P-104, P-106, P-107, P-108

・条件2（4〜6月合計400個以上）の検証：
　- P-103：95+145+200=440 ○
　- P-104：145+160+175=480 ○
　- P-106：75+85+95=255 → 除外
　- P-107：200+220+250=670 ○
　- P-108：130+150+165=445 ○

・条件3（6月÷4月≧1.2）の検証：
　- P-103：200÷95≒2.11 ○
　- P-104：175÷145≒1.21 ○
　- P-107：250÷200=1.25 ○
　- P-108：165÷130≒1.27 ○

・条件4（評価4.3以上）の検証：
　- P-103：4.5 ○
　- P-104：4.0 → 除外
　- P-107：4.1 → 除外
　- P-108：4.4 ○

【結論】
全条件を満たすのはP-103とP-108の2商品。
それぞれの6月販売数の合計：200 + 165 = 365個

したがって、正解は(4)である。

問2(正解:(3))
4つの条件を支店ごとに検証する。

【各支店の指標計算】
・店舗A：利益=8,500−6,800=1,700、顧客/社員=4,200÷12=350.0、評価4.1、成長率4％
・店舗B：利益=12,000−9,200=2,800、顧客/社員=5,800÷18≒322.2、評価3.9、成長率12％
・店舗C：利益=11,000−8,700=2,300、顧客/社員=5,200÷16=325.0、評価4.0、成長率6％
・店舗D：利益=9,800−7,500=2,300、顧客/社員=4,800÷14≒342.9、評価4.3、成長率3％
・店舗E：利益=7,200−5,900=1,300、顧客/社員=3,600÷10=360.0、評価4.4、成長率10％

【条件適用】
・条件1（利益≧1,500万円）：店舗Eが1,300で不可 → 残り A, B, C, D
・条件2（顧客/社員≧300）：全て300以上 → 残り A, B, C, D
・条件3（評価≧4.0）：店舗B(3.9)が不可 → 残り A, C, D
・条件4（成長率≧5％）：店舗A(4％)と店舗D(3％)が不可 → 残り 店舗C のみ

【選択肢の検討】
・(1)店舗A：成長率4％で条件4を満たさない
・(2)店舗B：評価3.9で条件3を満たさない
・(3)店舗C：全条件を満たす
・(4)店舗D：成長率3％で条件4を満たさない
・(5)店舗E：利益1,300万円で条件1を満たさない

したがって、正解は(3)である。
