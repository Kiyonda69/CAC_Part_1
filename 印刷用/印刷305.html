問1．商品Aと商品Bを合わせて6,000円で仕入れた。Aには原価の2割の利益を見込んだ価格をつけ、Bは原価の1割引の価格をつけて、どちらもその価格で売ったところ、全体で660円の利益となった。商品Aの原価はいくらか。次の(1)〜(5)から1つ選べ。（6点）
(1) 3,300円   (2) 4,200円   (3) 1,800円   (4) 5,040円   (5) 2,200円
問1(正解:(2))
2つの商品の原価がどちらも分かっていないが、仕入総額が与えられているため、Aの原価を a 円とおけばBの原価は (6,000 − a) 円と表せる。

【立式】
・Aの利益：原価の2割 → 0.2a 円の利益
・Bの損益：原価の1割引で売る → 原価の1割の損失 → 0.1(6,000 − a) 円の損失
・全体で660円の利益 → 0.2a − 0.1(6,000 − a) = 660

【求解】
0.2a − 600 + 0.1a = 660
0.3a = 1,260 → a = 1,260 ÷ 0.3 = 4,200円
（Bの原価は 6,000 − 4,200 = 1,800円）

【検算】
Aの利益 = 4,200 × 0.2 = 840円、Bの損失 = 1,800 × 0.1 = 180円
全体の利益 = 840 − 180 = 660円 となり、条件と一致する。

【選択肢の検討】
・(1) 3,300円 → Bの損失を考えず 0.2a = 660 とした値。不正解
・(2) 4,200円 → 0.3a = 1,260 を満たす。正解
・(3) 1,800円 → Bの原価との取り違え（AとBの条件を逆にして解いた場合もこの値になる）。不正解
・(4) 5,040円 → Aの売価（4,200 × 1.2）との取り違え。不正解
・(5) 2,200円 → −600 の移項を忘れて 0.3a = 660 とした値。不正解

したがって、正解は(2)である。
問2．ある店で、商品Aと商品Bを合わせて500個仕入れた。仕入値はAが1個300円、Bが1個200円で、仕入総額は128,000円であった。Aは原価の3割の利益を見込んだ定価をつけたところ、すべて定価で売れた。Bは原価の4割の利益を見込んだ定価をつけたが、定価で売れたのは仕入れた個数の6割にとどまり、残りはすべて定価の25％引きにして売り切った。このとき、全体の利益はいくらか。次の(1)〜(5)から1つ選べ。（6点）
(1) 31,360円   (2) 34,360円   (3) 35,760円   (4) 36,640円   (5) 42,800円
問2(正解:(4))
まずA・Bそれぞれの仕入個数を求め、次に商品ごと・価格帯ごとの利益を積み上げる2段構えの問題である。

【手順1：仕入個数を求める（つるかめ算）】
Aの個数を n 個とおくと、Bの個数は (500 − n) 個。仕入総額より
300n + 200(500 − n) = 128,000
100n + 100,000 = 128,000 → n = 280
よって、Aは280個、Bは 500 − 280 = 220個。

【手順2：各価格帯の利益を計算】
・A：定価 = 300 × 1.3 = 390円 → 1個あたり90円の利益 × 280個 = 25,200円
・B（定価分）：定価 = 200 × 1.4 = 280円 → 売れたのは 220 × 0.6 = 132個 → 1個あたり80円の利益 × 132個 = 10,560円
・B（割引分）：売価 = 280 × 0.75 = 210円 → 残り 220 − 132 = 88個 → 1個あたり 210 − 200 = 10円の利益 × 88個 = 880円
（1.4 × 0.75 = 1.05 なので割引後もわずかに原価を上回る点に注意）

【求解】
全体の利益 = 25,200 + 10,560 + 880 = 36,640円

【検算】
総売上 = 390 × 280 + 280 × 132 + 210 × 88 = 109,200 + 36,960 + 18,480 = 164,640円
利益 = 164,640 − 128,000 = 36,640円 となり、一致する。

【選択肢の検討】
・(1) 31,360円 → 25％引きを定価でなく原価に掛けて売価を150円（1個あたり50円の損失）とした値。不正解
・(2) 34,360円 → AとBの個数を取り違えた値（A220個・B280個で計算）。不正解
・(3) 35,760円 → 「4割増しの25％引きだから利益はちょうど0」と誤り、割引分の利益880円を落とした値（1.4 × 0.75 = 1.05 であり0にはならない）。不正解
・(4) 36,640円 → すべての条件を満たす。正解
・(5) 42,800円 → Bの売れ残りもすべて定価で売れたとして 25,200 + 80 × 220 と計算した値。不正解

したがって、正解は(4)である。
