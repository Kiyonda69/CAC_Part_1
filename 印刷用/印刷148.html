問1．次の資料に基づき、すべての条件を満たす配送業者として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
【資料1】配送業者の比較表
業者配送日数基本料金重量上限補償上限額
A社翌日2,800円30kg5万円
B社翌々日2,200円25kg10万円
C社翌日3,200円50kg8万円
D社翌々日1,800円20kg3万円
E社翌々日2,500円40kg6万円
【資料2】今回の配送条件
  • 商品の重量は 28kg
  • 商品の価値は 5万5千円（補償が必要な金額）
  • 到着は翌々日でも問題ない
  • 配送料は 2,700円以下 に収めたい
(1) B社　　(2) D社　　(3) E社　　(4) A社　　(5) C社
問1(正解:(3))

4つの条件を一つずつ各業者に当てはめ、すべてを満たす業者を絞り込む。

【条件①】重量28kg以下を扱える業者（重量上限が28kg以上）
・A社: 30kg ✓　B社: 25kg ✗　C社: 50kg ✓　D社: 20kg ✗　E社: 40kg ✓
→ 残るのは A・C・E社。

【条件②】補償上限が5万5千円以上
・A社: 5万円 ✗　C社: 8万円 ✓　E社: 6万円 ✓
→ A社は補償額が不足するため脱落。残るのは C・E社。

【条件③】翌々日着で問題ない（=翌日でも翌々日でも可）
・C社: 翌日 ✓　E社: 翌々日 ✓
→ 両社とも問題なし。

【条件④】配送料が2,700円以下
・C社: 3,200円 ✗　E社: 2,500円 ✓
→ C社は予算超過で脱落。残るのは E社のみ。

【検算】E社が4条件すべてを満たすことを確認
・重量: 40kg ≥ 28kg ✓　・補償: 6万円 ≥ 5万5千円 ✓
・日数: 翌々日 ✓　・料金: 2,500円 ≤ 2,700円 ✓

【選択肢の検討】
・(1) B社 → 重量制限25kg未満のため不適
・(2) D社 → 重量制限20kg未満のため不適
・(3) E社 → すべての条件を満たす（正解）
・(4) A社 → 補償上限が5万円で5万5千円に届かず不適
・(5) C社 → 配送料3,200円が予算2,700円を超過し不適

したがって、正解は(3)である。
問2．次の資料に基づき、利用者Pさんの今月の利用料金（税抜き）として正しいものを次の(1)〜(5)の中から1つ選べ。（6点）
【資料1】基本プラン
プラン月額基本料金データ通信容量
ライト3,000円5GB
スタンダード5,000円30GB
プレミアム7,500円無制限
【資料2】通話・オプション料金
  • 国内通話: 30秒あたり 22円
  • 国際通話: 30秒あたり 60円
  • 通話定額オプション（月額500円）: 加入で国内通話のみ無料
【資料3】データ超過料金
  • ライト・スタンダードは容量超過時、超過分1GBあたり 300円課金
  • プレミアムは超過料金なし
【資料4】各種割引（基本料金にのみ適用）
  • 家族割: 同一家族2回線目以降は基本料金から10%引き
  • 長期契約割引: 利用期間 24ヶ月以上 で5%引き
  • 学割: 学生は基本料金から15%引き（家族割と併用不可）
  • 自動引き落とし割引: 100円引き（定額）
  • ※割引は パーセント割引 → 定額割引 の順で適用
【資料5】Pさんの今月の利用状況
  • スタンダードプラン契約／通話定額オプション加入
  • 今月の通話: 国内通話多数（定額対象）／ 国際通話 8分
  • 今月のデータ通信量: 38GB
  • 家族割対象（2回線目）／利用期間 18ヶ月／学生ではない
  • 支払いは自動引き落とし
(1) 7,400円　　(2) 7,760円　　(3) 8,260円　　(4) 8,360円　　(5) 8,760円
問2(正解:(3))

料金は 「基本料金 → 割引適用 → オプション・通話料・データ超過料の加算」の順で計算する。

【ステップ1】基本料金
スタンダードプラン: 5,000円

【ステップ2】パーセント割引の適用
・家族割（2回線目）: -10% → 5,000 × 0.9 = 4,500円
・長期契約割引: 利用期間18ヶ月（24ヶ月未満のため 適用されない）
・学割: 学生ではないため対象外

【ステップ3】定額割引の適用
・自動引き落とし割引: -100円 → 4,500 - 100 = 4,400円

【ステップ4】オプション・通話料・データ超過料の加算
・通話定額オプション: +500円 → 4,400 + 500 = 4,900円
・国内通話: 定額対象につき 0円
・国際通話: 8分 = 480秒 = 16単位（30秒/単位）
　　16 × 60円 = 960円 → 4,900 + 960 = 5,860円
・データ超過: 38GB - 30GB = 8GB超過
　　8 × 300円 = 2,400円 → 5,860 + 2,400 = 8,260円

【検算】5,000×0.9 - 100 + 500 + 960 + 2,400 = 4,500 - 100 + 3,860 = 8,260円 ✓

【選択肢の検討】
・(1) 7,400円 → データ超過料を計算し忘れた場合の額に近い誤答
・(2) 7,760円 → 国際通話を「分単位」で誤計算した結果（8×60=480円）
・(3) 8,260円 → 上記の正しい計算による額（正解）
・(4) 8,360円 → 自動引き落とし割引を反映し忘れた誤答
・(5) 8,760円 → 長期契約割引を誤って加算しなかった上に他の誤りを含む

したがって、正解は(3)である。
