問1(正解:(3))

【解法】
A地区とB地区を合わせた学生利用者の割合を各年度で計算する。

【2017年の合算】
・A地区：学生250人、一般350人（合計600人）
・B地区：学生200人、一般300人（合計500人）
・合算：学生450人 / 合計1100人 = 40.9%

【2018年の合算】
・A地区：学生340人、一般260人（合計600人）
・B地区：学生280人、一般220人（合計500人）
・合算：学生620人 / 合計1100人 = 56.4%

【差の計算】
56.4 - 40.9 = 15.5ポイント増加

【注意点】
各地区の学生割合を単純に平均するのではなく、合算した人数から割合を計算する必要がある（加重平均）。
・単純平均の場合：(41.7+40.0)/2=40.85、(56.7+56.0)/2=56.35、差=15.5（この場合はたまたま近い値になる）

したがって、正解は(3) 15.5ポイントである。

問2(正解:(2))

【解法】
3年間の延べ利用者数を合算して割合を求める。

【2015〜2017年の3年間合算】
・学生（A地区）：200 + 180 + 250 = 630人
・学生（B地区）：160 + 140 + 200 = 500人
・学生合計：630 + 500 = 1130人
・利用者総数（A地区）：500 + 600 + 600 = 1700人
・利用者総数（B地区）：400 + 500 + 500 = 1400人
・総合計：1700 + 1400 = 3100人
・学生割合：1130 / 3100 = 36.5%

【2018〜2020年の3年間合算】
・学生（A地区）：340 + 380 + 310 = 1030人
・学生（B地区）：280 + 350 + 270 = 900人
・学生合計：1030 + 900 = 1930人
・利用者総数（A地区）：600 + 600 + 500 = 1700人
・利用者総数（B地区）：500 + 500 + 400 = 1400人
・総合計：1700 + 1400 = 3100人
・学生割合：1930 / 3100 = 62.3%

【差の計算】
62.3 - 36.5 = 25.8ポイント

【注意点】
各年の割合の平均ではなく、延べ人数から直接割合を計算する必要がある。各年で利用者総数が異なるため、単純平均では正確な値が得られない。

したがって、正解は(2) 25.8ポイントである。
