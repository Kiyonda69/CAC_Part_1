問1．次の表は、ある県のA〜E市における観光入込客数と観光消費額を、2023年と2024年についてまとめたものである。この表から確実にいえるものとして、最も妥当なのはどれか。（6点）
【表1】A〜E市の観光入込客数と観光消費額
市 観光入込客数（万人） 観光消費額（億円）
2023年 2024年 2023年 2024年
A市620682372409
B市540513297308
C市380418209230
D市760798456503
E市300342186205
(1) 2024年の観光入込客数の対前年増加率が最も大きいのはC市である。
(2) 2024年において、観光客1人当たりの観光消費額が最も高いのはE市である。
(3) A〜E市の観光消費額の合計は、2023年から2024年にかけて10%以上増加した。
(4) 2023年から2024年にかけて、観光入込客数と観光消費額がともに増加した市は3つである。
(5) 2024年のA〜E市の観光入込客数の合計に占めるD市の割合は、2023年に比べて低下した。
問1(正解:(5))
各選択肢について、表の数値をもとに計算して検証する。

【選択肢の検討】
・(1) 観光入込客数の対前年増加率を各市について計算すると、
　A市：(682−620)÷620 = 62÷620 = 10.0%
　B市：(513−540)÷540 = −27÷540 = −5.0%
　C市：(418−380)÷380 = 38÷380 = 10.0%
　D市：(798−760)÷760 = 38÷760 = 5.0%
　E市：(342−300)÷300 = 42÷300 = 14.0%
　最も大きいのはE市（14.0%）であり、C市（10.0%）ではない。→ 誤り

・(2) 2024年の観光客1人当たりの観光消費額（消費額÷客数）を比較する。
　A市：409÷682 ≒ 0.600　B市：308÷513 ≒ 0.600
　C市：230÷418 ≒ 0.550　D市：503÷798 ≒ 0.630
　E市：205÷342 ≒ 0.599
　最も高いのはD市（約6.3万円/人）であり、E市ではない。→ 誤り

・(3) 観光消費額の合計は、2023年が 372+297+209+456+186 = 1520億円、
　2024年が 409+308+230+503+205 = 1655億円。
　増加率は (1655−1520)÷1520 = 135÷1520 ≒ 8.9% で、10%に満たない。→ 誤り

・(4) 客数と消費額がともに増加したのは、A市・C市・D市・E市の4市である
　（B市のみ客数が540→513と減少）。3つではない。→ 誤り

・(5) 観光入込客数の合計は、2023年が 620+540+380+760+300 = 2600万人、
　2024年が 682+513+418+798+342 = 2753万人。
　D市の割合は、2023年が 760÷2600 ≒ 29.2%、2024年が 798÷2753 ≒ 29.0% であり、
　低下している。→ 正しい
　（別解：D市の増加率5.0%は合計の増加率 153÷2600 ≒ 5.9% より小さいため、
　実額を割らなくても構成比の低下が判定できる）

したがって、正解は(5)である。
問2．次の表は、ある国の製造業5部門について、2021年の生産額と、2022年から2024年までの各年における生産額の対前年増加率を示したものである。この表から確実にいえるものとして、最も妥当なのはどれか。（6点）

(1) 2024年の生産額が2021年の生産額を下回っている部門は2つある。
(2) 2023年の機械の生産額は、同年の食品の生産額の1.4倍を上回っている。
(3) 2022年において、生産額が前年より減少した部門の減少額の合計は、生産額が前年より増加した部門の増加額の合計の半分以下である。
(4) 2021年から2024年にかけての生産額の増加率が最も大きい部門は金属である。

問2(正解:(3))
対前年増加率の表なので、各年の生産額は2021年の実額に増加率を順に掛けて求める。
各部門の生産額（億円）を計算すると次のようになる。

　食品：2021年 2400 → 2022年 2520 → 2023年 2469.6 → 2024年 約2568.4
　化学：2021年 1800 → 2022年 1746 → 2023年 1850.8 → 2024年 約1897.0
　金属：2021年 1500 → 2022年 1620 → 2023年 1660.5 → 2024年 約1594.1
　機械：2021年 3200 → 2022年 3280 → 2023年 3444 → 2024年 約3650.6
　繊維：2021年 900 → 2022年 855 → 2023年 833.6 → 2024年 約900.3

【選択肢の検討】
・(1) 2024年と2021年を比べると、金属は 1594.1 > 1500、繊維も 900.3 > 900 と
　わずかに上回っており、5部門すべてが2021年を下回っていない。
　（繊維は 0.95×0.975×1.08 ≒ 1.0004 倍でぎりぎり増加）→ 誤り

・(2) 2023年は機械 3444、食品 2469.6。食品の1.4倍は 2469.6×1.4 ≒ 3457.4 であり、
　3444 はこれを下回る（3444÷2469.6 ≒ 1.39倍）。→ 誤り

・(3) 2022年に減少した部門は化学（1800→1746、減少額54）と繊維（900→855、減少額45）で、
　減少額の合計は 54+45 = 99億円。
　増加した部門は食品（+120）、金属（+120）、機械（+80）で、増加額の合計は 320億円。
　その半分は160億円であり、99 ≦ 160 が成り立つ。→ 正しい

・(4) 2021年から2024年の増加率は、
　食品：1.05×0.98×1.04 ≒ 1.070（+7.0%）　化学：0.97×1.06×1.025 ≒ 1.054（+5.4%）
　金属：1.08×1.025×0.96 ≒ 1.063（+6.3%）　機械：1.025×1.05×1.06 ≒ 1.141（+14.1%）
　繊維：≒ 1.000（+0.0%）
　最大は機械であり、金属ではない。→ 誤り

・(5) 2024年の合計は 2568.4+1897.0+1594.1+3650.6+900.3 ≒ 10610億円。
　機械の割合は 3650.6÷10610 ≒ 34.4% で、35%を超えていない。→ 誤り

したがって、正解は(3)である。
