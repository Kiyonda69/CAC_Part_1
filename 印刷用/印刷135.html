問1（正解: (1) 菱形）

【切断面の特定】

3点A、M（辺BFの中点）、N（辺DHの中点）を通る平面を求める。

平面の方程式: x + y = 2z（頂点Aを原点に、辺ABをx軸、辺ADをy軸、辺AEをz軸とする）


【断面頂点の特定】

平面が各辺と交わる点を求めると、A(0,0,0)、M(1,0,½)、G(1,1,1)、N(0,1,½) の4点。

すなわち断面は頂点A・M・G・Nを結ぶ四角形となる。


【断面の形状判定】

各辺の長さを計算する（立方体の1辺を1とする）:

　AM = √(1²+0²+(½)²) = √(5/4) = (√5)/2

　MG = √(0²+1²+(½)²) = √(5/4) = (√5)/2

　GN = √(1²+0²+(½)²) = √(5/4) = (√5)/2

　NA = √(0²+1²+(½)²) = √(5/4) = (√5)/2

4辺すべて等長 → 菱形または正方形の候補。


【正方形ではないことの確認】

対角線 AG = √(1²+1²+1²) = √3、対角線 MN = √(1²+1²+0²) = √2

AG ≠ MN のため、正方形ではなく菱形。

問2（正解: (3) 正六角形）

【切断面の特定】

3点P（辺BCの中点）、Q（辺DHの中点）、R（辺EFの中点）を通る平面を求める。

各点の座標: P(1, ½, 0)、Q(0, 1, ½)、R(½, 0, 1)

これらはすべて x＋y＋z = 3/2 を満たす（立方体の中心を通る平面）。


【断面頂点の特定】

平面 x＋y＋z = 3/2 が各辺と交わる点を求めると、

辺BCの中点P(1, ½, 0)、辺BFの中点(1, 0, ½)、辺EFの中点R(½, 0, 1)、

辺HEの中点(0, ½, 1)、辺DHの中点Q(0, 1, ½)、辺CDの中点(½, 1, 0) の6点。


【断面の形状判定】

隣接する2点間の距離はすべて √(½) = (√2)/2 で等しい。

この切断面は立方体の空間対角線に垂直な平面であり、

6辺が等長かつすべての内角が120°の正六角形となる。
