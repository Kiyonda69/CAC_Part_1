問1．次の表は、ある小売チェーンの5店舗（A〜E）における2025年4月から9月までの月別売上高（単位：万円）を示している。表中のC店舗7月の売上値が判読できなくなっている。表に示された各種合計値を用いて、空欄（？）に入る数値として最も適切なものを次の(1)〜(5)の中から1つ選べ。（6点）
【表1】5店舗の月別売上高（単位：万円）
店舗 4月5月6月7月8月9月 店舗計
A 120135148162175180 920

(1) 139　　(2) 142　　(3) 135　　(4) 137　　(5) 144
問1(正解:(1))

空欄に入る値（C店舗の7月売上）は、表中の3種類の合計値のいずれか1つから求めることができる。

【解法1】7月の月計から逆算
7月の月計は700万円。A・B・D・Eの7月売上を引けば良い。
700 −（162 + 122 + 105 + 172）= 700 − 561 = 139万円

【解法2】C店舗の店舗計から逆算
C店舗の店舗計は814万円。空欄以外の月の売上を引けば良い。
814 −（110 + 125 + 132 + 150 + 158）= 814 − 675 = 139万円

【解法3】総計から逆算
全体の総計は4,060万円。空欄以外の店舗計の和を引けば良い。
4,060 −（920 + 708 + 610 + 1,008）= 4,060 − 3,246 = 814（C店舗計）
814 − 675 = 139万円

3つの解法すべてで139万円となり、整合性が確認できる。

【選択肢の検討】
・(1) 139 → 上記の通り正解
・(2) 142 → 不整合（月計や店舗計と矛盾）
・(3) 135 → 不整合
・(4) 137 → 不整合
・(5) 144 → 不整合

したがって、正解は(1)である。

問2(正解:(3))

【ステップ1】田中氏に適用される割引率を求める
・会員ランク：ゴールド → 通常割引 10%（資料1）
・誕生月：3月、購入月：3月 → 誕生月加算 +5%（資料1）
・3月のキャンペーン：春の感謝祭 → 追加割引 +5%（資料2）
合計割引率 = 10% + 5% + 5% = 20%

【ステップ2】割引前の合計金額を求める
支払額24,400円は割引後の金額なので、割引前の金額は
24,400 ÷ (1 − 0.20) = 24,400 ÷ 0.80 = 30,500円

【ステップ3】判明している商品の小計を計算する
・P1（カテゴリX）：1,500 × 3 = 4,500円
・P3（カテゴリZ）：2,200 × 1 = 2,200円
・P4（カテゴリW）：3,500 × 2 = 7,000円
既知の合計 = 4,500 + 2,200 + 7,000 = 13,700円

【ステップ4】商品P2の小計と個数を求める
P2の小計 = 30,500 − 13,700 = 16,800円
カテゴリYの単価は2,800円（資料3）なので
個数 = 16,800 ÷ 2,800 = 6個

【検算】
割引前合計 = 4,500 + 2,800×6 + 2,200 + 7,000 = 30,500円
割引後 = 30,500 × 0.80 = 24,400円　○

したがって、正解は(3)である。
