問1．田中さんは家族で観光施設を訪問する計画を立てている。下記の【施設情報】と【来場者統計】を読み、田中さんの【希望条件】をすべて満たす施設を1〜5の中から選びなさい。（6点）
【施設情報】
  • 施設A：開館9:00-17:00、休館日は月曜、入場料（大人）1,500円、駐車場有料
  • 施設B：開館10:00-18:00、休館日は火曜、入場料（大人）1,200円、駐車場無料
  • 施設C：開館9:00-19:00、休館日は水曜、入場料（大人）2,000円、駐車場無料
  • 施設D：開館10:00-18:00、休館日は木曜、入場料（大人）1,200円、駐車場無料
  • 施設E：開館8:30-17:30、休館日なし、入場料（大人）1,000円、駐車場有料
【来場者統計】（前年度・単位：千人）
施設4-6月7-9月10-12月1-3月年合計
A55856545250
B40605030180

【希望条件】
  1. 火曜日に訪問できること
  2. 大人2名の入場料合計が2,500円以内であること
  3. 駐車場が無料であること
  4. 前年度の年間来場者数が20万人以上であること
（1）施設A　（2）施設B　（3）施設C　（4）施設D　（5）施設E
問1（正解：(4)）
条件を1つずつ各施設に適用し、すべての条件を満たす施設を特定する。

【条件①：火曜日に訪問可能】
火曜が休館日のBを除外。残：A・C・D・E

【条件②：大人2名の入場料合計が2,500円以内】
・A：1,500×2=3,000円 → 超過
・C：2,000×2=4,000円 → 超過
・D：1,200×2=2,400円 → OK
・E：1,000×2=2,000円 → OK
A・Cを除外。残：D・E

【条件③：駐車場無料】
Eは駐車場有料のため除外。残：D

【条件④：年間来場者数20万人以上】
Dは年間280千人=28万人 → 条件を満たす。

【選択肢の検討】
・(1) A：駐車場有料・入場料超過 → 不適
・(2) B：火曜休館 → 不適
・(3) C：入場料合計4,000円で超過 → 不適
・(4) D：4条件すべて満たす → 正解
・(5) E：駐車場有料・年間来場者15万人 → 不適

したがって、正解は(4)である。

【条件①：総支払額50,000円以下】

・Q：60,000円 → 超過

・S：30,000＋5,000＝35,000円 → OK（教材費別途に注意）

Rは「認定証発行なし」と明記されているため除外。残：P・S・T

【条件③：修了率80％以上】
・P：88％ → OK
・S：76％ → 不足
・T：92％ → OK
Sを除外。残：P・T

【条件④：満足度4.5以上】
・P：4.5 → OK
・T：4.4 → 不足
Tを除外。残：P

・(2) Q：受講料60,000円で総額超過 → 不適
・(3) R：認定証発行なし → 不適
・(4) S：教材費込み35,000円で総額OK／修了率76％で不足 → 不適
