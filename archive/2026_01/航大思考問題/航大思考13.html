問1 配点: 6点 / 目安時間: 3分
図1は、ある航空会社の国内線における年間乗客数と平均搭乗率の推移を示したものである。以下の(A)～(C)の値をそれぞれ求め、A+B+Cの合計値として正しいものを１つ選び、解答用紙にマークせよ。(6点)
【図1】国内線の年間乗客数と平均搭乗率の推移
(A) 乗客数が最も多い年の乗客数と、最も少ない年の乗客数の差
(B) 搭乗率が75%以上の年の乗客数の合計
(C) 2021年から2023年までの乗客数の平均（小数点以下切り捨て）
解説（正解: (4)）
【データの読み取り】
・2019年：乗客数850万人、搭乗率82%
・2020年：乗客数320万人、搭乗率45%
・2021年：乗客数480万人、搭乗率68%
・2022年：乗客数720万人、搭乗率78%
・2023年：乗客数780万人、搭乗率85%

【(A)の計算】
乗客数の最大は2019年の850万人、最小は2020年の320万人
A = 850 - 320 = 530

【(B)の計算】
搭乗率75%以上の年：2019年(82%)、2022年(78%)、2023年(85%)
B = 850 + 720 + 780 = 2350

【(C)の計算】
2021年～2023年の乗客数：480、720、780
平均 = (480 + 720 + 780) / 3 = 1980 / 3 = 660（小数点以下切り捨て）

【合計】
A + B + C = 530 + 2350 + 660 = 3540

正解：(4) 3540万
問2 配点: 6点 / 目安時間: 5分
図1と同じデータを用いて、以下の(A)～(D)の値をそれぞれ求め、A+B+C+Dの合計値として正しいものを１つ選び、解答用紙にマークせよ。なお、乗客数の下4桁は切り捨てて計算するものとする。(6点)
【図1】国内線の年間乗客数と平均搭乗率の推移（再掲）
(A) 前年比で乗客数が増加した年の数
(B) 各年について「乗客数（万人）×搭乗率（%）÷100」を計算し、その値が500以上となる年について、その値の合計（小数点以下切り捨て）
(C) 搭乗率が70%未満の年の乗客数の合計と、80%以上の年の乗客数の合計の差（大きい方から小さい方を引く）
(D) 2020年から2023年への乗客数の増加量
解説（正解: (4)）
【データの確認】
・2019年：乗客数850万人、搭乗率82%
・2020年：乗客数320万人、搭乗率45%
・2021年：乗客数480万人、搭乗率68%
・2022年：乗客数720万人、搭乗率78%
・2023年：乗客数780万人、搭乗率85%

【(A)の計算：前年比で増加した年の数】
・2019→2020：850→320（減少）
・2020→2021：320→480（増加）
・2021→2022：480→720（増加）
・2022→2023：720→780（増加）
A = 3（年）

【(B)の計算：乗客数×搭乗率÷100が500以上の年の合計】
・2019年：850×82÷100 = 697（500以上）
・2020年：320×45÷100 = 144（500未満）
・2021年：480×68÷100 = 326（500未満）
・2022年：720×78÷100 = 561（500以上）
・2023年：780×85÷100 = 663（500以上）
B = 697 + 561 + 663 = 1921

【(C)の計算：搭乗率70%未満と80%以上の乗客数の差】
・搭乗率70%未満：2020年(45%)、2021年(68%) → 320 + 480 = 800
・搭乗率80%以上：2019年(82%)、2023年(85%) → 850 + 780 = 1630
C = 1630 - 800 = 830

【(D)の計算：2020年から2023年への増加量】
D = 780 - 320 = 460

【合計】
A + B + C + D = 3 + 1921 + 830 + 460 = 3214

正解：(4) 3214
