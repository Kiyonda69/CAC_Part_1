問1 配点: 6点 / 目安時間: 3分
下図のように、A、B、C、Dの順に図形が並んでいる。この規則に従うとき、?に入る図形として正しいものを1つ選べ。
【図1】図形の系列
解説（正解: (3)）

図形の系列から2つの独立した規則を発見する必要がある。


【規則1の発見】外周の多角形

・A: 三角形（3辺）

・B: 四角形（4辺）

・C: 五角形（5辺）

・D: 六角形（6辺）

・?: 七角形（7辺）← 辺の数が1ずつ増加


【規則2の発見】内部の点の数

・A: 1個

・B: 3個

・C: 5個

・D: 7個

・?: 9個 ← 点の数が2ずつ増加（奇数列）


【選択肢の検討】

・(1) 七角形 + 7点 → 規則2に違反

・(2) 六角形 + 9点 → 規則1に違反

・(3) 七角形 + 9点 → 両規則を満たす

・(4) 八角形 + 9点 → 規則1に違反

・(5) 七角形 + 11点 → 規則2に違反


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下図において、2つの3x3グリッドに対して演算★を行うと、新たな3x3グリッドが得られる。例1と例2から演算★の規則を発見し、E★Fの結果として正しいものを1つ選べ。
【例1】A★B の結果
【例2】C★D の結果
【問題】E★F = ?
解説（正解: (3)）

2つの例から演算★の規則を帰納的に発見する必要がある。


【規則の発見】

例1を観察すると、AとBは全てのセルで色が異なり、結果は全て黒になっている。

例2を観察すると、CとDで同じ色のセルは白、異なる色のセルは黒になっている。


これらから、演算★の規則は以下の通り：

・同じ色（黒と黒、または白と白）→ 白

・異なる色（黒と白、または白と黒）→ 黒

（排他的論理和：XOR演算）


【E★Fの計算】

各セルについてXOR演算を適用する：

・1行目: (白,黒)→黒、(黒,黒)→白、(黒,白)→黒

・2行目: (黒,白)→黒、(白,白)→白、(白,黒)→黒

・3行目: (白,白)→白、(黒,白)→黒、(白,白)→白


【選択肢の検討】

・(1) AND演算の結果（両方黒のセルのみ黒）

・(2) OR演算の結果（どちらかが黒なら黒）

・(3) XOR演算の正しい結果

・(4) 計算ミスを誘う誤答

・(5) 一部のセルが異なる誤答


したがって、正解は(3)である。
