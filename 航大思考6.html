問1．以下の手順書に従って計算を行ったとき、最終的な「結果」の値として正しいものを1つ選び、解答用紙にマークせよ。(6点)
【手順書】
手順1: a に 10 を代入する 手順2: b に 4 を代入する 手順3: c に 2 を代入する 手順4: もし a が (b + c) より大きいならば a に (a - b) の値を代入する b に (b + c) の値を代入する そうでなければ c に (a - b) の値を代入する 手順5: もし b が c より大きいならば 結果 に (a + c) の値を代入する そうでなければ 結果 に (b - c) の値を代入する
(1) 8
(2) 6
(3) 10
(4) 12
(5) 4
問1(正解:(1))

【手順の追跡】
手順1〜3: a = 10, b = 4, c = 2

手順4の条件判定:
・a > (b + c) → 10 > (4 + 2) → 10 > 6 → 真
・条件が真なので:
　・a = a - b = 10 - 4 = 6
　・b = b + c = 4 + 2 = 6
・現在の値: a = 6, b = 6, c = 2

手順5の条件判定:
・b > c → 6 > 2 → 真
・条件が真なので:
　・結果 = a + c = 6 + 2 = 8

したがって、結果は8である。
正解:(1) 8
問2．以下の手順書に従って計算を行ったとき、最終的な「結果」の値として正しいものを1つ選び、解答用紙にマークせよ。(6点)
【手順書】
手順1: リスト に [3, 7, 2, 8, 4] を代入する 手順2: 合計 に 0 を代入する 手順3: 個数 に 0 を代入する 手順4: リストの各要素 x について、以下を繰り返す もし x が偶数 かつ x が 3 より大きいならば 合計 に (合計 + x) の値を代入する 個数 に (個数 + 1) の値を代入する そうではなく、もし x が奇数ならば 合計 に (合計 + 1) の値を代入する 手順5: 結果 に (合計 × 個数) の値を代入する
※ 偶数とは2で割り切れる整数（2, 4, 6, 8, ...）
※ 奇数とは2で割り切れない整数（1, 3, 5, 7, ...）
(1) 14
(2) 20
(3) 24
(4) 28
(5) 32
問2(正解:(4))

【手順の追跡】
手順1〜3: リスト = [3, 7, 2, 8, 4], 合計 = 0, 個数 = 0

手順4の繰り返し処理:

x = 3 のとき:
・3は奇数なので「偶数かつx>3」は偽
・「奇数」は真 → 合計 = 0 + 1 = 1
・現在: 合計 = 1, 個数 = 0

x = 7 のとき:
・7は奇数なので「偶数かつx>3」は偽
・「奇数」は真 → 合計 = 1 + 1 = 2
・現在: 合計 = 2, 個数 = 0

x = 2 のとき:
・2は偶数だが、2 > 3 は偽 → 「偶数かつx>3」は偽
・2は偶数なので「奇数」も偽
・どちらの条件も満たさない → 何もしない
・現在: 合計 = 2, 個数 = 0

x = 8 のとき:
・8は偶数かつ 8 > 3 → 「偶数かつx>3」は真
・合計 = 2 + 8 = 10, 個数 = 0 + 1 = 1
・現在: 合計 = 10, 個数 = 1

x = 4 のとき:
・4は偶数かつ 4 > 3 → 「偶数かつx>3」は真
・合計 = 10 + 4 = 14, 個数 = 1 + 1 = 2
・現在: 合計 = 14, 個数 = 2

手順5:
・結果 = 合計 × 個数 = 14 × 2 = 28

したがって、結果は28である。
正解:(4) 28
