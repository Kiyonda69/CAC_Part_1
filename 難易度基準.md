航空大学校入学試験 総合Part I 思考パズル問題
難易度目安

本文書は、問題作成時の難易度基準として使用するための参考資料です。
問題1（3分・標準難度）と問題2（5分・高難度）を基準として示します。

【問題1】二重規則パターン（3分・標準難度）

■ 問題概要

3つの図形（A→B→C→?）が順に並び、規則を発見して?を予測する。
各図形は「外周の多角形」と「中心からの放射線」の2要素で構成される。

■ 規則

規則1：放射線の本数が1本ずつ増加
  A:1本 → B:2本 → C:3本 → ?:4本

規則2：外周の多角形の辺が1つずつ増加
  A:三角形(3辺) → B:四角形(4辺) → C:五角形(5辺) → ?:六角形(6辺)

■ 難易度要素

・2つの独立した規則を同時に把握する必要がある
・どちらか一方の規則だけでは正解できない設計
・選択肢は各規則の片方だけ正しいものを配置

■ 選択肢設計

(1) 放射線4本 + 五角形 → 規則2違反
(2) 放射線3本 + 六角形 → 規則1違反
(3) 放射線4本 + 六角形 → 正解★
(4) 放射線5本 + 六角形 → 規則1違反
(5) 放射線4本 + 七角形 → 規則2違反

■ 想定解答プロセス

1. 図形が2つの要素で構成されていることに気づく（30秒）
2. 放射線の本数変化を発見する（20秒）
3. 多角形の辺数変化を発見する（30秒）
4. 両規則を?に適用して答えを導出（20秒）
5. 選択肢から該当するものを選択（20秒）
合計：約120秒（2分）、余裕を含めて3分

■ 測定能力

・パターン認識（2つの独立した規則の発見）
・視覚的分析（図形の構成要素の分離）
・論理的推論（規則の適用）


【問題2】排他的論理和(XOR)パターン（5分・高難度）

■ 問題概要

2つの3×3グリッド図形に対して演算★を行い、結果を予測する。
2つの例（A★B=結果1、C★D=結果2）から演算規則を帰納的に発見し、
問題（E★F=?）に適用する。

■ 規則

排他的論理和（XOR）演算：
・黒★黒 → 白（同じ色なら白）
・黒★白 → 黒（異なる色なら黒）
・白★黒 → 黒（異なる色なら黒）
・白★白 → 白（同じ色なら白）

言い換え：「同じ色なら白、異なる色なら黒」

■ 難易度要素

・規則が明示されず、2つの例から帰納的に発見する必要がある
・XOR（排他的論理和）という論理演算の理解が求められる
・9セル全てを正確に計算する必要があり、ミスが許されない
・AND演算やOR演算と混同しやすい
・紛らわしい誤答が多数配置されている

■ 選択肢設計

(1) AND演算結果（両方黒のみ黒）→ 演算規則の誤解を誘う
(2) OR演算結果（どちらか黒なら黒）→ 演算規則の誤解を誘う
(3) XOR結果の2箇所間違い → 計算ミスを誘う
(4) XOR演算の正しい結果 → 正解★
(5) XOR結果の左右反転 → 計算ミス・混乱を誘う

■ 想定解答プロセス

1. 例1を分析し、セルごとの対応関係を観察（40秒）
2. 「同じ色→白、異なる色→黒」という仮説を立てる（30秒）
3. 例2で仮説を検証する（40秒）
4. 問題E★Fの9セルを計算する（90秒）
5. 選択肢と照合して確認（30秒）
合計：約230秒（約4分）、余裕を含めて5分

■ 測定能力

・帰納的推論（例から規則を発見）
・論理演算の理解（XOR概念）
・注意力・正確性（9セルの計算）
・仮説検証能力（例2での確認）

問題作成時の指針

【3分問題（標準難度）を作成する場合】
・問題1を参考に、複数の単純な規則を組み合わせる
・規則自体は直感的に理解できるものを使用
・選択肢は「片方の規則だけ正しい」パターンで作成

【5分問題（高難度）を作成する場合】
・問題2を参考に、規則の発見自体を問題の核心とする
・例を複数提示し、帰納的推論を要求する
・論理演算、数学的関係、暗号的変換などを使用
・選択肢は類似の演算結果や部分的な間違いを配置
