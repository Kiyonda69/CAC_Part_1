# CLAUDE.md - プロジェクトガイドライン

## プロジェクト概要

航空大学校入学試験「総合Part I」科目の思考パズル問題を作成するプロジェクト。
論理的推論、空間視覚化、パターン認識、数学的思考を測定する問題を作成する。
Python検証により解の一意性を保証する。

## 言語

- 主要言語: **日本語**
- 問題内容、解説、ドキュメントはすべて日本語で記述

## 測定能力

- 論理推論
- 空間視覚化
- パターン認識
- 数学的思考
- 制約充足
- 資料読み取り

## 問題仕様

### 作成ルール
- **2問1セット**: 標準難度（問1）と高難度（問2）をペアで作成
- **出力形式**: 2問1セットで1つのHTMLファイルを生成
- **ファイル命名**: `航大思考{N}.html`（Nはセット番号: 1, 2, 3, ...）
  - 例: 航大思考1.html（問1・問2）、航大思考2.html（問1・問2）...

### 難易度別仕様

| 項目 | 問題1（標準） | 問題2（高難度） |
|------|-------------|----------------|
| 配点 | 6点 | 6点 |
| 制限時間 | 3分 | 5分 |
| 難易度 | ★★★★★ | ★★★★★★★★ |
| 解答時間目安 | 170秒 | 310秒 |

- 問題2は問題1の派生・高難度版として作成

## 作成ワークフロー

1. **設計**: 問題タイプを選択（制約充足、暗号解読、空間推論、パターン認識、データ分析）
2. **Python検証**: 総当たりで解の一意性を確認
3. **正解番号ランダム化**: `python3 -c "import random; print(random.randint(1, 5))"` を実行
4. **HTML作成**: `template.html` をベースに、指定セクションのみ編集

## ファイル構成

| ファイル | 説明 |
|---------|------|
| 航大思考{N}.html | 問題出力先（2問1セットで生成） |
| template.html | HTMLテンプレート（構造・スタイル変更禁止） |
| 難易度基準.md | 難易度基準の参考資料 |
| CLAUDE.md | Claude Code用ガイドライン（このファイル） |

## HTML編集ルール

- テンプレート: 必ず `template.html` を使用
- 編集範囲: `<!-- ここから問題セクション -->` から `<!-- 解答・解説セクション -->` の間のみ
- カラー: 白黒のみ（グレースケール可）
- 出力形式: 2問1セットで `航大思考{N}.html` として生成（Nはセット番号）
  - 各ファイルに問1（標準難度）と問2（高難度）を含める

### レイアウト構成

各問題は「問題文 → 図表 → 選択肢 → 解説」の順で配置する。解説は各問題の直下に配置し、まとめて最後に配置しない。

```
問1
├── 問題文
├── 図表
├── 選択肢
└── 解説（問1の解説をここに配置）

問2
├── 問題文
├── 図表
├── 選択肢
└── 解説（問2の解説をここに配置）
```

解説セクションのスタイル:
```html
<div style="margin-top: 30px; padding-top: 20px; border-top: 1px solid #ccc;">
    <!-- 解説内容 -->
</div>
```

## SVG座標ガイドライン

SVG図表作成時は、各SVGの前にコメントで座標系を文書化する:

```html
<!--
座標系定義:
- 原点: (50, 250)
- X軸: 右方向, 1単位=30px
- Y軸: 上方向, 1単位=40px (SVGは下が正なので減算)
-->
```

作成後はPythonで座標を検証し、軸ラベルとグラフ要素の位置が一致することを確認する。

```python
def verify_solution():
    """全ての可能性を検証し、唯一解を確認"""
    valid_solutions = []
    # 全パターンを総当たり
    # 制約条件を適用
    # 解の一意性を確認
    assert len(valid_solutions) == 1, f"解が{len(valid_solutions)}個存在"
    return valid_solutions[0]
```

## 品質チェックリスト

- [ ] Python検証で解が唯一であることを確認
- [ ] すべての条件が解導出に必要
- [ ] 矛盾する条件が存在しない
- [ ] SVG座標が意図した位置と一致
- [ ] 表のデータが計算結果と一致
- [ ] 正解番号がランダム化されている
- [ ] 170-310秒で解答可能
- [ ] 複数ステップの推論が必要

## 禁止事項

- 視覚的矛盾（グラフ値とテキスト値の不一致）
- 自明な解
- 航空専門知識を必要とする問題設計
- Python検証の省略
- 正解番号の手動選択（特に1番や5番への固定）
- フォーマルでない装飾文字の使用（✓、✗、◯、×などの記号）
