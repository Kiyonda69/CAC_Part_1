問1．5人のパイロット候補生A、B、C、D、Eがシミュレータ訓練を受け、それぞれ異なるスコア（60点、70点、80点、90点、100点のいずれか）を獲得した。以下の(ア)～(エ)の条件をすべて満たすとき、Aのスコアとして正しいものを１つ選び、解答用紙にマークせよ。(6点)
(ア) AのスコアはBのスコアより10点高い
(イ) CとDのスコアの合計は160点である
(ウ) Eのスコアは70点ではない
(エ) Cは最高点を獲得した
(1) 60点
(2) 70点
(3) 80点
(4) 90点
(5) 100点
問2．6人のパイロット候補生A、B、C、D、E、Fがシミュレータ訓練を受け、それぞれ異なるスコア（50点、60点、70点、80点、90点、100点のいずれか）を獲得した。以下の(ア)～(オ)の条件をすべて満たすとき、Cのスコアとして正しいものを１つ選び、解答用紙にマークせよ。(6点)
(ア) AとFのスコアの合計は150点である
(イ) BとEのスコアの合計は130点である
(ウ) CのスコアはDのスコアより10点高い
(エ) Aのスコアは80点より高い
(オ) Eのスコアは60点より高い
(1) 70点
(2) 90点
(3) 80点
(4) 100点
(5) 60点
問1(正解:(3))
条件を整理して論理的に各人のスコアを決定する。

【条件の整理】
・(エ)より: Cは最高点 → C = 100点(確定)
・(イ)より: C + D = 160 → 100 + D = 160 → D = 60点(確定)
・(ア)より: A = B + 10
・残りのスコア: 70点、80点、90点をA、B、Eに割り当て

【A、Bの決定】
・A = B + 10 を満たす組み合わせ:
　- B = 70 → A = 80
　- B = 80 → A = 90

【Eの決定】
・(ウ)より: Eは70点ではない
・B = 70, A = 80 の場合: E = 90（70点ではないので条件を満たす）
・B = 80, A = 90 の場合: E = 70（70点なので条件を満たさない）

【最終解】
A = 80点、B = 70点、C = 100点、D = 60点、E = 90点

したがって、Aのスコアは80点である。
正解:(3) 80点
問2(正解:(2))
複数の条件を組み合わせて段階的に解を導く。

【条件の整理】
・(エ)より: A > 80 → Aは90点か100点
・(ア)より: A + F = 150
　- A = 90 → F = 60
　- A = 100 → F = 50

【BとEの決定】
・(オ)より: E > 60 → Eは70点、80点、90点、100点のいずれか
・(イ)より: B + E = 130
　- E = 70 → B = 60
　- E = 80 → B = 50
　- E = 90 → B = 40（範囲外）
　- E = 100 → B = 30（範囲外）

【組み合わせの検討】
・A = 90, F = 60 の場合:
　- E = 70, B = 60 → Fと重複（不可）
　- E = 80, B = 50 → 残り C, D に 70, 100
　　C - D = 10 を満たす組み合わせなし（100-70=30）

・A = 100, F = 50 の場合:
　- E = 70, B = 60 → 残り C, D に 80, 90
　　(ウ) C - D = 10 → C = 90, D = 80（条件を満たす）
　- E = 80, B = 50 → Fと重複（不可）

【最終解】
A = 100点、B = 60点、C = 90点、D = 80点、E = 70点、F = 50点

したがって、Cのスコアは90点である。
正解:(2) 90点
