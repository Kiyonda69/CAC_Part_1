問1 配点: 6点 / 目安時間: 3分
飛行機の進行方向を映し、ある方位面の断面を示す垂直線と、ある高度面の断面を示す水平線が表示されている計器がある。下図に示す4つの表示は時間経過に伴う計器の遷移を示しており、表示と表示の間 A・B・C で機首上げ・機首下げ・右旋回・左旋回のいずれか1つの操作が行われた。A・B・C で行われた操作の組み合わせとして正しいものを選べ。
解説（正解: (2)）

計器の表示は飛行機（中央の白い四角）から見た方位面・高度面の相対位置を示す。


【操作と線の動きの対応】

・機首上げ … 機体が高度面より上昇し、水平線（高度面）は表示上「下」に動く。

・機首下げ … 機体が高度面より下降し、水平線は表示上「上」に動く。

・右旋回 … 機体が方位面の右側へ移動し、垂直線（方位面）は表示上「左」に動く。

・左旋回 … 機体が方位面の左側へ移動し、垂直線は表示上「右」に動く。


【各遷移で動いた線】

・A：水平線が「上」→「中央」へ下方向に移動 → 機首上げ

・B：垂直線が「左」→「中央」へ右方向に移動 → 左旋回

・C：水平線が「中央」→「下」へ下方向に移動 → 機首上げ


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ計器について、下図に示すように 5 つの表示が連続して観測された。表示と表示の間 A・B・C・D で機首上げ・機首下げ・右旋回・左旋回のいずれか1つの操作が行われた。線のずれ幅は均等に1段階ずつであり、どの遷移も垂直線か水平線のどちらか一方だけが動く。A・B・C・D で行われた操作の組み合わせとして正しいものを選べ。
解説（正解: (3)）

操作と線の動きの対応は問1と同じ。本問では線のずれは「やや左／中央」「中央／やや上」の2段階のみで、各遷移は1段階だけ動く。


【各遷移で動いた線】

・A：垂直線が「やや左」→「中央」へ右方向に移動 → 左旋回

・B：水平線が「中央」→「やや上」へ上方向に移動 → 機首下げ

・C：垂直線が「中央」→「やや左」へ左方向に移動 → 右旋回

・D：水平線が「やや上」→「中央」へ下方向に移動 → 機首上げ


【別解：観察ポイント】

D1とD5の表示は同一であり、4回の操作で元の位置に戻っている。よって「左旋回1回＋右旋回1回」と「機首下げ1回＋機首上げ1回」が必ず含まれる。この条件を満たすのは(2)と(3)のみで、Bの遷移で水平線が上に動く事実から(3)が正解と確定する。


したがって、正解は(3)である。
