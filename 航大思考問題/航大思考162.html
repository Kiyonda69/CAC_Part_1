問1 配点: 6点 / 目安時間: 3分
下の(1)～(5)は、それぞれ目が1～6までのサイコロの展開図の候補である。組み立てたとき、向かい合う面の数の和がすべて7になるサイコロになるものを1つ選べ。
解説（正解: (2)）

十字型展開図 [上] / [左][中][右][奥] / [下] では、組み立てたとき向かい合う面のペアは次の3組である。

・上 ⇔ 下（中央列の最上段と最下段）

・左 ⇔ 右（横一列の左端と中央右）

・中 ⇔ 奥（横一列の中央左と右端）

すなわち、横4マス [b][c][d][e] のうち b⇔d、c⇔e がそれぞれ向かい合う。


【選択肢の検討】

・(1) [上1, 左2, 中3, 右4, 奥5, 下6] → 1+6=7◯、2+4=6 ✗

・(2) [上2, 左3, 中1, 右4, 奥6, 下5] → 2+5=7◯、3+4=7◯、1+6=7◯ ★

・(3) [上1, 左3, 中2, 右5, 奥4, 下6] → 1+6=7◯、3+5=8 ✗

・(4) [上1, 左4, 中3, 右2, 奥5, 下6] → 1+6=7◯、4+2=6 ✗

・(5) [上3, 左1, 中2, 右5, 奥4, 下6] → 3+6=9 ✗


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下図は、目が1～6までのサイコロの展開図である（向かい合う面の数の和は7）。これを組み立てて、目が4の面を上、目が2の面を正面（手前）に向けたとき、右側面に来る目の数はいくつか。
【図】サイコロの展開図
解説（正解: (1)）

展開図から、向かい合う面のペアは 1⇔6、2⇔5、3⇔4。

題意より上面=4、前面=2 なので、下面=3、後面=5、残る左右は1と6のいずれかとなる。


【展開図から組み立てる手順】

展開図の中央 [1] を前面、その上 [3] を上面とすれば、左 [2] が左面、右 [5] が右面、その先 [6] が後面、下 [4] が下面となる。

初期状態:　上=3、下=4、前=1、後=6、左=2、右=5。


【回転手順】

① 上=4 にするため、サイコロを左方向に180°倒す（左に2回倒す）。

　 → 上=4、下=3、前=1、後=6、左=5、右=2。

② 前=2 にするため、上下軸まわりに反時計回りに90°回転させる（上から見て）。

　 → 上=4、下=3、前=2、後=5、左=1、右=6。


よって、上面=4、前面=2 のとき、右側面に来る目は6。


したがって、正解は(1)である。
