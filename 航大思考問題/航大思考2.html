問1 配点: 6点 / 目安時間: 3分
下の図は、3つの属性（形・塗り・点の数）を持つ図形を、ある規則に従って3×3のマトリクスに配置したものである。各行・各列において、それぞれの属性は異なる値をとる。「?」のマスに入る図形として正しいものを1つ選べ。
【図1】図形マトリクス
※ 各図形には「形」「塗り」「点の数」の3つの属性がある
解説（正解: (2)）

マトリクスの各マスの図形には「形」「塗り」「点の数」の3つの属性があり、各行・各列に3種類の値が1つずつ出現する（ラテン方陣構造）。


【規則1: 形の巡回】

行1: ○ → △ → □

行2: □ → ○ → △（1つシフト）

行3: △ → □ → ? → ?=○


【規則2: 塗りの巡回】

行1: 空 → 斜線 → 黒

行2: 斜線 → 黒 → 空（1つシフト）

行3: 黒 → 空 → ? → ?=斜線


【規則3: 点の数】

行1: 1, 2, 3 / 行2: 3, 1, 2 / 行3: 2, 3, ?

各行・各列に1,2,3が1つずつ → ?=1


したがって、正解は(2) ○・斜線・1点である。

問2 配点: 6点 / 目安時間: 5分
下の図は、3×3の白黒パターンを3×3のマトリクスの各マスに配置したものである。各行および各列のパターンの間には共通の規則がある。「?」のマスに入るパターンとして正しいものを1つ選べ。
【図2】白黒パターンマトリクス
※ 各マスには3×3の白黒パターンが入る
解説（正解: (3)）

【規則の発見】

各行の3つのパターンに着目する。1行目のパターンをA(列1), B(列2), C(列3)とすると、各マスを比較した場合、Cは「AとBの排他的論理和（XOR）」で得られる。


XOR規則: 2つのマスが同じ色→白、異なる色→黒


【1行目で検証】

A(行1列1)とB(行1列2)のXORがC(行1列3)と一致:

左上: A=黒, B=白 → 異なる → 黒 → C=黒（一致）

中央: A=黒, B=白 → 異なる → 黒 → C=黒（一致）

右下: A=黒, B=黒 → 同じ → 白 → C=白（一致）

（全9マスで一致を確認可能）


【列方向にも同じ規則】

1列目: A XOR D = G、2列目: B XOR E = H


【答えの導出】

行方向: G XOR H = ?

列方向: C XOR F = ?

いずれも同じ結果: □□□ / ■□■ / ■□■


したがって、正解は(3)である。
