問1 配点: 6点 / 目安時間: 3分
ある研修所で5人の研修生（甲〜戊）が3科目（科目X、科目Y、科目Z）の筆記試験を受験した。下表はその結果の一部をまとめたものである。表中の空欄ア〜ウに当てはまる数値の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。
【表1】研修生の科目別得点
研修生 科目X 科目Y 科目Z 合計
甲 78 ア 64
乙 イ 70 82
丙 85 62 ウ 220
丁 72 80 76 228
戊 68 74 70 212
科目合計 375 356 365
解説（正解: (1)）

【解法のポイント】

行（横）の合計と列（縦）の合計の両方を利用して、空欄の値を求める。


【ア（甲の科目Y）の求め方】

科目Yの合計が356であることを利用する。

ア + 70 + 62 + 80 + 74 = 356

ア = 356 - 286 = 70


【イ（乙の科目X）の求め方】

科目Xの合計が375であることを利用する。

78 + イ + 85 + 72 + 68 = 375

イ = 375 - 303 = 72


【ウ（丙の科目Z）の求め方】

丙の行合計が220であることを利用する。

85 + 62 + ウ = 220

ウ = 220 - 147 = 73


【検算】

科目Zの合計: 64 + 82 + 73 + 76 + 70 = 365（一致）


したがって、ア=70、イ=72、ウ=73であり、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
ある地域の5つの営業所（甲〜戊）について、4月から6月の月別売上高（万円）を下表にまとめた。ただし、丙営業所の4月の売上高は丁営業所の4月の売上高より10万円多いことが分かっている。表中の空欄ア〜エに当てはまる数値の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。
【表2】営業所別の月別売上高（万円）
営業所 4月 5月 6月 合計
甲 240 ア 280
乙 180 200 イ
丙 ウ 170 190
丁 エ 150 160
戊 120 130 140 390
月合計 870 910 970 2750
解説（正解: (2)）

【解法のポイント】

列（月）の合計から求められる空欄と、条件を組み合わせた連立方程式で求める空欄がある。


【ア（甲の5月売上）の求め方】

5月の合計が910であることを利用する。

ア + 200 + 170 + 150 + 130 = 910

ア = 910 - 650 = 260


【イ（乙の6月売上）の求め方】

6月の合計が970であることを利用する。

280 + イ + 190 + 160 + 140 = 970

イ = 970 - 770 = 200


【ウとエ（丙・丁の4月売上）の求め方】

4月の合計が870であることから、

240 + 180 + ウ + エ + 120 = 870

ウ + エ = 330 ・・・(i)


条件「丙の4月売上は丁の4月売上より10万円多い」から、

ウ = エ + 10 ・・・(ii)


(ii)を(i)に代入すると、

(エ + 10) + エ = 330

2エ = 320

エ = 160、ウ = 160 + 10 = 170


【検算】

4月合計: 240 + 180 + 170 + 160 + 120 = 870（一致）

総合計: 780 + 580 + 530 + 470 + 390 = 2750（一致）


【選択肢の検討】

・(1) ウ=160、エ=170 → 条件「丙 > 丁」に矛盾（160 < 170）

・(2) 全て一致 → 正解

・(3) ア=250 → 5月合計が900になり不一致

・(4) ウ+エ=330だが ウ-エ=30 → 条件「差が10」に不一致

・(5) イ=210 → 6月合計が980になり不一致


したがって、正解は(2)である。
