問1．ある「変換装置A」に3つの数字を入力すると、特定の法則に従って変換された3つの数字が出力される。図1の入出力例から法則を推論し、入力(4, 2, 5)に対する出力として正しいものを１つ選び、解答用紙にマークせよ。(6点)
【図1】変換装置Aの入出力例
入力 出力
(1, 2, 3) → (3, 2, 2)
(2, 3, 4) → (4, 3, 4)
(3, 1, 2) → (2, 1, 6)
(2, 5, 1) → (1, 5, 4)
※ 変換装置Aは2つの法則を組み合わせて数字を変換する
入力: (4, 2, 5) → 出力: ?
(1) (8, 2, 5)
(2) (5, 4, 8)
(3) (5, 2, 8)
(4) (8, 4, 5)
(5) (4, 2, 10)
問1(正解:(3))
入出力例から2つの法則を推論する。

【法則の発見】
例1: (1, 2, 3) → (3, 2, 2)
・入力の3番目「3」が出力の1番目に → 順序が逆転している可能性
・入力の1番目「1」が出力の3番目で「2」に → 2倍されている

例2: (2, 3, 4) → (4, 3, 4)
・入力「4」が出力1番目、入力「2」×2=4が出力3番目 → 仮説と一致

例3: (3, 1, 2) → (2, 1, 6)
・入力「2」が出力1番目、入力「3」×2=6が出力3番目 → 仮説と一致

【法則の確定】
法則1: 1番目の数字を2倍する
法則2: 数字の順序を逆にする

【解答の導出】
入力: (4, 2, 5)
法則1適用: (8, 2, 5) ← 4×2=8
法則2適用: (5, 2, 8) ← 逆順

したがって、出力は(5, 2, 8)である。
正解:(3) (5, 2, 8)
問2．「変換装置B」は変換装置Aよりも複雑な法則で数字を変換する。図2の入出力例から法則を推論し、入力(5, 2, 4)に対する出力として正しいものを１つ選び、解答用紙にマークせよ。(6点)
【図2】変換装置Bの入出力例
入力 出力
(2, 3, 1) → (2, 5, 4)
(3, 1, 2) → (4, 4, 6)
(4, 2, 3) → (6, 6, 8)
(1, 4, 2) → (4, 5, 4)
※ 変換装置Bは4つの法則を組み合わせて数字を変換する
※ 法則は一定の順序で適用される
入力: (5, 2, 4) → 出力: ?
(1) (8, 7, 8)
(2) (8, 7, 10)
(3) (8, 2, 8)
(4) (4, 7, 8)
(5) (8, 8, 7)
問2(正解:(1))
入出力例から4つの法則を推論する。

【法則の発見過程】
例1: (2, 3, 1) → (2, 5, 4)
例2: (3, 1, 2) → (4, 4, 6)
例3: (4, 2, 3) → (6, 6, 8)
例4: (1, 4, 2) → (4, 5, 4)

【観察1: 3番目の変化】
入力の3番目: 1→2, 2→4, 3→6, 2→4 → すべて2倍されている
ただし出力位置が異なる → 順序も変わっている

【観察2: 1番目の変化（偶奇による分岐）】
・入力2(偶数)→出力3番目が4=2×2
・入力3(奇数)→出力3番目が6=3+3
・入力4(偶数)→出力3番目が8=4×2
・入力1(奇数)→出力3番目が4=1+3
→ 偶数なら2倍、奇数なら+3

【観察3: 2番目の変化】
・(2,3,1): 3+2=5 → 出力2番目が5
・(3,1,2): 1+3=4 → 出力2番目が4
→ 2番目に元の1番目の値を加算

【法則の確定】
法則1: 1番目が偶数なら2倍、奇数なら+3
法則2: 2番目に元の1番目の値を足す
法則3: 3番目を2倍する
法則4: 全体を逆順にする

【解答の導出】
入力: (5, 2, 4)
法則1: 5は奇数 → 5+3=8 → (8, 2, 4)
法則2: 2+5=7 → (8, 7, 4)
法則3: 4×2=8 → (8, 7, 8)
法則4: 逆順 → (8, 7, 8)

したがって、出力は(8, 7, 8)である。
正解:(1) (8, 7, 8)
