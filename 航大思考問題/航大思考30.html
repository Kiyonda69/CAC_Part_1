問1 配点: 6点 / 目安時間: 3分
ある管制空域において、5機の航空機A機〜E機がレーダーに映っている。各機体は4方角（北・東・南・西）と3距離圏（第1圏・第2圏・第3圏）の交点に1機ずつ存在する。各パイロットから管制官への報告を下表に示す。ただしB機のみ通信不良のため報告が不明（？）である。次のア〜ウについて、正しいものの組み合わせを選べ。
【レーダー図】
機体報告内容
A機「私は北方向の第2圏にいる。」
B機？（通信不良）
C機「私は南方向の第3圏にいる。」
D機「私は西方向の第1圏にいる。第1圏には私を含めて3機がいる。」
E機「私は南方向の第1圏にいる。」
ア．B機の報告がなくても、5機全ての位置が確定する。
イ．B機の報告がない場合、B機の位置として2通りが考えられる。
ウ．B機の報告がない場合、B機以外の4機の位置は全て確定する。
解説（正解: (5)　イ、ウが正しい）

【各機体の位置を確定する】

各パイロットの報告（B機除く）から位置を特定する。

・A機：「北方向の第2圏」→ A機＝（北、第2圏）と確定。

・C機：「南方向の第3圏」→ C機＝（南、第3圏）と確定。

・E機：「南方向の第1圏」→ E機＝（南、第1圏）と確定。

・D機：「西方向の第1圏、第1圏は計3機」→ D機＝（西、第1圏）と確定。

　　第1圏の3機はD機（西）・E機（南）・B機（不明）→ B機は第1圏に存在。


【B機の候補を絞り込む】

B機は第1圏にいるが、方角が不明。

・西・第1圏 → D機が占有（不可）

・南・第1圏 → E機が占有（不可）

・北・第1圏 → 空き → 候補①

・東・第1圏 → 空き → 候補②

よってB機の候補は（北・第1圏）または（東・第1圏）の2通り。


【各命題の検討】

ア．B機報告なしで全位置確定するか？ → B機が2通りあり確定しない。ア：誤り。

イ．B機の位置として2通りが考えられるか？ → 北・第1圏 と 東・第1圏 の2通り。イ：正しい。

ウ．B機以外の4機は全て確定するか？ → A・C・D・E機は報告から一意に確定。ウ：正しい。


したがって、正解は(5)　イ、ウが正しい。

問2 配点: 6点 / 目安時間: 5分
教室に5つの席（席1〜席5）が左から右へ一列に並んでいる。5人の生徒P, Q, R, S, Tがそれぞれ異なる席に1人ずつ座る。次の条件1〜4が全て成り立つとき、以下のア〜ウについて正しいものの組み合わせを選べ。
【座席配置図】
条件内容
条件1Pは席2に座っている。
条件2QはPより右の席に座っている。
条件3RとSは隣り合う席に座っていない。
条件4Tは席1か席5のどちらかに座っている。
ア．Tが席1に座る場合、全員の席の配置は2通りある。
イ．Qの席として考えられるのは3通りである。
ウ．ある配置においてRとSの席を入れ替えても、全ての条件を満たす配置が存在する。
解説（正解: (5)　ア、ウが正しい）

【条件の整理】

条件1よりP＝席2が確定。条件2よりQは席3・席4・席5のいずれか。条件4よりTは席1か席5。


【T＝席1の場合を検討する】

P＝席2、T＝席1が確定。Q, R, Sが席3・席4・席5に入る。

条件3（RとS隣接不可）を検討する。

・Q＝席3 → R, Sは席4・席5 → 席4と席5は隣接 → 不可。

・Q＝席4 → R, Sは席3・席5 → 席3と席5は隣接しない → 可。

・Q＝席5 → R, Sは席3・席4 → 席3と席4は隣接 → 不可。

よってT＝席1のとき、Q＝席4が確定。R, Sは席3と席5（2通り）。


【T＝席5の場合を検討する】

P＝席2、T＝席5が確定。Q, R, Sが席1・席3・席4に入る。

条件2よりQ＝席3またはQ＝席4（席1はPより左なので不可）。

・Q＝席3 → R, Sは席1・席4 → 隣接しない → 可（2通り）。

・Q＝席4 → R, Sは席1・席3 → 隣接しない → 可（2通り）。

合計で4通りの配置が可能。


【全有効配置のまとめ（全6通り）】

T＝席1: (T,P,R,Q,S)と(T,P,S,Q,R)の2通り。

T＝席5: (R,P,Q,S,T), (S,P,Q,R,T), (R,P,S,Q,T)がない..ではなく

　(R,P,Q,S,T), (S,P,Q,R,T), (S,P,R,Q,T), (R,P,S,Q,T) の4通り。


【各命題の検討】

ア．T＝席1のとき配置は2通りか？ → R,Sの入れ替えで2通り。ア：正しい。

イ．Qの席は3通りか？ → Q＝席3またはQ＝席4の2通り。イ：誤り。

ウ．RとSを入れ替えても条件を満たす配置があるか？ → 例えば(R＝席1, S＝席4, Q＝席3, T＝席5)を入れ替えると(S＝席1, R＝席4, Q＝席3, T＝席5)で、席1と席4は隣接せず全条件を満たす。ウ：正しい。


したがって、正解は(5)　ア、ウが正しい。
