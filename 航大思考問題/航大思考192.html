問1 配点: 6点 / 目安時間: 3分
次の5社のECサイト月次データを読み、各社の実績に関する記述として正しいものを(1)〜(5)の中から1つ選べ。
あるグループ企業5社（A〜E）の自社ECサイトにおける当月実績である。A社はアクセス数200千件、購入件数4,000件、売上8,000千円、返品件数120件であった。B社はアクセス数150千件、購入件数3,600件、売上7,200千円、返品件数72件である。C社はアクセス数250千件、購入件数5,000件、売上9,000千円、返品件数200件であった。D社はアクセス数120千件、購入件数3,000件、売上6,600千円、返品件数60件にとどまる。E社はアクセス数180千件、購入件数3,600件、売上7,920千円、返品件数144件であった。なお、購入率はアクセス数に対する購入件数の割合、客単価は売上を購入件数で割った値、返品率は購入件数に対する返品件数の割合とする。

解説（正解: (3)）

各社の3指標を計算する。


【購入率】（購入件数÷アクセス数）

A 2.0% / B 2.4% / C 2.0% / D 2.5%（最高） / E 2.0%


【客単価】（売上÷購入件数）

A 2,000円 / B 2,000円 / C 1,800円 / D 2,200円 / E 2,200円


【返品率】（返品件数÷購入件数）

A 3.0% / B 2.0% / C 4.0% / D 2.0% / E 4.0%


【選択肢の検討】

・(1) 客単価最高はD(2,200円)とE(2,200円)の同値で「D社のみ」ではない → 誤

・(2) 返品率最高はC(4.0%)とE(4.0%)の同値で、C社単独ではない → 誤

・(3) 購入率最高はD(2.5%)でありB(2.4%)を上回る単独最高 → 正

・(4) 客単価最低はC(1,800円)でありB(2,000円)ではない → 誤

・(5) 返品率最低はB・D(2.0%)でありA(3.0%)ではない → 誤


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
次の6店舗の月次経営データ（表2・表3）を読み、ア〜オの記述の正誤を判定して、正しいものの組み合わせを(1)〜(5)の中から1つ選べ。

解説（正解: (5)）

売上＝来客数（千人）×客単価（円）として各指標を計算する。


【売上（千円）】

A 32,000 / B 30,000 / C 30,000 / D 30,000 / E 30,000 / F 32,400（最多）


【営業利益（千円）】（売上−原価−固定費）

A 2,800（最少） / B 4,500（最多） / C 3,000 / D 3,000 / E 3,000 / F 4,312


【利益率】（営業利益÷売上）

A 8.75%（最低） / B 15.0%（最高） / C 10.0% / D 10.0% / E 10.0% / F 約13.3%


【1日あたり来客数（人）】（来客数÷営業日数）

A 1,600 / B 1,000（最少） / C 2,000 / D 1,250 / E 2,500（最多） / F 1,500


【記述の検討】

・ア 売上最多(F)≠営業利益最多(B) → 誤

・イ 利益率最高(B)＝1日あたり来客最少(B) → 正

・ウ 営業利益最少(A)＝利益率最低(A) → 正

・エ 1日あたり来客最多(E)≠売上最多(F) → 誤

・オ 営業利益最多(B)＝利益率最高(B) → 正


正しい記述はイ・ウ・オ。したがって、正解は(5)である。
