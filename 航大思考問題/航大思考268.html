問1 配点: 6点 / 目安時間: 3分
図のように、マス目の上のマスAに、上の面が1、手前の面が2、右の面が3のサイコロが置かれている。向かい合う面の目の和は7である。このサイコロを、すべらないように矢印の順にマスBまで4回転がしたとき、マスBで上を向いている面の目として正しいものを、次の(1)〜(5)のうちから1つ選べ。
【図1】サイコロの置き方と転がす経路
問1（正解: (1)）

サイコロの転がし問題は、1回転がすごとに「上の面の目」を追跡すれば必ず解ける。向かい合う面の目の和は7なので、上の面が分かれば底面は「7−上の面」で求められる。


【最初の置き方の整理】

上=1・手前=2・右=3 であるから、向かい合う面の和7より、底=6・奥=5・左=4 である。


【転がしの基本規則】

・右へ転がすと、左の面が上に来る（左→上→右→底の順に一周する）。

・手前へ転がすと、奥の面が上に来る。

・同じ向きに2回続けて転がすと、上の面と底面がちょうど入れ替わる。


【上の面の追跡】

経路は「右へ2回 → 手前へ2回」である。

・1回目（右へ）: 左の面が上に来る → 上=4

・2回目（右へ）: もとの底面が上に来る → 上=7−1=6

・3回目（手前へ）: 奥の面が上に来る → 上=5

・4回目（手前へ）: 上下が再び入れ替わる → 上=7−6=1

すなわち上の面は 1→4→6→5→1 と変化し、マスBでは1となる。


【近道】

「同じ向きに2回転がすと上下が入れ替わる」ことに気づけば、右へ2回で上=7−1=6、続けて手前へ2回で上=7−6=1 と、2手で答えが出る。


【選択肢の検討】

・(2) 2 → 手前の面の目と混同した誤り

・(3) 4 → 1回転がしただけの途中の目

・(4) 5 → 3回転がした途中の目

・(5) 6 → 右へ2回転がした時点の途中の目


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ置き方（上の面が1、手前の面が2、右の面が3。向かい合う面の目の和は7）でマスAに置いたサイコロを、図の矢印の順にすべらないようにマスBまで6回転がす。マスAからマスBまでにサイコロが乗った7つのマスについて、それぞれのマスでサイコロの底面（マスに接していた面）の目をすべて合計した値として正しいものを、次の(1)〜(5)のうちから1つ選べ。
【図2】サイコロの置き方と転がす経路
問2（正解: (5)）

問1と同様に、1回転がすごとに上の面の目を追跡する。各マスでの底面の目は、向かい合う面の和が7であることから「7−上の面」で求められる。最初のマスAの底面も合計に含める点に注意する。


【転がす向きの整理】

経路の矢印より、転がす順は「右 → 手前 → 右 → 右 → 奥 → 右」の6回である。

・右へ転がすと左の面が上に来る。

・手前へ転がすと奥の面が上に来る。

・奥へ転がすと手前の面が上に来る。


【上の面と底面の追跡】

最初の置き方は 上=1・手前=2・右=3（底=6・奥=5・左=4）。

・(4) 25 → 転がしの追跡を途中で誤ったときに生じやすい近い値


したがって、正解は(5)である。
