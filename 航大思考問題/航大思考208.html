・P-107：670 ○ / ・P-108：445 ○

・P-103：200÷95≒2.11 ○ / ・P-104：175÷145≒1.21 ○ / ・P-107：1.25 ○ / ・P-108：1.27 ○

・P-103：4.5 ○ / ・P-104：4.0 → 除外 / ・P-107：4.1 → 除外 / ・P-108：4.4 ○


【結論】全条件を満たすのはP-103とP-108。

6月販売数の合計：200 + 165 = 365個

問2 配点: 6点 / 目安時間: 5分

【条件】下記すべてを満たす支店を選ぶ
  • 条件1：利益（売上−経費）が1,500万円以上である
  • 条件2：社員1人あたりの顧客数が300人以上である
  • 条件3：平均評価が4.0以上である
  • 条件4：前年比成長率が5％以上である
問2（正解: (3)）

4つの条件を支店ごとに検証する。


【各支店の指標計算】

・店舗A：利益=1,700、顧客/社員=350.0、評価4.1、成長率4％

・店舗B：利益=2,800、顧客/社員≒322.2、評価3.9、成長率12％

・店舗C：利益=2,300、顧客/社員=325.0、評価4.0、成長率6％

・店舗D：利益=2,300、顧客/社員≒342.9、評価4.3、成長率3％

・店舗E：利益=1,300、顧客/社員=360.0、評価4.4、成長率10％


【条件適用】

条件1（利益≧1,500）：店舗E(1,300)が不可 → 残り A, B, C, D

条件2（顧客/社員≧300）：全店舗OK

条件3（評価≧4.0）：店舗B(3.9)が不可 → 残り A, C, D

条件4（成長率≧5％）：店舗A(4％)と店舗D(3％)が不可 → 残り 店舗C のみ


【選択肢の検討】

・(1)店舗A：成長率4％で条件4を満たさない

・(2)店舗B：評価3.9で条件3を満たさない

・(3)店舗C：全条件を満たす

・(4)店舗D：成長率3％で条件4を満たさない

・(5)店舗E：利益1,300万円で条件1を満たさない


したがって、正解は(3)である。
