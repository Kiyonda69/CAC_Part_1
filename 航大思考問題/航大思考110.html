問1 配点: 6点 / 目安時間: 3分
図は、滑走路に着陸しようとしている飛行機から滑走路を見た前方の景色である。この地点から、滑走路へ直行し着陸するまでの景色として誤っているものを1つ選べ。
解説（正解: (5)）

滑走路は地面上の長方形であり、その上空から前方を見ると遠近法により台形に見える。


【透視図法の原理】

・観察者は滑走路の上空にいるため、滑走路の遠い側（奥）は視界の上方に、近い側（手前）は視界の下方に見える。

・遠いほど小さく見えるため、上辺（奥）は狭く、下辺（手前）は広く見える。

・すなわち、常に「上辺 ＜ 下辺」の関係が成立する。


【進入中の見え方の変化】

・飛行機が滑走路に直行し近づくほど、台形は相似的に拡大する。

・ただし、拡大しても「上辺 ＜ 下辺」の関係は崩れない。

・接地直前では、台形の下辺がフレーム下端に迫り、両側面が視界の端まで広がる。


【選択肢の検討】

・(1) 上辺が下辺より狭い台形で、進入中の正常な拡大段階 → 正常

・(2) (1)よりさらに拡大し、両側が枠に近づく → 接地に近い段階として正常

・(3) 上辺が下辺より狭い中程度の台形 → 正常

・(4) 少し拡大した台形 → 正常

・(5) 上辺が下辺より広い逆台形になっている。これは遠い側が手前より広く見えることを意味し、観察者が滑走路を下から見上げる状況でしか起こらない。進入中の飛行機（上空から見下ろす）にはありえない → 誤り


したがって、正解は(5)である。

問2 配点: 6点 / 目安時間: 5分
図は、滑走路に着陸しようとしている飛行機から滑走路を見た前方の景色である。画面の中ほどに水平線（地平線）があり、その直下に台形の滑走路が見える。この地点から、機首の向き（ピッチ）を一定に保ったまま滑走路へ直進し接地するまでの景色として誤っているものを1つ選べ。
解説（正解: (3)）

地平線は常に観察者の眼の高さ（水平視線の先）にある。


【地平線の見かけの位置】

・飛行機がピッチ（機首の上下角）を一定に保っている限り、機体基準（フレーム）に対する地平線の見かけの高さは変わらない。

・高度が変わっても、機首が同じ角度であれば水平視線の方向は同じであり、フレーム内の地平線の位置は固定される。

・滑走路の遠い端（奥）は地面上の最遠点なので、その上辺は常に地平線上にくる。


【進入中の見え方の変化】

・飛行機が近づくと、台形は下方に広がり拡大する。

・台形の上辺（奥の辺）は地平線上から動かない。下辺（手前の辺）は視界の下へ広がっていく。


【選択肢の検討】

・(1) 地平線は元の位置、台形は適度に拡大 → 正常

・(2) 地平線は元の位置、台形がより拡大 → 正常（進入中）

・(3) 地平線が下方に移動しており、ピッチを一定に保った直進降下ではありえない（地平線が下がるのは機首を上げた場合）。また、台形上辺も下がった地平線上に描かれており矛盾 → 誤り

・(4) 地平線は元の位置、台形が視界端まで広がる → 接地直前として正常

・(5) 地平線は元の位置、台形が少し拡大 → 正常


したがって、正解は(3)である。
