問1 配点: 6点 / 目安時間: 3分
次の「倉庫在庫管理月次レポート」を読み、空欄（ア）に当てはまる数値として正しいものを(1)〜(5)から1つ選べ。
倉庫在庫管理月次レポート（2026年3月分）

当倉庫では商品A〜Dの4品目を管理している。当月末時点の在庫数は以下の通りである。
・4品目の合計在庫数は1,200個であった。
・商品Aの在庫数は商品Bの在庫数の2倍であった。
・商品Cの在庫数は商品Aの在庫数より80個少ない（ア）個であった。
・商品Dの在庫数は商品Bと商品Cの在庫数の合計と等しかった。
解説（正解: (1)）

商品BをxとするとA=2x、C=2x−80、D=x+(2x−80)=3x−80。

合計: 2x＋x＋(2x−80)＋(3x−80) = 8x−160 = 1,200 → x=170

各在庫数: A=340、B=170、C=260（=ア）、D=430

合計確認: 340＋170＋260＋430 = 1,200個 ✓

したがって正解は(1)の260個である。

問2 配点: 6点 / 目安時間: 5分
次の「月間商品販売実績レポート」を読み、空欄（ウ）に当てはまる数値として正しいものを(1)〜(5)から1つ選べ。
月間商品販売実績レポート（2026年3月分）

当月は商品P〜Tの5品目の販売実績をまとめた。
・5品目の月間販売合計は500個であった。
・商品Pの販売数は商品Qの販売数より30個多かった。
・商品Qの販売数は商品Rの販売数の1.5倍であった。
・商品Rと商品Sの販売数の合計は160個であった。
・商品Sの販売数は商品Tの販売数の2倍であった。
・商品Tは（ウ）個販売された。
解説（正解: (4)）

商品TをtとするとS=2t、R=160−2t、Q=240−3t、P=270−3t。

合計: (270−3t)＋(240−3t)＋(160−2t)＋2t＋t = 670−5t = 500 → t=34

各販売数: T=34（=ウ）、S=68、R=92、Q=138、P=168

合計確認: 168＋138＋92＋68＋34 = 500個 ✓

【解法のポイント】全商品をtで表現し、合計500の方程式を解く。直接Tへ遡れないため、合計式の構成が鍵となる。

したがって正解は(4)の34個である。
