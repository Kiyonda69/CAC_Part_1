問1 配点: 6点 / 目安時間: 3分
ある商品に、原価の何割かの利益を見込んで定価をつけた。この商品を定価の1割引で売ると560円の利益が得られ、定価の2割5分引で売ると280円の損失が出る。この商品の原価はいくらか。次の(1)〜(5)から1つ選べ。
解説（正解: (1)）

問1（正解: (1)）

原価が不明なだけでなく、見込んだ利益の割合も与えられていないため、定価も未知数である。原価を x 円、定価を y 円とおき、2つの売り方の条件から連立方程式を立てる。


【立式】

・定価の1割引で売ると560円の利益 → 0.9y − x = 560 …①

・定価の2割5分引で売ると280円の損失 → x − 0.75y = 280 …②


【求解】

①と②を辺々加えると x が消去され、

0.9y − 0.75y = 560 + 280 → 0.15y = 840 → y = 5,600円

①に代入して x = 0.9 × 5,600 − 560 = 5,040 − 560 = 4,480円


【別解（差に着目）】

1割引の売価と2割5分引の売価の差は定価の15％分。一方、利益560円と損失280円の差は 560 + 280 = 840円。よって定価の15％ = 840円 → 定価 = 5,600円。あとは同様。


【検算】

定価5,600円の1割引 = 5,040円 → 利益 5,040 − 4,480 = 560円。

定価5,600円の2割5分引 = 4,200円 → 損失 4,480 − 4,200 = 280円。いずれも条件と一致する。


【選択肢の検討】

・(1) 4,480円 → ①②をともに満たす。正解

・(2) 4,200円 → 2割5分引の売価（0.75 × 5,600）との取り違え。不正解

・(3) 4,760円 → 1割引の売価5,040円から、利益560円ではなく損失280円を引いた誤り。不正解

・(4) 5,040円 → 1割引の売価（0.9 × 5,600）との取り違え。不正解

・(5) 5,600円 → 定価との取り違え。不正解


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
ある店で、同じ商品を300個仕入れ、原価の5割の利益を見込んで定価をつけて売り出した。何個かは定価で売れたが、その後、定価の2割引にしたところ、定価で売れた個数のちょうど半分の個数が売れた。それでも残った分は、すべて定価の半額にして売り切った。その結果、全体の利益は仕入総額のちょうど27％となった。定価の半額で売った個数は何個か。次の(1)〜(5)から1つ選べ。
解説（正解: (1)）

問2（正解: (1)）

原価の実額は与えられていないが、利益の条件が仕入総額に対する割合で示されているため、原価を1個あたり x 円とおけば x は式の両辺から消える。定価で売れた個数を n 個とおくと、2割引で売れた個数は n/2 個、半額で売った個数は 300 − n − n/2 = 300 − 1.5n 個と表せる。


【各売価を整理】

・定価 = 1.5x 円 → 1個あたりの利益 0.5x 円

・2割引の売価 = 1.5x × 0.8 = 1.2x 円 → 1個あたりの利益 0.2x 円

・半額の売価 = 1.5x × 0.5 = 0.75x 円 → 1個あたり 0.25x 円の損失


【立式と求解】

全体の利益 = 0.5x × n + 0.2x × (n/2) − 0.25x × (300 − 1.5n)

= (0.5n + 0.1n − 75 + 0.375n)x = (0.975n − 75)x 円

これが仕入総額 300x 円の27％、すなわち 81x 円に等しいので、

0.975n − 75 = 81 → 0.975n = 156 → n = 160個

よって、半額で売った個数 = 300 − 160 − 80 = 60個


【検算】

原価を800円とすると、仕入総額 = 240,000円。定価1,200円で160個 → 192,000円、960円で80個 → 76,800円、600円で60個 → 36,000円。売上合計 = 304,800円。

総利益 = 304,800 − 240,000 = 64,800円。64,800 ÷ 240,000 = 0.27 = 27％ となり、条件と一致する。


【選択肢の検討】

・(1) 60個 → 0.975n = 156 → n = 160 から導かれる値。正解

・(2) 40個 → 2割引で売れた個数を「定価で売れた個数の半分」ではなく「定価と同数」と誤読して立式（1.2n − 75 = 81 → n = 130、300 − 260 = 40）した値。不正解

・(3) 80個 → 2割引で売れた個数（160 ÷ 2）との取り違え。不正解

・(4) 140個 → 定価以外で売れた個数の合計（80 + 60）との取り違え。不正解

・(5) 160個 → 定価で売れた個数 n との取り違え。設問は半額で売った個数を問うている。不正解


したがって、正解は(1)である。
