問1 配点: 6点 / 目安時間: 3分
飛行機の進行方向を映し、ある方位面の断面を示す垂直線と、ある高度面の断面を示す水平線が表示されている計器がある。右図は飛行機が設定された方位と高度を適正に飛行している状態を示している。下の4つの表示の間A, B, Cで行われた操作として推測できる正しい組み合わせを1つ選べ。

※ 垂直線が中央→設定方位上、水平線が中央→設定高度上。各線は偏差に応じて中央からずれる。

（右図）正常な飛行状態
【4つの表示の変化】
解説（正解: (2)）

【計器の読み方】

・垂直線（方位偏差）：左偏位 = 機体が方位より右、右偏位 = 機体が方位より左

・水平線（高度偏差）：上偏位 = 機体が高度より低い、下偏位 = 機体が高度より高い


【操作A（表示1→表示2）】

垂直線：左→中央（方位右偏を修正、つまり左旋回）

水平線：中央→下（設定高度より高くなった、つまり上昇）

→ A は「左旋回しながら上昇」


【操作B（表示2→表示3）】

垂直線：中央→右（方位左偏が発生、つまり左旋回しすぎ）

水平線：下→中央（高度が正常に戻った、つまり降下）

→ B は「左旋回しながら降下」


【操作C（表示3→表示4）】

垂直線：右→中央（方位左偏を修正、つまり右旋回）

水平線：中央→上（設定高度より低くなった、つまり降下）

→ C は「右旋回しながら降下」


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ計器を使用している。右図は飛行機が設定された方位と高度を適正に飛行している状態を示している。下の5つの表示の間A, B, C, Dで行われた操作として推測できる正しい組み合わせを1つ選べ。
（右図）正常な飛行状態
【5つの表示の変化】
解説（正解: (5)）

【操作A（表示1→表示2）】

垂直線：右→中央（方位左偏を修正、つまり右旋回）

水平線：中央→下（設定高度より高くなった、つまり上昇）

→ A は「右旋回しながら上昇」


【操作B（表示2→表示3）】

垂直線：中央→左（方位右偏が発生、つまり右旋回しすぎ）

水平線：下→中央（高度が正常に戻った、つまり降下）

→ B は「右旋回しながら降下」


【操作C（表示3→表示4）】

垂直線：左→中央（方位右偏を修正、つまり左旋回）

水平線：中央→上（設定高度より低くなった、つまり降下）

→ C は「左旋回しながら降下」


【操作D（表示4→表示5）】

垂直線：中央→中央（方位変化なし）

水平線：上→中央（高度が正常に戻った、つまり上昇）

→ D は「上昇」


なお、(1)はAで垂直線が変化できない（V=右から左旋回すると線は右のまま飽和）。(2)はDが降下（表示4でH=上→降下するとH=上のまま）で矛盾。(3)(4)はAで水平線の向きが合わない。


したがって、正解は(5)である。
