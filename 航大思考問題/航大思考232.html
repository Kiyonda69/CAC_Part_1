問1配点: 6点 / 目安時間: 3分
積み木を組んでできた立体を真上から見て、各マスに積み上げた立方体の段数を記したものが【図1】である。0は積み木が無いことを表す。この立体の奥（鏡A）と左（鏡B）に、2枚の鏡を直角に合わせて垂直に立てる。2枚の鏡が出会う角をのぞき込むと、奥の鏡と左の鏡の両方で反射した像（合わせ鏡の像）が1つ見える。この像を同じように真上から見た図として正しいものを、(1)〜(5)から一つ選べ。
【図1】立体を真上から見た図と合わせ鏡（直角2枚）の位置
合わせ鏡に映った像を真上から見た図（クリックで解答）
問1（正解: (5)）
この問題の鍵は、直角に合わせた2枚の鏡（合わせ鏡）に映る像は、1枚の鏡のような「左右反転」にはならないという点である。光は奥の鏡と左の鏡で2回反射するため、反転が2回起こって打ち消し合い、像は鉛直軸まわりに180度回転したものとして見える（鏡文字にならない非反転像）。

真上から見た図では、180度回転＝各行を左右逆にし、さらに行の上下も逆にする（点対称）操作になる。

元の図（各行）：
3 1 2 ／ 1 0 3 ／ 2 2 1
180度回転（点対称）：
1 2 2 ／ 3 0 1 ／ 2 1 3
これは選択肢(5)と一致する。

【選択肢の検討】
・(1) 元の図のまま。何の変換もしていないので誤り。
・(2) 各行だけを左右逆にした図（1枚の鏡＝左右反転）。合わせ鏡では2回反射するため左右反転は起こらない。最も誤りやすい罠。
・(3) 行の並びだけを逆にした上下反転（前後方向の反転）。これも1枚の鏡（奥のみ）に相当し誤り。
・(4) 行と列を入れ替えた転置（対角線対称）。回転ではないので誤り。
・(5) 左右と上下を両方逆にした180度回転。これが合わせ鏡の像であり正解。

したがって、正解は(5)である。
問2配点: 6点 / 目安時間: 5分
【図2】は、積み木を組んでできた立体を真上から見て各マスの段数を記したものである。黒く塗ったマスには、目印のついた立方体が1段だけ置かれている。この立体の奥（鏡A）と左（鏡B）に、2枚の鏡を直角に合わせて垂直に立てる。2枚の鏡が出会う角をのぞき込んだとき見える合わせ鏡の像を真上から見た図として正しいものを、(1)〜(5)から一つ選べ。
【図2】立体を真上から見た図（黒マス＝目印）と合わせ鏡の位置
合わせ鏡に映った像を真上から見た図（クリックで解答）
問2（正解: (4)）
問1と同じく、直角に合わせた2枚の鏡（合わせ鏡）の像は2回反射により180度回転になる（左右反転ではない）。数字配置を点対称に並べ替えると同時に、黒い目印のマスも180度回転で対角の位置へ移る。

元の図：
2 0 3 1 ／ 1 3 0 2 ／ 3 1 2 0 ／ 0 2 1 3
目印は上から1行目・右端（右上の角）にある。
180度回転後：
3 1 2 0 ／ 0 2 1 3 ／ 2 0 3 1 ／ 1 3 0 2
目印は下から1行目・左端（左下の角）へ移る。
これは選択肢(4)と一致する。

【選択肢の検討】
・(1) 元の図のまま。目印も右上のままで誤り。
・(2) 各行を左右逆にした左右反転（1枚の鏡）。目印は右上→左上へ移る。合わせ鏡では起こらない最大の罠。
・(3) 行の並びを逆にした上下反転（1枚の鏡）。目印は右上→右下へ移り誤り。
・(4) 左右・上下を両方逆にした180度回転。目印は右上→左下へ移る。数字配置・目印ともに一致し正解。
・(5) 転置（対角線対称）。目印は左下の角に来るが、数字配置が180度回転と異なるため誤り。目印位置だけで判断すると引っかかる罠。

数字配置と目印の位置の両方が一致するのは(4)のみ。よって正解は(4)である。
