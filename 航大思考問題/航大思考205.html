問1 配点: 6点 / 目安時間: 3分
ある中学校2年A組の生徒11名に英語の小テスト（100点満点）を実施したところ、得点は以下のとおりであった。このデータについて、最小値・第1四分位数・中央値・第3四分位数・最大値（5数要約）と四分位範囲（Q3−Q1）の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。
【11名の得点（記入順）】
64, 36, 78, 50, 92, 70, 42, 75, 60, 55, 84
問1（正解: (2)）

11個のデータを昇順に並べ替えて5数要約を算出する。


【手順1：昇順に整列】

記入順: 64, 36, 78, 50, 92, 70, 42, 75, 60, 55, 84

整列後（11個）: 36, 42, 50, 55, 60, 64, 70, 75, 78, 84, 92

順位: 1番,2番,3番,4番,5番,6番,7番,8番,9番,10番,11番


【手順2：最小値・最大値】

・最小値 = 1番目 = 36　／　最大値 = 11番目 = 92


【手順3：中央値】

データ数は11（奇数）なので、中央値は (11+1)/2 = 6番目の値

中央値 = 64


【手順4：第1四分位数 Q1】

中央値を除いた下半分（1〜5番目）の中央値

下半分: 36, 42, 50, 55, 60　→ 3番目の値

Q1 = 50


【手順5：第3四分位数 Q3】

中央値を除いた上半分（7〜11番目）の中央値

上半分: 70, 75, 78, 84, 92　→ 上半分の3番目（全体の9番目）の値

Q3 = 78


【手順6：四分位範囲 IQR】

IQR = Q3 − Q1 = 78 − 50 = 28


【選択肢の検討】

・(1) Q1=55, Q3=75は4番目・8番目を四分位数と誤った。

・(2) 5数要約=36/50/64/78/92, IQR=28 で全て正しい。正解

・(3) 中央値を5番目の60と誤った。

・(4) Q3を8番目の75と誤った。

・(5) 最小値・最大値を2番目・10番目（42・84）と誤った。


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
ある会社の営業課に所属する9名（社員ア〜ケ）の今月の契約獲得件数を、昇順に並べると次の表のとおりであった。表中の a, b, c, d は値が伏せられている。下の【箱ひげ図情報】および【追加情報】を用いて、a + b + c + d の値として正しいものを、下の(1)〜(5)のうちから1つ選べ。
順位 1番2番3番4番5番 6番7番8番9番
社員 アイウエオ カキクケ
件数 a3845b58 c7075d
【箱ひげ図情報】（9名の契約獲得件数）
・最小値 = 28
・第1四分位数（Q1） = 41.5
・中央値 = 58
・第3四分位数（Q3） = 72.5
・最大値 = 88
【追加情報】
・9名の契約獲得件数の平均値は 58 件である。
・社員カの件数（c）は社員エの件数（b）よりちょうど20件多い。
問2（正解: (4)）

データは昇順に並んでいるので、5数要約と平均値の式から a, b, c, d を一つずつ確定する。


【ステップ1：最小値・最大値の確定】

データは昇順なので、1番目=最小値、9番目=最大値

・a = 最小値 = 28

・d = 最大値 = 88


【ステップ2：中央値の確認】

データ数9（奇数）なので、中央値 = 5番目の値 = 58

表中の5番目は58なので整合する（追加情報なし）。


【ステップ3：四分位数の確認】

中央値を除いた下半分（1〜4番目）は a, 38, 45, b

Q1 =（2番目＋3番目）÷2 = (38＋45)÷2 = 41.5 ✓（情報と一致）

中央値を除いた上半分（6〜9番目）は c, 70, 75, d

Q3 =（7番目＋8番目）÷2 = (70＋75)÷2 = 72.5 ✓（情報と一致）

※ Q1・Q3だけからは b, c は確定しない。


【ステップ4：平均値の利用】

9名の合計件数 = 平均×人数 = 58×9 = 522

28＋38＋45＋b＋58＋c＋70＋75＋88 = 522

402 + b + c = 522　∴　b + c = 120


【ステップ5：追加情報による連立】

・c = b + 20（カの件数はエより20件多い）

・b + c = 120

代入：b + (b+20) = 120　→　2b = 100　→　b = 50

よって c = 50＋20 = 70


【ステップ6：昇順性の確認】

28, 38, 45, 50, 58, 70, 70, 75, 88

※ 6番目と7番目が同値70だが、昇順（広義単調増加）として矛盾なし。


【ステップ7：和の計算】

a + b + c + d = 28 + 50 + 70 + 88 = 236


【選択肢の検討】

・(1) 218：bとcを取り違え b=40, c=60 などの誤算。

・(2) 226：Q1・Q3の式に1番目・4番目を含めた誤り（c=64, b=46 など）。

・(3) 232：平均値を9名でなく8名で計算した誤り。

・(4) 236：5数要約と平均値・追加情報すべてに整合。正解

・(5) 244：最小値・最大値を逆に解釈した誤り。


したがって、正解は(4)である。
