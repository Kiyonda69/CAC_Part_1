問1 配点: 6点 / 目安時間: 3分
飛行機が、地上の点Oの真上で、機首を真北に向けたまま鉛直面内で宙返り（縦に1回転）を行い、通過した経路に沿ってスモーク（煙）の輪を空中に残した。この輪は、点Oの真上にあり、南北方向に立てた鉛直な平面内の半径が一定の円で、しばらく形を変えずに空中にとどまるものとする。この輪を、（ア）真東の遠方から水平に見たとき、（イ）真上の遠方から見下ろしたとき、それぞれどのような形に見えるか。（ア）（イ）の組み合わせとして正しいものを、(1)〜(5)のうちから1つ選べ。ただし、遠近によるゆがみは考えないものとし、真上から見た図は北を上、真東から見た図は上方向を上とする。
【図1】宙返りのスモークの輪（斜めから見た模式図）
※ 各選択肢の上段ア＝真東から見た形、下段イ＝真上から見た形（北を上）
問1（正解: (4)）

ある方向から見たスモークの輪の形は、輪をその視線と垂直な平面に写した影（正射影）と同じである。輪は「南北方向に立てた鉛直な平面」の中にある半径一定の円であることに注意する。


【（ア）真東から水平に見る】

・視線は東西方向である。輪がのっている平面（南北×上下の面）は、この視線とちょうど正対する。

・正対する平面内の図形は形を変えずに見えるので、輪はそのまま真円に見える。


【（イ）真上から見下ろす】

・視線は上下方向であり、高さの違いは真上からは見えなくなる。

・輪の上のどの点も東西方向の位置は点Oの真上で共通であり、輪は東西方向にはまったく広がりを持たない。

・したがって地面に落とした影は、南北方向にのびる1本の線分になる（北を上とした図では縦の線分）。


【選択肢の検討】

・(1) 真上からも円に見えるとしている → 輪が水平に寝ている場合の見え方であり誤り。

・(2) アを線分、イを円としている → （ア）と（イ）の見え方を取り違えている。

・(3) アを鉛直の線分としている → 真南（または真北）から見たときの見え方との混同。輪の面は東西方向の視線に正対するので円に見える。

・(4) ア＝円、イ＝南北方向の線分 → 正しい。

・(5) イの線分の向きが東西方向になっている → 輪は南北方向に広がっているので誤り。


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
飛行機が、水平方向には地上の点Oの真上を中心とする半径が一定の円を真上から見て時計回りに描きながら、一定の割合で高度を上げ続け、ちょうど2周して、通過した経路に沿ってスモークの線を空中に残した。このスモークの軌跡を、（ア）真上の遠方から見下ろしたとき、（イ）真東の遠方から水平に見たとき、それぞれどのような形に見えるか。（ア）（イ）の組み合わせとして正しいものを、(1)〜(5)のうちから1つ選べ。ただし、遠近によるゆがみは考えないものとし、スモークは形を変えずに空中にとどまるものとする。
【図2】上昇しながら2周したスモークの軌跡（斜めから見た模式図）
※ 各選択肢の上段ア＝真上から見た形（北を上）、下段イ＝真東から見た形（上方向を上）
問2（正解: (5)）

軌跡の見え方は、視線方向の変化が消え、視線と垂直な方向の変化だけが残ると考えるとよい。軌跡の性質は「水平方向には半径一定の円周上を回り続ける」「高度は一定の割合で増え続ける」の2つである。


【（ア）真上から見下ろす】

・真上からは高度（上下方向）の違いは見えなくなり、水平方向の位置だけが残る。

・水平方向の位置は、1周目も2周目も常に点Oを中心とする同じ半径の円周上にある。

・したがって1周目と2周目の軌跡はぴったり重なり、1つの円に見える。渦巻きに見えるのは回転の半径が変化する場合だけであり、半径一定の本問では渦巻きにはならない。


【（イ）真東から水平に見る】

・真東からは東西方向の変化が見えなくなり、南北方向の位置と高度だけが残る。

・高度は一定の割合で増え続けるので、軌跡は下から上へ一方向に進む。

・一方、南北方向の位置は円運動により北端と南端の間を往復する（2周でちょうど2往復）。

・したがって、左右に波打ちながら上昇する曲線に見える。もし右上がりの直線に見えるなら、南北方向の位置が一方向に動き続けることになり、円を描いているという条件に反する。


【選択肢の検討】

・(1)(3) アを渦巻きとしている → 「上昇しながら回る＝渦巻き」という思い込み。半径が一定なら真上からは1つの円に重なる。

・(1)(2) イを右上がりの直線としている → 南北方向の往復（波打ち）を見落としている。

・(4) イを2つの円としている → 輪が円に見えるのは視線が輪の面と正対する場合に限られる。上昇しながらの円運動の軌跡は閉じた輪を作らない。

・(5) ア＝1つの円、イ＝波打ちながら上昇する曲線 → 正しい。


したがって、正解は(5)である。
