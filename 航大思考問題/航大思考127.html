問1 配点: 6点 / 目安時間: 3分
P島からT島の5つの島が空路で結ばれている。図は、島を楕円で、直行便を線分で表した、各島の位置と空路の概略図である。各島間の空路の関係について、アからエまでのことがわかっているとき、確実にいえるものはどれか。
ア　P島とQ島は直行便で結ばれている
イ　Q島とR島を結ぶ直行便はない
ウ　R島とS島は直行便で結ばれている
エ　T島とP島、T島とQ島は直行便で結ばれていない
解説（正解: (2)）

【確定情報の整理】

・ア：P島－Q島　→　直行便あり

・イ：Q島－R島　→　直行便なし

・ウ：R島－S島　→　直行便あり

・エ：T島－P島　→　直行便なし　／　T島－Q島　→　直行便なし


【T島の接続を分析する】

T島と接続できる可能性がある島を調べる。

・T－P：エより直行便なし（確定）

・T－Q：エより直行便なし（確定）

・T－R：条件に記載なし（不確定）

・T－S：条件に記載なし（不確定）

→ T島と繋がりうる島は最大でR島・S島の2島のみ。したがって「T島と直行便で結ばれている島の数は2つ以下」は条件の組み合わせに関わらず必ず成り立つ。


【その他の選択肢】

・(1) P島の確定接続はQ島のみ。P－R、P－Sは不確定のため「3つ以上」は確実にいえない。

・(3) Q－Sの接続は条件に記載なし。確実にいえない。

・(4) S－Tの接続は条件に記載なし。確実にいえない。

・(5) R島の確定接続はS島のみ。他の接続は不確定のため「2つ以上」は確実にいえない。


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下図は、ある立方体の展開図であり、A面からF面の各面には異なる模様が描かれている。この展開図を折り畳んで立方体に組み立てたとき、横縞（B面）と向かい合う面の模様はどれか。
【展開図】
解説（正解: (4)）

【展開図の構造】

S字型展開図の各面の隣接関係：A－B（横）、B－C（縦）、C－D（横）、D－E（縦）、E－F（縦）


【組み立て手順（C面を前面として）】

・B（C面の上側） → 上面

・A（B面の左側） → B面が上面になるとAは左面

・D（C面の右側） → 右面

・E（D面の下側） → D面が右面になるとEは下面

・F（E面の下側） → E面が下面になるとFは後面


【向かい合う面のペア】

・A（左面） ↔ D（右面）

・B（上面） ↔ E（下面）　← B面（横縞）の向かいはE面（点）

・C（前面） ↔ F（後面）


B面（横縞）と向かい合う面はE面（点模様）= (4)である。
