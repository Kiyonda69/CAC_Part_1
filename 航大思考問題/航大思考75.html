問1 配点: 6点 / 目安時間: 3分
下の図形ア～オのうち、一筆書きできるものをすべて選んだものはどれか。ただし、一筆書きとは、筆を紙から離さずに図形のすべての辺をちょうど一度ずつ通って描くことをいう。
解説（正解: (4)）

【一筆書きの条件】

グラフ理論において、一筆書きが可能な条件は次の通りである。

・すべての頂点の次数（その頂点につながる辺の数）が偶数 → 一筆書き可能（閉路）

・奇数次の頂点がちょうど2個 → 一筆書き可能（その2頂点が始点と終点）

・奇数次の頂点が3個以上 → 一筆書き不可


【各図形の判定】

・ア（正五角形）: 全5頂点の次数は2（偶数） → 一筆書き可能

・イ（五角形＋対角線1本）: 対角線の両端の2頂点が次数3（奇数）、他3頂点は次数2 → 奇数次頂点2個 → 一筆書き可能

・ウ（完全グラフK5）: 全5頂点の次数は4（偶数） → 一筆書き可能

・エ（正方形＋両対角線）: 全4頂点の次数は3（奇数） → 奇数次頂点4個 → 一筆書き不可

・オ（家形）: 左上と右上の2頂点が次数3（奇数）、他3頂点は次数2 → 奇数次頂点2個 → 一筆書き可能


一筆書きできるのはア、イ、ウ、オの4つである。


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
下の図は立方体の各辺をグラフとして表したものである。8個の頂点A～Hはすべて次数3（各頂点に3本の辺が接続）であり、辺は全部で12本ある。この図形を一筆書きできるようにするためには、既存の頂点間に新たに辺を最低何本追加すればよいか。
解説（正解: (3)）

【一筆書きの条件の確認】

一筆書きが可能であるためには、奇数次の頂点が0個（閉路）または2個（始点と終点が異なる経路）でなければならない。


【立方体グラフの分析】

立方体グラフでは、全8頂点の次数はすべて3（奇数）である。

・A: B, D, E に接続 → 次数3

・B: A, C, F に接続 → 次数3

・C: B, D, G に接続 → 次数3

・D: A, C, H に接続 → 次数3

・E: A, F, H に接続 → 次数3

・F: B, E, G に接続 → 次数3

・G: C, F, H に接続 → 次数3

・H: D, E, G に接続 → 次数3


奇数次の頂点が8個あるため、このままでは一筆書きできない。


【最小追加辺数の計算】

辺を1本追加すると、その両端の頂点の次数が1ずつ増え、奇数→偶数に変わる。

つまり、1本の辺追加で奇数次頂点を2個減らせる。


一筆書きには奇数次頂点を最大2個まで減らす必要がある。

現在8個の奇数次頂点を2個にするには: (8 - 2) / 2 = 3本の追加が必要。


【具体例】

例えば、辺A-C、B-D、E-Gを追加すると:

・A: 次数4（偶数）、B: 次数4（偶数）、C: 次数4（偶数）、D: 次数4（偶数）

・E: 次数4（偶数）、G: 次数4（偶数）

・F: 次数3（奇数）、H: 次数3（奇数）

奇数次頂点が2個（FとH）となり、FからHへの一筆書きが可能になる。


したがって、正解は(3)である。
