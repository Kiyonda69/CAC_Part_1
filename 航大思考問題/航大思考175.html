問1 配点: 6点 / 目安時間: 3分
下図は、ある空港の建物配置を真上から見た見取り図（平面図）である。空港には4つの建物があり、それぞれに高さが記載されている。この空港を真南から水平に見たときに見えるシルエット（建物の輪郭線）として最も適切なものを、選択肢(1)〜(5)の中から1つ選べ。なお、視点は十分に遠方かつ地上の高さにあるものとする。
【図1】空港の平面図（真上から見たもの）
【選択肢】南から見たシルエット（左：西側、右：東側）
解説（正解: (1)）

真南から空港を水平に見たとき、各列（西東方向の位置）について、その列にある最も高い建物の高さがシルエットの高さとなる。


【列ごとの最大高さの計算】

・列1（西端）: 格納庫A（高さ2）のみ → 高さ 2

・列2: 管制塔（高さ5）のみ → 高さ 5

・列3: ターミナル（高さ3）のみ → 高さ 3

・列4（東端）: 格納庫B（高さ2）のみ → 高さ 2


したがって、南からのシルエットは西から東へ [2, 5, 3, 2] となる。


【選択肢の検討】

・(1) [2, 5, 3, 2] → 上の計算と一致するため正解。

・(2) [2, 3, 5, 2] → 列2と列3の高さが入れ替わっている。

・(3) [2, 2, 5, 3] → 列3に管制塔があるかのような誤った配置。

・(4) [3, 5, 2, 2] → 列1に高さ3の建物があるかのような誤り。

・(5) [2, 5, 2, 3] → 列3と列4の高さが入れ替わっている。


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
ある空港には、高さの異なる4つの建物が4×4のマス目状の敷地に配置されている。建物は「管制塔（高さ5）」「旅客ターミナル（高さ4）」「第2ターミナル（高さ3）」「整備格納庫（高さ2）」の4つで、それぞれ1マスを占有する。下図は、この空港を真南および真東から水平に見たときに見えるシルエットである。これらの情報と整合する配置（真上から見た平面図）として最も適切なものを、選択肢(1)〜(5)の中から1つ選べ。なお、グリッドの列番号は西から東へ1〜4、行番号は北から南へ1〜4とする。
【図1】南および東から見たシルエット
【選択肢】空港の平面図（上=北、下=南、左=西、右=東）
凡例: 5=管制塔 4=旅客ターミナル 3=第2ターミナル 2=整備格納庫
解説（正解: (3)）

南からのシルエットは「各列で最も高い建物の高さ」、東からのシルエットは「各行で最も高い建物の高さ」を表す。各建物の高さは異なるため、シルエットの値から各建物の列・行が一意に決まる。


【南シルエット [2, 5, 4, 3] からの推論】

・列1の最大高さ=2 → 整備格納庫は列1

・列2の最大高さ=5 → 管制塔は列2

・列3の最大高さ=4 → 旅客ターミナルは列3

・列4の最大高さ=3 → 第2ターミナルは列4


【東シルエット [3, 5, 2, 4] からの推論】

・行1の最大高さ=3 → 第2ターミナルは行1

・行2の最大高さ=5 → 管制塔は行2

・行3の最大高さ=2 → 整備格納庫は行3

・行4の最大高さ=4 → 旅客ターミナルは行4


【配置の特定】

・管制塔: (列2, 行2)　・旅客ターミナル: (列3, 行4)

・第2ターミナル: (列4, 行1)　・整備格納庫: (列1, 行3)


【選択肢の検討】

・(1) 第2ターミナル(4,4)、整備(1,1) → 東シルエットで行1=2、行4=4となり不一致。

・(2) 旅客(4,1)、第2(3,4) → 南シルエットが [2,5,3,4] となり不一致。

・(3) 全ての建物が正しい位置 → 正解。

・(4) 整備(1,1)で行3の建物が無く、東シルエット行3=0となり不一致。

・(5) 管制塔(2,3)、整備(1,2) → 東シルエットが [3,2,5,4] となり不一致。


したがって、正解は(3)である。
