問1 配点: 6点 / 目安時間: 3分
下図は、立方体の展開図である。この展開図を組み立てて立方体にしたとき、面Bと向かい合う面はどれか。次のうちから選べ。
【展開図】立方体の展開図（T字＋下3連型）
問1（正解: (3)）の解説

【対面特定の基本ルール】

展開図上で「一直線に並ぶ3つの面」のうち、両端の2面は組み立てたとき向かい合う（対面となる）。


【本問の展開図における直線3連】

・横方向: A−B−C → A↔Cが対面

・縦方向: B−D−E−F の4連を「3連の重なり」として2通り見る

　─ B−D−E の3連 → B↔Eが対面

　─ D−E−F の3連 → D↔Fが対面


【対面ペア一覧】: A↔C、B↔E、D↔F


【別解：面の配置による検証】

Bを上面と決めると、A=左面、C=右面、D=前面、E=底面、F=後面となる。上面Bの対面は底面Eで一致する。


面Bと向かい合う面はEである。よって正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下図は、6つの正方形を連結した立方体の展開図である。この展開図を組み立てて立方体にしたとき、向かい合う面（対面）3組の組み合わせとして正しいものはどれか。次のうちから選べ。
【展開図】立方体の展開図（1−4−1型）
問2（正解: (2)）の解説

【展開図の構造】

・横方向: B−C−D−E の4連（中央段）

・縦方向: A−C（Cの上にA）、E−F（Eの下にF）


【対面特定の基本ルール】

① 一直線に並ぶ3つの面の両端は対面となる。

② 一直線に並ぶ4つの面は折り曲げてリング状の側面となり、1番目と3番目、2番目と4番目が対面となる。


【本問への適用】

・B−C−D−E の4連 → 1番目と3番目（B↔D）、2番目と4番目（C↔E）が対面

・残った面はAとF → 必然的にA↔Fが対面


【面の配置による検証】

Cを上面とすると、B=左面、D=右面、E=底面、A=前面、F=後面となる。

・上面C ↔ 底面E ✓

・左面B ↔ 右面D ✓

・前面A ↔ 後面F ✓


【対面ペア一覧】: A↔F、B↔D、C↔E


対面の組み合わせは「A−F、B−D、C−E」となる。よって正解は(2)である。
