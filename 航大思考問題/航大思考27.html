問1 配点: 6点 / 目安時間: 3分
下の資料は、A社・B社・C社の3社における2020年から2024年までの業績推移を示している。【資料1】は売上高（億円）の推移、【資料2】は営業利益率（%）の推移を表している。これらの資料に関する記述として正しいものを、(1)～(5)の中から1つ選べ。
【資料1】売上高の推移（億円）
【資料2】営業利益率の推移（%）
解説（正解: (2)）

【資料の読み取り】

売上高（億円）：A社 180→190→200→195→220 / B社 120→130→125→140→150 / C社 80→90→95→100→110

営業利益率（%）：A社 8→9→10→8→11 / B社 10→9→8→10→11 / C社 6→7→8→6→10


【各選択肢の検討】

(1) 誤り。 B社+C社の合計は常にA社を上回っている。

　例：2020年はA社=180億、B社+C社=200億（A社が下回っている）。


(2) 正しい。 B社の売上高は120→130（増加）→125（減少）→140（増加）→150（増加）と推移する。

　前年比で減少した年は2022年のみであり、正しい記述である。


(3) 誤り。 2024年の3社合計 = 220+150+110 = 480億円 ＜ 500億円。


(4) 誤り。 2022年の営業利益率はA社=10%が最高（B社=8%、C社=8%）。

　C社ではなくA社が最も高い。


(5) 誤り。 C社の営業利益（＝売上高×営業利益率）は以下の通り。

　80×6%=4.8、90×7%=6.3、95×8%=7.6、100×6%=6.0（減少）、110×10%=11.0

　2023年に前年比で減少しており、毎年増加とは言えない。


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下の資料は、P社・Q社・R社の3社における2020年から2024年までの業績推移を示している。【資料1】は売上高（百万円）の推移、【資料2】は売上総利益率（%）の推移を表している。これらの資料に関する記述として正しいものを、(1)～(5)の中から1つ選べ。
【資料1】売上高の推移（百万円）
【資料2】売上総利益率の推移（%）
解説（正解: (4)）

【資料の読み取り】

売上高（百万円）：P社 500→520→550→530→580 / Q社 350→360→340→380→400 / R社 200→220→240→250→260

売上総利益率（%）：P社 30→32→35→26→33 / Q社 25→24→22→24→30 / R社 20→22→24→20→25


【各選択肢の検討】

(1) 誤り。 Q社の売上高は2021年の360から2022年の340へ減少（正しい）。しかし売上総利益率は24%から22%へ低下しており、上昇していない。両条件ともに成立しないため誤り。


(2) 誤り。 P社の売上高は500→520→550→530（減少）→580と推移しており、2023年に前年比で減少している。「存在しない」は誤り。


(3) 誤り。 2024年の売上総利益を計算する。

　P社：580×33% = 191.4百万円

　Q社：400×30% = 120.0百万円

　R社：260×25% = 65.0百万円

　合計：376.4百万円 ＜ 400百万円 → 400百万円を超えていない。


(4) 正しい。 R社の売上高：200→220→240→250→260と、毎年増加している。


(5) 誤り。

　2020年の平均：(30+25+20)÷3 = 25.0%

　2023年の平均：(26+24+20)÷3 ≈ 23.3%

　2023年の平均（23.3%）は2020年の平均（25.0%）より低い。


したがって、正解は(4)である。
