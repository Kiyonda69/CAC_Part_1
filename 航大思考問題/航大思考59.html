問1 配点: 6点 / 目安時間: 3分
図は、ある市の公共スポーツ施設における年間利用者数の内訳を、男女別・年齢層別に示したものである。30代の利用者数は、利用者数全体のおよそ何%か。正しいものを1つ選べ。
【男女別・年齢層別スポーツ施設利用者数】
解説（正解: (1)）

男女それぞれの30代の人数を求め、全体に対する割合を計算する。


【男性の30代】

・9,450人 × 30% = 2,835人


【女性の30代】

・5,550人 × 20% = 1,110人


【全体に占める割合】

・30代合計：2,835 + 1,110 = 3,945人

・利用者総数：9,450 + 5,550 = 15,000人

・割合：3,945 / 15,000 = 26.3%


【注意点】

男女の30代割合を単純平均すると (30+20)/2 = 25.0% となるが、男性の利用者数が女性より多いため、男性寄りの値 26.3% が正解となる。


したがって、正解は(1) 26.3%である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ図を用いる。40代以上の利用者のうち、女性が占める割合はおよそ何%か。正しいものを1つ選べ。
【男女別・年齢層別スポーツ施設利用者数】（問1と同じ図）
解説（正解: (4)）

40代以上の利用者を男女それぞれ求め、そのうち女性が占める割合を計算する。


【男性の40代以上】

・40代 + 50代 + 60代以上 = 25% + 11% + 4% = 40%

・9,450 × 40% = 3,780人


【女性の40代以上】

・40代 + 50代 + 60代以上 = 28% + 15% + 7% = 50%

・5,550 × 50% = 2,775人


【女性の割合】

・40代以上合計：3,780 + 2,775 = 6,555人

・女性割合：2,775 / 6,555 = 42.3%


【注意点】

全体の女性比率37%をそのまま答えると(2)の誤答になる。女性は40代以上の割合が男性より高い（50% vs 40%）ため、40代以上に限れば女性比率は全体より高い42.3%となる。


したがって、正解は(4) 42.3%である。
