問1 配点: 6点 / 目安時間: 3分
左と右の図ではある規則がある。同様の規則性で並ぶとき、左端の図と対応するものを(1)～(5)から選べ。
【規則の例】
【問題】
解説（正解: (3)）

【規則の発見】

例題の左の図形（L字型）を90度時計回りに回転させると、右の図形と一致する。

したがって、この問題の規則は「90度時計回り回転」である。


【解法】

左端の図形を90度時計回りに回転させる。

回転の変換: 位置(i, j)の要素は位置(j, N-1-i)に移動する。


【選択肢の検討】

・(1) 反時計回り90度回転 → 回転方向が逆のため不正解

・(2) 180度回転 → 回転角度が異なるため不正解

・(3) 90度時計回り回転 → 規則に一致するため正解

・(4) 水平反転 → 回転ではなく反転のため不正解

・(5) 垂直反転 → 回転ではなく反転のため不正解


したがって、正解は(3)である。

下に示す2組の例では、左の図形にある共通の変換規則を適用すると右の図形になる。同じ規則を問題の図形に適用したとき、正しい結果を(1)～(5)から選べ。
【例題】
【問題】
解説（正解: (4)）

【規則の発見】

例題の2組を比較して、共通する変換規則を見つける。

例1: 左の図形を90度時計回りに回転し、さらに白黒を反転すると右の図形になる。

例2: 同じく90度時計回り回転+白黒反転で右の図形が得られる。

したがって、規則は「90度時計回り回転 + 白黒反転」の複合変換である。


【解法】

ステップ1: 問題の図形を90度時計回りに回転

ステップ2: 回転後の図形の白黒を反転（塗りつぶしと空白を入れ替え）


【選択肢の検討】

・(1) 白黒反転のみ（回転なし） → 不正解

・(2) 90度時計回り回転のみ（反転なし） → 不正解

・(3) 反時計回り回転+白黒反転 → 回転方向が逆のため不正解

・(4) 90度時計回り回転+白黒反転 → 規則に一致するため正解

・(5) 水平反転+白黒反転 → 変換が異なるため不正解


したがって、正解は(4)である。
