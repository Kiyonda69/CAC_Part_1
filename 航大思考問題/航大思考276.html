問1 配点: 6点 / 目安時間: 3分
円形の紙に、円の端から端まで届く直線を何本か引いて、紙をいくつかの領域に分けていく。ただし、どの2本の直線も円の内部で交わり、3本以上の直線が同じ点で交わることはないものとする。下の図のように、領域の数は直線が1本のとき2個、2本のとき4個、3本のとき7個になる。この規則で直線を7本引いたとき、紙は何個の領域に分けられるか。
【図】直線の本数と領域の数
問1（正解: (4)）

直線を1本増やしたとき、領域がいくつ増えるかに着目する。


【規則の発見】

n本目の直線を引くと、この直線はすでに引かれているn−1本のすべてと円の内部の異なる点で交わるから、n−1個の交点で n個の部分に分けられる。その各部分が、通っている領域を1つずつ2つに割るので、領域は n個増える。

図の値で確かめると、1本→2個、2本→2+2=4個、3本→4+3=7個となり一致する。


【計算】

4本: 7+4=11個　→　5本: 11+5=16個　→　6本: 16+6=22個　→　7本: 22+7=29個

（別解）領域の数 ＝ 1 ＋（直線の本数）＋（円の内部の交点の個数）。7本の直線からできる交点は 7×6÷2＝21個だから、1＋7＋21＝29個。


【選択肢の検討】

・(1) 16個 → 5本のときの値。

・(2) 22個 → 6本のときの値（最後の+7を加え忘れた誤り）。

・(3) 28個 → 7本目の増分を+6と誤った値。

・(4) 29個 → 正解。

・(5) 37個 → 8本のときの値（+8まで余分に加えた誤り）。


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ円形の紙に、次の条件をすべて満たすように直線を8本引く。
・8本のうち3本は互いに平行であり、この3本はどの2本も交わらない。
・別の3本は、円の内部の点Pで3本とも交わる。
・これ以外に平行な直線の組はなく、3本以上の直線が同じ点で交わることもない。すなわち、平行な3本どうしを除き、どの2本の直線も円の内部で交わる。

このとき、紙は何個の領域に分けられるか。
【図】条件の模式図（8本のうち6本のみ示す）
※ 図は条件を模式的に示したもので、残りの2本の直線は示していない。
問2（正解: (4)）

問1の規則を一般化する。新しく引く直線が、すでに引かれた直線と円の内部で k個の点で交わるならば、その直線は k+1個の部分に分けられるから、領域は k+1個増える。この規則で8本を順に引いて数える。


【順に数える】（平行な3本をa・b・c、その他の2本をd・e、点Pを通る3本をf・g・hとする）

・a: 交わる直線なし → +1 → 2個

・b: aと平行で交わらない → +1 → 3個

・c: 同様に → +1 → 4個

・d: a・b・cと3点で交わる → +4 → 8個

・e: a・b・c・dと4点で交わる → +5 → 13個

・f: a〜eの5本と5点で交わる → +6 → 19個

・g: a〜eと5点、fとは点Pで交わり計6点 → +7 → 26個

・h: a〜eと5点、f・gとは同じ点Pで交わるので交点は1個分、計6点 → +7 → 33個


【別解（一般配置との比較）】

8本がどの2本も交わり3本が1点で交わらない一般配置なら、増分は +1,+2,…,+8 で 1+8+28＝37個。本問では、平行な3本のうち2本目・3本目の増分が+2・+3ではなく+1・+1になり3個減、共点により8本目の増分が+8ではなく+7になり1個減。よって 37−3−1＝33個。共点では交点は2個減る（3個→1個）が、領域の減少は1個にとどまる点に注意する。


【選択肢の検討】

・(1) 29個 → Pで交わる3本のうち1本が領域を増やさないと誤解し、7本の一般配置（1+7+21）とした値。

・(2) 31個 → 点Pで3個の交点がすべて失われるとして、領域も3個減ると誤った値（37−3−3）。

・(3) 32個 → 点Pで交点が2個減るから領域も2個減ると誤った値（37−3−2）。

・(4) 33個 → 正解。

・(5) 37個 → 平行・共点の条件を見落とした一般配置の値。


したがって、正解は(4)である。
