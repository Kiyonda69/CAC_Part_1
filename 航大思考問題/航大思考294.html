問1 配点: 6点 / 目安時間: 3分
次の表は、ある県のA〜E市における観光入込客数と観光消費額を、2023年と2024年についてまとめたものである。この表から確実にいえるものとして、最も妥当なのはどれか。
【表1】A〜E市の観光入込客数と観光消費額
市 観光入込客数（万人） 観光消費額（億円）
2023年 2024年 2023年 2024年
A市620682372409
B市540513297308
C市380418209230
D市760798456503
E市300342186205
解説（正解: (5)）

問1（正解: (5)）

各選択肢について、表の数値をもとに計算して検証する。


【選択肢の検討】

・(1) 観光入込客数の対前年増加率は、A市：62÷620 = 10.0%、B市：−27÷540 = −5.0%、C市：38÷380 = 10.0%、D市：38÷760 = 5.0%、E市：42÷300 = 14.0%。最も大きいのはE市であり、C市ではない。→ 誤り

・(2) 2024年の観光客1人当たりの観光消費額（消費額÷客数）は、A市：409÷682 ≒ 0.600、B市：308÷513 ≒ 0.600、C市：230÷418 ≒ 0.550、D市：503÷798 ≒ 0.630、E市：205÷342 ≒ 0.599。最も高いのはD市であり、E市ではない。→ 誤り

・(3) 観光消費額の合計は、2023年が 372+297+209+456+186 = 1520億円、2024年が 409+308+230+503+205 = 1655億円。増加率は 135÷1520 ≒ 8.9% で、10%に満たない。→ 誤り

・(4) 客数と消費額がともに増加したのは、A市・C市・D市・E市の4市である（B市のみ客数が減少）。3つではない。→ 誤り

・(5) 客数の合計は、2023年が 2600万人、2024年が 2753万人。D市の割合は 760÷2600 ≒ 29.2% から 798÷2753 ≒ 29.0% へと低下している。→ 正しい


【時短テクニック】(5)は実額を割らなくても、D市の増加率（5.0%）が合計の増加率（153÷2600 ≒ 5.9%）より小さいことから、構成比の低下を判定できる。


したがって、正解は(5)である。

問2 配点: 6点 / 目安時間: 5分
次の表は、ある国の製造業5部門について、2021年の生産額と、2022年から2024年までの各年における生産額の対前年増加率を示したものである。この表から確実にいえるものとして、最も妥当なのはどれか。

解説（正解: (3)）

問2（正解: (3)）

対前年増加率の表なので、各年の生産額は2021年の実額に増加率を順に掛けて求める。

食品：2400 → 2520 → 2469.6 → 約2568.4

化学：1800 → 1746 → 1850.8 → 約1897.0

金属：1500 → 1620 → 1660.5 → 約1594.1

機械：3200 → 3280 → 3444 → 約3650.6

繊維：900 → 855 → 833.6 → 約900.3


【選択肢の検討】

・(1) 2024年と2021年を比べると、金属は 1594.1 > 1500、繊維も 0.95×0.975×1.08 ≒ 1.0004倍でわずかに上回っており、2021年を下回る部門は1つもない。→ 誤り

・(2) 2023年は機械 3444、食品 2469.6。食品の1.4倍は約3457.4であり、3444はこれを下回る（3444÷2469.6 ≒ 1.39倍）。→ 誤り

・(3) 2022年に減少した部門は化学（減少額54億円）と繊維（減少額45億円）で合計99億円。増加した部門は食品（+120）、金属（+120）、機械（+80）で合計320億円。その半分は160億円であり、99 ≦ 160 が成り立つ。→ 正しい

・(4) 2021年から2024年の増加率は、食品 +7.0%、化学 +5.4%、金属 +6.3%、機械 +14.1%、繊維 +0.0%。最大は機械であり、金属ではない。→ 誤り

・(5) 2024年の合計は約10610億円。機械の割合は 3650.6÷10610 ≒ 34.4% で、35%を超えていない。→ 誤り


したがって、正解は(3)である。
