問1 配点: 6点 / 目安時間: 3分
下図のように、一辺aの立方体の8つの頂点すべてについて、各頂点に集まる3つの辺の中点を通る平面で切り落とす。このとき、できる立体の面の数、辺の数、頂点の数の組み合わせとして正しいものを(1)～(5)から1つ選べ。
【図1】立方体の頂点切断
解説（正解: (2)）

【立体の構造分析】

一辺aの立方体の各頂点を、その頂点に集まる3辺の中点を通る平面で切り落とす操作を考える。


【頂点の数】

・元の立方体は12本の辺を持つ

・各辺の中点が新しい立体の頂点となる

・したがって頂点の数は 12


【面の数】

・元の立方体の6面 → 各面の4辺の中点を結ぶ正方形に縮小 → 6面（正方形）

・元の立方体の8頂点 → 各頂点の切断面が正三角形 → 8面（正三角形）

・合計: 6 + 8 = 14面


【辺の数】

・オイラーの多面体定理: V - E + F = 2

・12 - E + 14 = 2 → E = 24

・（確認: 三角形面8×3辺 + 正方形面6×4辺 = 48、各辺は2面で共有 → 48÷2 = 24）


したがって、正解は(2)である。

（この立体は「立方八面体（cuboctahedron）」と呼ばれるアルキメデスの立体の一種である。）

問2 配点: 6点 / 目安時間: 5分
下図のように、一辺4cmの立方体ABCD-EFGHがある。辺AE上の中点をM、辺BF上のBP=1cmとなる点をP、辺DH上のDQ=3cmとなる点をQとする。3点M、P、Qを通る平面でこの立方体を切断したとき、切断面の形状として正しいものを(1)～(5)から1つ選べ。
【図2】立方体の切断
解説（正解: (1)）

【座標設定】

A(0,0,0)、B(4,0,0)、C(4,4,0)、D(0,4,0)

E(0,0,4)、F(4,0,4)、G(4,4,4)、H(0,4,4)


【切断点の座標】

・M: 辺AEの中点 → M(0, 0, 2)

・P: 辺BF上でBP=1 → P(4, 0, 1)

・Q: 辺DH上でDQ=3 → Q(0, 4, 3)


【切断面の方程式】

ベクトルMP = (4, 0, -1)、ベクトルMQ = (0, 4, 1)

法線ベクトル = MP × MQ = (4, -4, 16) → 簡約化: (1, -1, 4)

平面の方程式: x - y + 4z = 8


【4つ目の交点】

辺CGとの交点R: x=4, y=4を代入 → 4 - 4 + 4z = 8 → z = 2

よって R(4, 4, 2) すなわち辺CGの中点


【切断面の形状判定】

4辺の長さを計算する:

・|MP| = sqrt(16+0+1) = sqrt(17)

・|PR| = sqrt(0+16+1) = sqrt(17)

・|RQ| = sqrt(16+0+1) = sqrt(17)

・|QM| = sqrt(0+16+1) = sqrt(17)

4辺すべてが等しい → ひし形または正方形


直角判定: MP・MQ = 4×0 + 0×4 + (-1)×1 = -1 ≠ 0

角度が直角でないため正方形ではない。


したがって、切断面はひし形（正方形でない）であり、正解は(1)である。
