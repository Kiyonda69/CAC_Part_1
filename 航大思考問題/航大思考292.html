問1 配点: 6点 / 目安時間: 3分
次の表は、ある市の公共施設5施設における年間利用者数の推移を示したものである。この表から確実にいえるものとして、最も妥当なのはどれか。次の(1)〜(5)から1つ選べ。
【表】公共施設の年間利用者数の推移（単位：千人）
施設2022年2023年2024年
図書館240264290
体育館180171190
美術館120132120
市民ホール100110121
プール160168180
※ 数値は各年の年間利用者数（千人）。
解説（正解: (3)）

問1（正解: (3)）

まず各年の5施設合計を求めておく。2022年：240＋180＋120＋100＋160＝800千人、2023年：264＋171＋132＋110＋168＝845千人、2024年：290＋190＋120＋121＋180＝901千人。


【選択肢の検討】

・(1) 図書館の対前年増加率は、2023年が（264−240）÷240＝24÷240＝10.0%、2024年が（290−264）÷264＝26÷264≒9.8%。増加数は24→26と増えているが、分母も240→264と大きくなるため、増加率は2024年の方が小さい。→ 誤り

・(2) 2022年に対する2024年の増加率は、図書館（290−240）÷240＝50÷240≒20.8%、体育館10÷180≒5.6%、美術館0%、市民ホール（121−100）÷100＝21.0%、プール20÷160＝12.5%。最大は市民ホール（21.0%）であり、図書館（20.8%）は僅差の2位。増加数（図書館の50千人）だけを見て図書館と即断すると誤る。→ 誤り

・(3) 2023年の合計845千人の20%は 845×0.2＝169千人。体育館は171千人で169千人を上回るから、割合は 171÷845≒20.2%＞20%。→ 正しい

・(4) 2022年比の増加率は（901−800）÷800＝101÷800≒12.6%。800×1.13＝904＞901 であるから、13%には届かない。→ 誤り

・(5) プールの3年間平均は（160＋168＋180）÷3＝508÷3≒169.3千人で、170千人を下回る。170×3＝510＞508 と比べれば割り算をせずに判定できる。→ 誤り


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
次の表は、ある工場における製品A〜Dの生産量の対前年増加率を示したものである。この表から確実にいえるものとして、最も妥当なのはどれか。次の(1)〜(5)から1つ選べ。
【表】製品A〜Dの生産量の対前年増加率（単位：%）
製品2021年2022年2023年2024年
A5.0△ 2.04.03.0
B△ 10.011.05.0△ 2.0
C20.0△ 10.02.00.0
D2.03.03.02.0
※ 数値は前年に対する生産量の増加率（%）。△はマイナス（減少）を表す。
解説（正解: (4)）

問2（正解: (4)）

表に示されているのは対前年増加率のみであり、生産量の実数はどの製品についても不明である。2020年の生産量を100とした指数を、増加率を順に掛けて求めると次のようになる。


・製品A：100 → 105.0 → 102.9 → 107.0 → 110.2（1.05×0.98×1.04×1.03）

・製品B：100 → 90.0 → 99.9 → 104.9 → 102.8（0.90×1.11×1.05×0.98）

・製品C：100 → 120.0 → 108.0 → 110.2 → 110.2（1.20×0.90×1.02×1.00）

・製品D：100 → 102.0 → 105.1 → 108.2 → 110.4（1.02×1.03×1.03×1.02）


【選択肢の検討】

・(1) 製品Bの2022年の指数は 90.0×1.11＝99.9＜100。「−10%の後に＋11%だから差し引き＋1%」と増加率を足し算すると誤る典型の罠で、10%減った後の11%増は、小さくなった値（90）に対する11%（9.9）しか戻らない。→ 誤り

・(2) 2020年に対する2024年の増加率は、製品C＋10.2%（指数110.16）に対し、製品D＋10.4%（指数110.38）が最大。増加率の単純合計では C：20−10＋2＋0＝＋12 が最大に見えるが、大きな増減を繰り返すほど掛け算の結果は目減りする（20%増→10%減は 1.2×0.9＝1.08 で＋8%にしかならない）。→ 誤り

・(3) この表からは各製品の実数（生産量そのもの）は分からない。指数はあくまで各製品自身の2020年を100とした相対値であり、製品間の生産量の大小は比較できない（例えばAが1万個、Cが100個でも表は成立する）。→ 確実にはいえない → 誤り

・(4) 2020年比＋10%以上（指数110以上）となるのは、製品A（110.23）・製品C（110.16）・製品D（110.38）の3製品。製品Bは102.8で届かない。増加率の単純合計（A：＋10、C：＋12、D：＋10）では境界上に見えるため、丁寧に掛け算して確かめる必要がある。→ 正しい

・(5) 製品Dの増加率は 2.0→3.0→3.0→2.0（%）と推移しており、2022年から2023年は横ばい、2023年から2024年は低下している。生産量自体は毎年増加しているが、「増加率が毎年上昇」とはいえない。→ 誤り


したがって、正解は(4)である。
