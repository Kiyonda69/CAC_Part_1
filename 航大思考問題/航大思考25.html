問1 配点: 6点 / 目安時間: 3分
平面上にある目標地点Xの位置を特定するために、以下の2種類の測量装置を使用する。
【装置の特性】
各装置は既知の地点に設置でき、複数台を異なる地点に設置することも可能である。Xは装置の設置地点とは異なる位置にあるものとする。Xの位置の特定について正しい記述を1つ選べ。
解説（正解: (2)）

【各装置の測定結果の幾何学的意味】

・装置A（距離計）：Xまでの距離dを測定 → Xは装置を中心とする半径dの円周上にある

・装置B（方位計）：Xへの方位角θを測定 → Xは装置から方位角θ方向の半直線上にある


【各選択肢の検討】

・(1) 装置A 1台：Xは半径dの円周上にある → 無限個の候補点が存在 → 一意に特定できない


・(2) 装置A+B 同一地点：円の中心から出る半直線は、その円と常に1点のみで交わる。距離dと方位角θから X = (d cos θ, d sin θ) と一意に定まる → 正しい


・(3) 装置A 2台（異なる位置）：2つの円は2点で交差する場合がある。例えば2台を6m離して設置し、各距離が5mのとき、2つの候補点が生じる → 常に一意とは限らない


・(4) 装置B 2台（異なる位置）：Xが2装置を結ぶ直線上にある場合、2本の半直線が同一直線上に重なり、一意に特定できない


・(5) 装置A+B 異なる地点：半直線が円と2点で交わる場合がある。例えば装置Bの半直線が装置Aの円を貫通するとき、2候補が生じる → 常に一意とは限らない


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下図のように、4×4のマス目が描かれた正方形の紙を2回折り畳んだ後、2か所に穴を開けた。紙を広げたとき、穴の位置の配置として正しいものを1つ選べ。
【折り畳み手順】
解説（正解: (1)）

【折り畳みの対称性】

・折り1（上半分を下に）：上下方向の対称性が生まれる（行1と行4、行2と行3が対応）

・折り2（右半分を左に）：左右方向の対称性が生まれる（列1と列4、列2と列3が対応）


【穴の展開】

折り畳んだ状態で見える範囲は、元の紙の左下1/4（列1-2、行3-4）に相当する。


・穴A（左上 = 列1, 行3の位置）：

　左右対称 → (列4, 行3) にも穴

　上下対称 → (列1, 行2) と (列4, 行2) にも穴

　→ 計4つの穴：(1,2), (1,3), (4,2), (4,3)


・穴B（右下 = 列2, 行4の位置）：

　左右対称 → (列3, 行4) にも穴

　上下対称 → (列2, 行1) と (列3, 行1) にも穴

　→ 計4つの穴：(2,1), (2,4), (3,1), (3,4)


【展開後のパターン】

行1: ○●●○　行2: ●○○●　行3: ●○○●　行4: ○●●○

上下・左右ともに対称な「枠」型のパターンとなる。


【誤答の分析】

・(2) 正解の反転パターン。穴と空白の位置が逆転している

・(3) 中央2列のみに穴。左右の折り返しの対応関係を誤っている

・(4) 外側2列のみに穴。穴AとBの位置対応を取り違えている

・(5) 上下2行のみに穴。折り1のみ展開し、折り2の左右対称を無視している


したがって、正解は(1)である。
