問1 配点: 6点 / 目安時間: 3分
ある企業が新人研修の受講者6人（A～F）に対して4科目の試験を実施し、以下の成績一覧表を作成した。また、下記の評価基準に基づいて各受講者を評価する。このとき、次の(1)～(5)の記述のうち正しいものを1つ選べ。
【表】新人研修 成績一覧（各100点満点）
受講者 ビジネスマナー 情報リテラシー 語学 プレゼンテーション
A 82 75 90 68
B 71 88 65 79
C 95 62 78 84
D 68 91 72 76
E 84 79 85 92
F 76 83 69 71
【評価基準】
・4科目の合計が310点以上の者を「優秀」と認定する。
・いずれかの科目で70点未満の者は「要再試験」の対象となる。
・「優秀」に該当し、かつ「要再試験」の対象でない者を「総合合格」とする。
解説（正解: (1)）

表の数値と評価基準を照合して、各受講者の評価を確定する。


【各受講者の合計点と評価】

・A: 82+75+90+68 = 315点 → 優秀(310以上)。プレゼンテーション68点(70未満) → 要再試験。

・B: 71+88+65+79 = 303点 → 優秀ではない。語学65点 → 要再試験。

・C: 95+62+78+84 = 319点 → 優秀。情報リテラシー62点 → 要再試験。

・D: 68+91+72+76 = 307点 → 優秀ではない。ビジネスマナー68点 → 要再試験。

・E: 84+79+85+92 = 340点 → 優秀。全科目70点以上 → 要再試験なし → 総合合格。

・F: 76+83+69+71 = 299点 → 優秀ではない。語学69点 → 要再試験。


【選択肢の検討】

・(1) 総合合格はEの1人だけ → 正しい

・(2) 要再試験はA, B, C, D, Fの5人であり、4人ではない → 誤り

・(3) Aの語学は90点で要再試験ではない（要再試験はプレゼンテーション68点） → 誤り

・(4) 合計最低はF(299点)であり、D(307点)ではない → 誤り

・(5) 情報リテラシー平均 = (75+88+62+91+79+83)/6 = 478/6 = 79.7点で80点以下 → 誤り


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
ある会社が取り扱う5つの商品（A～E）について、四半期ごとの売上データと年間目標をまとめた表と、商品評価の基準が以下のとおり定められている。このとき、次の(1)～(5)の記述のうち正しいものを1つ選べ。
【表】商品別 四半期売上実績（単位: 万円）
商品 Q1 Q2 Q3 Q4 年間目標
A 280 320 350 410 1,200
B 190 210 180 220 850
C 450 380 420 510 1,600
D 310 290 340 360 1,400
E 160 240 270 330 900
【商品評価基準】
・年間売上の合計が年間目標以上の商品を「目標達成」とする。
・Q1からQ4にかけて毎四半期、前期より売上が増加している商品を「成長優良」とする。
・いずれかの四半期で売上が200万円を下回った商品を「要改善」と判定する。
・「目標達成」かつ「成長優良」の商品を「最優秀商品」とする。ただし、「要改善」に該当する商品は「最優秀商品」から除外する。
解説（正解: (2)）

表の売上データと評価基準を照合して、各商品を評価する。


【各商品の年間売上と評価】

・A: 280+320+350+410 = 1,360万円（目標1,200 → 達成）。Q1→Q4で毎期増加 → 成長優良。全期200以上 → 要改善なし。

・B: 190+210+180+220 = 800万円（目標850 → 未達）。Q3(180)がQ2(210)より減少 → 成長優良でない。Q1(190), Q3(180)が200未満 → 要改善。

・C: 450+380+420+510 = 1,760万円（目標1,600 → 達成）。Q2(380)がQ1(450)より減少 → 成長優良でない。全期200以上 → 要改善なし。

・D: 310+290+340+360 = 1,300万円（目標1,400 → 未達）。Q2(290)がQ1(310)より減少 → 成長優良でない。全期200以上 → 要改善なし。

・E: 160+240+270+330 = 1,000万円（目標900 → 達成）。Q1→Q4で毎期増加 → 成長優良。Q1(160)が200未満 → 要改善。


【最優秀商品の判定】

・A: 目標達成かつ成長優良、要改善なし → 最優秀商品

・E: 目標達成かつ成長優良だが、要改善に該当 → 最優秀商品から除外


【選択肢の検討】

・(1) Eは要改善のため最優秀から除外、最優秀はAのみ → 誤り

・(2) 最優秀はAだけ → 正しい

・(3) 成長優良はAとE（Cは Q2でQ1より減少） → 誤り

・(4) D年間売上1,300 > E年間売上1,000だが、達成率はD(92.9%) < E(111.1%) → 誤り

・(5) 要改善はBとE（Eも Q1が160で200未満） → 誤り


したがって、正解は(2)である。
