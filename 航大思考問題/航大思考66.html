問1 配点: 6点 / 目安時間: 3分
下図のように、一辺の長さが2cmの正方形ABCDが直線l上を滑ることなく転がる。正方形が1回転（元の向きに戻る）したとき、頂点Aが描く軌跡の長さとして正しいものを(1)〜(5)から1つ選べ。
【図1】正方形の回転と頂点Aの軌跡
解説（正解: (4)）

【回転の分析】

正方形ABCDが直線l上を1回転するには、90°ずつ4回の回転が必要である。各回転では、地面に接している頂点を中心として正方形が倒れるように転がる。


【各段階での頂点Aの動き】

・段階1：頂点A（右下）を中心に90°回転 → Aは回転の中心なので動かない（軌跡 = 0）

・段階2：頂点Bを中心に90°回転 → AはBから距離2cm → 円弧 = 2 × π/2 = π cm

・段階3：頂点Cを中心に90°回転 → AはCから距離2√2 cm（対角線）→ 円弧 = π√2 cm

・段階4：頂点Dを中心に90°回転 → AはDから距離2cm → 円弧 = π cm


【合計】

軌跡の長さ = 0 + π + π√2 + π = π(2 + √2) cm


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
下図のように、半径1cmの円板が一辺6cmの正三角形の外側に沿って滑ることなく転がり、正三角形の周りを1周する。このとき、円板の中心が描く軌跡で囲まれた領域の面積として正しいものを(1)〜(5)から1つ選べ。
【図2】円板が正三角形の外側を転がる様子
解説（正解: (5)）

【軌跡の構成】

円板の中心は、正三角形の各辺から常に半径1cmだけ離れた位置を通る。この軌跡は以下の要素から構成される：

・直線部分：各辺に平行な線分（辺と同じ長さ6cm）が3本

・円弧部分：各頂点で120°の円弧（半径1cm）が3個


【面積の分解】

軌跡で囲まれた領域は、以下の3種類の部分に分解できる：


1. 元の正三角形の面積

S1 = (√3/4) × 6² = 9√3 cm²


2. 3つの長方形（辺に沿った帯状領域）

各辺（長さ6cm）に対して、幅1cmの帯が外側にできる。

S2 = 3 × 6 × 1 = 18 cm²


3. 3つの扇形（頂点での円弧部分）

正三角形の内角は60°なので、外側を回る際の転回角は180° - 60° = 120°。

3つの扇形の中心角の合計 = 120° × 3 = 360°（完全な円）

S3 = π × 1² = π cm²


【合計面積】

S = 9√3 + 18 + π cm²


したがって、正解は(5)である。
