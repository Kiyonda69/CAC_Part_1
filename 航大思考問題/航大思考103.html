問1 配点: 6点 / 目安時間: 3分
下図は立方体の展開図であり、各面にア〜カの記号が書かれている。展開図上の頂点Pに黒丸の印がつけられている。この展開図を折り曲げて立方体を組み立てたとき、頂点Pに集まる3つの面の記号の組み合わせとして正しいものを、次の(1)〜(5)の中から1つ選べ。
【展開図】
解説（正解: (4)）

展開図の中心にある「エ」を正面として立方体を組み立てると、各面の配置は次のようになる。


【面の配置】

・エ → 正面、イ → 上面、カ → 底面、ウ → 左側面、オ → 右側面、ア → 背面


【対面関係】

・エ ↔ ア、イ ↔ カ、ウ ↔ オ


【頂点Pの追跡】

Pはイの左上角にある。イが上面に折り畳まれるとき、イの下辺（エとの境界）が手前、上辺が奥、左辺が左側面側になる。

よって、イの左上角は「奥・左・上」の頂点 → ア（背面）・ウ（左側面）・イ（上面）が集まる頂点に対応する。


【選択肢の検討】

・(1) ア, オ, イ → 奥・右・上の角。Pとは異なる。

・(2) ウ, エ, イ → 手前・左・上の角。Pとは異なる。

・(3) ア, エ, イ → アとエは対面関係にあり、同じ頂点に集まることは不可能。

・(4) ア, ウ, イ → 奥・左・上の角。Pと一致。正解。

・(5) ア, ウ, カ → 奥・左・下の角。Pとは異なる。


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
下図は立方体の展開図であり、各面にア〜カの記号が書かれている。展開図上の4つの頂点にP, Q, R, Sの印がつけられている。この展開図を折り曲げて立方体を組み立てたとき、Pと同じ頂点に重なる印として正しいものを、次の(1)〜(5)の中から1つ選べ。
【展開図】
解説（正解: (3)）

このT字型展開図を「エ」を正面として組み立てる。


【面の配置】

・エ → 正面、イ(エの上) → 上面、ア(イの左) → 左側面、ウ(イの右) → 右側面

・オ(エの下) → 底面、カ(オの下) → 背面

・対面: エ↔カ、イ↔オ、ア↔ウ


【各マークの追跡】

・P（アの左上角）: アはイ(上面)の左辺から折り下げて左側面になる。アの左上角は「奥・左・下」の頂点、すなわち背面(カ)・左側面(ア)・底面(オ)が集まる点に移動。


・Q（イの右上角）: イは上面。イの右上角は「奥・右・上」の頂点 → 背面(カ)・右側面(ウ)・上面(イ)が集まる点。Pとは異なる。


・R（オの左下角 = カの左上角）: オは底面。オの下辺はカ(背面)側。オの左下角は「奥・左・下」の頂点 → Pと同じ頂点に移動。


・S（カの右下角）: カは背面。カの右下角は背面の奥・右・上に対応 → 背面(カ)・右側面(ウ)・上面(イ)が集まる点。Qと同じ頂点。Pとは異なる。


PとRが同じ頂点に集まり、QとSが別の同じ頂点に集まる。Pと同じ頂点に重なるのはRのみ。


したがって、正解は(3)である。
