問1 配点: 6点 / 目安時間: 3分
下の図のように、立方体ABCD-EFGHを、辺AD・DC・CG・GF・FE・EAの中点を通る平面で切断したところ、切り口は正六角形となった。この立方体を、前面ABFEを中央にした十字型の展開図に開いたとき、切り口の線（切断面の縁）はどのように現れるか。最も適切なものを下の選択肢から1つ選びなさい。
【図1】立方体と展開図の対応
解説（正解: (1)）

立方体を辺AD・DC・CG・GF・FE・EAの中点を通る平面で切断したときの正六角形の切り口を、十字型展開図上で特定する。


【切り落とされる頂点の特定】

各面で切り口の2本の中点線分は、共通の頂点で出会う。

・上面ABCD: 辺AD・DCの中点 → 頂点D

・後面DCGH: 辺DC・CGの中点 → 頂点C

・右面BCGF: 辺CG・GFの中点 → 頂点G

・下面EFGH: 辺GF・FEの中点 → 頂点F

・前面ABFE: 辺FE・EAの中点 → 頂点E

・左面ADHE: 辺EA・ADの中点 → 頂点A

切り落とされる6頂点はA・C・D・E・F・G。残るB・Hは体対角線BHの両端で、切断面はBHに垂直な中点平面である。


【展開図上の各パネルでの切り取り位置】

・上面パネル: D=左上 → 左上コーナー

・前面パネル: E=左下 → 左下コーナー

・下面パネル: F=右上 → 右上コーナー

・左面パネル: A=右上 → 右上コーナー

・右面パネル: G=右下 → 右下コーナー

・後面パネル: C=左上 → 左上コーナー


【選択肢の検討】

・(2) ⊥AGパターン（切り取り頂点B・C・D・E・F・H）→ 不正解

・(3) ⊥CEパターン（切り取り頂点A・B・D・F・G・H）→ 不正解

・(4) ⊥DFパターン（切り取り頂点A・B・C・E・G・H）→ 不正解

・(5) 中央4面に水平帯。これは正方形断面であり正六角形ではない → 不正解


したがって、正解は(1)である。

【図2】立方体と展開図の対応
