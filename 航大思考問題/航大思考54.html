表は、日本の港湾を国際戦略港湾、国際拠点港湾、重要港湾などの種類別にまとめた取扱状況のデータである。これらの区分のうち、入港船舶1隻あたりの貨物取扱量が2番目に多い区分はどれか。正しいものを1つ選べ。

各区分について「貨物取扱量 / 入港船舶数」を計算し、2番目に大きい値を求める。


【各区分の入港船舶1隻あたりの貨物取扱量】

・重要港湾: 285,350 / 245,600 ≒ 1.162 千トン/隻

・地方港湾: 82,480 / 186,320 ≒ 0.443 千トン/隻

・56条港湾: 3,280 / 8,650 ≒ 0.379 千トン/隻


【順位】

1位: 国際戦略港湾(2.724) → 2位: 国際拠点港湾(1.385) → 3位: 重要港湾(1.162) → 4位: 地方港湾(0.443) → 5位: 56条港湾(0.379)


2番目に多いのは国際拠点港湾であり、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
表は、日本の交通機関別の旅客輸送状況をまとめたものである。次のア～エの記述のうち、正しいものの組み合わせを1つ選べ。

（国土交通省：「令和5年度交通関連統計資料集」より作成）
ア　旅客1人あたりの平均移動距離（旅客キロ/旅客数）が最も長い交通機関は航空である。
イ　従業員1千人あたりの営業収入が最も高い交通機関は鉄道(JR)である。
ウ　旅客キロ1億人kmあたりの営業収入が最も高い交通機関はタクシーである。
エ　鉄道(民鉄)の従業員数は全交通機関の合計従業員数の15%を超える。
解説（正解: (2)）

【各記述の検証】


ア: 旅客1人あたりの平均移動距離が最も長い交通機関は航空

旅客キロ(億人km) / 旅客数(百万人) × 100 = 平均距離(km)

・鉄道(JR): (2,845 / 9,284) × 100 ≒ 30.6 km

・鉄道(民鉄): (1,326 / 5,482) × 100 ≒ 24.2 km

・バス: (428 / 3,856) × 100 ≒ 11.1 km

・タクシー: (85 / 1,248) × 100 ≒ 6.8 km

・航空: (1,128 / 1,056) × 100 ≒ 106.8 km

→ 航空が最も長い → 正しい


イ: 従業員1千人あたりの営業収入が最も高いのは鉄道(JR)

・鉄道(JR): 42,680 / 128 ≒ 333.4 億円

・航空: 34,520 / 62 ≒ 556.8 億円

→ 最も高いのは航空(556.8)であり鉄道(JR)ではない → 誤り


ウ: 旅客キロ1億人kmあたりの営業収入が最も高いのはタクシー

・タクシー: 15,680 / 85 ≒ 184.5 億円

・航空: 34,520 / 1,128 ≒ 30.6 億円

・バス: 12,340 / 428 ≒ 28.8 億円

→ タクシーが最も高い → 正しい


エ: 鉄道(民鉄)の従業員数は全合計の15%を超える

・合計従業員数: 128 + 86 + 185 + 295 + 62 = 756千人

・民鉄の割合: 86 / 756 ≒ 11.4%

→ 15%を超えない → 誤り


正しい記述はアとウであり、正解は(2)である。
