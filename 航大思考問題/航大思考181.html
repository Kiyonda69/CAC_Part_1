問1 配点: 6点 / 目安時間: 3分
次の月次売上レポートを読み、4店舗の業績に関する記述として正しいものを(1)〜(5)の中から1つ選べ。
小売業X社の当月レポート。東店では来客数8,500人のうち2,720人が購入に至り、売上は4,080千円となった。西店は来客数6,000人で購入者は1,800人、売上は3,150千円である。南店の来客数は7,500人で、そのうち2,400人が購入し、売上は3,360千円であった。北店は来客数5,000人、購入者数1,400人、売上は2,100千円である。なお、購入率は購入者数を来客数で割った値、客単価は売上を購入者数で割った値、来店者1人あたり売上は売上を来客数で割った値とする。
【表1】4店舗の月次売上データ
店舗来客数（人）購入者数（人）売上（千円）
東店8,5002,7204,080
西店6,0001,8003,150
南店7,5002,4003,360
北店5,0001,4002,100
解説（正解: (4)）

各店の指標を計算する。


【購入率】（購入者数÷来客数）

東店 2,720÷8,500 = 32.0% / 西店 1,800÷6,000 = 30.0%

南店 2,400÷7,500 = 32.0% / 北店 1,400÷5,000 = 28.0%


【客単価】（売上÷購入者数）

東店 1,500円 / 西店 1,750円 / 南店 1,400円 / 北店 1,500円


【来店者1人あたり売上】（売上÷来客数）

東店 480円 / 西店 525円 / 南店 448円 / 北店 420円


【選択肢の検討】

・(1) 売上最大は東店(4,080千円)であり西店ではない → 誤

・(2) 購入率32.0%は東店と南店の両方であり、東店のみとは言えない → 誤

・(3) 客単価最低は南店(1,400円)であり、北店ではない → 誤

・(4) 来店者1人あたり売上最大は西店(525円) → 正

・(5) 北店の購入率は28.0%で30%を下回る → 誤


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
次の品質管理報告書を読み、5生産ラインの実績に関する記述として正しいものを(1)〜(5)の中から1つ選べ。
Y食品工場では当月、5つの生産ライン（A〜E）で製品を製造した。Aラインは生産数12,000個に対し不良品240個が確認され、検査時間は60時間を要した。Bラインは生産数8,000個、不良品120個、検査時間50時間であった。Cラインは生産数15,000個、不良品450個、検査時間75時間である。Dラインは生産数6,000個、不良品90個、検査時間30時間にとどまった。Eラインは生産数10,000個、不良品200個、検査時間40時間であった。なお、不良品率は不良品数を生産数で割った値、検査効率は生産数を検査時間で割った値（1時間あたり検査個数）とする。
【表2】5生産ラインの月次実績
ライン生産数（個）不良品数（個）検査時間（h）
A12,00024060
B8,00012050
C15,00045075
D6,0009030
E10,00020040
解説（正解: (5)）

各ラインの指標を計算する。


【不良品率】（不良品数÷生産数）

A 2.0% / B 1.5% / C 3.0% / D 1.5% / E 2.0%


【検査効率】（生産数÷検査時間）

A 200個/h / B 160個/h / C 200個/h / D 200個/h / E 250個/h


【選択肢の検討】

・(1) 不良品率1.5%はBとDの2ラインに存在するため、「Bのみ」とは言えない → 誤

・(2) 検査効率最大はE(250個/h)であり、A(200個/h)ではない → 誤

・(3) 生産数の合計は12,000+8,000+15,000+6,000+10,000=51,000個であり、52,000個ではない → 誤

・(4) 不良品数最多はC(450個)であり、A(240個)ではない → 誤

・(5) 検査効率の最大はEの250個/hである。「250個/hを超える」とは250より大きい値を指すため、ちょうど250個/hのEを含めいずれも250を超えない → 正


【ポイント】選択肢(5)では「250個/hを超える」（>250）と「250個/h以上」（≧250）の違いに注意。Eは250ちょうどであり「超える」には該当しない。


したがって、正解は(5)である。
