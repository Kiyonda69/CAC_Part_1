問1 配点: 6点 / 目安時間: 3分
下図は、ある航空機を真上から見た図（上面図）と前方から見た図（正面図）である。この航空機を機体の右側面から見た図として最も適切なものを、(1)〜(5)から1つ選べ。
【図1】航空機の上面図と正面図
解説（正解: (3)）

上面図と正面図から、航空機の三次元的な特徴を読み取り、右側面図を構成する。


【上面図から読み取れる情報】

・機首は右側（細く尖った先端）

・機尾は左側（垂直尾翼の根元・水平尾翼が配置）

・主翼は胴体中央付近に左右対称に広がる（後退翼）

・水平尾翼は機尾近くに左右対称に広がる

・エンジンは主翼下の左右に2基


【正面図から読み取れる情報】

・垂直尾翼は胴体の上方に伸びる三角形

・エンジンは主翼より下方にぶら下がる円形


【右側面図に必要な特徴】

右側面図では、機首は右、機尾は左に来る。垂直尾翼は後部（左側）の上方、エンジンは主翼下の下方、機首は尖った形になる。


【選択肢の検討】

・(1) 機首が左を向いている → 方向が逆 → 不正解

・(2) 垂直尾翼が機首側（前方）にある → 不正解

・(3) すべての条件と一致 → 正解

・(4) 垂直尾翼が胴体の下方 → 正面図では尾翼は上方 → 不正解

・(5) 機首が尖っておらず平ら → 上面図では機首は尖っている → 不正解


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下図は、立方体の展開図である。立方体の6つの面には、それぞれ航空に関係する文字（A：エアクラフト、B：ブルースカイ、C：コントロール、D：ダイヤモンド、E：エンジン、F：フラップ）が描かれている。この展開図を組み立て、Bの面が正面に、Cの面が下になるように立方体を置いた。このとき、右側面に見える文字として正しいものを、(1)〜(5)から1つ選べ。
【図2】立方体の展開図
解説（正解: (2)）

展開図から立方体を組み立て、面の位置関係を空間的に把握する問題である。


【展開図の構造】

T字型展開図において、Cの面を中心とすると、上下左右に隣接する面は「Cを正面とした立方体」の各方向の面に対応する。

・C：中央（正面）

・B：Cの上に隣接 → 上面

・F：Cの下に隣接 → 下面

・A：Cの左に隣接 → 左側面

・D：Cの右に隣接 → 右側面

・E：Dの右端、折り返して後面（裏面）


【標準向き（Cが正面）の面配置】

正面：C、上：B、下：F、左：A、右：D、後：E


【操作: Bを正面、Cを下にする】

立方体を手前に90度倒す（前傾）。元の上面Bが正面に来て、元の正面Cが下に移る。

・正面：B（元の上）

・上：E（元の後）

・下：C（元の正面）

・後：F（元の下）

・左：A（不変）

・右：D（不変）


前後方向の回転では左右は変化しないため、AとDの位置は標準向きと同じである。


【選択肢の検討】

・(1) F：操作後は後面 → 不正解

・(2) D：操作後も右側面のまま → 正解

・(3) A：操作後も左側面のまま → 不正解

・(4) E：操作後は上面 → 不正解

・(5) C：操作後は下面 → 不正解


したがって、正解は(2)である。
