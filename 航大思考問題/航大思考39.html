問1 配点: 6点 / 目安時間: 3分
図のような正方形ABCDの頂点に次のア〜オの操作のいずれか1つを行ったとき、A,B,C,Dの並びが元の正方形と必ず一致する操作はどれか。正しいものを1つ選べ。
ア　AとBを入れ替える。
イ　AとC、BとDをそれぞれ入れ替える。
ウ　AとBを入れ替えた後、CとDを入れ替える。
エ　AをCの位置、BをAの位置、CをDの位置、DをBの位置にそれぞれ移す。
オ　ランダムに二つの頂点を選び、それらを入れ替える。これを2回行う。
解説（正解: (5)）

正方形ABCDの対称操作（回転・反転で元の図形と一致する操作）は以下の8通りである（対称群D4）。


【正方形の対称群D4】

恒等変換: (A,B,C,D)、90度回転: (D,A,B,C)、180度回転: (C,D,A,B)、270度回転: (B,C,D,A)

水平軸反転: (D,C,B,A)、垂直軸反転: (B,A,D,C)、対角線AC反転: (A,D,C,B)、対角線BD反転: (C,B,A,D)


【各操作の判定】

ア: AとBの交換 → (B,A,C,D) → D4に含まれない

イ: AC交換+BD交換 → (C,D,A,B) → D4に含まれる（180度回転）

ウ: AB交換後CD交換 → (B,A,D,C) → D4に含まれる（垂直軸反転）

エ: A→C, B→A, C→D, D→B → (B,D,A,C) → D4に含まれない

オ: 2回の交換で(C,A,B,D)等のD4外の置換が生じうるため、必ず一致するとは限らない


したがって、正解は(5) イ、ウである。

問2 配点: 6点 / 目安時間: 5分
5つの地区P,Q,R,S,Tにはそれぞれ施設があり、バス会社X,Y,Zがこれらの地区間で直通バスを運行している。これらの地区のうちいくつかは山岳地帯に位置している。次のことがわかっているとき、各地区と直通バスについて確実にいえるものはどれか。正しいものを1つ選べ。
(I) P〜Tのうち、山岳地帯でない地区は平野部に位置して互いに隣接しており、山岳地帯の地区はほかのどの地区とも隣接していない。
(II) X社は平野部の2地区と、山岳地帯の1地区の間で直通バスを運行している。
(III) Y社は平野部の1地区と、山岳地帯の2地区の間で直通バスを運行しており、そのうち1地区はTである。
(IV) Z社は平野部の2地区の間でのみ直通バスを運行している。
(V) Pは平野部であり、どのバス会社もPとそのほかの地区の間で直通バスを運行している。
(VI) すべての地区で、いずれかの会社が直通バスを運行している。
ア　2社の直通バスが運行されている路線は1組である
イ　Tは山岳地帯である
ウ　QR間はいずれかのバス会社が直通バスを運行している
エ　Z社の便のみ発着している地区がある
解説（正解: (1)）

【条件の整理】

条件(V)より、Pは平野部。条件(III)より、Yは山岳地帯の2地区を含みその1つがT → Tは山岳地帯。

条件(V)より、YはPを含む必要があるため、Yの平野部地区はP。よってY = {P, T, 他の山岳地区}。

山岳地帯はT以外にもう1つ必要。平野部は少なくとも2地区（Z社が2地区間で運行するため）。


【可能な構成の分析】

山岳地帯 = {T, Q or R or S}の組み合わせと、平野部の残り地区の配分により複数の有効な構成が存在する。


【選択肢の検証】

ア: 全有効構成で、2社が共通して運行する路線は常にちょうど1組 → 確実に真

イ: Tは全構成で山岳地帯 → 確実に真

ウ: QR間にバスが運行されない構成が存在 → 確実とはいえない

エ: Z社のみ発着する地区がない構成が存在 → 確実とはいえない


したがって、確実にいえるのはアとイであり、正解は(1)である。
