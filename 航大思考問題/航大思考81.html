問1 配点: 6点 / 目安時間: 3分
9個のマス目がついた透明なシートがある。マス目の中には記号がかかれている。下の例のように、2枚のシートを重ねると、両方の記号が見える。
【例】2枚のシートの重ね合わせ
次のアからオまでのシートのうち3枚を重ねたところ、下の目標パターンのように見えた。重ねたシートの組み合わせは次のうちどれか。正しいものを1つ選べ。ただし、シートは回転させないものとする。
【シート一覧】
【目標パターン】
解説（正解: (1)）

【目標パターンの分析】

目標パターンには9つの記号がある：

・1行目：●（黒丸）、□（白四角）、▲（黒三角）

・2行目：◇（白ひし形）、○（白丸）、■（黒四角）

・3行目：△（白三角）、▼（逆黒三角）、●（黒丸）


【検証：ア＋イ＋ウ】

・アの●(左上) + イの□(上中) + ウの▲(右上) → 1行目完成

・イの◇(左中) + アの○(中央) + ウの■(右中) → 2行目完成

・イの△(左下) + ウの▼(下中) + アの●(右下) → 3行目完成

すべてのセルが一致し、記号の衝突もない。


他の選択肢は目標パターンの全記号を再現できないか、同じセルに異なる記号が重なる。


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
9個のマス目がついた透明なシートがある。マス目の中には記号がかかれている。下の例のように、シートは回転させることができ、回転後に重ねると異なるパターンが得られる。ただし、裏返すことはできないものとする。
【例】回転と重ね合わせ
次のアからオまでのシートのうち3枚を選び、必要に応じて回転させて重ねたところ、下の目標パターンのように見えた。重ねたシートの組み合わせは次のうちどれか。正しいものを1つ選べ。
【シート一覧】
【目標パターン】
解説（正解: (5)）

【目標パターンの分析】

・1行目：■（黒四角）、▲（黒三角）、◇（白ひし形）

・2行目：△（白三角）、空白、○（白丸）

・3行目：●（黒丸）、□（白四角）、空白


【各シートの検討】

・シートア（回転なし）：■(左上)、○(右中) → 目標の位置に一致

・シートイ（回転なし）：▲(左中)、●(右下)

　→ 90°時計回り回転すると：▲(上中)、●(左下) → 目標の位置に一致

・シートウ（回転なし）：◇(右上)、△(左中)、□(下中) → 目標の位置に一致


【検証：ア(0°) ＋ イ(90°回転) ＋ ウ(0°)】

・アの■(左上) + イ(90°)の▲(上中) + ウの◇(右上) → 1行目完成

・ウの△(左中) + 空白(中央) + アの○(右中) → 2行目完成

・イ(90°)の●(左下) + ウの□(下中) + 空白(右下) → 3行目完成

すべてのセルが一致し、衝突もない。


他の選択肢はいかなる回転の組み合わせでも目標パターンを再現できない。


したがって、正解は(5)である。
