解説（正解: (4)）

【解法のポイント】

各部門の非正規割合は表に示されているが、2部門を合わせた非正規割合は単純に足したり平均したりするのではなく、合計人数から計算する必要がある。


【計算過程】

2016年:

・非正規合計 = 160 + 210 = 370人

・全従業員 = (400 + 160) + (290 + 210) = 560 + 500 = 1060人

・非正規割合 = 370 / 1060 = 34.9%


2017年:

・非正規合計 = 240 + 270 = 510人

・全従業員 = (300 + 240) + (250 + 270) = 540 + 520 = 1060人

・非正規割合 = 510 / 1060 = 48.1%


増加ポイント = 48.1 - 34.9 = 13.2ポイント


したがって、正解は(4) 13.2ポイントである。

問2 配点: 6点 / 目安時間: 5分
表は、あるスーパーマーケットチェーンの3つの地域（東部・中部・西部）の店舗数と1店舗あたり平均売上高を年度別に示したものである。全地域を合わせた1店舗あたり平均売上高は、2019年から2020年にかけておよそいくら減少したか。正しいものを1つ選べ。

解説（正解: (3)）

【解法のポイント】

3地域の平均売上高を単純に平均するのではなく、各地域の総売上高（= 店舗数 × 平均売上高）を合計し、全店舗数で割って加重平均を求める必要がある。店舗数が地域ごとに異なるため、単純平均と加重平均は一致しない。


【計算過程】

2019年:

・東部: 35店 × 90百万円 = 3,150百万円

・中部: 25店 × 80百万円 = 2,000百万円

・西部: 22店 × 76百万円 = 1,672百万円

・総売上高 = 6,822百万円、総店舗数 = 82店

・加重平均 = 6,822 / 82 = 83.2百万円


2020年:

・東部: 40店 × 72百万円 = 2,880百万円

・中部: 28店 × 65百万円 = 1,820百万円

・西部: 30店 × 60百万円 = 1,800百万円

・総売上高 = 6,500百万円、総店舗数 = 98店

・加重平均 = 6,500 / 98 = 66.3百万円


減少額 = 83.2 - 66.3 = 16.9百万円


【注意】

単純に3地域の平均売上高を平均すると、2019年は(90+80+76)/3=82.0、2020年は(72+65+60)/3=65.7となり、差は16.3百万円（選択肢(2)）となる。これは店舗数による重みを考慮していないため誤りである。


したがって、正解は(3) 16.9百万円である。
