問1 配点: 6点 / 目安時間: 3分
表は、日本の水道事業体を給水人口の規模別にまとめた運営状況のデータである。これらの区分のうち、職員1人あたりの年間総配水量が2番目に多い区分はどれか。正しいものを1つ選べ。
【水道事業の規模別運営状況】
事業規模 事業体数 給水人口
(千人)		 年間総配水量
(千m3)		 職員数
(人)
大規模(給水人口50万人以上) 12 18,456 4,128,500 28,450
中規模(10万～50万人) 78 16,234 3,245,600 18,920
やや中規模(5万～10万人) 115 8,124 1,586,400 9,850
小規模(1万～5万人) 482 10,856 2,124,800 12,340
極小規模(1万人未満) 685 3,248 645,200 5,680
合計 1,372 56,918 11,730,500 75,240
（総務省：「令和5年度地方公営企業年鑑」より作成）
解説（正解: (2)）

各区分について「年間総配水量 / 職員数」を計算し、2番目に大きい値を求める。


【各区分の職員1人あたり年間総配水量】

・大規模: 4,128,500 / 28,450 ≒ 145.1 千m3/人

・中規模: 3,245,600 / 18,920 ≒ 171.5 千m3/人

・やや中規模: 1,586,400 / 9,850 ≒ 161.1 千m3/人

・小規模: 2,124,800 / 12,340 ≒ 172.2 千m3/人

・極小規模: 645,200 / 5,680 ≒ 113.6 千m3/人


【順位】

1位: 小規模(172.2) → 2位: 中規模(171.5) → 3位: やや中規模(161.1) → 4位: 大規模(145.1) → 5位: 極小規模(113.6)


したがって、2番目に多いのは中規模(10万～50万人)であり、正解は(2)である。

ア　農業就業人口1千人あたりの農業産出額が最も高い地域は北海道である。
イ　経営体1つあたりの耕地面積が2番目に広い地域は九州である。
ウ　農業産出額に占める農業所得の割合が最も低い地域は北陸である。
エ　耕地面積1haあたりの農業産出額が最も高い地域は関東である。
解説（正解: (2)）

【各記述の検証】


ア: 農業就業人口1千人あたりの農業産出額

・北海道: 12,667 / 82 ≒ 154.5（最大）

・九州: 17,632 / 196 ≒ 90.0

・東海: 7,524 / 105 ≒ 71.7

→ 北海道が最も高い → 正しい


イ: 経営体1つあたりの耕地面積

・北海道: 1,143,000 / 36,200 ≒ 31.57 ha（1位）

・北陸: 168,200 / 52,800 ≒ 3.19 ha（2位）

・東北: 582,400 / 184,500 ≒ 3.16 ha（3位）

・九州: 348,200 / 132,400 ≒ 2.63 ha（4位）

→ 2位は北陸であり九州ではない → 誤り


ウ: 農業産出額に占める農業所得の割合

・北陸: 780 / 3,145 ≒ 24.8%（最低）

・中国四国: 2,034 / 7,856 ≒ 25.9%

・近畿: 1,425 / 5,280 ≒ 27.0%

→ 北陸が最も低い → 正しい


エ: 耕地面積1haあたりの農業産出額

・東海: 7,524 / 135,800 ≒ 0.0554 億円/ha（最大）

・近畿: 5,280 / 103,500 ≒ 0.0510

・関東: 19,280 / 418,600 ≒ 0.0461

→ 最も高いのは東海であり関東ではない → 誤り


正しい記述はアとウであり、正解は(2)である。
