問1 配点: 6点 / 目安時間: 3分
次の【資料】および【表】を読み、文中の空欄（ア）〜（エ）に入る語句・数値の組合せとして正しいものを、下の(1)〜(5)のうちから1つ選べ。
【資料】M市 公立図書館の貸出状況（2025年度）
M市内のA館・B館・C館・D館の4つの公立図書館について、2025年度の年間貸出冊数を「小説」「専門書」「児童書」の3カテゴリに分けて集計した。4館全体の小説の貸出冊数は（ア）冊にのぼり、最も貸出の多いカテゴリとなっている。各館の蔵書構成や利用者層は異なっており、児童書の貸出が館全体の貸出冊数に占める割合（児童書比率）が最も高いのは（イ）館であった。また、4館のうち専門書の貸出冊数が小説の貸出冊数を上回ったのは（ウ）館のみであり、専門的な学習ニーズが強いことがうかがえる。なお、4館全体の年間貸出冊数（全カテゴリの合計）は（エ）冊である。

※児童書比率＝（児童書の貸出冊数）÷（その館の貸出冊数の合計）。
解説（正解: (5)）

【表】の数値を用いて、資料文中の各空欄に入る値を順に確定する。


【各空欄の確定】

・（ア）4館合計の小説の貸出冊数：800 + 500 + 400 + 800 = 2,500冊（表のカテゴリ合計欄と一致）。

・（イ）児童書比率（児童書÷館合計）：A 300/1,500 = 20.0％、B 600/1,400 ≒ 42.9％、C 300/1,200 = 25.0％、D 700/1,800 ≒ 38.9％。最大はB館。なお児童書の冊数自体が最大なのはD館（700冊）であり、冊数と比率を取り違えないこと。

・（ウ）「専門書 ＞ 小説」となる館の数：A 400＜800（×）、B 300＜500（×）、C 500＞400（○）、D 300＜800（×）。該当は1館（C館のみ）。

・（エ）4館全カテゴリの貸出冊数の合計：1,500 + 1,400 + 1,200 + 1,800 = 5,900冊。


【選択肢の検討】

・(1) イ=Dは誤り（児童書「冊数」は最大だが、児童書「比率」最大はB館42.9％）。

・(2) ウ=2は誤り（「専門書＞小説」を満たすのはC館のみで1館）。

・(3) ア=1,900は児童書合計であり、小説合計ではない。

・(4) エ=1,900は児童書合計のみで、4館全カテゴリの合計ではない。

・(5) ア=2,500、イ=B、ウ=1、エ=5,900 → すべて整合 → 正しい


したがって、正解は(5)である。

【資料】K研修施設 4-6月 会議室利用レポート
K研修施設には収容人数の異なる4つの会議室α・β・γ・δがある。各室の月間総開館時間はいずれも180時間（30日×6時間）であり、3カ月間（4-6月）では540時間にあたる。2025年4-6月の3カ月間における4室合計の利用時間は（ア）時間に達した。各室の収益効率を比較するため、月別に「時間あたり収益（収益÷利用時間）」を算出したところ、6月において時間あたり収益が最も高かったのは（イ）室であり、収容人数の大きな会議室が高単価で利用された月であったことがわかる。また、3カ月平均の利用率（3カ月の利用時間÷540時間）が50％以上だった会議室は（ウ）室存在する。さらに、γ室の月平均収益（3カ月の収益÷3）は（エ）千円であり、4室全体の3カ月収益合計は（オ）千円となった。

解説（正解: (1)）

【表】の値を用いて、資料文中の各空欄を順に確定する。


【各空欄の確定】

・（ア）4室合計の利用時間（4-6月の3カ月分）：α 240、β 310、γ 300、δ 240 → 合計 240+310+300+240 = 1,090時間（件数合計179と取り違えないこと）。

・（イ）6月の時間あたり収益（収益÷利用時間）：α 250/100=2.5、β 360/120=3.0、γ 480/120=4.0、δ 450/100=4.5（千円/時） → δ室。3カ月収益合計はγ1,200＞δ1,080だが、6月単月の「時間あたり」ではδが上回る。

・（ウ）3カ月の利用率（利用時間÷540）：α 240/540≒44.4％（×）、β 310/540≒57.4％（○）、γ 300/540≒55.6％（○）、δ 240/540≒44.4％（×） → 50％以上は2室（β・γ）。

・（エ）γ室の月平均収益：（320+400+480）÷3 = 1,200÷3 = 400千円。1,200は3カ月合計であり、月平均ではない。

・（オ）4室全体の3カ月収益合計：α 600 + β 930 + γ 1,200 + δ 1,080 = 3,810千円。


【選択肢の検討】

・(1) ア=1,090、イ=δ、ウ=2、エ=400、オ=3,810 → すべて整合 → 正しい

・(2) イ=γは誤り（3カ月収益合計はγが最大だが、6月の「時間あたり収益」はδ4.5＞γ4.0）。

・(3) ウ=3は誤り（αとδはともに44.4％で50％未満）。

・(4) エ=1,200はγ室の3カ月合計収益であり、月平均ではない。

・(5) ア=179は3カ月の利用件数合計であり、利用時間合計ではない。


したがって、正解は(1)である。
