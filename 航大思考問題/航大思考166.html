問1 配点: 6点 / 目安時間: 3分
下の3つの図は、各面に1〜6の数字が一つずつ書かれた同じサイコロを、異なる方向から見たものである。このサイコロを、前面（数字「2」の面）を中央に置いた十字型の展開図に開いたとき、空欄(ア)・(イ)・(ウ)に入る数の組合せとして正しいものを、下の選択肢(1)〜(5)のうちから一つ選びなさい。
【図1】サイコロの3つの視点
【図2】十字型展開図（前面=2 を中央に配置）
解説（正解: (3)）

3つの視点から、サイコロの各面の対応関係を順に決定する問題である。


【視点1から判明する3面】

視点1: 上=1, 前=2, 右=3。上面・前面・右面の3面が確定。残り3面（左、後、下）は{4, 5, 6}のいずれか。


【視点2: 前後軸まわりの左回転（roll left）】

視点2: 上=3, 前=2, 右=6。前=2が共通なので、前後軸まわりに左へ90°回転（左に転がす）したもの。

左回転では「上→左、左→下、下→右、右→上」。新しい上=3は視点1の右=3と整合。新しい右=6は視点1の下面に相当 → 下=6。よって対面: 上↔下 = 1↔6。


【視点3: 前後軸まわりの右回転（roll right）】

視点3: 上=4, 前=2, 右=1。前=2が共通なので、前後軸まわりに右へ90°回転したもの。

右回転では「上→右、右→下、下→左、左→上」。新しい右=1は視点1の上=1と整合。新しい上=4は視点1の左面に相当 → 左=4。よって対面: 左↔右 = 4↔3。


【後面の決定】

残る面は後面のみ、未使用の数字は5。よって後=5（対面: 前↔後 = 2↔5）。

すべての対面の組: 1↔6、2↔5、3↔4（標準的な西洋サイコロと同じ配置）。


【十字型展開図】

上=1、左(ア)=4、前=2、右=3、後(イ)=5、下(ウ)=6。


【選択肢の検討】

・(1) (5,4,6): 視点3より左=4のはずが5 → 不正解

・(2) (4,6,5): 視点2より下=6のはずが5 → 不正解

・(3) (4,5,6): すべて整合 → 正解

・(4) (6,5,4): 視点3より左=4のはずが6 → 不正解

・(5) (5,6,4): 視点3より左=4のはずが5 → 不正解


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下の4つの図は、各面に1〜6の数字が一つずつ書かれた同じサイコロを、異なる方向から見たものである（このサイコロは標準的な西洋サイコロとは限らない）。このサイコロを、前面（数字「2」の面）を中央に置いた十字型の展開図に開いたとき、空欄(ア)・(イ)・(ウ)に入る数の組合せとして正しいものを、下の選択肢(1)〜(5)のうちから一つ選びなさい。
【図1】サイコロの4つの視点
【図2】十字型展開図（前面=2 を中央に配置）
解説（正解: (2)）

4視点から非標準サイコロの対面の組（上↔下、前↔後、左↔右）を決定する。

【視点1から3面確定】上=1, 前=2, 右=5。残り3面（下・後・左）は{3, 4, 6}のいずれか。

【視点2: 上=5, 前=2, 右=4】前=2共通 → 前後軸まわりに左回転（上→左、左→下、下→右、右→上）。新しい上=5は視点1の右=5と整合、新しい右=4は視点1の下面に相当 → 下=4（対面: 上↔下 = 1↔4）。

【視点3: 上=1, 前=5, 右=3】上=1共通 → 上下軸まわりに右回転（前→左、左→後、後→右、右→前）。新しい前=5は視点1の右=5と整合、新しい右=3は視点1の後面に相当 → 後=3（対面: 前↔後 = 2↔3）。

【視点4: 上=6, 前=2, 右=1】前=2共通 → 前後軸まわりに右回転（上→右、右→下、下→左、左→上）。新しい上=6は視点1の左面に相当 → 左=6（対面: 左↔右 = 6↔5）。

【十字型展開図】上=1、左(ア)=6、前=2、右=5、後(イ)=3、下(ウ)=4。

【選択肢の検討】

・(1) (3,6,4): 視点4より左=6、視点3より後=3に矛盾 → 不正解
・(2) (6,3,4): すべて整合 → 正解
・(3) (6,4,3): 視点2より下=4、視点3より後=3に矛盾 → 不正解
・(4) (4,3,6): 視点4より左=6、視点2より下=4に矛盾 → 不正解
・(5) (3,4,6): 視点4より左=6に矛盾 → 不正解

したがって、正解は(2)である。
