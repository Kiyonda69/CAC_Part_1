問1 配点: 6点 / 目安時間: 3分
下の表は、ブラックボックスPとブラックボックスQのそれぞれの入力と出力の関係を示している。下図のように、ブラックボックスPの出力をブラックボックスQの入力とする直列接続を考える。この直列接続に3を入力したとき、最終的な出力の値として正しいものを(1)～(5)から1つ選べ。
【図1】ブラックボックスの直列接続
【表1】各ブラックボックスの入出力
入力 x P の出力 Q の出力
150
273
398
41115
51324
解説（正解: (2)）

【ブラックボックスPの規則】

Pの出力は 5, 7, 9, 11, 13 と公差2の等差数列である。

P(x) = 2x + 3


【ブラックボックスQの規則】

Qの出力は 0, 3, 8, 15, 24 で、階差が 3, 5, 7, 9（公差2の等差数列）。

これは二次関数であり、Q(x) = x² - 1


【直列接続の計算】

入力 3 → P(3) = 2(3) + 3 = 9

Pの出力 9 をQに入力 → Q(9) = 9² - 1 = 81 - 1 = 80


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下の表は、ブラックボックスFとブラックボックスGの入出力の関係を示している。ただし、表中の(ア)～(ウ)は未知の値である。下図のように、ブラックボックスGの出力をブラックボックスFの入力とする直列接続 H(x) = F(G(x)) を考えるとき、H(4) の値として正しいものを(1)～(5)から1つ選べ。
【図2】ブラックボックスの直列接続 H(x) = F(G(x))
【表2】各ブラックボックスの入出力
入力 x F の出力 G の出力
132
2(ア)5
3158
4(イ)(ウ)
53514
解説（正解: (1)）

【ブラックボックスGの規則】

Gの既知の出力: G(1)=2, G(2)=5, G(3)=8, G(5)=14

階差: G(2)-G(1)=3, G(3)-G(2)=3（一定）→ 一次関数

G(x) = 3x - 1 → (ウ) = G(4) = 3(4) - 1 = 11


【ブラックボックスFの規則】

Fの既知の出力: F(1)=3, F(3)=15, F(5)=35

階差（2ステップ）: 15-3=12, 35-15=20、二階差分=8 → 二次関数

F(x) = ax² + bx + c として連立方程式を解くと a=1, b=2, c=0

F(x) = x² + 2x

（ア）= F(2) = 4+4 = 8、（イ）= F(4) = 16+8 = 24


【H(4) の計算】

H(4) = F(G(4)) = F(11) = 11² + 2(11) = 121 + 22 = 143


【選択肢の検討】

(1) 143 = F(11)：正解

(2) 132 = 11²+11：F(x) = x²+x と誤った場合

(3) 121 = 11²：F(x) = x² と誤った場合

(4) 99 = F(9)：G(4) = 9 と誤った場合

(5) 80 = F(8)：G(3) = 8 を G(4) と混同した場合


したがって、正解は(1)である。
