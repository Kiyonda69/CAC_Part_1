問1 配点: 6点 / 目安時間: 3分
図のように、黒と白の7つのマスからなる行が、ある規則にしたがって上から下のように変化している。図の規則と矛盾なく変化しているものは次のうちどれか。正しいものを1つ選べ。ただし、選択肢では矢印を省略している。
解説（正解: (3)）

【規則の発見】

例題の4行を観察すると、各行の次の行は以下の規則で生成されている:

「各マスの色は、そのマスと右隣のマス（最右端は最左端と比較）の色が異なれば黒、同じなら白」

これはXOR（排他的論理和）演算に相当する。


具体的に確認すると:

・行1の1番目(B)と2番目(B)は同色 → 次行1番目はW

・行1の2番目(B)と3番目(W)は異色 → 次行2番目はB

・行1の7番目(B)と1番目(B)は同色 → 次行7番目はW（端は循環）

→ 行2「WBBBWBW」と一致する。


【選択肢の検討】

・(1) 行1→行2: 規則を適用すると「BBBBBBW」→ 行2「WBBBWBW」と不一致

・(2) 行1→行2: 規則を適用すると「WBBBBBB」→ 行2「BWWBWWB」と不一致

・(3) 行1→行2, 行2→行3, 行3→行4 すべて規則に一致 → 正解

・(4) 行1→行2: 規則を適用すると「BWBBBBW」→ 行2「WBWWWWW」と不一致

・(5) 行1→行2は一致するが、行2→行3で4番目のマスが不一致


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
1辺の長さが2の立方体ABCD-EFGHがある（ABCDが上面、EFGHが底面で、A-E, B-F, C-G, D-Hがそれぞれ対応する頂点である）。辺ABの中点をP、辺FGの中点をQ、辺DHの中点をRとする。三角形PQRの面積として正しいものを(1)～(5)の中から1つ選べ。
解説（正解: (1)）

【座標の設定】

立方体の頂点に座標を設定する。

上面: A=(0,0,2), B=(2,0,2), C=(2,2,2), D=(0,2,2)

底面: E=(0,0,0), F=(2,0,0), G=(2,2,0), H=(0,2,0)


【中点の座標】

・P（ABの中点）= (1, 0, 2)

・Q（FGの中点）= (2, 1, 0)

・R（DHの中点）= (0, 2, 1)


【辺の長さの計算】

・PQ = √{(2-1)² + (1-0)² + (0-2)²} = √(1+1+4) = √6

・QR = √{(0-2)² + (2-1)² + (1-0)²} = √(4+1+1) = √6

・RP = √{(1-0)² + (0-2)² + (2-1)²} = √(1+4+1) = √6


3辺がすべて√6で等しいため、三角形PQRは正三角形である。


【面積の計算】

1辺√6の正三角形の面積 = (√3/4) × (√6)² = (√3/4) × 6 = 3√3/2


したがって、正解は(1)である。
