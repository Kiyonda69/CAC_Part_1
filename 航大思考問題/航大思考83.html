問1 配点: 6点 / 目安時間: 3分
次の図のような立方体の展開図を組み立てたとき、辺アイと重なる辺はどれか。
解説（正解: (3)）

展開図を③を正面として組み立てると、各面は以下の位置になります。


【面の配置】

・① → 上面（③の上に折り上げる）

・② → 左面（③の左に折る）

・④ → 右面（③の右に折る）

・⑤ → 背面（④のさらに右から後ろに折る）

・⑥ → 下面（③の下に折り下げる）


【頂点の対応】

組み立てると、以下の頂点が同一点に重なります。

・ア = ウ = キ（左上奥）

・イ = カ（右上奥）

・ク = シ = ス（左下奥）

・サ = セ（右下奥）


辺アイ（①の上辺）は立方体の上面奥の辺になります。

辺カキ（⑤の上辺）も同じく立方体の上面奥の辺です。

よって、辺アイと辺カキが重なります。


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下の見本のようなL字型の図形がある。各マスにはアルファベットが書かれている。この見本を時計回りに90°回転させた図形として正しいものを、(1)〜(5)から1つ選べ。ただし、回転に伴い文字の向きも変わることに注意せよ。
【見本】
解説（正解: (4)）

L字型図形を時計回りに90°回転させるとき、以下の2点に注意します。

1. セルの位置が90°回転する

2. セル内の文字の向きも90°回転する（文字が右に傾く）


【位置の変化】

元の配置を座標で表すと：

・F(0,2) → 左上、G(0,1) → 左中、H(0,0) → 左下、J(1,0) → 右下


90°時計回り回転 (x,y)→(y,-x) を正規化すると：

・F → (2,1) 右上、G → (1,1) 中上、H → (0,1) 左上、J → (0,0) 左下


つまり、回転後は上段にH, G, Fが左から右に並び、下段左にJが位置します。

各文字は90°時計回りに傾いた状態になります。


【選択肢の検討】

・(1) 180°回転した配置 → 不正解

・(2) 鏡像を回転した配置 → 不正解

・(3) 90°反時計回り回転 → 不正解

・(4) 位置・文字向きともに正しい90°時計回り回転 → 正解

・(5) 形は似ているがFとGの位置が入れ替わっている → 不正解


したがって、正解は(4)である。
