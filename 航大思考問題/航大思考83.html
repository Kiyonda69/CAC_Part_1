問1 配点: 6点 / 目安時間: 3分
次の図のような立方体の展開図を組み立てたとき、辺アイと重なる辺はどれか。
解説（正解: (3)）

③を正面として組み立てると、各面は以下の位置になります。

・① → 上面、② → 左面、④ → 右面、⑤ → 背面、⑥ → 下面


【頂点の対応】

・ア = ウ = キ（左上奥）、イ = カ（右上奥）

・ク = シ = ス（左下奥）、サ = セ（右下奥）


辺アイ（①の上辺）と辺カキ（⑤の上辺）はどちらも立方体の上面奥の辺であり、重なります。


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下の見本のようなL字型の図形がある。各マスにはアルファベットが書かれている。この見本を時計回りに90°回転させた図形として正しいものを、(1)〜(5)から1つ選べ。ただし、回転に伴い文字の向きも変わることに注意せよ。
【見本】
解説（正解: (4)）

L字型図形を時計回りに90°回転させるとき、セルの位置と文字の向きが両方変化します。


【位置の変化】

見本（列, 行）: F(0,0) G(0,1) H(0,2) J(1,2)

90°CW回転後: H(0,0) G(1,0) F(2,0) J(0,1)

→ 上段にH, G, Fが並び、下段左にJが位置する


【選択肢の検討】

・(1) 180°回転 → 不正解

・(2) Jが上のL字（鏡像）→ 不正解

・(3) 90°反時計回り回転 → 不正解

・(4) 90°時計回り回転 → 正解

・(5) FとGが入れ替わっている → 不正解


したがって、正解は(4)である。
