問1 配点: 6点 / 目安時間: 3分
同じ大きさの立方体を、縦3マス×横3マスの範囲の床の上にすき間なく積み重ねて立体を作った。この立体を正面から見た図と、右側面から見た図は、下の【図1】のようになった。立方体は床のマス目に沿って置かれ、上の段の立方体は必ず下の段の立方体の真上に載っている。また、立方体が1個も置かれていないマスがあってもよい。このとき、この立体を作るのに必要な立方体の個数として、考えられる最も少ない個数はどれか。
【図1】立体の見え方
問1（正解: (4)）

積み木の投影図問題では、床のマス目を上から見た3×3の表を考え、各マスに積む立方体の個数（高さ）を書き込んで考える。


【条件の整理】

・正面から見た図より、左・中央・右の各列の最も高い積み方は、それぞれ3個・1個・2個。

・右側面から見た図より、手前・中央・奥の各行の最も高い積み方は、それぞれ1個・3個・2個。

・つまり、各列（左右方向）の最大の高さが 3・1・2、各行（奥行き方向）の最大の高さが 2・3・1（奥から順）になればよい。


【最少個数の考え方】

・左の列には高さ3の積み方が少なくとも1マス必要で、その1マスだけで立方体3個を使う。同様に中央の列は1個、右の列は2個が最低限必要。列どうしは重ならないので、少なくとも 3＋1＋2＝6個は必要である。

・6個で実現できるかを確かめる。高さの内訳は正面図（3・1・2）と側面図（2・3・1）でどちらも「3・2・1が1つずつ」なので、同じ高さどうしを交差するマスに割り当てればよい。


【6個の配置例】（上から見た表・数字は積む個数、0は置かない）

・高さ3 → 左の列×中央の行の交点に3個

・高さ2 → 右の列×奥の行の交点に2個

・高さ1 → 中央の列×手前の行の交点に1個

奥　　： 0　0　2
中央　： 3　0　0
手前　： 0　1　0
　　　　（左・中央・右）

この配置は列の最大が左から 3・1・2、行の最大が奥から 2・3・1 となり、両方の見え方を満たす。使う立方体は 3＋2＋1＝6個。


【選択肢の検討】

・(1) 3個 → 最も高い列の3個だけでは、右の列（高さ2）や中央の列（高さ1）が作れない

・(2) 4個 → 左の列の3個と中央の1個だけでは、右の列の高さ2が作れない

・(3) 5個 → 各列に必要な最低数 3＋1＋2＝6個を下回るため不可能

・(4) 6個 → 上記の配置例により実現可能で、これより少なくはできない（正解）

・(5) 7個 → 実現可能だが最少ではない


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
同じ大きさの立方体を、縦3マス×横3マスの範囲の床の上にすき間なく積み重ねて立体を作った。真上から見ると、9つのマスすべてに立方体が見えた（どのマスにも少なくとも1個は積まれている）。この立体を正面から見た図と、右側面から見た図は、下の【図2】のようになった。立方体は床のマス目に沿って置かれ、上の段の立方体は必ず下の段の立方体の真上に載っている。このとき、この立体を作るのに使われた立方体の個数について、考えられる最も多い個数と最も少ない個数の差はいくらか。
【図2】立体の見え方
問2（正解: (1)）

問1と同じく、上から見た3×3の表に各マスの高さを書き込んで考える。正面図より各列の最大の高さは左から 2・4・3、右側面図より各行の最大の高さは奥から 4・2・3。さらに真上から見て9マスすべてに立方体が見えるので、どのマスも高さ1以上である。


【最も多い場合】

各マスの高さは、そのマスが属する列の最大値と行の最大値のどちらも超えられない。よって各マスの最大の高さは「列の値と行の値の小さい方」となる。

奥(4)　： 2　4　3 → 計9
中央(2)： 2　2　2 → 計6
手前(3)： 2　3　3 → 計8
　　　　　（左2・中央4・右3）

この配置は各列の最大が 2・4・3、各行の最大が 4・2・3 となり、実際に両方の見え方を満たす。よって最多個数は 9＋6＋8＝23個。


【最も少ない場合】

・9マスすべてに1個以上必要なので、まず9個は必ず使う。

・高さの内訳は正面図（2・4・3）と側面図（4・2・3）でどちらも「4・3・2が1つずつ」なので、同じ高さどうしを交差するマスに割り当てられる。高さ4は中央の列×奥の行、高さ3は右の列×手前の行、高さ2は左の列×中央の行の交点に置き、残りの6マスは1個ずつでよい。

奥(4)　： 1　4　1
中央(2)： 2　1　1
手前(3)： 1　1　3
　　　　　（左2・中央4・右3）

この配置も両方の見え方を満たし、合計は 4＋3＋2＋1×6＝15個。9マスの1個分に加えて、高さ4・3・2の柱に必要な上積み 3＋2＋1＝6個は必ずかかるため、これより少なくはできない。


【差の計算】

最多 23個 − 最少 15個 ＝ 8個


【選択肢の検討】

・(1) 8個 → 23−15＝8（正解）

・(2) 9個 → 最少を14個などと数え落とした場合の誤り

・(3) 10個 → 最少で高さの共有を1か所しか使わなかった場合の誤り

・(4) 11個 → 最少を12個（高さの合計だけ）とした場合の誤り

・(5) 12個 → 最多・最少の両方を誤った場合の値


したがって、正解は(1)である。
