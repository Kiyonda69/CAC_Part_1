問1 配点: 6点 / 目安時間: 3分
5人の航空大学校受験者A、B、C、D、Eが、月曜日から金曜日の5日間に、それぞれ異なる1日ずつ体力測定を受ける。下図の条件が与えられているとき、Dが体力測定を受ける曜日として正しいものを選べ。
【条件関係図】矢印は「より早い曜日に受ける」を表す
解説（正解: (5) 金曜日）

【順序制約の整理】

条件①②から B < A < C、条件③から E < C、条件⑤から C < D という順序が得られる。


【Dの曜日を確定】

B < A < C < D の4段階の順序があるため、Dは少なくとも4番目以降。

もし D=木曜日(4日目)なら B・A・C が1〜3日目に収まり、E < C を満たす日がなくなる（矛盾）。

よって D = 金曜日（5日目） が確定。


【残りの配置を確定】

C < D=金 かつ B < A < C の3段階があるため C = 木曜日 が確定。

B・A・Eは月〜水の3日に入り、B < A の条件と④「BとEは連続しない」を照合すると：

・(B=月, A=火, E=水)：|1-3|=2 → 連続しない ✓（唯一有効）

・(B=月, A=水, E=火)：|1-2|=1 → 連続する → 条件④違反

・(B=火, A=水, E=月)：|2-1|=1 → 連続する → 条件④違反


確定した配置：B=月、A=火、E=水、C=木、D=金


したがって、正解は(5) 金曜日である。

問2 配点: 6点 / 目安時間: 5分
3桁の暗証番号（各桁は1〜9の異なる数字）を当てるゲームを行った。各回の試行に対し、数字・位置ともに一致する桁数を「ヒット（H）」、数字は合っているが位置が異なる桁数を「ブロー（B）」として下表の結果が得られた。暗証番号として正しいものを選べ。
ヒット（H）：正しい桁に正しい数字がある個数　／　ブロー（B）：数字は正しいが桁の位置が異なる個数
【試行結果表】
試行 入力した数字 ヒット（H） ブロー（B）
第1回 1 2 3 1 0
第2回 4 5 6 1 0
第3回 7 8 9 0 1
第4回 3 2 7 2 0
解説（正解: (3) 4 2 7）

【第4回（3 2 7、2H 0B）から場合を整理】

{3,2,7}のうち2つが正しい位置にあり、残り1つは暗証番号に含まれない。

・場合(a) 1桁目=3 かつ 2桁目=2（7は含まれない）

・場合(b) 1桁目=3 かつ 3桁目=7（2は含まれない）

・場合(c) 2桁目=2 かつ 3桁目=7（3は含まれない）


【第1回（1 2 3、1H 0B）で絞り込み】

場合(a)：1桁目=3のとき、試行の3桁目=3が番号に含まれブロー → 1H 1Bとなり矛盾。除外。

場合(b)：1桁目=3のとき、試行の3桁目=3がブロー → 1H 1Bとなり矛盾。除外。

場合(c)：2が2桁目で試行の2桁目=2がヒット、3は番号に含まれない。1H 0B ✓


場合(c)のみ残る：2桁目=2、3桁目=7、3は番号に含まれない。


【第2回（4 5 6、1H 0B）で1桁目を決定】

5・6は0ブローより番号に含まれない。1ヒットは4が1桁目に一致する場合のみ。

→ 1桁目=4 が確定。


【第3回（7 8 9、0H 1B）で確認】

7は3桁目（試行では1桁目に入力）→ ブロー ✓　8・9は含まれない。0H 1B ✓


したがって、正解は(3) 4 2 7である。
