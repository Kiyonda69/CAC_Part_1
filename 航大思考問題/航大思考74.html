問1 配点: 6点 / 目安時間: 3分
下の表の「?」に当てはまる数字として正しいものを1つ選べ。
484146
383136
302328
46?44
解説（正解: (1)）

この数表は、各セルの値が「行の値＋列の値」で決まる加法的な構造を持っている。


【規則の発見】

任意の2列間の差は、すべての行で一定である。

・1列目 - 2列目: 48-41 = 38-31 = 30-23 = 46-? = 7

・2列目 - 3列目: 41-46 = 31-36 = 23-28 = ?-44 = -5


【解法】

1列目と2列目の差は常に7であるから:

46 - ? = 7 → ? = 39


検算: 2列目と3列目の差は常に-5であるから:

? - 44 = -5 → ? = 39（一致）


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
以下に1から12の数を表す図形が並んでいる。各図形は円を4等分したもので、各象限の色（白・灰・黒）によって数を表している。47を表す図形として正しいものを1つ選べ。
解説（正解: (5)）

各円は4つの象限（右上・右下・左下・左上）に分かれており、それぞれの色が「白=0」「灰=1」「黒=2」の値を持つ。


【規則の発見】

各象限は3進法の桁に対応し、数値は次のように計算される:

　右上 × 27 ＋ 右下 × 9 ＋ 左下 × 3 ＋ 左上 × 1


【検証（例）】

・1 = 0×27 + 0×9 + 0×3 + 1×1 → 右上:白, 右下:白, 左下:白, 左上:灰

・9 = 0×27 + 1×9 + 0×3 + 0×1 → 右上:白, 右下:灰, 左下:白, 左上:白

・12 = 0×27 + 1×9 + 1×3 + 0×1 → 右上:白, 右下:灰, 左下:灰, 左上:白


【47の計算】

47 = 1×27 + 2×9 + 0×3 + 2×1

　右上: 1 → 灰 / 右下: 2 → 黒 / 左下: 0 → 白 / 左上: 2 → 黒


【選択肢の検討】

・(1) 灰,灰,黒,黒 → 27+9+6+2 = 44

・(2) 灰,黒,白,灰 → 27+18+0+1 = 46

・(3) 灰,黒,灰,白 → 27+18+3+0 = 48

・(4) 灰,黒,灰,黒 → 27+18+3+2 = 50

・(5) 灰,黒,白,黒 → 27+18+0+2 = 47 → 正解


したがって、正解は(5)である。
