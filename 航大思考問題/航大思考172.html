問1 配点: 6点 / 目安時間: 3分
下図は、1辺1cmの立方体ブロックを組み合わせて作った立体を、平面図・正面図・側面図（右側面）の3方向から見た図である。この立体として正しいものを、次の選択肢から1つ選べ。
【図1】3面図
解説（正解: (3)）

3面図から立体を特定する。


【3面図の読解】

・平面図: 外形は2×2の正方形、中央の水平線は前後で高さが異なることを示す。

・正面図: 2×2の完全な正方形（前から見て上下左右すべて埋まる）。

・側面図: 右上が欠けたL字（右から見て上奥が空）。


【立体の推定】

・側面図の「右上欠け」＝奥列の上段が空。正面図が完全2×2なので、前列の上段は残っている。

・よって、欠けているのは奥列の上段2個（左奥・右奥の上段）。

・2×2×2の立方体から、奥の上端の1辺（2個）を取り除いた形。


【選択肢の検討】

・(1) 左列の上段2個欠け → 正面図が「左上欠けL」 → 不正解

・(2) 前列の上段2個欠け → 側面図が「左上欠けL」 → 不正解

・(3) 奥列の上段2個欠け → 全条件と一致 → 正解

・(4) 右列の上段2個欠け → 正面図が「右上欠けL」 → 不正解

・(5) 右奥上の1個のみ欠け → 側面・正面とも完全2×2 → 不正解


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下図は、1辺1cmの立方体ブロックを組み合わせて作った別の立体を、平面図・正面図・側面図（右側面）の3方向から見た図である。この立体として正しいものを、次の選択肢から1つ選べ。
【図2】3面図
解説（正解: (4)）

3個のブロックが取り除かれた非対称な立体を、3面図から特定する。


【3面図の読解】

・平面図: 外形は2×2、内部のL字線は奥左の1区画のみ高さ2、他3区画は高さ1であることを示す。

・正面図: 右上が欠けたL字（前から見て上段の右側が空）。

・側面図: 左上が欠けたL字（右から見て上段の前側が空）。


【立体の推定】

・奥左の上段ブロック(0,1,1)のみ存在し、他の上段ブロックは全て欠如すべきか？を検討。

・正面図「右上欠け」＝右列の上段が手前・奥とも空＝(1,0,1)・(1,1,1)欠如。

・側面図「左上欠け」＝前列の上段が左右とも空＝(0,0,1)・(1,0,1)欠如。

・両方の条件を統合すると、欠けるブロックは(0,0,1)、(1,0,1)、(1,1,1)の3個。

・残るブロックは底面4個 + 奥左上段(0,1,1)の計5個。


【選択肢の検討】

・(1) 奥左の上段が欠け、奥右上を残す配置 → 平面図の高い区画位置が一致しない → 不正解

・(2) 左列の上段2個と右奥上が欠ける → 正面図が「左上欠けL」 → 不正解

・(3) 右列の上段2個のみ欠ける（2個欠け） → 平面図の内部線が縦中央線になる → 不正解

・(4) 前列の上段2個と右奥上が欠ける → 全条件と一致 → 正解

・(5) 前列の上段2個と左奥上が欠ける → 平面図で右奥のみ高くなる → 不正解


したがって、正解は(4)である。
