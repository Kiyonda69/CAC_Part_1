問1 配点: 6点 / 目安時間: 3分
下図のように、4本の棒の高さを低・中・高の3段階で表すことによって整数を表す規則がある。1から12までの整数とそれに対応する図形が示されている。この規則に従うとき、整数65を表す図形として正しいものを1つ選べ。
【整数と図形の対応表】
解説（正解: (3)）

【規則の発見】

4本の棒はそれぞれ「低」「中」「高」の3段階の高さをとる。これは3進法（基数3）の各桁に対応している。


棒の高さを数値に置き換える:

・低 = 0　・中 = 1　・高 = 2


左から順に、27の位、9の位、3の位、1の位を表す:

・整数 n = (左の棒) x 27 + (左中の棒) x 9 + (右中の棒) x 3 + (右の棒) x 1 + 1


【整数65の計算】

65 - 1 = 64 を3進法で表す:

・64 = 2x27 + 1x9 + 0x3 + 1

・各桁: 2, 1, 0, 1 → 高, 中, 低, 中


【選択肢の検討】

・(1) 高中低低 → 2x27+1x9+0+0+1 = 64

・(2) 中高低中 → 1x27+2x9+0+1+1 = 47

・(3) 高中低中 → 2x27+1x9+0+1+1 = 65 → 正解

・(4) 高低中中 → 2x27+0+1x3+1+1 = 59

・(5) 高中中低 → 2x27+1x9+1x3+0+1 = 67


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
A、B、C、D、Eの5人が、5階建てマンションの1階から5階にそれぞれ1人ずつ住んでいる。次の条件がすべて成り立つとき、Dが住んでいる階として正しいものを1つ選べ。
【条件】
I．BはAより上の階に住んでいる。
II．DはCのちょうど2階上に住んでいる。
III．Eは偶数階に住んでいる。
IV．Aは1階に住んでいない。
V．BとEは隣の階に住んでいない。
【マンション】
解説（正解: (3)）

【条件の整理】

条件II「DはCのちょうど2階上」より、(C, D)の組み合わせは:

・C=1階, D=3階　・C=2階, D=4階　・C=3階, D=5階


【場合分け】


場合1: C=1階, D=3階

残りA, B, Eは2階, 4階, 5階。条件III「E偶数階」→ E=2階 or E=4階。

(a) E=2階: A, B={4,5}。条件I「B>A」→ A=4, B=5。条件V: |5-2|=3 → 適合。条件IV: A=4 → 適合。有効

(b) E=4階: A, B={2,5}。条件I → A=2, B=5。条件V: |5-4|=1 → 不適合


場合2: C=2階, D=4階

残り={1,3,5}。条件III「E偶数階」→ 該当なし → 不適合


場合3: C=3階, D=5階

残り={1,2,4}。条件III → E=2 or E=4。条件IV → A=2 or A=4。

(a) E=2: A,B={1,4}。条件IV → A=4, B=1。条件I: 1>4 → 不適合

(b) E=4: A,B={1,2}。条件IV → A=2, B=1。条件I: 1>2 → 不適合


【結論】

有効な配置: A=4階, B=5階, C=1階, D=3階, E=2階


したがって、正解は(3) 3階である。
