問1 配点: 6点 / 目安時間: 3分
下表は、ある小売店の商品A〜Eについて、10月から12月までの3ヶ月間の売上（単位：千円）を示している。下記の3つの条件をすべて満たす商品はどれか。次のうちから選べ。
【表1】商品別 月別売上（単位：千円）
商品10月11月12月
A200180160
B110140130
C130150170
D145135195
E160170165
【条件】
条件1: 3ヶ月の合計売上が400千円以上である。
条件2: 12月の売上が11月の売上より増加している。
条件3: 3ヶ月の中で最高売上月と最低売上月の差が50千円以下である。
問1（正解: (3)）の解説

各商品について3条件を順に検証する。


【各商品の検証】

・商品A: 合計=200+180+160=540（≧400 ✓）、12月160<11月180 → 条件2 ✗

・商品B: 合計=110+140+130=380（<400）→ 条件1 ✗

・商品C: 合計=130+150+170=450（≧400 ✓）、12月170>11月150 ✓、最高170−最低130=40（≦50 ✓）→ 全条件 ✓

・商品D: 合計=145+135+195=475（≧400 ✓）、12月195>11月135 ✓、最高195−最低135=60（>50）→ 条件3 ✗

・商品E: 合計=160+170+165=495（≧400 ✓）、12月165<11月170 → 条件2 ✗


3条件すべてを満たすのは商品Cのみ。したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下表は、ある企業の5部署について、5つの評価指標（売上、顧客満足度、業務効率、革新性、コスト管理）の評価点（100点満点）を示したものである。下記の4つの条件をすべて満たす部署はどれか。次のうちから選べ。
【表2】部署別 評価指標スコア
部署売上顧客
満足度		業務
効率		革新性コスト
管理
営業7080759570
開発9075857060
企画8085708075
総務6590656095
人事8570806580
【条件】
条件1: 5指標の平均点が75点以上である。
条件2: 5指標の最高点と最低点の差が25点以上である。
条件3: 4つ以上の指標で70点以上を獲得している。
条件4: 革新性が80点以上、または、コスト管理が80点以上である。
問2（正解: (1)）の解説

各部署について4条件を順に検証する。


【各部署の検証】

・営業: 平均=(70+80+75+95+70)/5=78（≧75 ✓）、最高95−最低70=25（≧25 ✓）、70以上=5つ（≧4つ ✓）、革新性95≧80 ✓ → 全条件 ✓

・開発: 平均=(90+75+85+70+60)/5=76（≧75 ✓）、差=90−60=30（≧25 ✓）、70以上=4つ（≧4つ ✓）、革新性70<80 かつ コスト60<80 → 条件4 ✗

・企画: 平均=(80+85+70+80+75)/5=78、差=85−70=15（<25）→ 条件2 ✗

・総務: 平均=(65+90+65+60+95)/5=75、差=95−60=35、70以上=2つ（満足度・コスト）（<4つ）→ 条件3 ✗

・人事: 平均=(85+70+80+65+80)/5=76、差=85−65=20（<25）→ 条件2 ✗


4条件すべてを満たすのは営業のみ。したがって、正解は(1)である。
