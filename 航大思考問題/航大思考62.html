問1 配点: 6点 / 目安時間: 3分
下図は立方体の展開図であり、各面にア〜カの記号と模様が描かれている。この展開図を組み立てたとき、丸（ア）が描かれた面と向かい合う面の模様として正しいものを、(1)〜(5)のうちから一つ選べ。
【展開図】
解説（正解: (2)）

【展開図の分析】

十字型展開図の各面の配置を確認する。中央の面「ア（丸）」を基準に展開図を組み立てる。


【対面関係の特定】

十字型展開図では、中央の面から2つ離れた面が向かい合う。

・ア（丸） と カ（菱形）：中央とその2つ下 → 対面

・イ（三角） と エ（星）：左右の面 → 対面

・ウ（四角） と オ（十字）：上下の面 → 対面


丸（ア）の向かい側は菱形（カ）である。


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下の(1)〜(5)は、いずれも6つの正方形を辺でつなげた図形である。このうち、折りたたんで立方体を組み立てることができないものを一つ選べ。
解説（正解: (3)）

【展開図の判別方法】

立方体の展開図には11種類の形がある。6つの正方形が辺でつながっていても、折りたたんだときに面が重なるものは展開図にならない。


【各選択肢の検討】

・(1) 十字型：最も基本的な展開図。中央を底面にすると上下左右と対面の6面が正しく構成される。 → 組み立て可能

・(2) S字型：段階的にずれた形。折りたたむと各面が異なる方向を向く。 → 組み立て可能

・(3) 2行3列の長方形：折りたたむと面が重なる。上段左と下段左を折り上げると同じ方向を向いてしまう。 → 組み立て不可能

・(4) 階段型：1マスずつずれながら伸びる形。正しく立方体を構成できる。 → 組み立て可能

・(5) S字型の変形：折りたたむと正しく6面全てが異なる方向を向く。 → 組み立て可能


【2行3列が不可能な理由】

2行3列の長方形では、一方の行の両端を折り上げると、もう一方の行の対応する面と重なってしまう。4つの面が同一の軸周りに折れるため、立方体の6方向を全てカバーできない。


したがって、正解は(3)である。
