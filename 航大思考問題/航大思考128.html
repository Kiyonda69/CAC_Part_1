問1 配点: 6点 / 目安時間: 3分
ある立体を真上から見ると図Aのようになり、アの方向から見ると図Bのようになった。この立体をイの方向から見た図として正しいものを1つ選べ。
図A（真上から）
図B（アの方向から）
解説（正解: (4)）

与えられた立体は2つのブロックからなる。

・ブロックA（右前・高い）：右半分×前半分×高さ3

・ブロックB（左全域・低い）：左半分×全奥行×高さ1


【上面図（図A）の確認】

真上から見ると、両ブロックの足跡がL字形を形成する。前半は全幅が存在し、後半は左半分のみ。


【正面図（図B）の確認】

ア方向から見ると左半分は高さ1（ブロックB）、右半分は高さ3（ブロックA）の階段形状。


【イ方向（右側面）の決定】

右側から見たとき、奥行き位置ごとの最大高さは：

・前部（手前半分）：ブロックA（高さ3）とブロックB（高さ1）の両方が存在 → 最大高さ＝3

・後部（奥半分）：ブロックBのみ（高さ1）→ 高さ＝1

よって手前が高く奥が低い階段形状となる。


・(1)(2) は高さが均一の長方形 → 誤　・(3) は前後逆 → 誤　・(5) は比率が違う → 誤


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
図のように、直方体ABCD-EFGHがある。AB=4、BC=3、AE=3とする。辺DHの中点をP、辺BFの中点をQとするとき、3点A、P、Qを含む平面でこの直方体を切断した。切り口の形として正しいものを1つ選べ。
解説（正解: (3)）

直方体の各頂点を座標で表す（AB=4をx軸、BC=3をy軸、AE=3をz軸）。

A(0,0,0)、B(4,0,0)、C(4,3,0)、D(0,3,0)

E(0,0,3)、F(4,0,3)、G(4,3,3)、H(0,3,3)

P（辺DH中点）＝ (0, 3, 1.5)、Q（辺BF中点）＝ (4, 0, 1.5)


【切断面と各辺との交点を求める】

3点A、P、Qを含む平面の法線ベクトル：

AP × AQ = (0,3,1.5) × (4,0,1.5) = (4.5, 6, −12)

平面の方程式：4.5x + 6y − 12z = 0

頂点G(4,3,3)を代入：4.5×4 + 6×3 − 12×3 = 18 + 18 − 36 = 0 ✓

よって、頂点Gもこの平面上にある。


切り口は四角形 A → Q → G → P の順に結んだ四角形AQGP。


【断面形状の特定】

・AQ = (4, 0, 1.5)　PG = G − P = (4, 0, 1.5)　→ AQ ∥ PG（平行かつ等長）

・AP = (0, 3, 1.5)　QG = G − Q = (0, 3, 1.5)　→ AP ∥ QG（平行かつ等長）

2組の対辺が平行かつ等長 → 平行四辺形


AQ · AP = 4×0 + 0×3 + 1.5×1.5 = 2.25 ≠ 0 なので長方形ではない。


したがって、正解は(3)である。
