問1 配点: 6点 / 目安時間: 3分
下図は、6つの面に異なる図形（●・○・■・□・▲・△）が描かれた立方体の展開図である。この展開図を組み立てて作られる立方体として正しいものを、(1)～(5)の中から1つ選べ。
【図】立方体の展開図
解説（正解: (5)）

展開図を組み立てると、向かい合う面のペアは ● ⇔ △、■ ⇔ ▲、○ ⇔ □ となる。

●を上、■を正面（手前）に向けたとき、左面=○、右面=□、後面=▲、下面=△。

上面が●のままで上下軸まわりに回転させると、(正面, 右面) は次のいずれかでなければならない。

　(■, □) ／ (□, ▲) ／ (▲, ○) ／ (○, ■)


【選択肢の検討】

・(1) 上=●, 正面=■, 右=○ → 正面=■のとき右面は□。○は左面 ✗

・(2) 上=●, 正面=□, 右=■ → 正面=□のとき右面は▲。■は左面 ✗

・(3) 上=●, 正面=■, 右=▲ → ■と▲は向かい合う面で同時に見えない ✗

・(4) 上=●, 正面=△, 右=□ → ●と△は向かい合う面で同時に見えない ✗

・(5) 上=●, 正面=■, 右=□ → 標準向きで矛盾なし ★


したがって、正解は(5)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ展開図の立方体について、初め●を上面、■を正面（手前）、□を右側面に向けて平らな机に置く。この立方体を「前へ転がす → 右へ転がす → 前へ転がす」の順に転がしたとき、最後に上面に来る図形はどれか。
【図】立方体の展開図
解説（正解: (4)）

展開図から、向かい合う面のペアは ● ⇔ △、○ ⇔ □、■ ⇔ ▲。

初期状態は次の通り。

　上=●、下=△、前=■、後=▲、左=○、右=□


【転がしのルール】

・前へ転がす：上→前、前→下、下→後、後→上（左右は不変）

・右へ転がす：上→右、右→下、下→左、左→上（前後は不変）


【手順】

① 前へ転がす → 上=▲、前=●、下=■、後=△、左=○、右=□

② 右へ転がす → 上=○、前=●、下=□、後=△、左=■、右=▲

③ 前へ転がす → 上=△、前=○、下=●、後=□、左=■、右=▲


よって、最後に上面に来る図形は△である。


【選択肢の検討】

・(1) ● → 最終的に下面

・(2) ○ → 最終的に前面

・(3) ■ → 最終的に左面

・(4) △ → 上面 ★

・(5) ▲ → 最終的に右面


したがって、正解は(4)である。
