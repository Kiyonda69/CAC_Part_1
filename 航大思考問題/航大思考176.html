問1 配点: 6点 / 目安時間: 3分
下図は、ある空港の貨物ターミナルで3×3に区切られたエリアに積まれた航空貨物用コンテナ（ULD）の段数を真上から見た平面図である。各マスの数字はそのマスに積まれているコンテナの段数を表す。観察者が南側（図の下方向）からこのコンテナ群を正面に見たときに見えるシルエット（各列の最も高い段数で形作られる輪郭）として正しいものを、(1)〜(5)のうちから1つ選べ。
【図1】貨物コンテナの平面図（上から見た図）
解説（正解: (3)）

真上から見た平面図の各マスの数字は、そのマスに積まれているコンテナの段数を表す。観察者が南側（図の下方向）からコンテナ群を見ると、列ごとに最も高い段数のコンテナのシルエットが見える。手前のコンテナが奥のコンテナを隠すため、各列で最大段数のみがシルエットとして残る。


【列ごとの最大段数を求める】

・左列（列1）：上から 2、1、2 → 最大値は 2

・中央列（列2）：上から 1、2、3 → 最大値は 3

・右列（列3）：上から 3、1、2 → 最大値は 3


よって、南側から見える正面シルエットは左から「2、3、3」となる。


【選択肢の検討】

・(1) 2、3、2 → 右列が一段不足（実際は最大3段）。不正解。

・(2) 3、3、3 → 左列が一段超過（実際は最大2段）。不正解。

・(3) 2、3、3 → すべての列の最大段数と一致。正解。

・(4) 1、2、3 → 左・中央列がいずれも不足。不正解。

・(5) 3、2、3 → 左列が一段超過、中央列が一段不足。不正解。


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
問1と同様に、ある空港の貨物ターミナルで3×3に区切られたエリアにコンテナが積まれている。今度は平面図（上面図）が分からない。観察者が「南側から見た正面シルエット」と「東側から見た右側面シルエット」を観測したところ、下図のようになった。このとき、コンテナの配置を真上から見た平面図として正しいものを、(1)〜(5)のうちから1つ選べ。なお、各マスには0〜3段のコンテナが積まれているものとする。
【図1】観測された2方向のシルエット
選択肢（上から見た平面図、数字は段数。上方向が北、下方向が南）
解説（正解: (4)）

正面シルエット（南→北方向に見た図）は、各列の最大段数を示す。右側面シルエット（東→西方向に見た図）は、各行の最大段数を示す。観測されたシルエットから次の条件が導かれる。


【正面シルエットから（列の最大段数）】

・列1（左）の最大段数 = 3

・列2（中）の最大段数 = 3

・列3（右）の最大段数 = 2 → 列3のすべてのマスは2段以下


【側面シルエットから（行の最大段数）】

・行1（北）の最大段数 = 2 → 行1のすべてのマスは2段以下

・行2（中）の最大段数 = 3

・行3（南）の最大段数 = 3


【選択肢の検討】

・(1) 列の最大は 3, 3, 3 → 列3が要求値2を超えるため正面と矛盾。不正解。

・(2) 列の最大は 3, 3, 2（一致）。しかし行の最大は 2, 3, 2 で行3が要求値3に届かず側面と矛盾。不正解。

・(3) 列の最大は 3, 2, 3 で列2・列3が要求値と異なる。正面と矛盾。不正解。

・(4) 列の最大は 3, 3, 2（一致）、行の最大は 2, 3, 3（一致）。正解。

・(5) 列の最大は 3, 3, 2（一致）。しかし行の最大は 3, 2, 3 で側面と矛盾。不正解。


したがって、正解は(4)である。
