問1 配点: 6点 / 目安時間: 3分
5人の訓練生A、B、C、D、Eが、1番目から5番目の順にフライト訓練を受ける。以下の(ア)～(オ)の条件をすべて満たすとき、3番目に訓練を受ける人物として正しいものを１つ選び、解答用紙にマークせよ。(6点)
【図1】訓練順序
(ア) AとEは隣り合って並ぶ
(イ) CはDのすぐ後に訓練を受ける
(ウ) Bは5番目に訓練を受ける
(エ) AはCより前に訓練を受ける
(オ) Eは1番目ではない
解説（正解: (5)）
問1（正解: (5)）
条件を整理して論理的に順序を決定する。

【条件の整理】
・(ウ)より: Bは5番目（確定）
・(イ)より: DとCは隣接し、D→Cの順
・(オ)より: Eは1番目ではない
・(ア)より: AとEは隣接

【AとEの位置の決定】
・AとEは隣接し、Eは1番目ではない
・Bが5番目なので、A-Eのペアは1-2、2-3、3-4のいずれか
・(エ)より AはCより前。(イ)よりDはCのすぐ前
・D-Cペアが存在し、AはCより前なので、AはDより前になる

【配置の確定】
・Eが1番目ではないので、A-Eのペアは「A→E」の順で1-2番目
・D-Cのペアが3-4番目（AがCより前という条件を満たす）
・Bが5番目

【最終順序】
1番目: A → 2番目: E → 3番目: D → 4番目: C → 5番目: B

したがって、3番目に訓練を受けるのはDである。
正解: (5) D
問2 配点: 6点 / 目安時間: 5分
航空貨物の積み込みにおいて、6つの荷物P、Q、R、S、T、Uを1番目から6番目の順に積み込む。以下の(ア)～(キ)の条件をすべて満たすとき、4番目に積み込まれる荷物として正しいものを１つ選び、解答用紙にマークせよ。(6点)
【図2】積み込み順序
(ア) PはQより前に積み込む
(イ) RとSの間には、ちょうど2つの荷物がある
(ウ) TはUの直後に積み込む
(エ) Qは6番目に積み込む
(オ) SはPより前に積み込む
(カ) Uは1番目ではない
(キ) RはTより後に積み込む
解説（正解: (3)）
問2（正解: (3)）
条件を整理して論理的に順序を決定する。

【確定条件の整理】
・(エ)より: Q = 6番目（確定）
・(ア)より: P < Q なので、P は 1〜5番目のいずれか
・(オ)より: S < P なので、S, P の順序が確定
・(ウ)より: T = U + 1（TはUの直後で隣接）
・(カ)より: U は 1番目ではない

【U-Tペアの位置決定】
・Uは1番目ではないので、U-Tペアは (2,3)、(3,4)、(4,5) のいずれか
・(キ)より: R > T
・(イ)より: |R - S| = 3（間に2つの荷物）

【場合分けによる検証】
・Q = 6番目が確定
・S < P < 6 かつ R > T を満たす配置を探す
・|R - S| = 3 より、(S, R) = (1,4) または (4,1) または (2,5) または (5,2) など
・S < P かつ R > T を考慮すると、S = 1番目、R = 4番目

【配置の確定】
・S = 1番目、R = 4番目（間に2つの荷物: 2,3番目）
・U-T ペアは R より前で、U は 1番目ではない → (U,T) = (2,3)
・P = 5番目（S < P < Q を満たす残りの位置）
・Q = 6番目

【最終順序】
1: S → 2: U → 3: T → 4: R → 5: P → 6: Q

したがって、4番目に積み込まれるのはRである。
正解: (3) R
