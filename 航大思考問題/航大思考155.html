問1 配点: 6点 / 目安時間: 3分

立方体の展開図を、以下の文章の指示にしたがって作る。なお、この問題には図が示されていないので、解答者は文章をもとに自分で展開図を描いて考えること。

合同な6枚の正方形を平面上に次のように配置する。

  • 中央に正方形Aを置く。
  • Aの上辺に接して上方向に正方形Bを置く。
  • Bの上辺に接して上方向に正方形Cを置く。
  • Aの下辺に接して下方向に正方形Dを置く。
  • Aの右辺に接して右方向に正方形Eを置く。
  • Aの左辺に接して左方向に正方形Fを置く。

これら6枚を辺で折りたたんで立方体を組み立てるとき、面Bと向かい合う面（対面）はどれか。

解説（正解: (1)）

文章にしたがって展開図を描くと、次のような十字型（中央列が4枚、両側に1枚ずつ）になる。

[C] [B] [F] [A] [E] [D]

【組み立て手順】

正方形Aを底面として机に置き、隣接する正方形を順に折り上げる。

・BはAの上辺で90°折り上がり、奥側の側面（北の壁）になる。

・CはBの上辺でさらに折られ、立方体の上面（天井）になる。

・DはAの下辺で折り上がり、手前側の側面（南の壁）になる。

・EはAの右辺で折り上がり、右の側面（東の壁）になる。

・FはAの左辺で折り上がり、左の側面（西の壁）になる。


【対面のペア】

・A（底面）⇔ C（上面）

・B（北の壁）⇔ D（南の壁）

・E（東の壁）⇔ F（西の壁）


したがって、面Bと向かい合う面は(1) Dである。

問2 配点: 6点 / 目安時間: 5分

立方体の展開図を、以下の文章の指示にしたがって作る。なお、この問題には図が示されていないので、解答者は文章をもとに自分で展開図を描いて考えること。

合同な6枚の正方形を平面上に次のように配置する。

  • 中央の行に左から順に、正方形A、B、C、Dを横一列に並べる（4枚が辺で連結）。
  • Bの上辺に接して上方向に正方形Eを置く。
  • Cの下辺に接して下方向に正方形Fを置く。

この6枚を組み立てて立方体としたとき、各面には1から6までの異なる整数が1つずつ書かれており、向かい合う2面の数字の和はいずれも等しいことが分かった。

A=1、B=2、E=3 であるとき、面C、面D、面Fの数字をこの順に並べたものとして正しいものはどれか。

解説（正解: (5)）

文章にしたがって展開図を描くと、横一列4枚に上下から1枚ずつ食い違いに付いたジグザグ型になる。

[E] [A] [B] [C] [D] [F]

【対面ペアの特定】

正方形Bを底面として机に置き、隣接する正方形を順に折り上げて考える。

・AはBの左隣 → 西の側面となる。

・CはBの右隣 → 東の側面となる。

・DはCの右隣（Cが東の壁になった後、Cの上辺に当たる）→ 上面（天井）となる。

・EはBの上辺隣 → 北の側面となる。

・FはCの下辺隣（Cが東の壁になると、Cの下辺は南向きの縁）→ 南の側面となる。


以上から、対面ペアは次の通り。

・A（西）⇔ C（東）

・B（底面）⇔ D（上面）

・E（北）⇔ F（南）


【数字の決定】

1〜6の総和は 1+2+3+4+5+6 = 21。対面3組の和がいずれも等しいので、各組の和は 21÷3 = 7（標準サイコロの規則）。

・A + C = 7 → 1 + C = 7 → C = 6

・B + D = 7 → 2 + D = 7 → D = 5

・E + F = 7 → 3 + F = 7 → F = 4

{C, D, F} = {6, 5, 4} は1〜6のうち未使用の数字{4,5,6}とちょうど一致する。


【選択肢の検討】

・(1) (5, 6, 4) → A↔D・B↔Cが対面と誤認した結果。

・(2) (4, 5, 6) → 順序の入れ替え（FをCに、CをDに、DをFに割り当てた）。

・(3) (5, 4, 6) → A↔Bや隣接面を対面と誤認した結果。

・(4) (6, 4, 5) → DとFを取り違えた結果。

・(5) (6, 5, 4) → 正しい対面ペアから導かれる正解。


したがって、正解は(5)である。
