問1 配点: 6点 / 目安時間: 3分
下図のように、4×4の格子パターン（黒マスと白マス）に対して、次の2つの操作を順に行う。操作後のパターンとして正しいものを(1)～(5)の中から1つ選べ。
【操作の流れ】
解説（正解: (3)）

【解法】

2つの操作を順に追跡する。


操作1: 90°右回転（時計回り）

元パターンの各列を下から読み取り、回転後の各行に対応させる。


操作2: 左右反転

中間結果の各行の左右を入れ替える。


【合成操作の性質】

90°右回転＋左右反転 = 主対角線に関する転置（行と列の入れ替え）と同じ結果になる。つまり、元パターンの(i行, j列)のマスが、結果の(j行, i列)に移る。


【検証】

元(0,0)=黒 → 結果(0,0)=黒、元(0,3)=黒 → 結果(3,0)=黒、元(1,0)=黒 → 結果(0,1)=黒

選択肢(3)がすべての条件を満たす。


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
下の表は、3×3のマス目の各位置に積み上げた単位立方体の個数を示している。この立体を右側面（右列側）から見たとき、正しい側面図を(1)～(5)の中から1つ選べ。
【各位置の積み上げ個数】
左列 中列 右列
奥 1 0 1
中 0 3 2
手前 2 1 0
※ 数値はその位置に積まれた単位立方体の個数（0は立方体なし）
※ 正面は「手前」の側、右側面は「右列」の側から見た図
【参考: 正面図】
解説（正解: (2)）

【右側面図の求め方】

右側面（右列側）から立体を見ると、各行（手前・中・奥）の断面が見える。各行について、左列・中列・右列の中で最も高い積み上げ数が、その行の見える高さになる。


【各行の計算】

手前行: max(2, 1, 0) = 2

中行: max(0, 3, 2) = 3

奥行: max(1, 0, 1) = 1


右側面図では、左側が手前、右側が奥になる。

よって、左から順に高さ 2, 3, 1 の図になる。


【選択肢の検討】

(1) [2,3,2]: 奥行の高さが2 → 実際はmax=1なので誤り

(2) [2,3,1]: 全行の最大値と一致 → 正解

(3) [3,2,1]: 手前の高さが3 → 実際はmax=2なので誤り

(4) [1,3,2]: 手前と奥が逆 → 誤り

(5) [2,2,1]: 中行の高さが2 → 実際はmax=3なので誤り


したがって、正解は(2)である。
