問1 配点: 6点 / 目安時間: 3分
次の条件で定まる立方体について、指定された3点を通る平面で切断したときの断面の形として正しいものを次の(1)〜(5)の中から1つ選べ。
【立方体の構成】
  • 立方体 ABCD-EFGH において、ABCD は上面の正方形、EFGH は下面の正方形である。
  • 頂点を上から見たとき、A・B・C・D は反時計回りに並んでおり、A の真下が E、B の真下が F、C の真下が G、D の真下が H である。
【切断する3点】
  • 頂点 A
  • 頂点 C
  • 辺 FG の中点 M
上記の3点 A・C・M を通る1つの平面で立方体を切断したときに現れる切り口（断面）の形として最も適切なものを選べ。
解説（正解: (1) 等脚台形）

立方体の一辺を 1 とし、A(0,0,1)、B(1,0,1)、C(1,1,1)、D(0,1,1)、E(0,0,0)、F(1,0,0)、G(1,1,0)、H(0,1,0) と座標を設定する。辺 FG の中点は M(1, 0.5, 0)。


【切断面の特定】

3点 A、C、M を通る平面と立方体の各面の交わりを調べると:

・上面 ABCD: 対角線 AC が断面の一辺

・側面 BCGF: 頂点 C と点 M を結ぶ線分 CM

・底面 EFGH: 点 M と 辺 EF 上の点 N(0.5, 0, 0) を結ぶ線分 MN

・側面 ABFE: 点 N と 頂点 A を結ぶ線分 NA

よって断面は 4頂点 A・C・M・N からなる四角形である。


【辺の長さ（一辺=1のとき）】

・AC = √2 ≈ 1.414

・CM = √(0.25 + 1) = √1.25 ≈ 1.118

・MN = √(0.25 + 0.25) = √0.5 ≈ 0.707

・NA = √(0.25 + 1) = √1.25 ≈ 1.118


【形状の判定】

・MN の方向 (-0.5, -0.5, 0) と AC の方向 (1, 1, 0) は平行 ⇒ 1組の対辺 AC ∥ MN を持つ 台形。

・残る2辺 CM = NA = √1.25 で長さが等しい ⇒ 等脚台形。

・AC ≠ MN なので長方形・平行四辺形ではない。


したがって、正解は(1) 等脚台形である。

問2 配点: 6点 / 目安時間: 5分
次の条件で定まる立方体について、指定された3点を通る平面で切断したときの断面の形として正しいものを次の(1)〜(5)の中から1つ選べ。
【立方体の構成】
  • 立方体 ABCD-EFGH において、ABCD は上面の正方形、EFGH は下面の正方形である。
  • 頂点を上から見たとき、A・B・C・D は反時計回りに並んでおり、A の真下が E、B の真下が F、C の真下が G、D の真下が H である。
【切断する3点】
  • 辺 AB の中点 P
  • 辺 CG の中点 Q
  • 辺 HE の中点 R
上記の3点 P・Q・R を通る1つの平面で立方体を切断したときに現れる切り口（断面）の形として最も適切なものを選べ。
解説（正解: (2) 正六角形）

立方体の一辺を 1 とし、A(0,0,1)、B(1,0,1)、C(1,1,1)、D(0,1,1)、E(0,0,0)、F(1,0,0)、G(1,1,0)、H(0,1,0) と座標を設定する。

・P = AB の中点 = (0.5, 0, 1)　・Q = CG の中点 = (1, 1, 0.5)　・R = HE の中点 = (0, 0.5, 0)


【切断面の方程式】

3点 P・Q・R を通る平面の法線ベクトルを計算すると n = (-1, 1, 1) となり、整理すると −x + y + z = 0.5。

この平面は立方体の中心 (0.5, 0.5, 0.5) を通る（−0.5+0.5+0.5=0.5）ことに注目する。


【各辺との交点】

立方体の12本の辺について平面と交わる点をすべて求めると、次の 6つの中点 が得られる:

・辺 AB の中点 P(0.5, 0, 1)　・辺 BC の中点 (1, 0.5, 1)

・辺 CG の中点 Q(1, 1, 0.5)　・辺 GH の中点 (0.5, 1, 0)

・辺 HE の中点 R(0, 0.5, 0)　・辺 AE の中点 (0, 0, 0.5)

これら6点を順に結ぶと六角形が得られる。


【正六角形であることの確認】

・隣り合う2点間の距離はいずれも √((0.5)²+(0.5)²) = √0.5 ≈ 0.707 で等しい。

・各内角を計算すると、すべて 120° となる。

6辺すべてが等しく、6つの内角もすべて 120° なので、断面は 正六角形である。


【補足】

立方体の中心を通り、互いに平行でない3つの辺の中点を通る平面は、立方体の対称性により必ず正六角形を切り出す。これは空間幾何の有名な結果である。


したがって、正解は(2) 正六角形である。
