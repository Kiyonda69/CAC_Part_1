問1 配点: 6点 / 目安時間: 3分
図のように、黒と白の6つのマスからなる行が、ある規則にしたがって上から下のように変化している。図と同じ規則にしたがって変化しているものは次のうちどれか。正しいものを(1)～(5)から1つ選べ。ただし、選択肢では矢印を省略している。
解説（正解: (3)）

【規則の発見】

この問題の規則は「各マスの次の行の値は、そのマスと右隣のマスの排他的論理和（XOR）で決まる」というものである。6マス目の右隣は1マス目に循環する。


具体的には、現在の行を a[1], a[2], ..., a[6] とすると、次の行は:

・b[i] = a[i] XOR a[i+1]（i = 1, 2, ..., 5）

・b[6] = a[6] XOR a[1]（循環）

ここで、同じ色なら白、異なる色なら黒となる。


【選択肢の検討】

・(1) 行2→行3: 3番目が不一致 → 不正解

・(2) 行2→行3: 6番目が不一致 → 不正解

・(3) 全ての行間でXOR規則が成立 → 正解

・(4) 行1→行2: 6番目が不一致 → 不正解

・(5) 行2→行3: 1番目が不一致 → 不正解


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
1辺の長さが1の立方体ABCD-EFGHがある（ABCDが上面、EFGHが下面で、A-E、B-F、C-G、D-Hがそれぞれ対応する頂点）。8つの頂点のうち3つを選んで三角形を作るとき、正三角形となるものは全部で何個あるか。正しいものを(1)～(5)から1つ選べ。
解説（正解: (2)）

【解法】

立方体の8頂点間の距離は3種類ある:

・辺の長さ: 1（隣接する2頂点）

・面の対角線: sqrt(2)（同じ面上の対角頂点）

・空間対角線: sqrt(3)（立方体の対角頂点）


正三角形を作るには、3辺が全て等しくなければならない。


【辺の長さ1の正三角形】

立方体のグラフは二部グラフであり、奇数サイクルを含まないため、辺のみで三角形を作ることは不可能。


【空間対角線sqrt(3)の正三角形】

3頂点が互いに全てsqrt(3)離れることはない（座標計算から不可能）。


【面の対角線sqrt(2)の正三角形】

立方体には2つの正四面体が内接する:

・正四面体1: A, C, F, H（4頂点が互いにsqrt(2)で結ばれる）

・正四面体2: B, D, E, G（4頂点が互いにsqrt(2)で結ばれる）

各正四面体から3頂点を選ぶ方法は C(4,3) = 4通り。

合計 4 + 4 = 8個の正三角形が存在する。


したがって、正解は(2)の8個である。
