問1 配点: 6点 / 目安時間: 3分
下図は、立方体（正六面体）の展開図であり、六つの面にそれぞれアルファベット A・B・C・D・E・F が記されている。この展開図を組み立てて立方体を作るとき、面 F と向かい合う（平行に対面する）面はどれか。正しいものを1つ選べ。
【図1】立方体の展開図
解説（正解: (2)）

【展開図の組み立て】

この展開図は「1-4-1 型」に近いが、下端の E にさらに F が連結した縦長型である。中央の面 C を前面（基面）として折り畳むと、各面は次のように配置される。

・C の上の A を折り込む → 上面 = A

・C の左の B を折り込む → 左面 = B

・C の右の D を折り込む → 右面 = D

・C の下の E を折り込む → 下面 = E

・E のさらに下の F：E がすでに下面となっているため、F は E から 90° さらに折れ、前面 C と反対側、すなわち 背面 に回り込む → 背面 = F


【対面ペアの整理】

立方体の平行な対面の組は以下の3組である。

・上面 A ⇔ 下面 E

・左面 B ⇔ 右面 D

・前面 C ⇔ 背面 F


【Fの位置に関する注意点】

展開図上では F が E の真下にあるため、直感的には「F の対面＝A」と誤認しやすい。しかし F は E に折りで連結するため、実際の立方体では F は背面、A は上面となり、F の対面は A ではなく、前面の C となる。


【選択肢の検討】

・(1) A → 上面。F（背面）と隣接するが対面ではない。誤り

・(2) C → 前面。F（背面）と平行な対面。正解

・(3) D → 右面。F と隣接する。誤り

・(4) E → 下面。F と隣接する（展開図の折り目に由来）。誤り

・(5) B → 左面。F と隣接する。誤り


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ展開図を組み立てた立方体を、面 E が底面となるように机の上に置く。このとき前面（手前側）には C が、右側面には D が見えている。この立方体を次の手順で転がしたとき、上面（最上面）に現れる文字はどれか。正しいものを1つ選べ。

【手順】
① 手前から奥方向へ 90° 転がす（前面が下面となるように倒す）。
② 続いて、①の状態から右方向へ 90° 転がす（右側面が下面となるように倒す）。
【図2】展開図（再掲）と初期配置
解説（正解: (3)）

【初期配置】

問1で求めたとおり、立方体の対面ペアは A⇔E、C⇔F、B⇔D。

初期状態「底面 E・前面 C・右面 D」より、上面 A・背面 F・左面 B となる。


【①手前から奥方向へ 90° 転がす】

前下辺を軸に回転するため、各面は次のように遷移する。

・上面 → 前面：A が前面に

・前面 → 下面：C が下面に

・下面 → 背面：E が背面に

・背面 → 上面：F が上面に

・左右は不変：左 B・右 D

→ 上 F・前 A・右 D・下 C・後 E・左 B


【②右方向へ 90° 転がす】

右下辺を軸に回転。前後は不変で上下左右が循環する。

・上面 → 右面：F が右面に

・右面 → 下面：D が下面に

・下面 → 左面：C が左面に

・左面 → 上面：B が上面に

・前後は不変：前 A・後 E

→ 最終: 上 B・前 A・右 F・下 D・後 E・左 C


【検算】

対面ペア（上B・下D）（前A・後E）（右F・左C）は、初期の B⇔D、A⇔E、C⇔F と一致する。


【選択肢の検討】

・(1) A → 最終前面。誤り

・(2) D → 最終下面（上面 B の対面）。誤り

・(3) B → 最終上面。正解

・(4) F → ①直後の上面に該当するが、②で右面へ移る。誤り

・(5) C → ①直後の下面、②で左面へ移る。誤り


したがって、正解は(3)である。
