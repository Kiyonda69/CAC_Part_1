問1 配点: 6点 / 目安時間: 3分
下図は立方体の展開図であり、6つの面にはア〜カの記号が書かれている。この展開図を組み立てて立方体にしたとき、向かい合う面（対面）の組み合わせとして正しいものを、次の(1)〜(5)の中から1つ選べ。
【展開図】
解説（正解: (5)）

階段型(2-2-2)の展開図は、立方体の11種類の展開図のうちでも対面関係が直感的に分かりにくい形である。中央の「カ」を正面（手前）として組み立てるのが手がかりとなる。


【折り畳みの手順】

・「カ」を正面（前面）として固定する。

・「ウ」は「カ」の右に隣接するので、右側へ折り畳む → 右面。

・「ア」は「カ」の上に隣接するので、上方向へ折り畳む → 上面。

・「エ」は「ア」の左に隣接するので、「ア」（上面）の左辺を蝶番として下方向へ折る → 左面。

・「オ」は「ウ」の下に隣接するので、「ウ」（右面）の下辺を蝶番として手前方向へ折る → 底面。

・「イ」は「オ」の右に隣接するので、「オ」（底面）の奥辺を蝶番として上方向へ折る → 背面。


【面の3D配置】

・カ → 正面、イ → 背面

・ア → 上面、オ → 底面

・エ → 左面、ウ → 右面


【対面ペア】

・カ ↔ イ（前 ↔ 後）

・ア ↔ オ（上 ↔ 下）

・エ ↔ ウ（左 ↔ 右）


【選択肢の検討】

・(1) ア-ウ、イ-エ、オ-カ → ア-ウは隣接、誤り。

・(2) ア-イ、ウ-カ、エ-オ → ア-イは隣接、誤り。

・(3) ア-カ、イ-ウ、エ-オ → ア-カは隣接、誤り。

・(4) ア-エ、イ-オ、ウ-カ → ア-エ、ウ-カは隣接、誤り。

・(5) ア-オ、イ-カ、ウ-エ → 全て対面ペアと一致、正解。


したがって、正解は(5)である。

問2 配点: 6点 / 目安時間: 5分
下図は立方体の展開図であり、5つの頂点にP、Q、R、S、Tの印が付けられている。この展開図を組み立てて立方体にしたとき、これら5つの印は立方体の何個の異なる頂点に重なるか。次の(1)〜(5)の中から1つ選べ。
【展開図と頂点マーク】
解説（正解: (5)）

問1の対面関係（カ↔イ、ア↔オ、エ↔ウ）を踏まえ、各面を立方体上の位置に配置すると次の通り。

・カ → 正面、イ → 背面、ア → 上面、オ → 底面、エ → 左面、ウ → 右面


立方体の8頂点を「前後・左右・上下」の組み合わせ（例：前-左-下）で表す。各マークの折り畳み後の3D頂点を1つずつ追跡する。


【マークの追跡】


P（エの左下角）：

エは左面。エが「ア（上面）の左辺から下に折り下げて左面」になる過程を追うと、エの左下角は立方体の「前-左-下」の頂点に重なる。


Q（アの右上角）：

アは上面。アの「右上角」は、展開図上で奥側（後）の右にあたる。よって立方体の「後-右-上」の頂点に重なる。


R（オの右下角＝イの左下角）：

オは底面。オの「右下角」は、ウ（右面）から見て手前ではなく奥（後）側の左にあたる。よって立方体の「後-左-下」の頂点。


S（イの右上角）：

イは背面。イが「オ（底面）の奥辺から上に折り上げて背面」になる過程を追うと、イの右上角は背面の右上、すなわち「後-右-上」の頂点に重なる。これはQと同じ頂点。


T（カの左下角）：

カは正面。カの「左下角」は、立方体の「前-左-下」の頂点に重なる。これはPと同じ頂点。


【まとめ】

・P と T → 「前-左-下」（同一頂点）

・Q と S → 「後-右-上」（同一頂点）

・R → 「後-左-下」（単独）


したがって、5つの印が重なる立方体の異なる頂点は3個であり、正解は(5)である。
