問1 配点: 6点 / 目安時間: 3分
次の展開図を組み立てて立方体を作るとき、面Uと向かい合う面の記号はどれか。正しいものを1つ選べ。
解説（正解: (5)）

【展開図の組み立て】

面Rを前面（基面）として考える。

・面P は Rの上に接続 → 上面

・面T は Rの下に接続 → 下面

・面Q は Rの左に接続 → 左面

・面S は Rの右に接続 → 右面

・面U は Tの下にさらに接続 → Tを下面として折り畳むと、Uは更に折り曲がって立方体の背面に回り込む


したがって、組み立てた立方体の各面対応は次のようになる。

・前面＝R　背面＝U

・上面＝P　下面＝T

・左面＝Q　右面＝S


【対面の特定】

立方体において「前面」と「背面」は対面関係にある。よって、面Uと向かい合う面は面Rである。


【選択肢の検討】

・(1) Q → 左面。Uと隣接しており対面ではない。誤り

・(2) T → 下面。Uは展開図上でTの下に連結するがTと隣接するため対面ではない。誤り

・(3) S → 右面。Uと隣接しており対面ではない。誤り

・(4) P → 上面。Uと隣接しており対面ではない。誤り

・(5) R → 前面。Uの対面（背面と前面の関係）。正解


したがって、正解は(5)である。

問2 配点: 6点 / 目安時間: 5分
立方体の展開図にアルファベットを図のように記入する。この立方体を組み立て、図のように鏡をおく。目の方向からみると鏡に映る立方体はどのように見えるか。正しいものを1つ選べ。
解説（正解: (1)）

【展開図から立方体の組み立て】

面Pを前面（基面）として展開図を組み立てると、各面は次のように対応する。

・P = 前面、Y = 背面（PとYは対面）

・H = 左面、R = 右面（HとRは対面）

・S = 上面、W = 下面（SとWは対面）

※ Y は W の下に連結するため、W を下面として折るとYは更に折れて背面に回り込む。


【鏡像の原理】

鏡は立方体の左側に置かれ、目は右側にある。目は鏡に映った立方体の像を見る。


(a) 見える面の選定: 鏡は立方体の左面に面しているため、鏡像には前面(P)・上面(S)に加えて、実物の左面(H)が映る。右面(R)ではなく左面(H)が映ることに注意。


(b) 文字の反転: 鏡に映ると、各面の文字は左右反転（鏡文字）になる。S・P・H のすべてが反転する。


【選択肢の検討】

・(1) 上面に反転S、前面に反転P、側面に反転H → 正解

・(2) 側面にRの鏡文字 → 左面Hではなく右面Rが表示されており誤り

・(3) 前面のPが反転されていない → 鏡像では全ての文字が反転するため誤り

・(4) 上面のSが反転されていない → 鏡像では全ての文字が反転するため誤り

・(5) 全ての文字が反転されていない → 鏡像の原理に反するため誤り


したがって、正解は(1)である。
