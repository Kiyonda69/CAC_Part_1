問1 配点: 6点 / 目安時間: 3分
下の3×3の図形行列において、各図形には一定の規則がある。「？」に当てはまる図形を(1)〜(5)の中から1つ選べ。

各行・各列のパターンには、それぞれ独立した規則がある。横方向と縦方向の両方の規則を見つけ、「？」の図形を求めよ。

【図形行列】
問1（正解: (2)）

【横方向の規則】

左から右に進むと、塗りつぶされた区画のパターンが90°時計回りに回転する。

 回転の対応: 左上→右上→右下→左下→左上


確認例（行①）:

 列①: 左上のみ黒 → 列②: 右上のみ黒（90°時計回り）→ 列③: 右下のみ黒（さらに90°）


【縦方向の規則】

上から下に進むと、時計回り順で塗りつぶし区画が1つずつ増える。

 列③の確認: 行①:右下(1区画) → 行②:右下+左下(2区画) → 行③:右下+左下+左上(3区画)


【答えの導出】

列③ 行③の図形を横方向の規則から求める:

 列③行②のパターン {右上,左下,右下} を90°時計回りに回転:

  右上→右下、左下→左上、右下→左下

 → {左上, 左下, 右下} = 左上・左下・右下が黒、右上が白


縦方向の規則でも: 列③の時計回り追加順は 右下→左下→左上 → {右下,左下,左上} = {左上,左下,右下} ✓


よって正解は(2)（左上・左下・右下が黒）である。

問2 配点: 6点 / 目安時間: 5分
下の3行3列の表において、各行の左2つの図形から右の図形が生成される規則がある。その規則を発見し、「？」に当てはまる図形を(1)〜(5)の中から1つ選べ。

行①と行②から規則を発見せよ。行①では2つの図形が重ならないため、行②も参考にして規則を特定すること。

【図形演算表】
問2（正解: (5)）

【規則の発見】

各行の「結果」がどのように生成されるか調べる。


行①の観察: 図A={左上}、図B={右上,左下}、結果={左上,右上,左下}

 → AとBに共通する区画がないため、結果=A∪B（和集合）に見える。

 → ただし「和集合」か「対称差」か、この行だけでは判断できない。


行②で規則を特定: 図A={左上,右上}、図B={右上,右下}、結果={左上,右下}

 → 和集合ならば {左上,右上,右下}（3区画）になるはずだが、実際は2区画。

 → 共通の「右上」が消えている → 規則は「対称差（⊕）」と確定。


【対称差の定義】

 A ⊕ B ＝ AとBの一方にのみ含まれる区画の集合

 （両方に含まれる区画は除外される）


【行③への適用】

 図A = {左上, 右上, 左下}、図B = {左下, 右下}

 A∩B（共通） = {左下}

 A∪B（合計） = {左上, 右上, 左下, 右下}

 A⊕B = A∪B − A∩B = {左上, 右上, 右下}


左上・右上・右下が黒で左下が白の図形は選択肢(5)に一致する。


よって正解は(5)（左上・右上・右下が黒）である。
