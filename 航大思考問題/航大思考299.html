問1 配点: 6点 / 目安時間: 3分
下の【図1】のような立方体ABCD−EFGHがあり、底面がABCD、上面がEFGHで、頂点Eは頂点Aの真上にある。辺ABと残り11本の辺との位置関係を考えるとき、辺ABと平行な辺の本数（ア）と、辺ABとねじれの位置にある辺の本数（イ）の組合せとして正しいものはどれか。ただし、辺どうしの位置関係は、それぞれの辺を含む直線どうしの位置関係で考えるものとする。
【図1】立方体ABCD−EFGH
解説（正解: (4)）

問1(正解:(4))

空間内の2直線の位置関係は、「交わる」「平行」「ねじれの位置」の3種類に分類される。2直線が同一平面上にあれば「交わる」か「平行」のどちらかであり、同一平面上にないとき、その2直線は「ねじれの位置」にある。辺ABを含む直線と、残り11本の辺を含む直線との関係を、もれなく重複なく数え上げればよい。


【数え上げ】

・平行な辺（ア）：辺ABと同じ向きの辺は、DC・EF・HG の3本。同じ面内にあるDC・EFは気づきやすいが、上面の奥にあるHGを見落としやすい。

・交わる辺：頂点Aを共有する AD・AE と、頂点Bを共有する BC・BF の4本。

・ねじれの位置にある辺（イ）：残りの CG・DH・EH・FG の4本。いずれも辺ABと平行でなく、延長しても交わらない（同一平面上にない）。

・検算：3＋4＋4＝11本で、残りの辺の総数と一致する。


【選択肢の検討】

・(1) ア2本・イ5本 → 上面の奥のHGを平行と気づかず、ねじれに数えた誤り

・(2) ア3本・イ2本 → ねじれをCG・DHの2本だけとし、EH・FGを見落とした誤り

・(3) ア3本・イ8本 → 「平行でない辺はすべてねじれ」とし、頂点で交わる4本まで含めた誤り

・(4) ア3本・イ4本 → 正しい

・(5) ア1本・イ6本 → 平行を同じ面内のDCだけとし、EF・HGをねじれに数えた誤り


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
下の【図2】のような正六角柱ABCDEF−GHIJKLがあり、底面ABCDEFと上面GHIJKLはともに正六角形で、頂点Gは頂点Aの真上にある。辺ABと残り17本の辺との位置関係を考えるとき、辺ABと交わる辺の本数（ア）と、辺ABとねじれの位置にある辺の本数（イ）の組合せとして正しいものはどれか。ただし、辺どうしの位置関係は、それぞれの辺を含む直線どうしの位置関係で考えるものとする。
【図2】正六角柱ABCDEF−GHIJKL
解説（正解: (1)）

問2(正解:(1))

問1と同じ分類を正六角柱で行う高難度版である。最大のポイントは、「辺として離れていても、同一平面上にあって平行でなければ、直線として延長すると必ず交わる＝ねじれの位置ではない」という点である。


【数え上げ】

・平行な辺：底面で向かい合う DE、真上の GH、その向かいの JK の3本。（正六角形では、向かい合う辺どうしだけが平行である。）

・交わる辺（ア）：頂点を共有する FA・BC（底面）と AG・BH（側面の縦の辺）の4本に加えて、CD・EF の2本、合わせて6本。CDやEFは辺ABと離れているが、どちらも辺ABと同じ底面上にあり、平行ではないので、直線として延長すると必ず交わる。ここを「離れているからねじれ」と誤ると(2)を選んでしまう。

・ねじれの位置にある辺（イ）：上面の HI・IJ・KL・LG の4本と、側面の縦の辺 CI・DJ・EK・FL の4本、合わせて8本。上面の4本は底面と平行な別の平面上にあって辺ABと平行でないため、また縦の4本は辺ABと同一平面上にないため、いずれも延長しても交わらない。

・検算：3＋6＋8＝17本で、残りの辺の総数と一致する。


【選択肢の検討】

・(1) ア6本・イ8本 → 正しい

・(2) ア4本・イ10本 → CD・EFを「離れているからねじれ」と誤分類した誤り（最大の罠）

・(3) ア6本・イ4本 → ねじれを上面の4本だけとし、側面の縦の辺CI・DJ・EK・FLを見落とした誤り

・(4) ア8本・イ6本 → 縦の辺CI・FLを「頂点C・FがA・Bのとなりだから交わる」と誤解した誤り

・(5) ア4本・イ12本 → CD・EFに加えGH・JK（平行）までねじれに数えた誤り


なお、各辺の位置関係は、座標を定めてベクトルの計算（平行判定と同一平面判定）を行うことでも確かめられる。


したがって、正解は(1)である。
