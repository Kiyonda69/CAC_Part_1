問1 配点: 6点 / 目安時間: 3分
下の表の数は、ある規則に従って並べられている。この規則に従うとき、?に入る数として正しいものを(1)～(5)から1つ選べ。
【表】
1列 2列 3列 4列
ア 1 3 9 27
イ 2 7 22 67
ウ 3 11 35 107
エ 4 ? 48 147
解説（正解: (4)）

【横方向の規則】

各行は「前の数×3＋加算値」の規則に従っている。


・ア行（加算値0）: 1 → 1×3+0=3 → 3×3+0=9 → 9×3+0=27

・イ行（加算値1）: 2 → 2×3+1=7 → 7×3+1=22 → 22×3+1=67

・ウ行（加算値2）: 3 → 3×3+2=11 → 11×3+2=35 → 35×3+2=107

・エ行（加算値3）: 4 → 4×3+3=15 → 15×3+3=48 → 48×3+3=147


加算値は行ごとに0, 1, 2, 3と1ずつ増加している。


【縦方向の規則（別解）】

各列は等差数列: 2列目は 3, 7, 11, 15（公差4）


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
下図の3×3のマトリックスには一定の規則がある。?に入る図形として正しいものを(1)～(5)から1つ選べ。
【マトリックス】
解説（正解: (3)）

【4つの性質の発見】

各セルの図形には4つの独立した性質がある:

・形: 丸、三角、四角

・大きさ: 小、中、大

・塗りつぶし: 白抜き、灰色、黒塗り

・印(小さい点)の位置: 上、右、下


【ラテン方陣の規則】

各行・各列において、4つの性質がそれぞれ3種類全て揃う。


第3行: 四角・大・灰色・右 / 丸・中・白抜き・下 → 不足は 三角・小・黒・上

第3列: 四角・中・灰色・下 / 丸・大・白抜き・右 → 不足は 三角・小・黒・上


【選択肢の検討】

・(1) 三角・大・黒・上 → 大きさが違う

・(2) 三角・小・黒・右 → 印位置が違う

・(3) 三角・小・黒・上 → 全性質が正しい

・(4) 三角・小・灰・上 → 塗りが違う

・(5) 丸・小・黒・上 → 形が違う


したがって、正解は(3)である。
