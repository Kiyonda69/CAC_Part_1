問1 配点: 6点 / 目安時間: 3分
表は、ある地域の5つの農場（A～E）の年間収穫データを示したものである。1ヘクタールあたりの出荷額が高い順に農場を並べたものはどれか。正しいものを1つ選べ。
【農場別年間収穫データ】
農場 作付面積
(ha)		 収穫量
(t)		 農薬使用量
(kg)		 従業員数
(人)		 出荷額
(万円)
A1284368420
B181087212540
C872166504
D151056010480
E1060255360
解説（正解: (5)）

【解法のポイント】

各農場の出荷額を作付面積で割り、1ヘクタールあたりの出荷額を求めて比較する。


【計算過程】

・A: 420 / 12 = 35.0万円/ha

・B: 540 / 18 = 30.0万円/ha

・C: 504 / 8 = 63.0万円/ha

・D: 480 / 15 = 32.0万円/ha

・E: 360 / 10 = 36.0万円/ha


【順位の決定】

高い順: C(63.0) → E(36.0) → A(35.0) → D(32.0) → B(30.0)


したがって、正解は(5) C→E→A→D→Bである。

問2 配点: 6点 / 目安時間: 5分
表は、ある企業の5つの支店（甲～戊）の上半期・下半期の売上高と経費、および従業員数を示したものである。従業員1人あたりの年間利益額（年間利益＝上半期利益＋下半期利益、各半期の利益＝売上高－経費）が大きい順に支店を並べたものはどれか。正しいものを1つ選べ。
【支店別半期業績データ】　　　　（単位：百万円）
支店 上半期 下半期 従業員数
(人)
売上高 経費 売上高 経費
甲18012024015030
乙25017020013040
丙1509019011025
丁30021028019050
戊20014022014035
解説（正解: (2)）

【解法のポイント】

各支店の上半期・下半期それぞれの利益を求め、年間利益を算出した後、従業員数で割る。年間売上高の大きさと1人あたり利益額は必ずしも一致しない点に注意。


【計算過程】

甲: (180-120) + (240-150) = 60 + 90 = 150 → 150/30 = 5.00百万円

乙: (250-170) + (200-130) = 80 + 70 = 150 → 150/40 = 3.75百万円

丙: (150-90) + (190-110) = 60 + 80 = 140 → 140/25 = 5.60百万円

丁: (300-210) + (280-190) = 90 + 90 = 180 → 180/50 = 3.60百万円

戊: (200-140) + (220-140) = 60 + 80 = 140 → 140/35 = 4.00百万円


【順位の決定】

大きい順: 丙(5.60) → 甲(5.00) → 戊(4.00) → 乙(3.75) → 丁(3.60)


【注意】

年間利益の総額だけを見ると丁(180)が最大だが、従業員数(50人)が多いため1人あたりでは最下位となる。甲と乙は年間利益が同額(150)だが、従業員数の差(30人 vs 40人)により1人あたりでは甲が上位となる。


したがって、正解は(2) 丙→甲→戊→乙→丁である。
