問1 配点: 6点 / 目安時間: 3分
正方形のタイルからなるいくつかのピースを組み合わせると、下の図のような形になった。この形にもう1つのピースを加えて、穴の無い長方形を完成させたい。加えるべきピースはどれか。正しいものを(1)～(5)から1つ選べ。ただし、ピースは回転させてよいが、裏返してはならない。
解説（正解: (4)）

【目標の特定】

図の形は5列×4行の範囲に収まっており、15マス分のタイルがある。5×4=20マスの長方形を完成させるには、5マス分のピース（ペントミノ）を加えればよい。


【空きの形の分析】

空いている5マスの位置を確認すると、F型ペントミノの形状である。


【選択肢の検討】

(1) T型：十字型であり、空きの形と一致しない

(2) F'型（Fの鏡像）：Fに似ているが左右反転。裏返し禁止なので不可

(3) P型：2×2ブロック＋1マスの形状で不一致

(4) F型：180度回転すると空きの形と完全に一致する

(5) W型：階段状の形状で不一致


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
一辺の長さがaの正四面体ABCDにおいて、辺ABの中点をM、辺CDの中点をNとする。線分MNについて、次の記述ア～エのうち正しいものをすべて含む組み合わせはどれか。正しいものを(1)～(5)から1つ選べ。
ア　MNの長さは a√2 / 2 である
イ　MNは辺ABと辺CDの両方に垂直である
ウ　線分MNの中点は正四面体ABCDの重心と一致する
エ　線分MNは面ABCと面ABDのなす二面角の二等分面上にある
解説（正解: (2)）

正四面体の辺の長さをaとし、座標を設定する。

A=(0,0,0)、B=(a,0,0)、C=(a/2, a√3/2, 0)、D=(a/2, a√3/6, a√(2/3))

M = (a/2, 0, 0)、N = (a/2, a√3/3, a√(2/3)/2)


【ア の検証】

MN² = (a√3/3)² + (a√(2/3)/2)² = a²/3 + a²/6 = a²/2

MN = a√2/2 → 正しい


【イ の検証】

MNベクトルとABベクトルの内積 = 0 → MN⊥AB

MNベクトルとCDベクトルの内積 = 0 → MN⊥CD

→ 正しい


【ウ の検証】

重心G = (A+B+C+D)/4 = (a/2, a√3/6, a√(2/3)/4)

MN中点 = (a/2, a√3/6, a√(2/3)/4) = G

→ 正しい


【エ の検証】

面ABCと面ABDの外向き法線から二面角の二等分面を求めると、点Nからこの面までの距離は0でない。

MNはABの垂直二等分面上にあるが、二面角の二等分面上にはない → 誤り


正しいのはア、イ、ウの3つ。したがって、正解は(2)である。
