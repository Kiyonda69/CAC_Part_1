問1 配点: 6点 / 目安時間: 3分
下図は立方体の展開図であり、各面にはア〜カの文字が記されている。この展開図を組み立てて立方体を作ったとき、アの面と向かい合う面の文字と、イの面と向かい合う面の文字の組み合わせとして正しいものを1つ選べ。
【展開図】
解説（正解: (2)）

【展開図の折りたたみ】

この展開図は階段型の配置である。ウを正面（手前の面）として固定し、順に折りたたむ。


ステップ1: ウを正面に固定する。

ステップ2: イはウの上辺に接続 → 上面に折れる。

ステップ3: アはイの左辺に接続 → イが上面になった後、アは左面に折れる。

ステップ4: エはウの右辺に接続 → 右面に折れる。

ステップ5: オはエの下辺に接続 → エが右面になった後、オは下面に折れる。

ステップ6: カはオの右辺に接続 → オが下面になった後、カは背面に折れる。


【各面の位置】

・ア = 左面、イ = 上面、ウ = 正面

・エ = 右面、オ = 下面、カ = 背面


【向かい合う面】

・ア（左面）↔ エ（右面）

・イ（上面）↔ オ（下面）

・ウ（正面）↔ カ（背面）


したがって、アの向かいはエ、イの向かいはオであり、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
1辺の長さが1の立方体ABCD-EFGHがある。底面ABCDの頂点はA、B、C、Dであり、上面EFGHの頂点はそれぞれA、B、C、Dの真上にE、F、G、Hがある。頂点A、C、F、Hを結んでできる四面体について確実にいえるものを1つ選べ。
【立方体ABCD-EFGH】
解説（正解: (1)）

【座標設定】

立方体の頂点に座標を設定する。

・底面: A(0,0,0)、B(1,0,0)、C(1,1,0)、D(0,1,0)

・上面: E(0,0,1)、F(1,0,1)、G(1,1,1)、H(0,1,1)


【四面体ACFHが正四面体であることの確認】

全ての辺の長さを計算すると、6本全てがsqrt(2)で等しい。

|AC|=|AF|=|AH|=|CF|=|CH|=|FH|=sqrt(2)

よって四面体ACFHは正四面体である。


【体積の計算】

立方体から四面体ACFHを切り出すと、4つの直角三角錐が残る。

残る4つの三角錐は頂点B、D、E、Gにそれぞれ1つずつ存在する。


例えば頂点Bの直角三角錐B-ACFは、3辺BA、BC、BFが互いに直交し、各辺の長さが1であるから、

体積 = (1/6) x 1 x 1 x 1 = 1/6


他の3つ（D-ACH、E-AFH、G-CFH）も同様に体積1/6である。


4つの直角三角錐の体積の合計 = 4 x 1/6 = 2/3


四面体ACFHの体積 = 立方体の体積 - 直角三角錐の合計 = 1 - 2/3 = 1/3


したがって、正解は(1)である。
