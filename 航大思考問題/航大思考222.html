問1 配点: 6点 / 目安時間: 3分
下の【図1】は、立方体の形をした航空貨物コンテナ（ULD）の展開図である。6つの面には、それぞれ A・B・C・D・E・F の記号がかかれている。
この展開図を辺に沿って折り、すき間なく立方体に組み立てたとき、面【C】とちょうど向かい合う（平行に正対する）面はどれか。下の(1)〜(5)から1つ選べ。
【図1】コンテナの展開図
問1（正解: (3)）

展開図の真ん中の横一列 B・C・D・E は、立方体の側面を一周する4つの面になる。一周する4面では、2つ離れた面どうしが向かい合う。


【向かい合う面の組】

・横一列 B-C-D-E … C と E が2つ離れで向かい合い、B と D が向かい合う

・残る A（Cの上）と F（Cの下） … この2面が向かい合う


よって対面の組は C↔E、B↔D、A↔F の3組。面Cと向かい合うのは面Eである。

A や F は折るとCに隣り合う面（Cと辺を共有する面）になるため、向かい合う面ではない点に注意。


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
別の立方体コンテナを考える。このコンテナの6面には P・Q・R・S・T・U の記号がかかれている。
下の(1)〜(5)は、いずれもこのコンテナの展開図の候補である（記号の配置だけが異なる）。これらを辺に沿って折って立方体に組み立てたとき、面【P】と面【U】がちょうど向かい合う（平行に正対する）ものはどれか。1つ選べ。
問2（正解: (5)）

折ったとき、ある面と向かい合う面は「1マスはさんで反対側にくる面」または「一列に並ぶ4マスのうち2つ離れた面」として求められる。各候補で P の対面を調べる。


【各選択肢の P の対面】

・(1) … P の対面は T（P と U は折ると隣り合う面になる）

・(2) … P の対面は R

・(3) … P の対面は T

・(4) … P の対面は Q

・(5) … 横一列 U・S・P・T のうち P と U が2つ離れで向かい合う。P の対面は U


条件「P と U が向かい合う」を満たすのは(5)だけである。したがって、正解は(5)である。
