問1 配点: 6点 / 目安時間: 3分
図は、1995年から2020年までの25年間におけるある地域の農産物の国内出荷量および輸出量の推移を5年ごとに示したものである。国内出荷量と輸出量の合計が、前回の調査時点（5年前）と比較して減少した年はいつか。正しいものを1つ選べ。
【農産物出荷量の推移】
（農林水産省「農産物流通統計」より作成）
解説（正解: (4)）

各年の国内出荷量と輸出量の合計を計算し、前回（5年前）と比較する。


【各年の合計】

・1995年: 2200 + 600 = 2800万トン

・2000年: 2800 + 900 = 3700万トン

・2005年: 3500 + 1400 = 4900万トン

・2010年: 3800 + 1800 = 5600万トン

・2015年: 3200 + 2300 = 5500万トン

・2020年: 3600 + 2800 = 6400万トン


【前回比の変化】

・2000年: 3700 - 2800 = +900（増加）

・2005年: 4900 - 3700 = +1200（増加）

・2010年: 5600 - 4900 = +700（増加）

・2015年: 5500 - 5600 = -100（減少）

・2020年: 6400 - 5500 = +900（増加）


2015年は輸出量が2010年より500万トン増加したが、国内出荷量が600万トン減少したため、合計では100万トンの減少となった。


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
表は、ある会社の従業員3000人について、職種（営業・技術）と勤務地（東京・大阪・名古屋・福岡・札幌）別の人数の割合を示したものである。技術職のうち名古屋勤務の人数と、営業職のうち福岡勤務の人数の差は何人か。正しいものを1つ選べ。
【従業員の勤務地別内訳（計3000人）】
（単位：%）
勤務地
職種		 東京 大阪 名古屋 福岡 札幌 合計
営　業 40.0 25.0 10.0 15.0 10.0 100.0
技　術 30.0 20.0 20.0 20.0 10.0 100.0
全　体 34.0 22.0 16.0 18.0 10.0 100.0
解説（正解: (4)）

まず、営業職と技術職それぞれの人数を求める。


【職種別人数の算出】

営業職の人数をS人、技術職の人数をT人とすると、

・S + T = 3000 ……(i)


東京の全体人数に着目すると、

・営業の東京勤務: S × 40.0% = 0.40S

・技術の東京勤務: T × 30.0% = 0.30T

・全体の東京勤務: 3000 × 34.0% = 1020人


したがって、0.40S + 0.30T = 1020 ……(ii)


(i)より T = 3000 - S を(ii)に代入すると、

0.40S + 0.30(3000 - S) = 1020

0.40S + 900 - 0.30S = 1020

0.10S = 120

S = 1200人（営業職）、T = 1800人（技術職）


【求める差の計算】

・技術職の名古屋勤務: 1800 × 20.0% = 360人

・営業職の福岡勤務: 1200 × 15.0% = 180人

・差: 360 - 180 = 180人


したがって、正解は(4)である。
