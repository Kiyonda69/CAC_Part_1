問1 配点: 6点 / 目安時間: 3分
図のような直方体の航空貨物コンテナがあり、幅4m、奥行3m、高さ2mである。床面の頂点Aから、Aと空間的に正反対の位置にある上面の頂点Bまで、コンテナの表面に沿って固定用ベルトをたるみなく張る。ベルトの長さが最も短くなるように張ったとき、その長さとして正しいものを(1)～(5)の中から1つ選べ。
【図1】貨物コンテナ（幅4m・奥行3m・高さ2m）
問1（正解: (5)）

ベルトはコンテナの表面に沿って張られるため、ベルトが通る2つの面を1つの平面に開いて考えると（展開図）、最短のベルトは展開図上の「直線」になる。AからBへ表面を通る行き方は、どの面を通るかによって3通りの展開が考えられ、それぞれ直角三角形の斜辺の長さとして計算できる。


【3通りの展開と長さの比較】

・底面（4×3）→ 奥の側面（4×2）：展開すると縦(3+2)=5m、横4mの長方形の対角線 → √(5²+4²) = √41 ≈ 6.40m

・底面（4×3）→ 右の側面（3×2）：展開すると縦3m、横(4+2)=6mの長方形の対角線 → √(3²+6²) = √45 ≈ 6.71m

・前面（4×2）→ 上面（4×3）のように幅4mの辺をまたぐ場合：縦2m、横(4+3)=7m → √(2²+7²) = √53 ≈ 7.28m


【展開図】底面と奥の側面を開いた場合（最短経路）

【選択肢の検討】

・(1) √29 m：空間対角線 √(4²+3²+2²)。内部を貫く直線であり、表面に沿うベルトでは実現できない → 不正解

・(2) √45 m：右の側面を通る展開の長さ。√41より長い → 不正解

・(3) √53 m：幅4mの辺をまたぐ展開の長さ。最も長い → 不正解

・(4) 9 m：辺に沿って 4+3+2 とたどった長さ。面を斜めに横切る方が短い → 不正解

・(5) √41 m：3通りの展開の中で最短 → 正解


したがって、正解は(5)である。

問2 配点: 6点 / 目安時間: 5分
図のような直方体の格納庫の内部空間があり、奥行30m、幅12m、高さ12mである。手前側の壁面上の点A（左右方向の中央で、天井から1m下）から、奥側の壁面上の点B（左右方向の中央で、床から1m上）まで、壁面・天井・床の表面だけに沿って通信ケーブルをたるみなく張る。ケーブルの長さが最も短くなるように張ったとき、その長さとして正しいものを(1)～(5)の中から1つ選べ。
【図2】格納庫の内部空間（奥行30m・幅12m・高さ12m）
・点A：手前側の壁面上。左右方向の中央、天井から1m下
・点B：奥側の壁面上（図では白丸）。左右方向の中央、床から1m上
問2（正解: (1)）

問1と同様に、ケーブルが通る面を平面に開いて（展開図）、直線の長さを比較する。ただし本問では通る面の組み合わせが多く、「一見遠回りに見える経路」が最短になる点に注意が必要である。


【主な経路の比較】

・経路ア（天井をまっすぐ越える）：Aから天井まで1m＋天井30m＋奥壁を11m下る → 1+30+11 = 42m

・経路イ（側壁をまっすぐ通る）：手前壁→側壁→奥壁と開くと、横(6+30+6)=42m、縦(11-1)=10m → √(42²+10²) = √1864 ≈ 43.2m

・経路ウ（天井→側壁と通る）：手前壁→天井→側壁→奥壁と開くと、2辺が(1+30+6)=37mと(6+11)=17m → √(37²+17²) = √1658 ≈ 40.7m

・経路エ（天井→側壁→床と5つの面を横断）：手前壁→天井→側壁→床→奥壁と開くと、横(1+30+1)=32m、縦(6+12+6)=24m → √(32²+24²) = √1600 = 40m（最短）


【展開図】経路エ（5面横断）を開いた直角三角形

【選択肢の検討】

・(1) 40m：天井→側壁→床と5つの面を横断する経路エの長さで最短 → 正解

・(2) 約31.6m：√(30²+10²)。空間を貫く直線距離であり、表面に沿う条件では実現できない → 不正解

・(3) 約40.7m：経路ウ（天井→側壁）の長さ。40mより長い → 不正解

・(4) 42m：経路ア（天井をまっすぐ越える）の長さ。直感的には最短に見えるが、40mより長い → 不正解

・(5) 約43.2m：経路イ（側壁をまっすぐ通る）の長さ → 不正解


したがって、正解は(1)である。
