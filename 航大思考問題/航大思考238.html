問1 配点: 6点 / 目安時間: 3分
図のように、飛行機の胴体を一様な円筒とみなす。胴体の側面には1本のストライプ（帯状の線）が塗られており、ストライプは機首側の端の真上の点から始まり、まず右舷側（進行方向に向かって右）へ回り込みながら一定の割合で巻きつき、ちょうど1周して尾部側の端の真上の点で終わる。この円筒を真下の母線（図の「切り開く線」）に沿ってまっすぐ切り開き、外側の面を正面から見た展開図（左が機首側）を作る。展開図として正しいものを(1)～(5)の中から1つ選べ。なお、各選択肢の中央の横破線は「真上の線」の位置を示す。
【図1】胴体（円筒）とストライプ ― 右舷側から見た側面図
・ストライプの実線部分は手前（右舷側）、破線部分は向こう側（左舷側）を通る部分を表す
問1（正解: (3)）

切り開いた展開図がどうなるかは、切り口と各位置の対応を押さえれば計算なしで判断できる。


【展開図の縦方向の対応】

真下の母線に沿って切り開くので、展開図の上端と下端はどちらも「真下の線」（切り開く前は同じ1本の線）になる。下端から上へたどると、真下 → 右舷の真横 → 真上（中央の破線） → 左舷の真横 → 真下（上端）の順に対応する。つまり下半分の帯が右舷側、上半分の帯が左舷側である。


【ストライプの追跡】

・ストライプは機首側の端の「真上」から始まる → 展開図では左端の中央から始まる。

・まず右舷側（下半分の帯）へ回り込む → 線は右下がりに進む。

・一定の割合でちょうど1周巻くので、胴体の長さのちょうど半分の位置で「真下」＝切り口に達する → 展開図では中央の横位置で下端に達し、同じ横位置の上端から続きが現れる（2本に分かれる）。

・残りの半周は左舷側（上半分の帯）を通り、尾部側の端の「真上」＝右端の中央で終わる。巻く割合が一定なので、2本の線分は同じ傾きの平行な右下がりになる。


【選択肢の検討】

・(1) 1本の対角線：ストライプが切り口を一度も横切らない形。切り開く線がストライプの始点・終点（真上）を通る場合の図であり、真下で切る本問では誤り → 不正解

・(2) 右上がりの2本：巻き込む向きが逆（左舷側から先に回り込む場合の図）→ 不正解

・(3) 右下がりの2本が、同じ横位置で下端と上端につながる → 正解

・(4) 2本に分かれる横位置が上端と下端でずれている：上端と下端は切り開く前は同じ1本の線なので、ストライプが切り口を横切る点は必ず同じ横位置に現れる → 不正解

・(5) V字：線が途中で折り返しており、一方向に巻き続けるストライプにはならない → 不正解


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ円筒の胴体に、同じストライプに加えて、右舷側の真横の高さに4個の窓（①～④）が機首側から等間隔に並んでおり、さらに尾部寄りの「真下」に点検口（小さな正方形）が1つある。問1と同様に真下の母線に沿って切り開き、外側の面を正面から見た展開図（左が機首側）を作る。ストライプ・窓・点検口のすべてが正しく表されている展開図を(1)～(5)の中から1つ選べ。なお、各選択肢の中央の横破線は「真上の線」の位置を示す。
【図2】胴体（円筒）とストライプ・窓・点検口 ― 右舷側から見た側面図
・窓①～④は右舷側の真横（手前）の高さにある。点検口は真下（切り開く線の上）にある
問2（正解: (4)）

ストライプ・窓・点検口の3つの要素を、切り口との対応関係に注意して順に確認する。


【要素1: ストライプ】

問1と同じく、左端の中央（機首端の真上）から右下がりに進み、中央の横位置で切り口に達して2本に分かれ、同じ横位置の上端から続きが現れて右端の中央で終わる。右下がりの平行な2本で、分かれる横位置が上下で一致していなければならない。


【要素2: 窓列】

展開図の下端が真下、中央の破線が真上なので、右舷の真横はそのちょうど中間、すなわち下から1/4の高さの水平な帯に対応する（上から1/4の帯は左舷側）。窓4個はこの下側の帯に等間隔に並ぶ。またストライプは胴体の1/4の長さの位置で右舷の真横を通過するので、展開図では窓①と窓②の間で窓の列を横切る。


【要素3: 点検口】

点検口は「真下」、つまり切り開く線の上にある。切り開くと点検口は2つに割れ、展開図では下端と上端の同じ横位置（尾部寄り）に半分ずつ、計2か所に現れる。下端だけに1つ現れる図は誤りである。


【選択肢の検討】

・(1) ストライプが右上がり：巻き込む向きが逆。窓と点検口は正しいが誤り → 不正解

・(2) 窓が上から1/4の帯（左舷側）に並んでいる：右舷と左舷の取り違え → 不正解

・(3) ストライプが1本の対角線で、点検口が下端の1か所だけ：切り口の扱いが二重に誤り → 不正解

・(4) ストライプ（右下がり2本・分割位置一致）・窓（下側の帯・①②の間を横断）・点検口（上下両端に2つ）のすべてが正しい → 正解

・(5) ストライプの分かれる横位置が上端と下端でずれている：切り口の連続性に反する → 不正解


したがって、正解は(4)である。
