問1 配点: 6点 / 目安時間: 3分
下図はある立方体の展開図であり、各面には異なる模様が1つずつ描かれている。この展開図を組み立てて立方体を作るとき、丸（●）が描かれた面と向かい合う面の模様として正しいものを、(1)～(5)から1つ選べ。
【展開図】
解説（正解: (1)）

丸（●）が描かれた面を基準（正面）として組み立てを考える。


【各面の位置】

・星（★）は丸の左 → 左面

・三角（△）は丸の右 → 右面

・ひし形（◇）は丸の下 → 底面

・四角（□）は三角の上 → 三角が右面になると、四角は上に折れて上面

・十字（＋）はひし形の下 → ひし形が底面になると、十字は奥に折れて背面


【向かい合う面のペア】

・丸（●） ↔ 十字（＋）（正面 ↔ 背面）

・星（★） ↔ 三角（△）（左面 ↔ 右面）

・四角（□） ↔ ひし形（◇）（上面 ↔ 底面）


したがって、丸の向かい側は十字（＋）であり、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
下図は、3×3×3の格子空間に1辺1の立方体ブロックを配置したときの正面図と右側面図を示している。黒色の部分はブロックが見える位置、白色の部分はブロックが見えない位置を表す。2つの投影図がともに正しくなるようにブロックを配置するとき、必要なブロックの最小個数として正しいものを、(1)～(5)から1つ選べ。
解説（正解: (3)）

【解法の方針】

各高さ（y=1, 2, 3）ごとに、正面図と右側面図を同時に満たす最小のブロック配置を求める。


正面図で(x, y)が黒 = 高さyのx列にブロックが1つ以上ある

右側面図で(z, y)が黒 = 高さyのz行にブロックが1つ以上ある


【y=1（最下段）】

正面図: x=2のみ黒（必要列数: 1）

右側面図: z=1, z=2が黒（必要行数: 2）

最小ブロック数 = max(1, 2) = 2個

配置例: (2,1,1), (2,1,2)


【y=2（中段）】

正面図: x=1, 2, 3すべて黒（必要列数: 3）

右側面図: z=2, z=3が黒（必要行数: 2）

最小ブロック数 = max(3, 2) = 3個

配置例: (1,2,3), (2,2,2), (3,2,2)


【y=3（最上段）】

正面図: x=1, x=3が黒（必要列数: 2）

右側面図: z=1, z=2が黒（必要行数: 2）

最小ブロック数 = max(2, 2) = 2個

配置例: (1,3,2), (3,3,1)


【合計】

2 + 3 + 2 = 7個


各高さで、正面図のr列と右側面図のc行を同時に満たすには最低max(r, c)個のブロックが必要であり、これは達成可能である。


したがって、正解は(3)である。
