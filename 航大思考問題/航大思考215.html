問1 配点: 6点 / 目安時間: 3分
ある飛行機が、地点Oから機首方位「北」の状態で水平飛行を開始し、下の手順1〜5の順に飛行した。ここで「旋回」とはその場で機首の向きだけを変える操作であり、「右」「左」は進行方向に向かっての向きを表す。手順5を終えた時点における、地点Oから最終地点までの直線距離と、そのときの機首方位の組合せとして正しいものを、(1)〜(5)のうちから一つ選べ。
【飛行手順】
手順1．そのまま 4km 直進する。
手順2．右に 90° 旋回し、6km 直進する。
手順3．左に 90° 旋回し、3km 直進する。
手順4．左に 90° 旋回し、3km 直進する。
手順5．左に 90° 旋回し、3km 直進する。
【図1】方位と旋回のきまり
問1（正解: (1)）

機首方位と位置を手順ごとに追跡する。地点Oを基準として、東西方向と南北方向の移動量を分けて記録するのがポイントである。


【手順ごとの追跡】

・手順1: 機首方位「北」のまま4km直進 → Oから（東0km、北4km）

・手順2: 北から右に90°旋回すると「東」。6km直進 → （東6km、北4km）

・手順3: 東から左に90°旋回すると「北」。3km直進 → （東6km、北7km）

・手順4: 北から左に90°旋回すると「西」。3km直進 → （東3km、北7km）

・手順5: 西から左に90°旋回すると「南」。3km直進 → （東3km、北4km）


【直線距離の計算】

最終地点はOから東へ3km、北へ4kmの位置にある。三平方の定理より、Oからの直線距離は √(32 + 42) = √25 = 5km。最終の機首方位は手順5の旋回で決まった「南」である。


【選択肢の検討】

・(2) 4km・南 → 北方向の成分4kmだけを直線距離と誤認したもの

・(3) 5km・北 → 距離は正しいが、手順3を終えた時点の方位のまま止まっている

・(4) 7km・南 → 東3kmと北4kmを単純に足した値で、直線距離ではない

・(5) 5km・西 → 手順5の旋回を見落とし、手順4の方位「西」のままとした誤り


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
別の飛行機が、地点Oから機首方位「東」の状態で水平飛行を開始し、下の手順1〜6の順に飛行した。旋回のきまりは問1と同じであり、方位は図2の8方位で表す。なお、斜め方向（北東・南東・南西・北西）に 2√2 km 直進すると、東西方向に2km・南北方向に2km進んだ位置に到達する。手順6を終えた飛行機が、最終地点から出発点Oの真上まで一直線に戻るためには、どちらへ何度旋回してから何km直進すればよいか。正しいものを(1)〜(5)のうちから一つ選べ。
【飛行手順】
手順1．そのまま 4km 直進する。
手順2．左に 45° 旋回し、2√2 km 直進する。
手順3．左に 45° 旋回し、3km 直進する。
手順4．左に 90° 旋回し、8km 直進する。
手順5．左に 45° 旋回し、4√2 km 直進する。
手順6．右に 135° 旋回し、5km 直進する。
【図2】方位と旋回のきまり（問1と共通）
問2（正解: (1)）

問1と同様に機首方位と位置を手順ごとに追跡する。45°旋回が含まれるため方位は8方位で管理し、斜め方向の移動は東西成分と南北成分に分解する（d√2 kmの斜め移動 ＝ 東西方向d km ＋ 南北方向d km）。


【手順ごとの追跡】

・手順1: 機首方位「東」のまま4km直進 → Oから（東4km、北0km）

・手順2: 東から左に45°旋回すると「北東」。2√2 km直進＝東へ2km・北へ2km → （東6km、北2km）

・手順3: 北東から左に45°旋回すると「北」。3km直進 → （東6km、北5km）

・手順4: 北から左に90°旋回すると「西」。8km直進 → （西2km、北5km）

・手順5: 西から左に45°旋回すると「南西」。4√2 km直進＝西へ4km・南へ4km → （西6km、北1km）

・手順6: 南西から右に135°旋回すると、南西→西→北西→北と45°ずつ3段階回って「北」。5km直進 → （西6km、北6km）


【帰投経路の計算】

最終地点はOから西へ6km、北へ6kmの位置にある。Oへ戻るには、ここから東へ6km・南へ6km移動すればよい。東向きと南向きの成分が等しいので、戻る方向はちょうど「南東」である。

戻る距離は √(62 + 62) = √72 = 6√2 km（約8.5km）。

手順6を終えた時点の機首方位は「北」なので、南東を向くには北→北東→東→南東と右回りに45°×3＝135°旋回すればよい（左回りでは225°必要で、遠回りになる）。


【選択肢の検討】

・(2) 左135° → 「南西」を向き、Oから遠ざかる

・(3) 右45° → 「北東」を向いてしまい、Oの方向（南東）と一致しない

・(4) 12km → 東6kmと南6kmを単純に足した値で、直線距離ではない

・(5) 右90°・6km → 「東」を向いて東西成分だけ進む誤り。南北成分6kmが残る


したがって、正解は(1)である。
