問1 配点: 6点 / 目安時間: 3分
図Ⅰのように、三角柱、四角柱、五角柱、…と、底面の辺の数を1本ずつ増やした角柱を順につくる。各角柱について、2つの頂点を結ぶ線分のうち、どの面の上にもない線分を「空間対角線」と呼ぶことにする。空間対角線の本数を数えると、三角柱は0本、四角柱は4本、五角柱は10本である。この規則が続くとき、十角柱の空間対角線の本数として、正しいものを1つ選べ。
【図Ⅰ】角柱と空間対角線
※ 破線は隠れて見えない辺、灰色の細線は空間対角線を表す。
※ 五角柱の図では、10本のうち頂点Aから出る空間対角線2本のみを示している。
解説（正解: (3)）

問1(正解:(3))

空間対角線の本数を順に並べ、増え方（階差）に注目する。


【規則の発見】

・三角柱 0本 → 四角柱 4本 → 五角柱 10本 と増えており、増え方は 4本、6本。

・増え方が「2ずつ大きくなる」（4、6、8、10、…）と考えて先へ延ばすと、三角柱0 → 四角柱4 → 五角柱10 → 六角柱18 → 七角柱28 → 八角柱40 → 九角柱54 → 十角柱70。

・よって十角柱の空間対角線は 70本 である。


【別解：数え上げによる確認】

・十角柱の頂点は上下10個ずつの計20個。1つの頂点Aから他の頂点へ引いた線分のうち、同じ底面上の9頂点へ引いたもの（辺2本・底面の対角線7本）はすべて面の上にある。

・もう一方の底面の10頂点のうち、Aの真上（真下）の1頂点へは側面の辺、その両どなりの2頂点へは側面の対角線となるから、Aから引ける空間対角線は 10−3＝7本。

・各頂点から7本ずつ、両端で2回ずつ数えているので、20×7÷2＝70本。

・一般に n角柱の空間対角線は n(n−3)本 となり、図の 0本・4本・10本とも一致する。


【選択肢の検討】

・(1) 35本 → 多角形の対角線の公式 n(n−3)÷2 をそのまま使った誤り（頂点は2n個ある）。

・(2) 54本 → 九角柱の本数。階差を1回分早く止めた誤り。

・(3) 70本 → 上記のとおり正解。

・(4) 80本 → 1頂点から引ける本数を10−2＝8本と数え、側面の対角線2本を含めた誤り。

・(5) 140本 → 20×7＝140本のまま、両端の2重数えを2で割り忘れた誤り。


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ角柱の列について、今度は、2つの頂点を結ぶ線分のうち辺ではないものすべて、すなわち、面の上にある対角線（面の対角線）と空間対角線を合わせた「対角線の総数」を数える。図Ⅱのように、1つの頂点からは面の対角線と空間対角線が引け、対角線の総数は、三角柱は6本、四角柱は16本、五角柱は30本である。この規則が続くとき、対角線の総数が初めて200本を超えるのは、どの角柱か。正しいものを1つ選べ。
【図Ⅱ】四角柱の例（頂点Aから引ける対角線）と対角線の総数
角柱 三角柱 四角柱 五角柱 …
対角線の総数 6本 16本 30本 …
※ 三角柱の底面（三角形）には面の対角線がないため、総数6本はすべて側面の対角線である。
解説（正解: (2)）

問2(正解:(2))

対角線の総数の増え方（階差）から規則を見つけ、200本を初めて超える角柱を探す。


【規則の発見】

・総数は 6本 → 16本 → 30本 と増えており、増え方は 10本、14本。

・増え方が「4ずつ大きくなる」（10、14、18、22、…）と考えて表を延ばすと、

　三角柱6 → 四角柱16 → 五角柱30 → 六角柱48 → 七角柱70 → 八角柱96 → 九角柱126 → 十角柱160 → 十一角柱198 → 十二角柱240。

・十一角柱は198本で200本を超えない（あと2本足りない）。

・十二角柱で240本となり、初めて200本を超える。


【別解:公式による確認】

・n角柱の頂点は2n個。2つの頂点の結び方は全部で 2n(2n−1)÷2＝n(2n−1)通りあり、このうち辺は3n本だから、対角線の総数＝n(2n−1)−3n＝2n(n−2)本。

　（三角柱 2×3×1＝6本、四角柱 2×4×2＝16本、五角柱 2×5×3＝30本と一致）

・2n(n−2)＞200 すなわち n(n−2)＞100 となる最小のnを探すと、n＝11では 11×9＝99 で足りず、n＝12で 12×10＝120 となり初めて満たす。


【選択肢の検討】

・(1) 十一角柱 → 198本。200本にあと2本届かない。表の見誤りに注意（最大の罠）。

・(2) 十二角柱 → 240本で初めて200本を超える。正解。

・(3) 十三角柱 → 286本で200本は超えるが、「初めて」ではない。

・(4) 十五角柱 → 空間対角線を数え忘れ、面の対角線だけ（n²−n本）で計算した場合の誤り。

・(5) 十六角柱 → 面の対角線を数え忘れ、空間対角線だけ（n(n−3)本）で計算した場合の誤り。


したがって、正解は(2)である。
