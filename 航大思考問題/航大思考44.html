問1 配点: 6点 / 目安時間: 3分
図は、ある市の公立図書館における1990年から2020年までの30年間の蔵書数および年間貸出冊数の推移を5年ごとに示したものである。蔵書数と年間貸出冊数の合計が、前回の調査時点（5年前）と比較して減少した年はいつか。正しいものを1つ選べ。
【蔵書数と年間貸出冊数の推移】
（文部科学省：「社会教育調査（令和3年度）」より作成（一部改変））
解説（正解: (2)）

【解法のポイント】

各年の蔵書数と貸出冊数の合計を求め、5年前の値と比較する。


【計算過程】

・1990年: 80 + 180 = 260

・1995年: 105 + 230 = 335（前回比 +75、増加）

・2000年: 130 + 280 = 410（前回比 +75、増加）

・2005年: 155 + 250 = 405（前回比 -5、減少）

・2010年: 180 + 300 = 480（前回比 +75、増加）

・2015年: 210 + 340 = 550（前回比 +70、増加）

・2020年: 240 + 370 = 610（前回比 +60、増加）


蔵書数は一貫して増加しているが、2005年は貸出冊数が2000年の280万冊から250万冊に減少しており、蔵書数の増加分（+25）を上回る減少（-30）が生じたため、合計が前回より減少した唯一の年である。


したがって、正解は(2) 2005年である。

問2 配点: 6点 / 目安時間: 5分
表は、ある国際空港の年間利用客4500人について、旅行目的（出張・観光）と渡航先地域別の人数の割合を示したものである。出張客が観光客を上回っている渡航先地域をすべて特定し、それらの地域の出張客の合計は何人か。正しいものを1つ選べ。
【旅行目的別・渡航先地域別利用客の割合（計4500人）】
（単位：%）
地域 目的
北米 ヨーロッパ アジア オセアニア その他 合計
出　張 40.0 20.0 25.0 8.0 7.0 100.0
観　光 20.0 35.0 15.0 18.0 12.0 100.0
全　体 28.0 29.0 19.0 14.0 10.0 100.0
（国土交通省：「航空輸送統計年報（令和7年度）」より作成（一部改変））
解説（正解: (5)）

【解法のポイント】

まず、全体の割合と各目的の割合から連立方程式を立て、出張客と観光客の人数を求める。次に、各地域の出張客数と観光客数を比較する。


【出張客数・観光客数の算出】

出張客をB人、観光客をL人とすると、B + L = 4500

北米について: 0.40B + 0.20L = 0.28 × 4500 = 1260

0.40B + 0.20(4500 - B) = 1260

0.40B + 900 - 0.20B = 1260

0.20B = 360

B = 1800（出張客）、L = 2700（観光客）


【地域別比較】

・北米: 出張 0.40 × 1800 = 720、観光 0.20 × 2700 = 540 → 出張 > 観光

・ヨーロッパ: 出張 0.20 × 1800 = 360、観光 0.35 × 2700 = 945 → 出張 < 観光

・アジア: 出張 0.25 × 1800 = 450、観光 0.15 × 2700 = 405 → 出張 > 観光

・オセアニア: 出張 0.08 × 1800 = 144、観光 0.18 × 2700 = 486 → 出張 < 観光

・その他: 出張 0.07 × 1800 = 126、観光 0.12 × 2700 = 324 → 出張 < 観光


【出張客合計】

北米（720）＋ アジア（450）＝ 1170人


したがって、正解は(5) 1170人である。
