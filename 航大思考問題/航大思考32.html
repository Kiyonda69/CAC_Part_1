問1 配点: 6点 / 目安時間: 3分
2枚の透明なシート（シートA、シートB）がある。各シートには4×4のマス目が描かれており、一部のマスが黒く塗られている。シートAを時計回りに90度回転させた後、シートBの上に重ねたとき、黒いマスの配置として正しいものを(1)～(5)から1つ選べ。
解説（正解: (4)）

【回転操作】

4×4グリッドを時計回りに90度回転すると、元の座標(行r, 列c)は新しい座標(列c, 3-行r)に移る。


【シートAの回転結果】

シートAの黒マス: (0,1), (1,0), (1,1), (3,2)

回転後: (1,3), (0,2), (1,2), (2,0)

回転後のシートA: ..■. / ..■■ / ■... / ....


【重ね合わせ】

シートBの黒マス: (0,3), (2,2), (3,0)

回転したシートAとシートBの和集合を取ると：

行0: ..■■ / 行1: ..■■ / 行2: ■.■. / 行3: ■...


【各選択肢の検討】

・(1) シートAを回転せずそのまま重ねたパターン → 不正解

・(2) シートAを反時計回りに回転したパターン → 回転方向が逆

・(3) シートAを180度回転したパターン → 回転角度が異なる

・(4) 時計回り90度回転の正しいパターン → 正解

・(5) 正解に近いが、行2の(2,0)が(2,3)に誤っている


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
下図のような一辺の長さが6の立方体ABCD-EFGHがある。辺AB、辺CG、辺EHのそれぞれの中点をP、Q、Rとする。3点P、Q、Rを通る平面でこの立方体を切断したとき、切断面の形状と面積の組み合わせとして正しいものを(1)～(5)から1つ選べ。
【立方体ABCD-EFGH】
解説（正解: (1)）

【切断面の頂点を求める】

立方体の頂点座標: A(0,0,0), B(6,0,0), C(6,6,0), D(0,6,0), E(0,0,6), F(6,0,6), G(6,6,6), H(0,6,6)

中点: P(3,0,0), Q(6,6,3), R(0,3,6)


【平面の方程式】

ベクトルPQ=(3,6,3)、PR=(-3,3,6)から法線ベクトルを求める。

PQ×PR = (27,-27,27) → 簡約化: (1,-1,1)

平面方程式: x - y + z = 3


【各辺との交点】

この平面は立方体の6つの辺と交わる。全て辺の中点で交わるのが特徴。

・辺AB上: P(3,0,0)　・辺BC上: S(6,3,0)　・辺CG上: Q(6,6,3)

・辺GH上: T(3,6,6)　・辺HE上: R(0,3,6)　・辺AE上: U(0,0,3)


【形状の判定】

各辺の長さ: PS=SQ=QT=TR=RU=UP = 3√2 （全て等しい）

→ 切断面は正六角形


【面積の計算】

正六角形の面積 = (3√3/2) × (3√2)² = (3√3/2) × 18 = 27√3


【誤答の分析】

・(2) 3点P,Q,Rのみを結ぶ三角形と誤解。実際は6辺と交わる六角形

・(3) 切断面の形状を長方形と誤認

・(4) 正六角形は正しいが、辺の長さを6√2と誤り面積が4倍

・(5) 切断面をひし形と誤認


したがって、正解は(1)である。
