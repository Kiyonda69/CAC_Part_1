問1 配点: 6点 / 目安時間: 3分
下の図1のように、直線ℓと、1辺1cmの方眼のマス目に沿って描かれた図形が同じ平面上にある。図形は直線ℓに接している。この図形を、直線ℓを軸として1回転させてできる立体を、直線ℓを含む平面で切断したとき、切り口の形として正しいものを、次の(1)〜(5)のうちから1つ選べ。
【図1】直線ℓと図形（方眼の1マスは1辺1cm）
問1（正解: (2)）

平面図形を直線ℓの周りに1回転させてできる回転体を、ℓを含む平面で切断すると、切り口は「元の図形」と「元の図形をℓに関して折り返した鏡像」を合わせた、ℓについて左右対称な図形になる。このことを、立体を具体的に思い浮かべて確かめる。


【回転体の形を把握する】

図形は下から、幅3cm×高さ1cm、幅2cm×高さ1cm、幅1cm×高さ2cmの3段の階段形で、ℓに接している。これをℓの周りに1回転させると、

・下段 → 半径3cm・高さ1cmの円柱

・中段 → 半径2cm・高さ1cmの円柱

・上段 → 半径1cm・高さ2cmの円柱

を積み重ねた、3段の「ウェディングケーキ形」の立体ができる。


【切り口を求める】

この立体をℓを含む平面で切ると、各円柱の切り口は「幅＝直径、高さ＝円柱の高さ」の長方形になる。半径ではなく直径になる点に注意すると、下から

・幅6cm×高さ1cm（下段の円柱）

・幅4cm×高さ1cm（中段の円柱）

・幅2cm×高さ2cm（上段の円柱）

の長方形を軸をそろえて積んだ、左右対称な階段形になる。これは(2)の図形である。


【選択肢の検討】

・(1) 元の図形と同じ非対称な階段形 → 鏡像の側（軸の反対側）を忘れている → 不正解

・(2) 下から幅6cm・4cm・2cm(高さ2cm)の左右対称な階段形 → 正解

・(3) 左右対称だが上下が逆（上ほど広い形） → 図形の上下を取り違えている → 不正解

・(4) 左右対称だが最上段の高さが1cmしかない → 上段は高さ2cmなので一致しない → 不正解

・(5) 同じ向きの階段を2つ並べた形 → 鏡像ではなく平行移動した複製であり、左右対称でない → 不正解


したがって、正解は(2)である。

問2 配点: 6点 / 目安時間: 5分
下の図2のように、直線ℓと、1辺4cmの正方形の板が同じ平面上にある。板の辺は直線ℓに平行で、板はℓから1cm離れている。また、板には1辺1cmの正方形の穴が1つあいており、その位置は、ℓに近い側の辺から1cm、上の辺から1cmである。この板を、直線ℓを軸として1回転させてできる立体を、直線ℓを含む平面で切断したとき、切り口の形として正しいものを、次の(1)〜(5)のうちから1つ選べ。
【図2】直線ℓと穴のあいた正方形の板（板内の格子は1cm間隔）
問2（正解: (2)）

問1と同じく「回転体をℓを含む平面で切った切り口は、元の図形とその鏡像を合わせた左右対称な図形」という原理を使うが、本問では板が軸から離れており、さらに穴があるため、次の3点を正確に処理する必要がある。


【ポイント1: 軸との隙間は消えない】

板はℓから1cm離れているので、回転体は「内半径1cm・外半径5cm・高さ4cm」の、中心に円柱状の空洞をもつ筒（パイプ）状の立体になる。ℓを含む平面で切ると、切り口は軸の左右に1cmずつ離れた2つの部分に分かれ、中央には幅2cm（1cm×両側）の隙間ができる。


【ポイント2: 穴は埋まらずに残る】

板の穴は、回転させると立体の内部にドーナツ状（環状）の空洞を作る。ℓを含む平面はこの空洞を左右2か所で横切るので、切り口の左右両方の四角形に1辺1cmの穴が現れる。「回転させれば他の部分が通るから穴は埋まる」と考えるのは誤りである（板は常に同じ位置関係を保ったまま回るため、穴の軌跡はくり抜かれたまま残る）。


【ポイント3: 左側は鏡像（左右反転）になる】

軸の左側に現れる四角形は、元の板を平行移動したものではなく、ℓに関して折り返した鏡像である。したがって左側の穴は「軸に近い側の辺から1cm・上の辺から1cm」の位置、すなわち右側の穴と左右対称の位置にくる。上下の位置（上の辺から1cm）は変わらない。


【選択肢の検討】

・(1) 2つの四角形が軸で接している → ℓと板の間の1cmの隙間を忘れている → 不正解

・(2) 中央に幅2cmの隙間があり、穴が左右対称（どちらも内側の辺から1cm・上から1cm）→ 正解

・(3) 隙間は正しいが、左側の穴が外側の辺寄りにある → 鏡像でなく平行移動した複製 → 不正解

・(4) 隙間は正しいが穴がない → 回転で穴が埋まるという誤解 → 不正解

・(5) 隙間は正しいが、左側の穴が下寄り（下の辺から1cm）にある → 左右反転に加えて上下まで反転させた点対称の誤り → 不正解


したがって、正解は(2)である。
