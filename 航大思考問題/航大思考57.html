問1 配点: 6点 / 目安時間: 3分
図は、2010年から2020年までの10年間における国内出荷額および海外出荷額の推移を2年ごとに示したものである。国内出荷額と海外出荷額の合計が、前回の調査時点（2年前）と比較して減少した年はいつか。正しいものを1つ選べ。
【出荷額の推移】
（経済産業省：「電子部品・デバイス出荷統計」より作成（一部改変））
解説（正解: (4)）

【解法】

各年の国内出荷額と海外出荷額の合計を求め、前回（2年前）と比較する。


【各年の合計出荷額】

・2010年：3200 + 1800 = 5000（億円）

・2012年：3800 + 2400 = 6200（億円）→ 前回比 +1200

・2014年：4500 + 3100 = 7600（億円）→ 前回比 +1400

・2016年：4200 + 2900 = 7100（億円）→ 前回比 -500（減少）

・2018年：5000 + 3500 = 8500（億円）→ 前回比 +1400

・2020年：4600 + 4200 = 8800（億円）→ 前回比 +300


合計が前回より減少したのは2016年のみである。

したがって、正解は(4) 2016年である。

問2 配点: 6点 / 目安時間: 5分
表は、ある企業の従業員3600人について、職種（営業部・技術部）と勤務地域（東京・大阪・名古屋・福岡・その他）別の人数の割合を示したものである。5つの地域における技術部の総数は何人か。正しいものを1つ選べ。
【従業員の地域別内訳（計3600人）】
（単位：%）
職種 ＼ 地域 東京 大阪 名古屋 福岡 その他 合計
営 業 部 36.0 24.0 18.0 15.0 7.0 100.0
技 術 部 42.0 18.0 24.0 9.0 7.0 100.0
全　体 38.0 22.0 20.0 13.0 7.0 100.0
解説（正解: (1)）

【解法】

営業部の人数をx人、技術部の人数をy人とする。


【連立方程式の設定】

・x + y = 3600 …(i)

・東京の列から：0.36x + 0.42y = 0.38 × 3600 = 1368 …(ii)


【計算過程】

(i)より x = 3600 - y を(ii)に代入：

0.36(3600 - y) + 0.42y = 1368

1296 - 0.36y + 0.42y = 1368

0.06y = 72

y = 1200


【検算】

・x = 3600 - 1200 = 2400

・大阪の検算：0.24 × 2400 + 0.18 × 1200 = 576 + 216 = 792 = 0.22 × 3600（整合）


したがって、正解は(1) 1200人である。
