問1 配点: 6点 / 目安時間: 3分
ある化学プラントでは、反応容器内の温度を遠方から視覚的に把握するために、容器の側面に4つの感温表示パネル（左からP1、P2、P3、P4）が横一列に設置されている。各パネルにはサーモクロミック材料（温度により色が変わる特殊素材）が使われており、パネル周辺の温度がそれぞれの閾値を超えると白色（○）から黒色（●）に自動的に変色する。これは電気的な制御ではなく、材料自体の物理的性質を利用した光学的な構造である。4つのパネルの閾値温度はそれぞれ異なり、P1は180°C、P2は200°C、P3は220°C、P4は240°Cに設定されている。温度が上昇すると、閾値の低いP1から順に黒色に変わっていく。
この反応容器の適正温度範囲は200〜220°Cであり、この範囲ではP1とP2が黒色、P3とP4が白色となり、「●●○○」が正常な表示状態である。温度が高すぎると反応が暴走する危険があり、低すぎると反応効率が著しく低下する。なお、加熱出力だけを上げて攪拌速度を上げない場合、容器内に温度むらが生じ局所的な過熱が起きるため、加熱と攪拌は必ず同時に行うことが規定されている。
【操作一覧】
ア：加熱出力を上げ、攪拌速度を上げる
イ：現状の設定を維持する
ウ：加熱出力を上げる
エ：攪拌速度を下げる
オ：加熱出力を下げる
【状況】
解説（正解: (3)）

【表示の読み取り】

・P1(180°C)、P2(200°C)、P3(220°C)、P4(240°C)がそれぞれの閾値で白→黒に変色する

・正常状態「●●○○」は温度が200〜220°Cであることを示す

・変化後「●○○○」は温度が180〜200°Cに低下したことを示す


【必要な操作の分析】

・温度を200〜220°C（正常範囲）に戻す必要がある → 加熱出力を上げる

・しかし、加熱出力だけを上げると容器内に温度むらが生じ局所過熱が起きる

・規定により、加熱と攪拌は必ず同時に行う必要がある


【選択肢の検討】

・(1) ア、ウ → アは加熱+攪拌の複合操作で適切だが、ウ（追加加熱）を重ねると過加熱になる → 不正解

・(2) イ → 現状維持では温度低下のまま反応効率が低下する → 不正解

・(3) ア → 加熱出力を上げつつ攪拌速度も上げる。温度を均一に保ちながら正常範囲に戻せる → 正解

・(4) ウ → 加熱のみで攪拌を上げないため、規定に違反し局所過熱のリスクがある → 不正解

・(5) オ → 加熱出力を下げるとさらに温度が低下する → 不正解


したがって、正解は(3)である。

問2 配点: 6点 / 目安時間: 5分
問1の化学プラントでは、温度に加えて圧力も同時に監視する必要がある。温度表示パネル（P1〜P4）の下段に、圧力表示パネル（Q1、Q2、Q3、Q4）が横一列に追加設置されている。各パネルは圧力が閾値を超えると白色（○）から黒色（●）に変色する。Q1は0.5 MPa、Q2は1.0 MPa、Q3は1.5 MPa、Q4は2.0 MPaが閾値であり、適正圧力範囲は1.0〜1.5 MPa、正常表示は「●●○○」である。
操作員は以下の5つの操作を使用できる。同時に2つまでの操作を行うことができ、その効果は加算される。温度・圧力がともに正常な「●●○○」に戻る操作の組み合わせを選べ。
操作 内容 温度への影響 圧力への影響
ア加熱出力を上げる1段階上昇1段階上昇
イ加熱出力を下げる1段階下降1段階下降
ウ冷却水バルブを開く1段階下降変化なし
エ圧力逃し弁を開く変化なし1段階下降
オ圧力逃し弁を閉じる変化なし1段階上昇
【状況】
解説（正解: (4)）

【状況の分析】

・温度: ●●○○ → ●●●○（1段階上昇、高温側に逸脱）→ 温度を1段階下げる必要がある

・圧力: ●●○○ → ●○○○（1段階下降、低圧側に逸脱）→ 圧力を1段階上げる必要がある

・必要な変化: (温度 -1, 圧力 +1)


【各選択肢の効果】

・(1) ア、ウ → ア(温度+1, 圧力+1) + ウ(温度-1, 圧力±0) = (温度±0, 圧力+1) → 温度が戻らない → 不正解

・(2) イ、オ → イ(温度-1, 圧力-1) + オ(温度±0, 圧力+1) = (温度-1, 圧力±0) → 圧力が戻らない → 不正解

・(3) ウ、エ → ウ(温度-1, 圧力±0) + エ(温度±0, 圧力-1) = (温度-1, 圧力-1) → 圧力がさらに低下 → 不正解

・(4) ウ、オ → ウ(温度-1, 圧力±0) + オ(温度±0, 圧力+1) = (温度-1, 圧力+1) → 両方とも正常に戻る → 正解

・(5) イ、ア → イ(温度-1, 圧力-1) + ア(温度+1, 圧力+1) = (温度±0, 圧力±0) → 何も変わらない → 不正解


【ポイント】

・温度と圧力で逆方向の補正が必要な場合、加熱出力の操作（ア・イ）は両方に同時に影響するため使えない

・冷却水バルブ（ウ）は温度のみ、圧力逃し弁（オ）は圧力のみに影響するため、これらを組み合わせて独立に制御できる


したがって、正解は(4)である。
