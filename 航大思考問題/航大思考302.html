問1 配点: 6点 / 目安時間: 3分
同じ大きさの球を、下の図のように、上から第1段目に1個、第2段目に3個、第3段目に6個、…と、各段の球が正三角形状にすき間なく並ぶように積み上げて、第1番目、第2番目、第3番目、…と規則的に立体を作っていく。n段からなる立体を第n番目の立体と呼ぶと、第1番目の立体には1個、第2番目の立体には4個、第3番目の立体には10個の球が使われている。このとき、第6番目の立体に使われている球は全部で何個か。
【図1】規則的に作っていく立体（第1番目〜第3番目）
解説（正解: (1)）

問1(正解:(1))

各段の球の個数の規則を見つけ、第6番目の立体（6段）まで合計する問題である。


【規則の発見】

・各段の球は正三角形状に並ぶので、上から第m段目の球の個数は 1＋2＋3＋…＋m ＝ m(m＋1)/2（個）となる。

・すなわち各段の球数は、第1段目から順に 1, 3, 6, 10, 15, 21, …（個）。

・隣り合う段の球数の差が 2, 3, 4, 5, 6, … と1つずつ増えていることからも確認できる。


【計算】

第n番目の立体の球の総数は、第1段目から第n段目までの球数の合計である。

・第1番目：1個

・第2番目：1＋3＝4個

・第3番目：4＋6＝10個

・第4番目：10＋10＝20個

・第5番目：20＋15＝35個

・第6番目：35＋21＝56個

（第1〜3番目の値が問題文の 1個・4個・10個 と一致することも確認できる）


【選択肢の検討】

・(1) 56個 → 上記の計算どおり。正解

・(2) 66個 → 途中の累計を重複して加えるなどの計算ミス（56＋10）

・(3) 84個 → 第7番目の立体の球数。1段多く数えている

・(4) 91個 → 各段を「1辺m個の正方形状（m×m個）」と誤解した場合の合計（1＋4＋9＋16＋25＋36）

・(5) 120個 → 第8番目の立体の球数。2段多く数えている


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
問1と同じ規則で球を12段に積み上げて、第12番目の立体を作る。この立体は、下の図のように、底面1つと斜面3つの合わせて4つの面をもつ三角錐の形になり、それぞれの面には球が正三角形状に並んで現れている。第12番目の立体に使われている球のうち、これら4つの面のいずれの上にもない球（外から見えず、立体の内部に完全に隠れている球）は全部で何個か。
【図2】第12番目の立体の形（三角錐）と4つの面
解説（正解: (3)）

問2(正解:(3))

「内部に隠れている球」を段ごとに数え上げる問題である。


【考え方：段ごとに内部の球を数える】

・最下段（第12段目）の球は、すべて底面の上にあるから、内部の球は0個。

・上から第m段目（m≦11）は、1辺m個の正三角形状に球が並んでいる。この三角形の周（3つの辺）の上にある球は、3つの斜面のいずれかに現れて外から見える。

・内部に隠れるのは、周を除いた内側の球だけであり、それらは1辺が(m−3)個の正三角形状に並ぶから、その個数は (m−3)(m−2)/2（個）（m≧4。m≦3の段では0個）。


【計算】

第4段目から第11段目までの内部の球を合計すると、

・第4段目：1個　・第5段目：3個　・第6段目：6個　・第7段目：10個

・第8段目：15個　・第9段目：21個　・第10段目：28個　・第11段目：36個

合計　1＋3＋6＋10＋15＋21＋28＋36＝120個


【別解】

内部の球の各段の個数 1, 3, 6, 10, … は問1の各段の球数と同じ並びである。つまり内部の球だけを取り出すと、ちょうど第(12−4)＝8番目の立体と同じ形（8段の三角錐）になり、その総数は 8×9×10÷6＝120（個）と求めることもできる（各斜面で1段分ずつ、底面と最上部で合わせて4段分小さくなる）。


【選択肢の検討】

・(1) 56個 → 第6番目の立体の総数（内部を2段分小さく見積もった誤り）

・(2) 84個 → 第7番目の立体の総数。内部の立体を7段と誤った場合

・(3) 120個 → 上記の計算どおり。正解

・(4) 165個 → 最下段の内側の球45個を「隠れている」と誤って加えた場合（120＋45）

・(5) 220個 → 第10番目の立体の総数。斜面上の球の除外が不完全な場合


したがって、正解は(3)である。
