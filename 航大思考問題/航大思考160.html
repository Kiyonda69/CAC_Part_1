問1 配点: 6点 / 目安時間: 3分
下の立方体を3点（●）を通る平面で切ったとき、切り口の図形は(1)～(5)のどれになりますか。
解説（正解: (1)）

立方体の1辺を1とし、頂点を次のように設定する。

V1(0,0,0)、V2(1,0,0)、V3(1,1,0)、V4(0,1,0)（底面）

V5(0,0,1)、V6(1,0,1)、V7(1,1,1)、V8(0,1,1)（上面）


【3点の座標】

・P1：辺V1V2の中点(1/2, 0, 0)

・P2：辺V3V7の中点(1, 1, 1/2)

・P3：辺V8V5の中点(0, 1/2, 1)


【平面の式】

3点を通る平面は x − y + z = 1/2。


【切り口の頂点】

平面と立方体の各辺の交点は次の6点：

(1/2,0,0)、(1,1/2,0)、(1,1,1/2)、(1/2,1,1)、(0,1/2,1)、(0,0,1/2)

立方体の中心(1/2,1/2,1/2)から等距離(√(1/2))にあり、

隣り合う頂点間の距離はすべて √(1/2) で等しい。


【選択肢の検討】

・(1) 正六角形 → 6辺等長、内角120°、空間対角線V1V7に垂直 → 正解

・(2) 三角形 → 切り口の頂点数が6なので不適

・(3) 正方形 → 4頂点ではないので不適

・(4) 五角形 → 平面は立方体の6面すべてと交わるため5辺ではない

・(5) 台形 → 4辺の図形は不適


したがって、正解は(1)である。

問2 配点: 6点 / 目安時間: 5分
下の立方体を3点（●）を通る平面で切ったとき、切り口の図形は(1)～(5)のどれになりますか。
解説（正解: (4)）

問1と同じ頂点設定を用いる。


【3点の座標】

・P1：辺V1V2を3:1に内分する点(3/4, 0, 0)

・P2：辺V8V5の中点(0, 1/2, 1)

・P3：辺V5V6を1:3に内分する点(1/4, 0, 1)


【平面の式】

3点を通る平面は 2x + y + z = 3/2。


【頂点の半空間】

各頂点で 2x+y+z の値を計算：

V1=0、V2=2、V3=3、V4=1、V5=1、V6=3、V7=4、V8=2

3/2未満：V1、V4、V5（3頂点・L字配置）。残り5頂点は3/2より大。

よって平面が切るのは 5辺 → 切り口は五角形。


【切り口の頂点】

(3/4,0,0)、(1/4,1,0)、(0,1,1/2)、(0,1/2,1)、(1/4,0,1)

辺の長さは順に √(5)/4、√(5)/2、√(5)/2、√(5)/4、√(2)/2。

P1とP3を結ぶ線分の中点を通る軸に対して線対称な五角形。


【選択肢の検討】

・(1) 正五角形 → 5辺すべて等長ではないので不適

・(2) 等脚台形 → 4辺の図形は不適

・(3) 正六角形 → 6辺の図形は不適

・(4) 線対称な五角形 → 形状一致 → 正解

・(5) 菱形 → 4辺の図形は不適


したがって、正解は(4)である。
