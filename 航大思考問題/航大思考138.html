問1 配点: 6点 / 目安時間: 3分
縦3行×横3列のマス目の各マスに、次のように矢印が1本ずつ配置されている。
  • 1行目（上段）：左から順に 右向き・上向き・左向き
  • 2行目（中段）：左から順に 上向き・右向き・下向き
  • 3行目（下段）：左から順に 右向き・右向き・上向き
この矢印が書かれた紙を、紙面に対して垂直な軸を中心として反時計回りに90°回転させた。回転後の矢印の配置として正しいものはどれか。
問1（正解：(4)）

紙を反時計回り（CCW）に90°回転させると、以下の2つの変化が同時に起きる。


【位置の変化】

元の右端列（3列目）が新しい上段行（1行目）になる。

元の中央列（2列目）が新しい中段行（2行目）になる。

元の左端列（1列目）が新しい下段行（3行目）になる。

各列は上から下に読んだ順が、新しい行の左から右の順に対応する。


【矢印の向きの変化】

反時計回り90°の回転では：右向き→上向き、上向き→左向き、左向き→下向き、下向き→右向きに変わる。


【検証】

元の3列目（上から）：左向き・下向き・上向き → CCW回転後：下向き・右向き・左向き → 新しい1行目

元の2列目（上から）：上向き・右向き・右向き → CCW回転後：左向き・上向き・上向き → 新しい2行目

元の1列目（上から）：右向き・上向き・右向き → CCW回転後：上向き・左向き・上向き → 新しい3行目


したがって、正解は(4)である。

問2 配点: 6点 / 目安時間: 5分
一辺3cmの立方体がある。この立方体の主対角線（向かい合う2頂点を結ぶ直線）を考える。
  • 主対角線は、立方体の体の対角線であり、立方体の重心（中心点）を通る
  • 主対角線と垂直な方向を向く平面で、かつ立方体の重心を通る平面を考える
  • この平面によって立方体を切断する
切断後に現れる断面の形として正しいものはどれか。
問2（正解：(1)）

立方体の主対角線（体の対角線）とは、向かい合う2頂点、たとえば頂点A(0,0,0)と頂点B(3,3,3)を結ぶ直線である。


【切断平面の特定】

主対角線の方向ベクトルは (1,1,1)。重心は (1.5,1.5,1.5)。

切断平面の方程式: x + y + z = 4.5（方向(1,1,1)に垂直で重心を通る）


【交点の計算】

この平面は立方体の12本の辺のうち、6本の辺の中点を通過する。

各辺の中点での交点は：(1.5,3,0), (3,1.5,0), (3,0,1.5), (1.5,0,3), (0,1.5,3), (0,3,1.5)


【断面形状の確認】

隣接する2点間の距離はすべて 3√2/2 ≈ 2.12cm で等しく、内角もすべて120°。

したがって断面は正六角形である。


（直感的理解：立方体の主対角線は、3軸に対して等しく傾いている。その対称性により、垂直断面に6面体の各面が1辺ずつ均等に現れ、正六角形を形成する。）


したがって、正解は(1)である。
