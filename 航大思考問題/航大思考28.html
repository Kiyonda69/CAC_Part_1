問1 配点: 6点 / 目安時間: 3分
下の表は、アジア太平洋地域における4つの航空会社の2021年から2023年までの主要経営指標を示している。この表に関する記述ア〜オについて、正しいものの組み合わせとして最も適切なものを、(1)〜(5)の中から1つ選べ。
【表】アジア太平洋地域 主要航空会社 経営指標
航空会社 指標 2021年 2022年 2023年
P航空 旅客数（万人） 280320350
営業収入（億円） 1,2001,4501,600
営業利益率（%） 5.08.010.0
座席利用率（%） 707882
Q航空 旅客数（万人） 210180230
営業収入（億円） 900820980

ア：2022年において、旅客数が前年より減少した航空会社はQ航空とR航空の2社である。
イ：3年間を通じて、P航空の座席利用率は全4社の中で常に最も高かった。
ウ：2023年において、全4社の中で営業利益（営業収入×営業利益率）が最も大きい航空会社はP航空である。
エ：調査期間中に、R航空の営業収入が前年を下回った年は存在しない。
オ：2023年において、座席利用率が75%を超えた航空会社は2社である。
解説（正解: (3)）

【表の読み取り】

旅客数（万人）：P航空 280→320→350 / Q航空 210→180→230 / R航空 150→130→140 / S航空 80→90→100

営業収入（億円）：P航空 1,200→1,450→1,600 / Q航空 900→820→980 / R航空 650→700→620 / S航空 380→420→460

営業利益率（%）：P航空 5.0→8.0→10.0 / Q航空 4.0→2.0→6.0 / R航空 3.0→5.0→2.0 / S航空 1.0→3.0→5.0

座席利用率（%）：P航空 70→78→82 / Q航空 72→65→79 / R航空 68→74→70 / S航空 55→62→68


【各記述の検討】

ア（正しい）：2022年の旅客数を前年比で確認する。

　Q航空：210万→180万（減少）、R航空：150万→130万（減少）

　P航空：280万→320万（増加）、S航空：80万→90万（増加）

　減少した会社はQ航空とR航空の2社であり、正しい。


イ（誤り）：2021年の座席利用率はP航空=70%、Q航空=72%。

　Q航空（72%）がP航空（70%）を上回っており、P航空が常に最高とは言えない。


ウ（正しい）：2023年の営業利益（営業収入×営業利益率）を計算する。

　P航空：1,600億×10.0% = 160億円（最大）

　Q航空：980億×6.0% = 58.8億円 / R航空：620億×2.0% = 12.4億円 / S航空：460億×5.0% = 23億円

　P航空の160億円が最も大きく、正しい。


エ（誤り）：R航空の営業収入は 650→700→620（億円）と推移する。

　2023年（620億円）は2022年（700億円）を下回っており、記述は誤り。


オ（正しい）：2023年の座席利用率を確認する。

　P航空82%、Q航空79%（いずれも75%超）、R航空70%、S航空68%（いずれも75%以下）

　75%を超えた航空会社はP航空とQ航空の2社であり、正しい。


したがって、正しいものはア・ウ・オであり、正解は(3)である。

ア：2022年から2023年にかけて、旅客数が前年より減少した航空会社はE航空のみである。
イ：2023年における全5社の営業利益（営業収入×営業利益率）の合計は600億円を超える。
ウ：3年間を通じて、B航空の旅客数はC航空の旅客数を常に上回っていた。
エ：2023年において、座席利用率が最も高い航空会社と最も低い航空会社の差は20ポイント以上である。

解説（正解: (1)）

【表の読み取り】

旅客数（万人）：A航空 520→580→620 / B航空 380→350→410 / C航空 250→270→280 / D航空 160→180→200 / E航空 90→110→100

営業収入（億円）：A航空 2,400→2,800→3,000 / B航空 1,800→1,650→1,950 / C航空 1,100→1,200→1,250 / D航空 750→850→950 / E航空 420→480→450

営業利益率（%）：A航空 6.0→9.0→11.0 / B航空 4.0→2.0→7.0 / C航空 3.0→4.0→5.0 / D航空 2.0→4.0→6.0 / E航空 −1.0→1.0→3.0

座席利用率（%）：A航空 72→80→84 / B航空 68→62→75 / C航空 65→70→74 / D航空 60→66→72 / E航空 50→58→62


【各記述の検討】

ア（正しい）：2022年→2023年の旅客数を確認する。

　E航空：110万→100万（減少）

　A航空：580万→620万（増加）、B航空：350万→410万（増加）、C航空：270万→280万（増加）、D航空：180万→200万（増加）

　前年比減少はE航空のみであり、正しい。


イ（誤り）：2023年の全5社の営業利益を計算する。

　A航空：3,000×11.0% = 330.0億円

　B航空：1,950×7.0% = 136.5億円

　C航空：1,250×5.0% = 62.5億円

　D航空：950×6.0% = 57.0億円

　E航空：450×3.0% = 13.5億円

　合計：330.0+136.5+62.5+57.0+13.5 = 599.5億円 ＜ 600億円

　600億円を超えないため、誤り。（0.5億円だけ届かない点に注意）


ウ（正しい）：B航空とC航空の旅客数を年ごとに比較する。

　2021年：B航空380万 ＞ C航空250万 ○

　2022年：B航空350万 ＞ C航空270万 ○

　2023年：B航空410万 ＞ C航空280万 ○

　3年間を通じてB航空の旅客数はC航空を常に上回っており、正しい。


エ（正しい）：2023年の座席利用率の最高・最低を確認する。

　最高：A航空 84% / 最低：E航空 62%

　差：84 − 62 = 22ポイント ≥ 20ポイント → 正しい。


オ（誤り）：E航空の営業利益率を確認する。

　2021年：−1.0%（赤字）、2022年：1.0%（黒字）、2023年：3.0%（黒字）

　2022年・2023年には0%を上回っており、記述は誤り。


したがって、正しいものはア・ウ・エであり、正解は(1)である。
