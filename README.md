## 運用ガイド

### 問題作成サイクル

1. **航大思考1〜20**まで作成
2. ユーザーが`航大思考問題/`と`印刷用/`フォルダをアーカイブ
3. 再び航大思考1から新規作成を開始

### 新規問題セット作成手順

1. Python検証で解の一意性を確認
2. 正解番号をランダム生成: `python3 -c "import random; print(random.randint(1, 5))"`
3. `印刷用/印刷{N}.html` を作成（template_static.htmlベース）
4. `航大思考問題/航大思考{N}.html` を作成（template_dynamic.htmlベース）
5. `問題一覧.md` を更新
6. 品質チェック: `python3 scripts/quality_check.py`

### スクリプトの使用方法

#### 問題一覧の自動更新

```bash
python3 scripts/update_index.py
```

#### 品質チェック

```bash
# 全ファイルをチェック
python3 scripts/quality_check.py

# 特定ファイルのみチェック
python3 scripts/quality_check.py 航大思考問題/航大思考1.html
```

#### 解の一意性検証

```bash
# テンプレートをコピーして使用
cp scripts/verify_template.py scripts/verify_21.py
# 編集後に実行
python3 scripts/verify_21.py
```

### アーカイブ後のリセット手順

アーカイブ後に再作成を開始する際は以下を実施:

1. `航大思考問題/` フォルダを空にする（または新規作成）
2. `印刷用/` フォルダを空にする（または新規作成）
3. `問題一覧.md` をリセット（統計を0に戻す）
4. 航大思考1・印刷1から作成を再開

- 視覚的矛盾（グラフ値とテキスト値の不一致）

- 正解番号の手動選択（特に1番や5番への固定）
