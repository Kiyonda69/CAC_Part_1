# CAC_Part_1

航空大学校入学試験「総合Part I」思考パズル問題作成プロジェクト

## 概要

高度な論理的思考力を測定する思考パズル問題を作成するプロジェクト。
Python検証により解の一意性を保証する。

## 問題仕様

### 作成ルール
- **2問ずつ作成**: 標準難度と高難度をペアで作成
- **出力先**: すべて `main.html` に作成
- **問題番号**: 連番で管理（問1, 問2, 問3, ...）

### 難易度別仕様

| 項目 | 問題1（標準） | 問題2（高難度） |
|------|-------------|----------------|
| 配点 | 6点 | 6点 |
| 制限時間 | 3分 | 5分 |
| 難易度 | ★★★★★ | ★★★★★★★★ |
| 解答時間目安 | 170秒 | 310秒 |

## 測定能力

- 論理推論
- 空間視覚化
- パターン認識
- 数学的思考
- 制約充足
- 資料読み取り

## ファイル構成

```
main.html        - 問題出力先（全問題をここに作成）
CLAUDE.md        - Claude Code用ガイドライン
README.md        - プロジェクト説明（このファイル）
難易度基準.md    - 難易度基準の参考資料
template.html    - HTMLテンプレート
```

## 作成プロセス

### 1. 問題設計

問題タイプを選択:
- 制約充足
- 暗号解読
- 空間推論
- パターン認識
- データ分析

### 2. Python検証（必須）

```python
def verify_solution():
    """全ての可能性を検証し、唯一解を確認"""
    valid_solutions = []
    # 全パターンを総当たり
    # 制約条件を適用
    assert len(valid_solutions) == 1, f"解が{len(valid_solutions)}個存在"

```

### 3. 正解番号のランダム配置

```bash
python3 -c "import random; print(random.randint(1, 5))"
```

- 各問題ごとに実行
- 手動での番号決定は禁止

### 4. HTML作成

- `template.html` の構造・スタイルを一切変更しない
- 編集範囲: `<!-- ここから問題セクション -->` 〜 `<!-- 解答・解説セクション -->` のみ
- カラー: 白黒（グレースケール可）
- **出力先**: `main.html` に全問題を連番で作成

## 品質チェックリスト

- [ ] Python検証で解が唯一
- [ ] すべての条件が解導出に必要
- [ ] 矛盾する条件が存在しない
- [ ] SVG座標が意図した位置と一致
- [ ] 正解番号がランダム化されている
- [ ] 複数ステップの推論が必要

## 禁止事項

- 視覚的矛盾
- 自明な解
- 航空専門知識を必要とする設計
- Python検証の省略
- 正解番号の手動選択
