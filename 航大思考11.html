問1．あるコンビニエンスストアチェーンでは、5つの支店（A〜E）の週間売上データを集計している。表1は各支店の第1週から第4週までの売上（単位：万円）を示したものである。4週間の平均売上が87万円以上の支店のうち、第4週の売上が最も高い支店として正しいものを1つ選び、解答用紙にマークせよ。(6点)
【表1】支店別週間売上データ（単位：万円）
支店 第1週 第2週 第3週 第4週
A 85 92 88 95
B 78 84 91 87
C 90 86 94 98
D 82 89 85 92
E 75 80 86 93
(1) 支店A
(2) 支店C
(3) 支店B
(4) 支店D
(5) 支店E
問1（正解：(2)）

【各支店の4週間平均を計算】
支店A：(85 + 92 + 88 + 95) / 4 = 360 / 4 = 90.0万円 → 87万円以上
支店B：(78 + 84 + 91 + 87) / 4 = 340 / 4 = 85.0万円 → 87万円未満
支店C：(90 + 86 + 94 + 98) / 4 = 368 / 4 = 92.0万円 → 87万円以上
支店D：(82 + 89 + 85 + 92) / 4 = 348 / 4 = 87.0万円 → 87万円以上
支店E：(75 + 80 + 86 + 93) / 4 = 334 / 4 = 83.5万円 → 87万円未満

【条件を満たす支店】
平均87万円以上の支店：A、C、D

【第4週の売上を比較】
支店A：95万円
支店C：98万円
支店D：92万円

第4週の売上が最も高いのは支店C（98万円）である。
正解：(2) 支店C
問2．表1のデータを用いて、各支店の週間成長率（前週比）を分析する。ある支店が「安定成長型」と判定されるのは、以下の2つの条件を同時に満たす場合である。
　条件1：第1週から第4週まで、毎週前週比でプラス成長している（3週連続でプラス）
　条件2：第1週から第4週までの総成長率（(第4週-第1週)/第1週）が20%以上である
「安定成長型」と判定される支店として正しいものを1つ選び、解答用紙にマークせよ。(6点)
【表1】支店別週間売上データ（単位：万円）（再掲）
支店 第1週 第2週 第3週 第4週
A 85 92 88 95
B 78 84 91 87
C 90 86 94 98
D 82 89 85 92
E 75 80 86 93
(1) 支店A
(2) 支店B
(3) 支店C
(4) 支店E
(5) 支店D
問2（正解：(4)）

【各支店の週間成長率を計算】

支店A：
第1→2週：(92-85)/85 = +8.2%
第2→3週：(88-92)/92 = -4.3% → マイナス成長あり
→ 条件1を満たさない

支店B：
第1→2週：(84-78)/78 = +7.7%
第2→3週：(91-84)/84 = +8.3%
第3→4週：(87-91)/91 = -4.4% → マイナス成長あり
→ 条件1を満たさない

支店C：
第1→2週：(86-90)/90 = -4.4% → マイナス成長あり
→ 条件1を満たさない

支店D：
第1→2週：(89-82)/82 = +8.5%
第2→3週：(85-89)/89 = -4.5% → マイナス成長あり
→ 条件1を満たさない

支店E：
第1→2週：(80-75)/75 = +6.7%
第2→3週：(86-80)/80 = +7.5%
第3→4週：(93-86)/86 = +8.1%
→ 3週連続プラス成長 → 条件1を満たす

総成長率：(93-75)/75 = 18/75 = 24.0% → 20%以上 → 条件2を満たす

【結論】
条件1と条件2を同時に満たすのは支店Eのみである。
正解：(4) 支店E
